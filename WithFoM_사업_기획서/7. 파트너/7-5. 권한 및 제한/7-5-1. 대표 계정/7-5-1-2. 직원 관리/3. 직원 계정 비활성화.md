# 3: 직원 계정 비활성화

## 개요

대표 계정이 직원 계정을 비활성화/삭제하는 방법과 규칙을 정의합니다.

## 비활성화 vs 삭제

| 유형 | 설명 | 복구 가능 |
|---|---|---|
| 비활성화 | 일시적 접근 차단 | ✅ 가능 |
| 삭제 | 계정 완전 삭제 | ❌ 불가 |

## 비활성화 프로세스

| 단계 | 내용 |
|---|---|
| 1 | 설정 > 계정 관리 > 직원 목록 접근 |
| 2 | 비활성화할 직원 선택 |
| 3 | 비활성화 버튼 클릭 |
| 4 | 확인 팝업에서 확인 클릭 |
| 5 | 직원 즉시 로그아웃 처리 |

## 비활성화 영향

| 항목 | 영향 |
|---|---|
| 로그인 | 즉시 로그아웃, 재로그인 불가 |
| 데이터 | 활동 이력 유지 |
| 복구 | 대표가 언제든 재활성화 가능 |

## 삭제 프로세스

| 단계 | 내용 |
|---|---|
| 1 | 설정 > 계정 관리 > 직원 목록 접근 |
| 2 | 삭제할 직원 선택 |
| 3 | 삭제 버튼 클릭 |
| 4 | 최종 확인 팝업에서 확인 클릭 |
| 5 | 계정 영구 삭제 |

## 삭제 주의사항

| 항목 | 내용 |
|---|---|
| 복구 불가 | 삭제된 계정은 복구 불가 |
| 데이터 보존 | 활동 이력은 익명 처리하여 보존 |
| 재가입 | 동일 이메일/연락처로 재초대 가능 |
