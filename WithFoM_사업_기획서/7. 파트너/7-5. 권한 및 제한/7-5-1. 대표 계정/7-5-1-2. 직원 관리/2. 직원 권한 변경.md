　　　**1.2.** 직원 권한 변경

## 개요

대표 계정이 직원 계정의 권한을 변경하는 방법과 규칙을 정의합니다.

## 권한 변경 가능 범위

| 변경 유형 | 가능 여부 |
|---|---|
| 권한 추가 | ✅ 가능 |
| 권한 축소 | ✅ 가능 |
| 프리셋 변경 | ✅ 가능 |
| 대표 권한 부여 | ❌ 불가 |

## 권한 변경 프로세스

| 단계 | 내용 |
|---|---|
| 1 | 설정 > 계정 관리 > 직원 목록 접근 |
| 2 | 권한 변경할 직원 선택 |
| 3 | 권한 수정 화면에서 권한 변경 |
| 4 | 저장 버튼 클릭 |
| 5 | 직원에게 권한 변경 알림 자동 발송 |

## 권한 변경 즉시 반영

| 항목 | 내용 |
|---|---|
| 반영 시점 | 저장 즉시 반영 |
| 세션 영향 | 직원 앱 자동 새로고침 |
| 제한된 기능 | 즉시 접근 불가 |

## 직원 알림

| 변경 유형 | 알림 내용 |
|---|---|
| 권한 추가 | "[권한명] 권한이 추가되었습니다." |
| 권한 축소 | "[권한명] 권한이 제거되었습니다." |
