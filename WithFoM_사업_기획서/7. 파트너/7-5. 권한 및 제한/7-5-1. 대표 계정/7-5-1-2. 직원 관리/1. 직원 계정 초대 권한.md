# 1: 직원 계정 초대 권한

## 개요

대표 계정이 직원 계정을 초대할 때 설정 가능한 권한을 정의합니다.

## 부여 가능 권한

| 권한 영역 | 권한 항목 | 부여 가능 |
|---|---|---|
| 대시보드 | 리워드 승인 | ✅ 가능 |
| 대시보드 | 결제 승인 | ✅ 가능 |
| 일일 소진 보고 | 소진량 입력 | ✅ 가능 |
| 일일 소진 보고 | 소진량 수정 | ✅ 가능 |
| 컵홀더 발주 | 발주 요청 | ✅ 가능 |
| 컵홀더 발주 | 입고 확인 | ✅ 가능 |
| 메뉴 관리 | 메뉴 등록/수정 | ✅ 가능 |
| 메뉴 관리 | 품절 설정 | ✅ 가능 |
| 리워드 | 리워드 설정 변경 | ⚠️ 선택 |
| 매장 정보 | 기본 정보 수정 | ⚠️ 선택 |

## 부여 불가 권한 (대표 전용)

| 권한 영역 | 권한 항목 | 사유 |
|---|---|---|
| 정산 | 정산 내역 조회 | 재무 정보 보호 |
| 정산 | 이의신청 | 법적 책임 |
| 계정 관리 | 직원 계정 관리 | 대표 전용 |
| 설정 | 서비스 타입 변경 | 계약 관련 |
| 설정 | 탈퇴 요청 | 계약 관련 |

## 직원 권한 프리셋

| 프리셋 | 포함 권한 |
|---|---|
| 매장 관리자 | 리워드/결제 승인, 소진 보고, 발주, 메뉴 관리 |
| 일반 직원 | 리워드/결제 승인만 |
| 사용자 정의 | 개별 권한 선택 |
