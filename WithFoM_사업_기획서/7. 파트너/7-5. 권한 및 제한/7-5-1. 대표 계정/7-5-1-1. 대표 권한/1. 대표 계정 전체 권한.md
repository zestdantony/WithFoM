　　**1-1.1.** 대표 계정 전체 권한

## 개요

파트너 대표 계정이 가진 모든 권한을 정의합니다.

## 대표 계정 권한 목록

| 권한 영역 | 권한 항목 | 설명 |
|---|---|---|
| 계정 관리 | 직원 계정 초대 | 직원 계정 생성 및 초대 |
| 계정 관리 | 직원 권한 설정 | 직원별 권한 범위 설정 |
| 계정 관리 | 직원 계정 삭제 | 직원 계정 비활성화/삭제 |
| 정산 | 정산 내역 조회 | 월별 정산 내역 확인 |
| 정산 | 이의신청 | 정산 이의신청 제출 |
| 정산 | 계좌 정보 변경 | 정산 계좌 정보 수정 |
| 설정 | 서비스 타입 변경 | 리워드형/오더형/복합형 변경 |
| 설정 | 계약 관련 설정 | 계약 정보 조회/갱신 |
| 설정 | 탈퇴 요청 | 파트너 탈퇴 요청 |
| 운영 | 모든 운영 권한 | 일일 소진 보고, 리워드 승인 등 |

|---|---|
| 정산 내역 조회 | ❌ 불가 |
| 직원 계정 관리 | ❌ 불가 |
| 서비스 타입 변경 | ❌ 불가 |
| 계약/탈퇴 관련 | ❌ 불가 |
