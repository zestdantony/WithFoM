　　　**1.1.** 직원 계정 기본 권한

## 개요

직원 계정이 가질 수 있는 권한과 제한을 정의합니다.

## 기본 권한 (최소 권한)

| 권한 영역 | 권한 항목 | 기본 부여 |
|---|---|---|
| 대시보드 | 대시보드 조회 | ✅ 기본 |
| 대시보드 | 알림 확인 | ✅ 기본 |
| 매장 정보 | 매장 정보 조회 | ✅ 기본 |

## 대표가 부여 가능한 권한

| 권한 영역 | 권한 항목 | 부여 시 가능 작업 |
|---|---|---|
| 대시보드 | 리워드 승인 | 리워드 사용 승인/거절/복구 |
| 대시보드 | 결제 승인 | 결제 승인/거절 |
| 일일 소진 보고 | 소진량 입력 | 일일 소진량 입력 |
| 일일 소진 보고 | 소진량 수정 | 전일 소진량 수정 |
| 컵홀더 발주 | 발주 요청 | 컵홀더 추가 발주 요청 |
| 컵홀더 발주 | 입고 확인 | 입고 확인 처리 |
| 메뉴 관리 | 메뉴 등록/수정 | 메뉴 정보 관리 |
| 메뉴 관리 | 품절 설정 | 품절 ON/OFF |
| 리워드 | 리워드 설정 변경 | 쿠폰/포인트/스탬프 설정 |
| 매장 정보 | 기본 정보 수정 | 영업시간 등 기본 정보 수정 |

## 직원 접근 불가 (대표 전용)

| 권한 영역 | 사유 |
|---|---|
| 정산 내역 조회 | 재무 정보 보호 |
| 직원 계정 관리 | 대표 전용 권한 |
| 서비스 타입 변경 | 계약 관련 설정 |
| 탈퇴/계약 관련 | 법적 책임 |
| 계좌 정보 변경 | 재무 정보 보호 |
