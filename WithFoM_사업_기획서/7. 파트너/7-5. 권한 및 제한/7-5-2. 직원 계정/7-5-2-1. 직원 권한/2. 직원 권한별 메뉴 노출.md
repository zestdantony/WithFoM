　　　**1.2.** 직원 권한별 메뉴 노출

## 개요

직원 계정에 부여된 권한에 따라 앱에서 노출되는 메뉴를 정의합니다.

## 메뉴 노출 규칙

| 사이드바 메뉴 | 필요 권한 |
|---|---|
| 홈(대시보드) | 기본 (항상 노출) |
| 일일 소진 보고 | 소진량 입력 권한 |
| 컵홀더 발주 | 발주 요청 권한 |
| 메뉴 오더 | 결제 승인 권한 |
| 메뉴 관리 | 메뉴 등록/수정 권한 |
| 리워드 | 리워드 승인 또는 설정 권한 |
| 매장 정보 | 기본 (조회만) 또는 수정 권한 |
| 정산·등급 | ❌ 비노출 (대표 전용) |
| 도움말·문의 | 기본 (항상 노출) |
| 설정·계정 관리 | 기본 (자신 계정만) |

## 대시보드 보드 노출 규칙

| 보드 | 필요 권한 |
|---|---|
| 리워드 사용 내역 보드 | 리워드 승인 권한 |
| 결제 내역 보드 | 결제 승인 권한 |
| 주문 현황 보드 | 결제 승인 권한 |
| 리워드 현황 요약 보드 | 리워드 승인 또는 설정 권한 |
| 컵홀더 현황 보드 | 발주 요청 권한 |

## 권한 없는 메뉴 접근 시

| 상황 | 표시 |
|---|---|
| 메뉴 클릭 | 접근 권한 없음 팝업 표시 |
| 메시지 | "해당 메뉴에 대한 접근 권한이 없습니다. 대표에게 문의하세요." |
