**2.1.** 직원 계정 제한 사항

## 개요

직원 계정에 적용되는 제한 사항을 정의합니다.

## 기능 제한

| 제한 항목 | 제한 내용 |
|---|---|
| 정산 조회 불가 | 모든 정산 관련 정보 열람 불가 |
| 계정 관리 불가 | 다른 직원 계정 관리 불가 |
| 서비스 설정 불가 | 서비스 타입, 계약 관련 설정 불가 |
| 탈퇴 요청 불가 | 파트너 탈퇴 요청 불가 |
| 계좌 변경 불가 | 정산 계좌 정보 변경 불가 |

## 데이터 조회 제한

| 데이터 유형 | 직원 조회 가능 |
|---|---|
| 일일 매출 요약 | ⚠️ 대표가 허용 시 |
| 상세 정산 내역 | ❌ 불가 |
| 인센티브/비용 정보 | ❌ 불가 |
| 등급 상세 점수 | ⚠️ 대표가 허용 시 |

## 알림 제한

| 알림 유형 | 직원 수신 |
|---|---|
| 리워드/결제 승인 요청 | ✅ (권한 있을 시) |
| 일일 소진 보고 알림 | ✅ (권한 있을 시) |
| 재고 부족 경고 | ✅ (발주 권한 있을 시) |
| 정산 완료 알림 | ❌ 불가 |
| 등급 변동 알림 | ❌ 불가 |
| 계약 관련 알림 | ❌ 불가 |

## 계정 관련 제한

| 항목 | 제한 내용 |
|---|---|
| 비밀번호 변경 | ✅ 자신 계정만 가능 |
| 프로필 수정 | ✅ 자신 계정만 가능 |
| 알림 설정 | ✅ 자신 알림만 설정 가능 |
| 로그아웃 | ✅ 자신 계정 로그아웃 가능 |
