**1.1.** 등급별 혜택 및 제한 요약

## 개요

파트너 등급(플래티넘/골드/실버/브론즈)에 따른 혜택과 제한을 정의합니다.

## 등급 산정 기준

| 항목 | 가중치 |
|---|---|
| 보고율 | 45% |
| 스캔율 | 45% |
| 결제 기능 점수 | 10% |

## 등급별 혜택

| 등급 | 컵홀더 | 멤버십 할인 | 인센티브 | 캠페인 매칭 (등급 매칭 라벨) |
|---|---|---|---|---|
| 플래티넘 | 무료 제공 | 30% 할인 | 개당 10원 | 최우선 |
| 골드 | 무료 제공 | 30% 할인 | - | 우선 |
| 실버 | 무료 제공 | - | - | 일반 |
| 브론즈 | 50% 할인 | - | - | 후순위 |

## 캠페인 매칭 우선순위 점수 체계

### 종합 매칭 점수 = 등급 점수 + 멤버십 점수

| 등급 | 등급 매칭 점수 | 플랜 | 멤버십 매칭 점수 |
|---|---|---|---|
| 플래티넘 | 4 | 프리미엄 | 3 |
| 골드 | 3 | 플러스 | 2 |
| 실버 | 2 | 베이직 | 1 |
| 브론즈 | 1 | | |

### 매칭 우선순위 그룹

| 우선순위 그룹 | 점수 범위 | 해당 조합 |
|---|---|---|
| 최우선 매칭 | 6~7점 | 플래티넘+프리미엄(7), 플래티넘+플러스(6), 골드+프리미엄(6) |
| 우선 매칭 | 5점 | 플래티넘+베이직(5), 골드+플러스(5), 실버+프리미엄(5) |
| 일반 매칭 | 4점 | 골드+베이직(4), 실버+플러스(4), 브론즈+프리미엄(4) |
| 후순위 매칭 | 2~3점 | 실버+베이직(3), 브론즈+플러스(3), 브론즈+베이직(2) |

### 캠페인 매칭 순서 (모든 캠페인 공통)

| 순서 | 기준 |
|---|---|
| 1단계 | 광고주 타겟팅 일치도순 (지역, 상권, 연령층, 성별, 고객 구성, 음료 가격) |
| 2단계 | 각 풀 내에서 종합 매칭 점수순 (등급 + 멤버십 + 포기 감점) |
| 3단계 | 동일 점수 내에서 미매칭 기간 우선순 (미매칭 기간이 긴 파트너 우선) |
| 4단계 | 타이브레이커 (오더 이용 → 플랫폼 기여도 → 랜덤) |
| 수량 충족 | 대기 파트너로 수량 부족 시 진행 중 파트너 부분 기간 매칭 (종료 가까운 순) |

### 파트너 표시

| 항목 | 설명 |
|---|---|
| 파트너 등급 매칭 | 등급별 매칭 라벨 (최우선/우선/일반/후순위) |
| 멤버십 플랜 매칭 | 플랜별 매칭 라벨 (우선/일반/후순위) |
| 나의 종합 매칭 등급 | 합산 점수 기반 그룹 라벨 (최우선/우선/일반/후순위 매칭) |

※ 실제 점수, 그룹 내 순위, 동점 해소 과정은 시스템 내부에서만 운영

## 등급별 제한

| 등급 | 제한 사항 |
|---|---|
| 플래티넘 | 없음 |
| 골드 | 없음 |
| 실버 | 프리미엄 캠페인 매칭 제한 |
| 브론즈 | 프리미엄 캠페인 매칭 제한, 컵홀더 비용 발생 |

## 등급 변동 규칙

| 항목 | 내용 |
|---|---|
| 산정 주기 | 캠페인 종료 시 |
| 적용 시점 | 익월 1일 |
| 변동 알림 | 등급 변동 시 파트너에게 알림 |

## 등급 유지 조건

| 등급 | 유지 조건 |
|---|---|
| 플래티넘 | 종합 점수 73점 이상, 1년 유지, 위드폼 오더 산정 기간 내 항상 이용 |
| 골드 | 종합 점수 55점 이상, 6개월 유지 |
| 실버 | 종합 점수 36점 이상, 6개월 유지 |
| 브론즈 | 종합 점수 36점 미만 |

## 등급 유지 및 등급 변경

| 조건 | 결과 |
|---|---|
| 6개월 이내 캠페인 매칭 | 현재 등급 유지, 유예 기간 재시작 |
| 6개월 연속 미매칭 | 한 단계 등급 변경(다운) (플래티넘→골드→실버→브론즈) |
| 브론즈에서 미매칭 | 브론즈 유지 (최하 등급) |
| 등급 혜택 | 변동 전까지 현재 등급 혜택 유지 |

## 등급 변경 후 재산정

| 항목 | 내용 |
|---|---|
| 재매칭 시 | 해당 캠페인 포함 이전 6개월(플래티넘 1년) 재산정 |
| 조건 충족 시 | 즉시 업그레이드 가능 |

## 중도 포기 처리

| 항목 | 내용 |
|---|---|
| 포기 전까지 | 참여 기간은 등급 산정에 포함 |
| 포기 시점 | 미매칭 기간 카운트 시작 |
