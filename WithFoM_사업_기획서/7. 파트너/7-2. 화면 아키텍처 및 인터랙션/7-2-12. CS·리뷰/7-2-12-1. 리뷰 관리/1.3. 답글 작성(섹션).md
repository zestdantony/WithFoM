**1.3.** 답글 작성(섹션)  


**1.3.1.** 입력 영역  

**1.3.1.1.** 답글 입력  
1: 답글 내용 입력 (최대 500자)  
2: 글자 수 카운터 표시  
3: 줄바꿈 가능  


**1.3.2.** 버튼 영역  

**1.3.2.1.** 등록  
1: 클릭 시 답글 등록  
2: "답글이 등록되었습니다" 메시지 표시  
3: 소비자에게 알림 발송  

**1.3.2.2.** 취소  
1: 클릭 시 입력 내용 초기화  
2: 확인 팝업 표시 (입력 내용이 있는 경우)  


**1.3.3.** 답글 수정/삭제  

**1.3.3.1.** 수정  
1: 등록된 답글 수정 가능  
2: 수정 완료 시 "답글이 수정되었습니다" 메시지 표시  

**1.3.3.2.** 삭제  
1: 등록된 답글 삭제 가능  
2: 확인 팝업 표시  
3: 삭제 완료 시 "답글이 삭제되었습니다" 메시지 표시  


• 참조: 7-4-4-1. 리뷰 확인
