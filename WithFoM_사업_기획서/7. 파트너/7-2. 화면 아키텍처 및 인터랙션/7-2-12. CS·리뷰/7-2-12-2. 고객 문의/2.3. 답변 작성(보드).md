**2.3.** 답변 작성(보드)  


**2.3.1.** 입력 영역  

**2.3.1.1.** 답변 입력  
1: 답변 내용 입력 (최대 1000자)  
2: 글자 수 카운터 표시  
3: 줄바꿈 가능  


**2.3.2.** 버튼 영역  

**2.3.2.1.** 등록  
1: 클릭 시 답변 등록  
2: "답변이 등록되었습니다" 메시지 표시  
3: 소비자에게 알림 발송  

**2.3.2.2.** 임시저장  
1: 클릭 시 임시저장  
2: "임시저장되었습니다" 메시지 표시  
3: 임시저장된 답변은 다음 접속 시 불러오기  

**2.3.2.3.** 취소  
1: 클릭 시 입력 내용 초기화  
2: 확인 팝업 표시 (입력 내용이 있는 경우)  


**2.3.3.** 답변 후 상태 변경  

**2.3.3.1.** 상태 업데이트  
1: 문의 목록에서 해당 문의 답변 상태 "완료"로 변경  
2: 소비자 앱에서 답변 확인 가능  


• 참조: 7-4-3. 소비자 → 파트너 문의
