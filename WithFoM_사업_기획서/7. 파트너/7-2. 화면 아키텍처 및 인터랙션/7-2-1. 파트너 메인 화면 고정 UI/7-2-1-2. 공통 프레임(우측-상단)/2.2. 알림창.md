**2.2.** 알림창  


**2.2.1.** 알림 아이콘 위치  

**2.2.1.1.** 배치 위치  
1: 우측 상단 공통 프레임 내  
2: 검색창(돋보기 아이콘) 우측에 배치  
3: 배치 순서: 검색창(돋보기 아이콘) → 알림창(종 아이콘) (나란히 배치)  

**2.2.1.2.** 아이콘 표시  
1: 종(벨) 아이콘으로 표시  
2: 미확인 알림 존재 시: 아이콘 우측 상단에 빨간색 점(dot) 또는 숫자 뱃지 표시  
3: 아이콘 크기: 화면 크기에 따라 자동 조정 (반응형)  


**2.2.2.** 호버 툴팁  

**2.2.2.1.** 마우스 커서 호버 시  
1: 툴팁 메시지 표시: "클릭하면 알림 목록이 표시됩니다"  
2: 미확인 알림이 있는 경우: "N개의 새로운 알림이 있습니다"  
3: 툴팁 위치: 종 아이콘 바로 아래  
4: 툴팁 표시 시간: 0.5초 후 자동 표시  
5: 커서 이동 시 툴팁 자동 사라짐  


**2.2.3.** 알림(벨) 아이콘 클릭 시 표시 항목  

**2.2.3.1.** 알림 드롭다운 표시  
1: 클릭 시 알림 목록 드롭다운 노출  
2: 드롭다운 위치: 종 아이콘 아래  
3: 드롭다운 너비: 320px ~ 400px (반응형)  
4: 최대 표시 개수: 10개 (스크롤 가능)  

**2.2.3.2.** 알림 유형별 표시 (우선순위 순서)  
1: 일일 소진 보고 미완료 (우선순위: 높음)  
　A. 메시지: "오늘 소진 보고가 아직 완료되지 않았습니다"  
　B. 아이콘: 경고 아이콘 (노란색)  
　C. 클릭 시: 소진 보고 페이지로 이동  

2: 재고 부족 경고 (우선순위: 중간)  
　A. 메시지: "컵홀더 재고가 N개 남았습니다. 발주를 권장합니다"  
　B. 아이콘: 재고 아이콘 (주황색)  
　C. 클릭 시: 캠페인 발주 페이지로 이동  

3: 이상치 경고 (우선순위: 중간)  
　A. 메시지: "보고량 급증/급감이 감지되었습니다. 확인해주세요"  
　B. 아이콘: 차트 아이콘 (주황색)  
　C. 클릭 시: 소진 보고 상세 페이지로 이동  

4: 등급 변동 알림 (우선순위: 낮음)  
　A. 메시지: "다음 달 등급이 [등급]으로 변경될 예정입니다"  
　B. 아이콘: 등급 아이콘 (파란색)  
　C. 클릭 시: 파트너 등급 페이지로 이동  

5: 정산 완료 알림 (우선순위: 낮음)  
　A. 메시지: "이번 달 정산이 완료되었습니다"  
　B. 아이콘: 정산 아이콘 (녹색)  
　C. 클릭 시: 정산 관리 페이지로 이동  

6: 캠페인 신규 매칭 (우선순위: 낮음)  
　A. 메시지: "새로운 캠페인이 매칭되었습니다"  
　B. 아이콘: 캠페인 아이콘 (파란색)  
　C. 클릭 시: 캠페인 발주 페이지로 이동  

**2.2.3.3.** 알림 항목 표시 정보  
1: 알림 아이콘 (유형별 색상)  
2: 알림 메시지 (최대 2줄)  
3: 발생 시간 (예: "5분 전", "1시간 전", "어제")  
4: 읽음/안읽음 상태 표시 (안읽음: 굵은 글씨, 읽음: 일반 글씨)  


**2.2.4.** 알림 드롭다운 추가 기능  

**2.2.4.1.** 드롭다운 헤더  
1: 제목: "알림"  
2: 모두 읽음 처리 버튼: 우측에 "모두 읽음" 버튼 배치  
3: 클릭 시 모든 알림 읽음 처리  

**2.2.4.2.** 드롭다운 푸터  
1: "전체 알림 보기" 버튼  
2: 클릭 시: 알림 전체 페이지로 이동 (/partner/notifications)  

**2.2.4.3.** 알림 없음 상태  
1: 알림이 없는 경우: "새로운 알림이 없습니다" 메시지 표시  
2: 빈 상태 아이콘 표시 (종 아이콘 + 체크 표시)  


**2.2.5.** 드롭다운 닫기  

**2.2.5.1.** 닫기 동작  
1: 드롭다운 외부 클릭: 드롭다운 닫힘  
2: ESC 키 입력: 드롭다운 닫힘  
3: 알림 항목 클릭: 해당 페이지로 이동 후 드롭다운 닫힘  


**2.2.6.** 모바일 환경 고려사항  

**2.2.6.1.** 터치 인터랙션  
1: 종 아이콘 탭: 알림 드롭다운 표시 또는 알림 전체 페이지로 이동  
　A. 옵션 1: 드롭다운 형태로 표시 (바텀 시트)  
　B. 옵션 2: 알림 전체 페이지로 바로 이동  

2: 호버 툴팁: 모바일에서는 표시하지 않음 (터치만으로 충분)  

**2.2.6.2.** 레이아웃 조정  
1: 모바일: 종 아이콘 크기 축소 (예: 20px)  
2: 태블릿: 종 아이콘 크기 중간 (예: 24px)  
3: 데스크톱: 종 아이콘 크기 전체 (예: 28px)  
4: 알림 드롭다운: 모바일에서는 바텀 시트 형태로 표시
