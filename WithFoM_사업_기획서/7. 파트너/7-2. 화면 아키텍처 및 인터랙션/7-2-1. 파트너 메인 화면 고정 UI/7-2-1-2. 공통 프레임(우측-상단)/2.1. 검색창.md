**2.1.** 검색창  


**2.1.1.** 검색 아이콘 위치  

**2.1.1.1.** 배치 위치  
1: 우측 상단 공통 프레임 내  
2: 알림창(종 아이콘) 좌측에 배치  
3: 배치 순서: 검색창(돋보기 아이콘) → 알림창(종 아이콘) (나란히 배치)  

**2.1.1.2.** 아이콘 표시  
1: 돋보기 아이콘으로 표시  
2: 기본 상태: 돋보기 아이콘만 노출  
3: 아이콘 크기: 화면 크기에 따라 자동 조정 (반응형)  


**2.1.2.** 호버 인터랙션  

**2.1.2.1.** 마우스 호버 시 동작  
1: 돋보기 아이콘 위에 마우스 커서 호버 시 입력창 표시  
2: 입력창 표시 방식  
　A. 돋보기 아이콘 좌측으로 입력창이 슬라이드 확장  
　B. 입력창 너비: 200px ~ 300px (반응형)  
　C. 애니메이션: slide-left 또는 expand 애니메이션 (0.2초)  

3: 입력창 상태  
　A. placeholder 텍스트: "검색어를 입력하세요"  
　B. 입력창 포커스: 자동 포커스 없음 (호버 상태에서는 포커스 없음)  
　C. 커서 이동 시 입력창 유지: 입력창 영역 내에 커서가 있으면 유지  

**2.1.2.2.** 마우스 아웃 시 동작  
1: 입력창 영역 밖으로 커서 이동 시  
　A. 입력창 숨김 (돋보기 아이콘만 노출)  
　B. 애니메이션: slide-right 또는 collapse 애니메이션 (0.2초)  

2: 입력창에 텍스트가 입력된 상태에서 마우스 아웃  
　A. 입력창 유지 (텍스트가 있으면 닫히지 않음)  
　B. 입력창 외부 클릭 시 입력창 닫힘 및 텍스트 초기화  


**2.1.3.** 클릭 인터랙션  

**2.1.3.1.** 돋보기 아이콘 클릭 시  
1: 입력창 표시 및 자동 포커스  
　A. 입력창이 닫혀 있는 경우: 입력창 열림 + 자동 포커스  
　B. 입력창이 열려 있는 경우: 입력창 포커스 활성화  

2: 입력창 상태  
　A. 커서 활성화: 입력 가능 상태  
　B. 키섹션 입력 대기 상태  

**2.1.3.2.** 입력창 클릭 시  
1: 입력창 포커스 활성화  
　A. 커서 활성화: 입력 가능 상태  
　B. 텍스트 입력 가능  


**2.1.4.** 검색 기능  

**2.1.4.1.** 검색 대상  
1: 파트너 포털 내 메뉴 및 페이지  
　A. 캠페인 관리, 리워드 설정, 정산 내역 등 주요 메뉴  
　B. 매장 정보, 고객 관리, 주문 관리 등 기능 페이지  

2: 캠페인 정보  
　A. 캠페인명, 캠페인 코드, 브랜드명  
　B. 캠페인 상태 (진행 중, 종료, 대기 중)  

3: 고객 정보 (멤버십 구독 시)  
　A. 고객 닉네임, 전화번호 뒷자리  
　B. 리워드 사용 내역  

**2.1.4.2.** 검색 결과 표시  
1: 드롭다운 형태로 검색 결과 표시  
　A. 입력창 아래에 드롭다운 목록 노출  
　B. 최대 10개 결과 표시  
　C. 더보기 버튼: 결과가 10개 이상일 경우 표시  

2: 검색 결과 카테고리  
　A. 메뉴/페이지: 해당 페이지로 이동  
　B. 캠페인: 캠페인 상세 페이지로 이동  
　C. 고객: 고객 상세 정보 페이지로 이동  

3: 검색 결과 없음  
　A. "검색 결과가 없습니다" 메시지 표시  
　B. 추천 검색어 또는 도움말 링크 제공  

**2.1.4.3.** 검색 실행  
1: Enter 키 입력 시 검색 실행  
2: 검색 결과 항목 클릭 시 해당 페이지로 이동  
3: ESC 키 입력 시 입력창 닫힘 및 텍스트 초기화  


**2.1.5.** 모바일 환경 고려사항  

**2.1.5.1.** 터치 인터랙션  
1: 돋보기 아이콘 탭: 전체 화면 검색 모드 진입  
　A. 모바일에서는 호버 대신 탭으로 동작  
　B. 전체 화면 또는 상단 고정 검색창 표시  

2: 검색 결과 탭: 해당 페이지로 이동  
3: 취소 버튼 또는 외부 탭: 검색 모드 종료  

**2.1.5.2.** 레이아웃 조정  
1: 모바일: 돋보기 아이콘 크기 축소 (예: 20px)  
2: 태블릿: 돋보기 아이콘 크기 중간 (예: 24px)  
3: 데스크톱: 돋보기 아이콘 크기 전체 (예: 28px)
