**2.3.** 액션 버튼 및 화면 전환  


**2.3.1.** 검색창 클릭 시 화면 전환  

**2.3.1.1.** 돋보기 아이콘 클릭 시 동작  
1: 입력창 확장  
　A. 트리거: 돋보기 아이콘 클릭  
　B. 동작: 입력창 슬라이드 확장 + 자동 포커스  
　C. 애니메이션: slide-left 또는 expand (0.2초)  

2: 입력창이 이미 열린 상태  
　A. 동작: 입력창에 포커스 활성화  
　B. 커서 활성화: 입력 가능 상태  

**2.3.1.2.** 검색 결과 항목 클릭 시 화면 전환  
1: 메뉴/페이지 결과 클릭  
　A. 이동 경로: 해당 메뉴/페이지 경로  
　B. 이동 방식: 페이지 전환  
　C. 검색창 상태: 닫힘 및 텍스트 초기화  

2: 캠페인 결과 클릭  
　A. 이동 경로: /partner/campaign/{campaign_id}  
　B. 이동 방식: 페이지 전환  
　C. 전달 파라미터: campaign_id  

3: 고객 결과 클릭 (멤버십 구독 시)  
　A. 이동 경로: /partner/customer/{customer_id}  
　B. 이동 방식: 페이지 전환  
　C. 전달 파라미터: customer_id  


**2.3.2.** 알림창 클릭 시 화면 전환  

**2.3.2.1.** 종 아이콘 클릭 시 동작  
1: 알림 드롭다운 표시  
　A. 트리거: 종 아이콘 클릭  
　B. 동작: 알림 목록 드롭다운 노출  
　C. 애니메이션: fade-in + slide-down (0.2초)  

2: 드롭다운이 이미 열린 상태  
　A. 동작: 드롭다운 닫힘  
　B. 애니메이션: fade-out + slide-up (0.2초)  

**2.3.2.2.** 알림 항목 클릭 시 화면 전환  
1: 일일 소진 보고 미완료 알림 클릭  
　A. 이동 경로: /partner/campaign/report  
　B. 이동 방식: 페이지 전환  
　C. 해당 알림 읽음 처리  

2: 재고 부족 경고 알림 클릭  
　A. 이동 경로: /partner/campaign/order  
　B. 이동 방식: 페이지 전환  
　C. 해당 알림 읽음 처리  

3: 이상치 경고 알림 클릭  
　A. 이동 경로: /partner/campaign/report/detail  
　B. 이동 방식: 페이지 전환  
　C. 해당 알림 읽음 처리  

4: 등급 변동 알림 클릭  
　A. 이동 경로: /partner/grade  
　B. 이동 방식: 페이지 전환  
　C. 해당 알림 읽음 처리  

5: 정산 완료 알림 클릭  
　A. 이동 경로: /partner/settlement  
　B. 이동 방식: 페이지 전환  
　C. 해당 알림 읽음 처리  

6: 캠페인 신규 매칭 알림 클릭  
　A. 이동 경로: /partner/campaign/order  
　B. 이동 방식: 페이지 전환  
　C. 해당 알림 읽음 처리  

**2.3.2.3.** 전체 알림 보기 클릭  
1: 알림 전체 페이지 이동  
　A. 이동 경로: /partner/notifications  
　B. 이동 방식: 페이지 전환  
　C. 드롭다운 닫힘  


**2.3.3.** 호버 툴팁 표시  

**2.3.3.1.** 검색 아이콘 호버  
1: 호버 시 동작  
　A. 트리거: 마우스 커서가 돋보기 아이콘 위에 위치  
　B. 동작: 입력창 슬라이드 확장 (포커스 없음)  
　C. 애니메이션: slide-left 또는 expand (0.2초)  

2: 호버 해제 시 동작  
　A. 입력창 영역 밖으로 커서 이동  
　B. 동작: 입력창 숨김 (텍스트가 없는 경우)  
　C. 텍스트가 있는 경우: 입력창 유지  

**2.3.3.2.** 알림 아이콘 호버  
1: 호버 시 동작  
　A. 트리거: 마우스 커서가 종 아이콘 위에 위치  
　B. 표시 시간: 0.5초 후 자동 표시  
　C. 툴팁 메시지: "클릭하면 알림 목록이 표시됩니다" 또는 "N개의 새로운 알림이 있습니다"  
　D. 툴팁 위치: 종 아이콘 바로 아래  
　E. 커서 이동 시: 툴팁 자동 사라짐  


**2.3.4.** 모바일 환경 고려사항  

**2.3.4.1.** 터치 인터랙션  
1: 돋보기 아이콘 탭  
　A. 전체 화면 검색 모드 진입  
　B. 호버 동작 대신 탭으로 입력창 표시  

2: 종 아이콘 탭  
　A. 알림 드롭다운 표시 (바텀 시트 형태)  
　B. 또는 알림 전체 페이지로 바로 이동  

3: 호버 툴팁  
　A. 모바일에서는 표시하지 않음 (터치만으로 충분)  

**2.3.4.2.** 모달/드롭다운 표시 방식  
1: 모바일: 바텀 시트 형태로 표시  
2: 데스크톱: 드롭다운 또는 팝오버 형태로 표시  

**2.3.4.3.** 레이아웃 조정  
1: 모바일: 검색 아이콘, 알림 아이콘이 한 줄에 배치되도록 크기 조정  
2: 화면 크기에 따라 아이콘 크기 자동 조정 (반응형)  
3: 공간 부족 시: 아이콘 크기 축소, 아이콘 간격 최소화  


**2.3.5.** 뒤로가기 시 동작  

**2.3.5.1.** 검색 결과 페이지에서 뒤로가기  
1: 이전 페이지로 이동 (메인 화면 또는 대시보드)  

**2.3.5.2.** 알림 전체 페이지에서 뒤로가기  
1: 이전 페이지로 이동 (메인 화면 또는 대시보드)  

**2.3.5.3.** 알림 드롭다운이 열린 상태에서 뒤로가기  
1: 드롭다운 닫힘  
2: 원래 화면 유지
