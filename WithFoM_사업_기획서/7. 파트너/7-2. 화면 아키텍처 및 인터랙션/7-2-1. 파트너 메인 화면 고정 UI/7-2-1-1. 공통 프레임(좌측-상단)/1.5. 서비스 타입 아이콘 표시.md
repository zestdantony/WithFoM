**1.5.** 서비스 타입 아이콘 표시  


**1.5.1.** 서비스 타입 아이콘 위치  

1: 아이콘 위치  
　A. 좌측 상단 공통 프레임 내  
　B. 파트너 로고와 매장명과 같은 줄에 배치 (2줄)  
　C. 배치 순서: 파트너 로고 → 매장명 → 등급 마크 → 멤버십 플랜 뱃지 → 서비스 타입 아이콘 (나란히 배치)  

2: 배치 순서  
　A. 1줄: 위드폼 로고 + 파트너 포털 표기  
　B. 2줄: 파트너 로고 + 매장명 + 등급 마크 + 멤버십 플랜 뱃지 + 서비스 타입 아이콘 (나란히 배치)  


**1.5.2.** 서비스 타입별 아이콘 표시  

1: 리워드형  
　A. 아이콘: 리워드 아이콘만 표시  
　B. 아이콘 스타일: 리워드 관련 아이콘 (예: 선물 아이콘, 쿠폰 아이콘)  
　C. 색상: 리워드 테마 색상  
　D. 크기: 등급 마크 및 멤버십 뱃지와 유사한 크기 (예: 24px ~ 32px)  

2: 오더형  
　A. 아이콘: 오더 아이콘만 표시  
　B. 아이콘 스타일: 주문/결제 관련 아이콘 (예: 카트 아이콘, 주문 아이콘)  
　C. 색상: 오더 테마 색상  
　D. 크기: 등급 마크 및 멤버십 뱃지와 유사한 크기 (예: 24px ~ 32px)  

3: 복합형  
　A. 아이콘: 리워드 아이콘 + 오더 아이콘 나란히 표시  
　B. 아이콘 배치: 두 아이콘을 작은 간격으로 나란히 배치 (예: 4px 간격)  
　C. 아이콘 스타일: 각각 리워드/오더 아이콘 사용  
　D. 색상: 리워드 테마 색상 + 오더 테마 색상  
　E. 크기: 각 아이콘은 등급 마크 및 멤버십 뱃지보다 약간 작게 (예: 20px ~ 24px)  


**1.5.3.** 호버 툴팁  

1: 마우스 커서 호버 시  
　A. 툴팁 메시지 표시: 서비스 타입 상세 정보  
　B. 툴팁 내용  

| 서비스 타입 | 툴팁 내용 |
|---|---|
| 리워드형 | "리워드 서비스 활성화<br>QR 스캔 후 게임/리워드 이용 가능" |
| 오더형 | "오더 서비스 활성화<br>앱 주문·결제 이용 가능" |
| 복합형 | "리워드 + 오더 서비스 활성화<br>QR 스캔 게임/리워드 및 앱 주문·결제 모두 이용 가능" |

　C. 툴팁 위치: 서비스 타입 아이콘 바로 아래 또는 우측  
　D. 툴팁 표시 시간: 0.5초 후 자동 표시  
　E. 커서 이동 시 툴팁 자동 사라짐  


**1.5.4.** 서비스 타입 아이콘 클릭 시 동작  

1: 클릭 시 동작  
　A. 서비스 타입 설명 모달/팝오버 표시  
　B. 서비스 타입별 아이콘 설명 및 서비스 제공 내용 간단 설명 표시  
　C. 모달/팝오버 타입: 팝오버 또는 센터 모달  
　D. 배경 딤 처리: 모달 표시 시 배경 어둡게 처리  

2: 모달/팝오버 내용  

| 서비스 타입 | 아이콘 설명 | 서비스 제공 내용 |
|---|---|---|
| 리워드형 | 리워드 아이콘: 선물/쿠폰 아이콘으로 표시, 리워드 테마 색상 적용 | • QR 스캔 후 게임 참여 가능<br>• 광고주 리워드: 제품 할인 쿠폰, 무료 증정, E-Com 포인트, 샘플 키트<br>• 파트너 리워드: 매장 전용 혜택 (사이즈업, 디저트 증정 등)<br>• 스탬프 적립: 특정 금액 이상 결제 시 적립, 완성 시 보상 쿠폰 발급<br>• 앱 주문·결제 기능은 사용 불가 |
| 오더형 | 오더 아이콘: 카트/주문 아이콘으로 표시, 오더 테마 색상 적용 | • 앱 주문·결제 기능 제공<br>• 메뉴 오더 및 메뉴 관리 기능 사용 가능<br>• 실시간 주문 리스트 확인<br>• 카테고리/메뉴/옵션 관리<br>• QR 스캔 게임/리워드 기능은 사용 불가 |
| 복합형 | 리워드 아이콘 + 오더 아이콘 나란히 표시 | • 리워드형과 오더형의 모든 기능 사용 가능<br>• QR 스캔 게임/리워드 및 앱 주문·결제 모두 이용 가능<br>• 쿠폰/포인트 결제 시 자동 적용 |

3: 모달/팝오버 닫기  
　A. 닫기 버튼 클릭  
　B. 배경 클릭  
　C. ESC 키 입력  

4: 클릭 가능 여부  
　A. 서비스 타입 아이콘은 클릭 가능한 버튼 형태  
　B. 호버 시 커서 포인터 표시  
　C. 클릭 시 서비스 타입 설명 모달/팝오버 표시  


**1.5.5.** 서비스 타입 연동  

1: 리워드 설정 연동  
　A. 서비스 타입과 리워드 참여 여부(리워드 설정 > 참여 여부)는 연동  
　B. 리워드 설정 화면에서 리워드를 Off 하면 서비스 타입도 연동되어 리워드 아이콘 비활성화 또는 숨김  
　C. 오더만 활성화된 경우: 오더 아이콘만 표시  

2: 실시간 반영  
　A. 서비스 타입 변경 시 아이콘 즉시 업데이트  
　B. 리워드 설정 변경 시 아이콘 상태 즉시 반영
