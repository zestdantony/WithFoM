**1.6.** 액션 버튼 및 화면 전환  


**1.6.1.** 위드폼 로고 클릭 시 화면 전환  

1: 홈 화면(대시섹션)으로 이동  
　A. 트리거: 위드폼 로고 클릭  
　B. 이동 경로: /partner/dashboard 또는 /partner/home  
　C. 이동 방식: 페이지 전환 (새 페이지로 이동)  
　D. 전달 파라미터: 없음  

2: 현재 페이지 확인  
　A. 현재 페이지가 홈 화면인 경우: 클릭해도 동일 화면 유지  
　B. 페이지 새로고침 없이 현재 상태 유지  

3: 호버 효과  
　A. 마우스 커서 호버 시: 포인터 커서 표시  
　B. 로고 약간 확대 효과 (선택 사항)  


**1.6.2.** 등급 마크 클릭 시 화면 전환  

1: 등급 설명 모달/팝오버 표시  
　A. 트리거: 등급 마크 클릭  
　B. 모달 타입: 팝오버 또는 센터 모달  
　C. 배경 딤 처리: 모달 표시 시 배경 어둡게 처리  
　D. 모달 내용: 현재 등급 정보 및 등급별 혜택 상세 설명 표시  
　E. 간단한 등급 산정 안내 설명 표시  

2: 모달/팝오버 닫기  
　A. 닫기 버튼 클릭: 모달 닫힘, 원래 화면 유지  
　B. 배경 클릭: 모달 닫힘, 원래 화면 유지  
　C. ESC 키 입력: 모달 닫힘, 원래 화면 유지  


**1.6.3.** 멤버십 뱃지 클릭 시 화면 전환  

1: 멤버십 플랜 설명 모달/팝오버 표시  
　A. 트리거: 멤버십 뱃지 클릭  
　B. 모달 타입: 팝오버 또는 센터 모달  
　C. 배경 딤 처리: 모달 표시 시 배경 어둡게 처리  

2: "플러스로 업그레이드" 또는 "프리미엄으로 업그레이드" 버튼 클릭  
　A. 이동 경로: /partner/membership/subscribe  
　B. 이동 방식: 페이지 전환 (새 페이지로 이동)  
　C. 전달 파라미터  

| 파라미터 | 값 | 설명 |
|---|---|---|
| plan | plus 또는 premium | 업그레이드 대상 플랜 |
| source | header_badge | 진입 경로 (헤더 뱃지에서 진입) |

3: 모달/팝오버 닫기  
　A. 닫기 버튼 클릭: 모달 닫힘, 원래 화면 유지  
　B. 배경 클릭: 모달 닫힘, 원래 화면 유지  
　C. ESC 키 입력: 모달 닫힘, 원래 화면 유지  


**1.6.4.** 서비스 타입 아이콘 클릭 시 화면 전환  

1: 서비스 타입 설명 모달/팝오버 표시  
　A. 트리거: 서비스 타입 아이콘 클릭  
　B. 모달 타입: 팝오버 또는 센터 모달  
　C. 배경 딤 처리: 모달 표시 시 배경 어둡게 처리  

2: 모달/팝오버 내용  
　A. 리워드형: 리워드 아이콘 설명 + 리워드형 서비스 제공 내용 간단 설명  
　B. 오더형: 오더 아이콘 설명 + 오더형 서비스 제공 내용 간단 설명  
　C. 복합형: 리워드 아이콘 + 오더 아이콘 설명 + 복합형 서비스 제공 내용 간단 설명  

3: 모달/팝오버 닫기  
　A. 닫기 버튼 클릭: 모달 닫힘, 원래 화면 유지  
　B. 배경 클릭: 모달 닫힘, 원래 화면 유지  
　C. ESC 키 입력: 모달 닫힘, 원래 화면 유지  


**1.6.5.** 뒤로가기 시 동작  

1: 등급 설명 모달에서 뒤로가기  
　A. 모달 닫힘  
　B. 원래 화면 유지  

2: 멤버십 플랜 설명 모달에서 뒤로가기  
　A. 모달 닫힘  
　B. 원래 화면 유지  

3: 등급 상세 페이지에서 뒤로가기  
　A. 이전 페이지로 이동 (메인 화면 또는 대시섹션)  

4: 멤버십 구독 페이지에서 뒤로가기  
　A. 이전 페이지로 이동 (메인 화면 또는 대시섹션)  

5: 서비스 타입 설명 모달에서 뒤로가기  
　A. 모달 닫힘  
　B. 원래 화면 유지  


**1.6.6.** 호버 툴팁 표시  

1: 등급 마크 호버  
　A. 트리거: 마우스 커서가 등급 마크 위에 위치  
　B. 표시 시간: 0.5초 후 자동 표시  
　C. 툴팁 메시지: "클릭하면 등급 설명이 표시됩니다"  
　D. 툴팁 위치: 등급 마크 바로 아래 또는 우측  
　E. 커서 이동 시: 툴팁 자동 사라짐  

2: 멤버십 뱃지 호버  
　A. 트리거: 마우스 커서가 멤버십 뱃지 위에 위치  
　B. 표시 시간: 0.5초 후 자동 표시  
　C. 툴팁 메시지: "클릭하면 멤버십 플랜 설명이 표시됩니다"  
　D. 툴팁 위치: 멤버십 뱃지 바로 아래 또는 우측  
　E. 커서 이동 시: 툴팁 자동 사라짐  

3: 서비스 타입 아이콘 호버  
　A. 트리거: 마우스 커서가 서비스 타입 아이콘 위에 위치  
　B. 표시 시간: 0.5초 후 자동 표시  
　C. 툴팁 메시지: 서비스 타입별 상세 정보 표시  
　D. 툴팁 위치: 서비스 타입 아이콘 바로 아래 또는 우측  
　E. 커서 이동 시: 툴팁 자동 사라짐  


**1.6.7.** 모바일 환경 고려사항  

1: 터치 인터랙션  
　A. 등급 마크 탭: 등급 설명 모달 표시  
　B. 멤버십 뱃지 탭: 멤버십 플랜 설명 모달 표시  
　C. 서비스 타입 아이콘 탭: 서비스 타입 설정 페이지로 이동  
　D. 호버 툴팁: 모바일에서는 표시하지 않음 (터치만으로 충분)  

2: 모달 표시 방식  
　A. 모바일: 바텀 시트 형태로 표시  
　B. 데스크톱: 센터 모달 또는 팝오버 형태로 표시  

3: 레이아웃 조정  
　A. 모바일: 파트너 로고, 매장명, 등급 마크, 멤버십 뱃지, 서비스 타입 아이콘이 한 줄에 배치되도록 크기 조정  
　B. 화면 크기에 따라 로고, 매장명, 아이콘 크기 자동 조정 (반응형)  
　C. 공간 부족 시: 로고 크기 축소, 매장명 텍스트 크기 축소, 아이콘 간격 최소화
