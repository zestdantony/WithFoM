**1.1.** 좌측 상단에 위드폼 로고와 파트너 포털 표기  


**1.1.1.** 위드폼 로고 표시  

**1.1.1.1.** 로고 위치  
1: 좌측 상단 공통 프레임 내 최상단  
2: 클릭 가능한 버튼 형태로 표시  
3: 호버 시 커서 포인터 표시  

**1.1.1.2.** 로고 클릭 동작  
1: 위드폼 로고 클릭 시 파트너 포털 홈 화면(대시섹션)으로 이동  
2: 이동 경로: /partner/dashboard 또는 /partner/home  
3: 현재 페이지가 홈 화면인 경우 클릭해도 동일 화면 유지  


**1.1.2.** 파트너 포털 표기  

**1.1.2.1.** 표기 위치  
1: 위드폼 로고의 오른쪽에 배치  
2: 로고와 적절한 간격 유지 (예: 12px ~ 16px)  
3: 로고보다 작은 크기로 표기 (예: 로고 높이의 60~70%)  

**1.1.2.2.** 표기 스타일  
1: 텍스트 색상: 회색 또는 보조 색상  
2: 폰트 크기: 로고보다 작게 (예: 12px ~ 14px)  
3: 폰트 굵기: 일반 또는 중간 굵기  
4: 클릭 불가능한 텍스트 (로고만 클릭 가능)  


**1.1.3.** 레이아웃 구성  

**1.1.3.1.** 수평 배치  
1: 위드폼 로고 (왼쪽) + 파트너 포털 표기 (오른쪽)  
2: Flexbox 또는 Grid 레이아웃 사용  
3: 수직 정렬: 중앙 정렬 (align-items: center)  

**1.1.3.2.** 반응형 고려사항  
1: 모바일: 로고 크기 축소, 파트너 포털 표기 폰트 크기 조정  
2: 태블릿: 중간 크기 유지  
3: 데스크톱: 전체 크기 표시
