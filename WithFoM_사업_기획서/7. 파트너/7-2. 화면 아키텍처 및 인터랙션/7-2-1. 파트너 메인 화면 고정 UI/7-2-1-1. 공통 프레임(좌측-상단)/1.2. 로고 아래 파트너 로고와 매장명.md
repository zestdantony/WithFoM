**1.2.** 로고 아래 파트너 로고와 매장명  


**1.2.1.** 파트너 로고와 매장명 배치  

**1.2.1.1.** 배치 위치  
1: 좌측 상단 공통 프레임 내  
2: 위드폼 로고와 파트너 포털 표기 아래 위치 (2줄)  
3: 파트너 로고 + 매장명 + 등급 마크 + 멤버십 플랜 뱃지 + 서비스 타입 아이콘을 같은 줄에 나란히 배치  

**1.2.1.2.** 배치 순서  
1: 1줄: 위드폼 로고 + 파트너 포털 표기  
2: 2줄: 파트너 로고 + 매장명 + 등급 마크 + 멤버십 플랜 뱃지 + 서비스 타입 아이콘 (나란히 배치)  


**1.2.2.** 파트너 로고 표시  

**1.2.2.1.** 로고 위치 및 크기  
1: 파트너 로고는 매장명 왼쪽에 배치  
2: 로고 크기: 파트너가 업로드한 로고 이미지의 원본 비율 유지  
3: 최대 높이: 매장명 텍스트 높이와 동일하게 조정  
4: 최대 너비: 화면 공간에 맞게 자동 조정 (예: 최대 80px)  
5: 로고와 매장명 사이 간격: 적절한 간격 유지 (예: 8px ~ 12px)  

**1.2.2.2.** 로고 크기 조정 로직  
1: 로고 원본 비율 유지  
2: 매장명 텍스트 높이에 맞춰 높이 조정  
3: 너비는 비율에 따라 자동 계산  
4: 화면 공간 부족 시 로고 크기 축소 (최소 32px 높이)  


**1.2.3.** 매장명 표시  

**1.2.3.1.** 매장명 위치 및 스타일  
1: 매장명은 파트너 로고 오른쪽에 배치  
2: 매장명과 등급 마크 사이 간격: 적절한 간격 유지 (예: 12px ~ 16px)  
3: 텍스트 크기: 기본 폰트 크기 (예: 16px ~ 18px)  
4: 폰트 굵기: 중간 굵기 (예: 500)  

**1.2.3.2.** 매장명 길이에 따른 크기 조정  
1: 매장명이 짧은 경우: 기본 크기 유지  
2: 매장명이 긴 경우: 텍스트 크기 자동 축소 (최소 12px)  
3: 텍스트 오버플로우 처리: 말줄임표(...) 표시 또는 텍스트 크기 축소  
4: 화면 공간에 맞게 동적 조정  
5: 등급 마크, 멤버십 뱃지, 서비스 타입 아이콘 공간 확보 우선  

**1.2.3.3.** 반응형 처리  
1: 모바일: 매장명 텍스트 크기 축소 (예: 14px)  
2: 태블릿: 매장명 텍스트 크기 중간 (예: 16px)  
3: 데스크톱: 매장명 텍스트 크기 전체 (예: 18px)  
4: 화면 크기에 따라 로고와 매장명 크기 자동 조정
