**1.3.** 현재 등급을 나타내는 마크  


**1.3.1.** 등급 마크 표시  

**1.3.1.1.** 등급 마크 위치  
1: 좌측 상단 공통 프레임 내  
2: 파트너 로고와 매장명과 같은 줄에 배치 (2줄)  
3: 배치 순서: 파트너 로고 → 매장명 → 등급 마크 → 멤버십 플랜 뱃지 → 서비스 타입 아이콘 (나란히 배치)  
4: 등급별 색상 및 아이콘으로 구분  

**1.3.1.2.** 등급별 표시  
1: 브론즈: 브론즈 색상 + 브론즈 아이콘  
2: 실버: 실버 색상 + 실버 아이콘  
3: 골드: 골드 색상 + 골드 아이콘  
4: 플래티넘: 플래티넘 색상 + 플래티넘 아이콘  


**1.3.2.** 호버 툴팁  

**1.3.2.1.** 마우스 커서 호버 시  
1: 툴팁 메시지 표시: "클릭하면 등급 설명이 표시됩니다"  
2: 툴팁 위치: 등급 마크 바로 아래 또는 우측  
3: 툴팁 표시 시간: 0.5초 후 자동 표시  
4: 커서 이동 시 툴팁 자동 사라짐  


**1.3.3.** 등급 마크 클릭 시 파트너 등급 설명 노출  

**1.3.3.1.** 클릭 시 동작  
1: 등급 설명 모달/팝오버 표시  
2: 현재 등급 정보 표시  
3: 등급별 혜택 상세 설명 표시  
4: 간단한 등급 산정 안내 설명 표시  

**1.3.3.2.** 모달/팝오버 내용  

| 항목 | 내용 |
|---|---|
| 현재 등급 | 현재 등급 마크 + 등급명 표시 |
| 등급 유지 기간 | 등급 유지 기간 정보 |
| 등급별 혜택 상세 | 현재 등급의 모든 혜택 상세 설명 (1.3.4. 등급 설명 상세 내용 참조) |
| 등급 산정 안내 | 등급은 일일 컵홀더 소진 보고율과 동일 캠페인 벤치마크 대비 QR 스캔율을 기준으로 자동 산정됨 |
| 등급 유지 조건 | 6개월 이내 새 캠페인 매칭 시 등급 유지 |
| 미매칭 경고 | 미매칭 3개월 이상 시 경고 배지 표시 (1.3.5. 참조) |

**1.3.3.3.** 모달/팝오버 닫기  
1: 닫기 버튼 클릭  
2: 배경 클릭  
3: ESC 키 입력  


**1.3.4.** 등급 설명 상세 내용  

**1.3.4.1.** 브론즈 등급 혜택  
1: 컵홀더 단가의 50% 할인가로 구매  

**1.3.4.2.** 실버 등급 혜택  
1: 컵홀더 무료 제공  
2: 등급 유지 시 지속 혜택  

**1.3.4.3.** 골드 등급 혜택  
1: 컵홀더 무료 제공  
2: 멤버십 플랜 30% 할인  

**1.3.4.4.** 플래티넘 등급 혜택  
1: 컵홀더 무료 제공  
2: 멤버십 플랜 30% 할인  
3: 컵홀더 개당 10원 인센티브  


**1.3.5.** 미매칭 경고 표시  

**1.3.5.1.** 경고 배지 표시 조건  
1: 미매칭 3개월 이상 경과 시 노란색 경고 배지 표시  
2: 미매칭 5개월 3주차 이후 빨간색 경고 배지 표시  
3: 캠페인 매칭 완료 시 경고 배지 숨김  

**1.3.5.2.** 경고 배지 위치 및 스타일  
1: 등급 마크 우측 상단에 작은 배지로 표시  
2: 노란색 (3개월+): 주의 아이콘 (⚠️)  
3: 빨간색 (5개월 3주차+): 긴급 아이콘 (🚨)  

**1.3.5.3.** 경고 배지 클릭 시  
1: 미매칭 상태 상세 모달 표시  
2: 남은 유예 기간 표시  
3: 캠페인 매칭 안내 버튼 표시  
4: 다운그레이드 예정일 표시
