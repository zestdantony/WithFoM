**2.2.** 상품 리스트  


**2.2.1.** 리스트 레이아웃  

**2.2.1.1.** 뷰 모드  
1: 그리드 뷰 (기본)  
　A. 4열 배치 (데스크톱)  
　B. 2열 배치 (태블릿)  
　C. 1열 배치 (모바일)  
2: 리스트 뷰  
　A. 1행 1상품 형태  
　B. 상세 정보 더 많이 표시  

**2.2.1.2.** 정렬 옵션  
1: 인기순 (기본)  
2: 최신순  
3: 가격 낮은순  
4: 가격 높은순  
5: 리뷰 많은순  


**2.2.2.** 상품 카드 구성  

**2.2.2.1.** 그리드 뷰 카드  
1: 상품 이미지  
　A. 썸네일 이미지 (정사각형 또는 4:3 비율)  
　B. 호버 시 서브 이미지 전환 (선택적)  
2: 상품 정보  
　A. 상품명  
　B. 브랜드명 (선택적)  
　C. 가격 (단가 기준)  
　D. 단위 (예: 50개입, 1박스)  
3: 추가 표시  
　A. 베스트 뱃지 (인기 상품)  
　B. NEW 뱃지 (신규 상품)  
　C. 할인 뱃지 (할인 중)  
　D. 품절 오버레이 (재고 없음)  
4: 빠른 액션  
　A. 장바구니 담기 버튼 (호버 시 표시)  
　B. 찜하기 아이콘 (선택적)  

**2.2.2.2.** 리스트 뷰 행  
1: 좌측: 상품 이미지 (썸네일)  
2: 중앙: 상품 정보  
　A. 상품명  
　B. 상세 설명 (1-2줄)  
　C. 규격/사양  
3: 우측: 가격 및 액션  
　A. 단가  
　B. 수량 입력  
　C. [장바구니 담기] 버튼  


**2.2.3.** 검색 및 필터  

**2.2.3.1.** 검색  
1: 검색 대상  
　A. 상품명  
　B. 상품 설명  
　C. 브랜드명  
2: 자동완성 지원  
3: 최근 검색어 표시  

**2.2.3.2.** 카테고리 필터  
1: 좌측 사이드바  
　A. 대분류 카테고리 리스트  
　B. 클릭 시 해당 카테고리 상품만 표시  
2: 서브 카테고리  
　A. 대분류 확장 시 서브 카테고리 표시  
　B. 다중 선택 가능  

**2.2.3.3.** 추가 필터  
1: 가격대 필터  
　A. 범위 슬라이더 또는 직접 입력  
2: 배송 유형  
　A. 일반배송 / 빠른배송  
3: 재고 연동 필터  
　A. 내 재고 관리에 등록된 상품만 표시  


**2.2.4.** 상품 카드 인터랙션  

**2.2.4.1.** 카드 클릭  
1: 클릭 시 동작  
　A. 상품 상세 페이지로 이동  

**2.2.4.2.** 빠른 장바구니 담기  
1: 호버 시  
　A. [장바구니] 버튼 표시  
　B. 수량 선택 없이 기본 수량(1) 담기  
2: 클릭 후  
　A. 장바구니 담기 완료 토스트  
　B. 장바구니 아이콘 뱃지 숫자 증가  

**2.2.4.3.** 품절 상품  
1: 표시  
　A. 이미지에 품절 오버레이  
　B. 장바구니 담기 버튼 비활성화  
2: 클릭 시  
　A. 상품 상세 페이지로 이동 (정보 확인 가능)  
　B. [재입고 알림 신청] 버튼 제공  


**2.2.5.** 페이지네이션  

**2.2.5.1.** 페이지네이션 옵션  
1: 무한 스크롤 (기본)  
　A. 스크롤 시 자동으로 다음 페이지 로드  
　B. 로딩 인디케이터 표시  
2: 페이지 번호 방식 (선택 가능)  
　A. 하단에 페이지 번호 표시  
　B. 이전/다음 버튼  
3: 페이지당 표시 개수  
　A. 20 / 40 / 60개 선택 가능
