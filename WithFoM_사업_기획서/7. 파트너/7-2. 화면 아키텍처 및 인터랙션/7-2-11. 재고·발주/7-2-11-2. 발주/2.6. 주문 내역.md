**2.6.3.3.** 배송 문제 문의 (큰 구조 — 추후 상세 세팅 예정)

　A. 배송중 또는 배송완료 상태에서 [배송 문의] 버튼 표시  
　B. 캠페인 현황 페이지 > 캠페인 참여 섹션의 [배송 문의 →] 클릭 시 이 페이지로 이동  
2: 문의 양식 (추후 세팅)  
　A. 문제 유형 선택: 미배송 / 배송 지연 / 파손 / 수량 불일치 / 기타  
　B. 문제 상황 설명 (텍스트 입력)  
　C. 사진 첨부 (선택)  
3: 문의 발송 대상  
　A. 협력사에게 문의 내용 전송  
　B. 관리자에게 자동 알림: "파트너 {매장명}이 주문 #{번호} 배송 문제를 문의했습니다"  
4: 문의 결과 표시 (추후 세팅)  
　A. 협력사 답변 표시  
　B. 상태 변경 이력 표시  
　C. 추후 2.1. 캠페인 참여 섹션의 배송 현황과 연동

**2.6.4.3.** 협력사 입고 상태 복구 (큰 구조 — 추후 상세 세팅 예정)  
1: 복구 조건  
　A. 물류 사고 등으로 실제 미입고 확인 시 협력사가 입고완료를 이전 상태(배송준비 또는 배송중)로 복구 가능  
　B. 파트너의 배송 문제 문의 + 협력사 확인 후 복구 조치  
2: 복구 시 관리자 알림  
　A. 관리자에게 자동 알림: "협력사 {사명}이 주문 #{번호} 입고 상태를 {이전}→{복구}로 변경했습니다"  
　B. 중요도: 중요  
3: 파트너 화면 자동 업데이트  
　A. 캠페인 현황 페이지 > 캠페인 참여 섹션의 배송 현황이 자동으로 복구된 상태로 갱신  
　B. 파트너 알림: "배송 상태가 {이전}에서 {복구}로 변경되었습니다. 배송 현황을 확인해 주세요."
