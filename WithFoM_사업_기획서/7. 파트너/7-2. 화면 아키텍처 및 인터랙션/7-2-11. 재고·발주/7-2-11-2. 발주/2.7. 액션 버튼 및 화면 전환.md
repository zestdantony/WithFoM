**2.7.** 액션 버튼 및 화면 전환  


**2.7.1.** 상품 리스트 페이지  

**2.7.1.1.** 헤더 액션  
1: 장바구니 아이콘  
　A. 클릭 시: 장바구니 페이지로 이동  
　B. 뱃지: 담긴 상품 수 표시  
2: 검색 버튼  
　A. 클릭 시: 검색 모달 또는 검색창 포커스  

**2.7.1.2.** 상품 카드 액션  
1: 카드 클릭  
　A. 상품 상세 페이지로 이동  
2: 장바구니 버튼 (호버 시)  
　A. 클릭 시: 장바구니에 추가 (기본 수량 1)  
　B. 성공 토스트: "장바구니에 추가되었습니다"  


**2.7.2.** 상품 상세 페이지  

**2.7.2.1.** 구매 버튼  
1: [장바구니 담기]  
　A. 선택 수량만큼 장바구니에 추가  
　B. 성공 토스트 + 장바구니 아이콘 뱃지 업데이트  
2: [바로 구매]  
　A. 주문 확인 페이지로 바로 이동  
　B. 해당 상품만 주문 프로세스 진행  

**2.7.2.2.** 네비게이션  
1: 뒤로 가기  
　A. 이전 페이지 (상품 리스트)로 복귀  
2: 탭 전환  
　A. 상품설명 / 상품정보 / 배송정보 / 리뷰 탭 클릭 시 해당 섹션으로 스크롤  


**2.7.3.** 장바구니 페이지  

**2.7.3.1.** 상품 관리 액션  
1: 수량 변경  
　A. [-]/[+] 버튼 또는 직접 입력  
　B. 즉시 반영, 총액 재계산  
2: 상품 삭제  
　A. X 버튼 클릭  
　B. 확인 없이 즉시 삭제 (또는 간단 확인)  
3: 선택 삭제  
　A. 체크된 상품 일괄 삭제  
　B. 확인 팝업 표시  

**2.7.3.2.** 주문 진행  
1: [주문하기]  
　A. 선택된 상품으로 주문 확인 페이지 이동  
　B. 비활성화 조건: 선택 상품 없음  
2: [쇼핑 계속하기]  
　A. 상품 리스트 페이지로 이동  


**2.7.4.** 주문 확인 페이지  

**2.7.4.1.** 결제 버튼  
1: [결제하기]  
　A. 결제 프로세스 진행  
　B. PG 결제창 호출 또는 인앱 결제  
　C. 결제 완료 시 주문 완료 페이지로 이동  
2: [취소]  
　A. 장바구니 페이지로 복귀  
　B. 입력 정보 유지  


**2.7.5.** 주문 내역 페이지  

**2.7.5.1.** 주문 카드 액션  
1: [상세보기]  
　A. 주문 상세 페이지로 이동  
2: [재주문]  
　A. 해당 주문 상품을 장바구니에 추가  
　B. 장바구니 페이지로 이동  
3: [배송조회]  
　A. 배송 추적 모달 표시 또는 택배사 페이지 링크  

**2.7.5.2.** 주문 상세 액션  
1: [입고 확인]  
　A. 자동 입고 확인 팝업 표시  
　B. 확인 완료 시 상태 업데이트  
2: [주문취소]  
　A. 취소 확인 팝업  
　B. 취소 사유 선택  
　C. 취소 완료 처리  
3: [반품/교환 신청]  
　A. 반품/교환 신청 모달 표시  


**2.7.6.** 화면 전환 애니메이션  

**2.7.6.1.** 페이지 이동  
1: 리스트 → 상세  
　A. 좌측에서 우측으로 슬라이드 (300ms)  
2: 장바구니 → 주문확인 → 주문완료  
　A. 좌측에서 우측으로 슬라이드 (300ms)  
3: 뒤로 가기  
　A. 우측에서 좌측으로 슬라이드 (300ms)  

**2.7.6.2.** 모달  
1: 열기  
　A. 페이드인 + 슬라이드업 (250ms)  
　B. 배경 딤 처리  
2: 닫기  
　A. 페이드아웃 + 슬라이드다운 (200ms)  


**2.7.7.** 토스트 메시지  

**2.7.7.1.** 성공 토스트 (녹색)  
1: "장바구니에 추가되었습니다"  
2: "주문이 완료되었습니다"  
3: "입고 확인이 완료되었습니다"  

**2.7.7.2.** 알림 토스트 (파란색)  
1: "배송이 시작되었습니다"  
2: "배송이 완료되었습니다. 입고를 확인해 주세요."  

**2.7.7.3.** 경고 토스트 (주황색)  
1: "품절된 상품은 주문할 수 없습니다"  
2: "최소 주문 수량을 확인해 주세요"  

**2.7.7.4.** 에러 토스트 (빨간색)  
1: "결제에 실패했습니다. 다시 시도해 주세요."  
2: "주문 처리에 실패했습니다."
