**2.5.** 주문 확인 및 결제  


**2.5.1.** 페이지 구성  

**2.5.1.1.** 레이아웃  
1: 좌측/중앙: 주문 정보 입력 영역  
　A. 배송지 정보  
　B. 주문 상품 목록  
　C. 결제 수단 선택  
2: 우측: 결제 요약 영역  
　A. 최종 결제 금액  
　B. [결제하기] 버튼  


**2.5.2.** 배송지 정보  

**2.5.2.1.** 기본 배송지  
1: 매장 주소 자동 입력  
　A. 파트너 등록 시 입력한 매장 주소  
　B. 수령인: 매장명 또는 대표자명  
　C. 연락처: 등록된 연락처  
2: 배송지 변경  
　A. [변경] 버튼 클릭 시 수정 모드  
　B. 새 배송지 입력 또는 저장된 배송지 선택  

**2.5.2.2.** 배송 요청사항  
1: 선택 옵션  
　A. 드롭다운: 일반적인 요청사항 목록  
　B. 예: "부재 시 경비실에 맡겨주세요"  
2: 직접 입력  
　A. 텍스트 입력 (100자 이내)  


**2.5.3.** 주문 상품 목록  

**2.5.3.1.** 상품 표시  
1: 각 상품별 표시  
　A. 상품 이미지 (썸네일)  
　B. 상품명  
　C. 옵션 정보  
　D. 수량  
　E. 금액  
2: 수량 변경 불가  
　A. 이 단계에서는 수량 변경 불가  
　B. [장바구니로 돌아가기] 링크 제공  


**2.5.4.** 결제 수단  

**2.5.4.1.** 결제 방법 선택  
1: 신용/체크카드  
　A. 카드 정보 입력 또는 저장된 카드 선택  
　B. PG사 연동 결제  
2: 계좌이체  
　A. 실시간 계좌이체  
3: 간편결제  
　A. 네이버페이, 카카오페이 등  
4: 후불결제 (해당 시)  
　A. 월말 정산 방식  
　B. 승인된 파트너에 한해 제공  

**2.5.4.2.** 자동 결제 설정 (자동 발주용)  
1: 자동 발주 결제 수단 등록  
　A. 카드 등록  
　B. 자동 발주 시 해당 카드로 자동 결제  
2: 변경/삭제  
　A. 계정 설정에서 관리 가능  


**2.5.5.** 결제 요약  

**2.5.5.1.** 요약 항목  
1: 상품 금액  
　A. 주문 상품 총액  
2: 배송비  
　A. 배송비 (또는 무료)  
3: 할인 금액  
　A. 적용된 할인  
4: 최종 결제 금액  
　A. 대형 폰트로 강조  

**2.5.5.2.** 결제 버튼  
1: [결제하기]  
　A. 결제 진행  
　B. 결제 완료 시 주문 완료 페이지로 이동  
2: 이용약관 동의  
　A. 체크박스: "주문 내용을 확인하였으며, 결제에 동의합니다"  
　B. 체크 필수  


**2.5.6.** 주문 완료  

**2.5.6.1.** 완료 페이지  
1: 주문 완료 메시지  
　A. "주문이 완료되었습니다"  
　B. 주문번호 표시  
2: 주문 정보 요약  
　A. 주문 상품 목록  
　B. 배송지 정보  
　C. 결제 금액  
　D. 예상 배송일  
3: 버튼  
　A. [주문 내역 확인] - 주문 내역 페이지로 이동  
　B. [쇼핑 계속하기] - 상품 리스트로 이동  

**2.5.6.2.** 알림 발송  
1: 주문 확인 알림  
　A. 푸시 알림  
　B. 이메일 (선택)  
　C. SMS (선택)
