**1.8.** 액션 버튼 및 화면 전환  


**1.8.1.** 재고 관리 메인 페이지  

**1.8.1.1.** 헤더 액션 버튼  
1: 품목 추가 버튼  
　A. 클릭 시: 품목 등록 모달 또는 페이지로 이동  
2: 일괄 등록 버튼  
　A. 클릭 시: 엑셀 템플릿 다운로드 및 업로드 모달  
3: 설정 버튼  
　A. 클릭 시: 재고 관리 설정 페이지로 이동  
　B. 포함: 카테고리 관리, 자동 발주 일괄 설정, 알림 설정  


**1.8.2.** 품목 리스트 인터랙션  

**1.8.2.1.** 행 클릭  
1: 클릭 동작  
　A. 품목 상세 패널 표시 (우측 또는 하단)  
2: 더블클릭 동작  
　A. 빠른 수량 조정 모달 표시  

**1.8.2.2.** 액션 버튼  
1: 상세 버튼  
　A. 클릭 시: 품목 상세 페이지로 이동  
2: 발주 버튼  
　A. 클릭 시: 빠른 발주 모달 표시  
　B. 비활성화 조건: 관리자 미연동 품목  
3: 조정 버튼  
　A. 클릭 시: 수량 조정 모달 표시  
4: 입고 확인 뱃지 클릭  
　A. 클릭 시: 자동 입고 확인 팝업 표시  


**1.8.3.** 품목 상세 페이지  

**1.8.3.1.** 헤더 액션 버튼  
1: 수정 버튼  
　A. 클릭 시: 품목 정보 수정 모드 활성화  
2: 발주하기 버튼  
　A. 클릭 시: 해당 품목 발주 페이지 또는 모달로 이동  
　B. 비활성화 조건: 관리자 미연동 품목  
3: 수량 조정 버튼  
　A. 클릭 시: 수량 조정 모달 표시  
4: 삭제 버튼  
　A. 클릭 시: 삭제 확인 팝업 표시  
　B. 확인 시: 품목 삭제 (복구 불가 경고)  


**1.8.4.** 빠른 발주 모달  

**1.8.4.1.** 모달 구성  
1: 품목 정보 표시  
　A. 품목명, 현재 재고, 안전 재고  
2: 발주 수량 입력  
　A. 숫자 입력 필드  
　B. 권장 수량 표시 (안전 재고까지 채울 수량)  
3: 버튼  
　A. [장바구니에 추가]: 발주 페이지 장바구니에 추가  
　B. [바로 발주]: 즉시 발주 처리  
　C. [취소]: 모달 닫기  


**1.8.5.** 수량 조정 모달  

**1.8.5.1.** 모달 구성  
1: 조정 타입 선택  
　A. 라디오 버튼: 입고 / 출고 / 실사 조정 / 폐기·손실  
2: 조정량 입력  
　A. 숫자 입력 필드  
　B. 조정 후 예상 재고량 표시  
3: 사유 입력  
　A. 드롭다운 또는 텍스트 입력  
　B. 폐기·손실 시 필수 입력  
4: 버튼  
　A. [조정 완료]: 재고 즉시 반영  
　B. [취소]: 모달 닫기  


**1.8.6.** 화면 전환 애니메이션  

**1.8.6.1.** 페이지 이동  
1: 리스트 → 상세 페이지  
　A. 좌측에서 우측으로 슬라이드 전환 (300ms)  
2: 상세 페이지 → 리스트  
　A. 우측에서 좌측으로 슬라이드 전환 (300ms)  

**1.8.6.2.** 모달  
1: 모달 열기  
　A. 페이드인 + 스케일업 (200ms)  
　B. 배경 딤 처리  
2: 모달 닫기  
　A. 페이드아웃 + 스케일다운 (200ms)  


**1.8.7.** 토스트 메시지  

**1.8.7.1.** 성공 토스트 (녹색)  
1: "품목이 등록되었습니다"  
2: "재고가 조정되었습니다"  
3: "입고 확인이 완료되었습니다"  
4: "발주가 접수되었습니다"  

**1.8.7.2.** 경고 토스트 (주황색)  
1: "재고가 부족합니다. 발주를 검토해 주세요."  
2: "자동 발주가 생성되었습니다."  

**1.8.7.3.** 에러 토스트 (빨간색)  
1: "처리에 실패했습니다. 다시 시도해 주세요."  
2: "발주 가능 상품이 아닙니다."  


**1.8.8.** 뒤로 가기 동작  

**1.8.8.1.** 복귀 경로  
1: 품목 상세 → 품목 리스트  
　A. 브라우저 뒤로 가기 또는 상단 뒤로 버튼  
2: 모달 닫기  
　A. ESC 키  
　B. 모달 외부 클릭  
　C. X 버튼 클릭
