**1.4.** 품목 등록  


**1.4.1.** 등록 방식  

**1.4.1.1.** 개별 등록  
1: 등록 버튼 위치  
　A. 품목 리스트 상단 [+ 품목 추가] 버튼  
　B. 카테고리 내 [+ 추가] 버튼  
2: 등록 모달/페이지 구성  
　A. 기본 정보 입력 폼  
　B. 초기 재고 설정  
　C. 자동 발주 설정 (관리자 연동 품목만)  

**1.4.1.2.** 일괄 등록 (엑셀)  
1: 템플릿 다운로드  
　A. 품목 등록 엑셀 템플릿 제공  
　B. 컬럼: 품목명, 카테고리, 단위, 초기 재고, 안전 재고, 구매 단가  
2: 파일 업로드  
　A. 작성된 엑셀 파일 업로드  
　B. 데이터 검증 후 미리보기 표시  
　C. 오류 항목 표시 및 수정 가능  
　D. [등록 완료] 버튼으로 일괄 등록  


**1.4.2.** 등록 폼 구성  

**1.4.2.1.** 기본 정보  
1: 품목명 (필수)  
　A. 텍스트 입력  
　B. 중복 체크 (동일 품목명 경고)  
2: 카테고리 (필수)  
　A. 드롭다운 선택  
　B. 신규 카테고리 추가 옵션  
3: 단위 (필수)  
　A. 드롭다운 선택: ea, 카드, 봉지, kg, g, L, ml 등  
　B. 직접 입력 가능  
4: 품목 이미지 (선택)  
　A. 이미지 업로드  
　B. 권장 크기: 200×200px  

**1.4.2.2.** 재고 설정  
1: 초기 재고 (필수)  
　A. 숫자 입력  
　B. 기본값: 0  
2: 안전 재고 (권장)  
　A. 숫자 입력  
　B. 이 수량 이하 시 "위험" 상태 표시  
　C. 자동 발주 기준점  
3: 구매 단가 (선택)  
　A. 숫자 입력 (원)  
　B. 원가 계산 시 사용  

**1.4.2.3.** 자동 발주 설정  
1: 자동 발주 활성화  
　A. 토글: ON/OFF  
　B. 관리자 연동 품목만 활성화 가능  
2: 자동 발주 수량  
　A. 재고가 안전 재고 이하 시 발주할 수량  
　B. 숫자 입력  

**1.4.2.4.** 추가 정보  
1: 메모/설명 (선택)  
　A. 품목에 대한 추가 설명  
　B. 최대 200자  
2: 공급업체 (선택)  
　A. 텍스트 입력  
　B. 참고용 정보  


**1.4.3.** 등록 완료 후  

**1.4.3.1.** 등록 완료 동작  
1: 품목 리스트에 즉시 추가  
2: 성공 토스트 메시지 표시  
3: 레시피 연동 안내 (해당 시)  
　A. "이 품목을 메뉴 레시피에 연동하시겠습니까?"  
　B. [레시피 관리로 이동] / [나중에]  


**1.4.4.** 카테고리 확장성  

**1.4.4.1.** 새 카테고리 추가  
1: 품목 등록 시 카테고리 드롭다운에서 "새 카테고리 추가" 선택  
2: 카테고리명 입력  
3: 즉시 생성 및 선택 적용  

**1.4.4.2.** 카테고리 관리  
1: 설정 > 카테고리 관리에서  
　A. 카테고리 추가/수정/삭제  
　B. 카테고리 순서 변경  
　C. 카테고리별 아이콘/색상 설정  


• 참조: 7-2-11-1-3. 품목 상세 페이지
