**1.3.** 품목 상세 페이지  


**1.3.1.** 페이지 구성  

**1.3.1.1.** 상단 헤더  
1: 품목 기본 정보  
　A. 품목 이미지 (썸네일)  
　B. 품목명  
　C. 품목코드  
　D. 카테고리  
　E. 재고 상태 뱃지 (정상/주의/위험)  
2: 액션 버튼  
　A. [수정] - 품목 정보 수정  
　B. [발주하기] - 빠른 발주 (관리자 연동 품목만)  
　C. [수량 조정] - 재고 수동 조정  
　D. [삭제] - 품목 삭제 (확인 필요)  


**1.3.2.** 재고 현황 섹션  

**1.3.2.1.** 현재 재고 카드  
1: 표시 정보  
　A. 현재 재고량 (대형 숫자)  
　B. 단위  
　C. 안전 재고 대비 비율 프로그레스 바  
　D. 최종 갱신 일시  
2: 재고 수준 시각화  
　A. 프로그레스 바 색상: 녹색(정상) → 노란색(주의) → 빨간색(위험)  
　B. 안전 재고 라인 표시  

**1.3.2.2.** 예상 소진 카드  
1: 표시 정보  
　A. 일평균 소진량  
　B. 주평균 소진량  
　C. 예상 소진일 (D-day 형태)  
　D. 권장 발주 시점  
2: 알림 표시  
　A. 예상 소진일 7일 이내: 주황색 경고  
　B. 예상 소진일 3일 이내: 빨간색 긴급 경고  


**1.3.3.** 재고 변동 이력  

**1.3.3.1.** 이력 테이블  
1: 표시 컬럼  
　A. 일시  
　B. 변동 유형 (입고/출고/조정/폐기)  
　C. 변동량 (+/- 표시)  
　D. 변동 후 재고  
　E. 사유/메모  
　F. 처리자  
2: 필터 기능  
　A. 기간별 필터  
　B. 변동 유형별 필터  

**1.3.3.2.** 변동 유형 상세  
1: 입고 (녹색)  
　A. 발주 배송 완료 (자동)  
　B. 수동 입고 입력  
2: 출고 (파란색)  
　A. 판매 연동 자동 차감 (레시피 기반)  
　B. 수동 출고 입력  
3: 조정 (노란색)  
　A. 재고 실사 조정  
　B. 초기 재고 설정  
4: 폐기/손실 (빨간색)  
　A. 유통기한 만료  
　B. 파손/손실  
　C. 기타 폐기  


**1.3.4.** 발주 이력 섹션  

**1.3.4.1.** 이력 테이블  
1: 표시 컬럼  
　A. 발주일  
　B. 발주 수량  
　C. 발주 금액  
　D. 배송 상태 (주문확인/배송중/배송완료)  
　E. 배송 완료일  
　F. 입고 처리 상태 (자동입고/수동입고/미처리)  
2: 필터 및 정렬  
　A. 기간별 필터  
　B. 배송 상태별 필터  


**1.3.5.** 자동 발주 설정  

**1.3.5.1.** 설정 패널  
1: 자동 발주 활성화 토글  
　A. ON: 재고가 설정값 이하로 떨어지면 자동 발주  
　B. OFF: 수동 발주만 가능  
2: 발주 기준 설정  
　A. 최소 재고량 (자동 발주 트리거): 숫자 입력 + 단위  
　B. 발주 수량: 숫자 입력 + 단위  
　C. 발주 시점: 재고가 최소 재고량 이하가 되는 즉시 / 특정 시간에 일괄  
3: 알림 설정  
　A. 자동 발주 시 푸시 알림 ON/OFF  
　B. 자동 발주 시 이메일 알림 ON/OFF  
　C. 대표 계정에 발주 알림 ON/OFF  

**1.3.5.2.** 관리자 연동 조건  
1: 자동 발주 기능 제한  
　A. 관리자(위드폼)가 등록한 상품만 자동 발주 가능  
　B. 수동 관리 품목은 자동 발주 설정 비활성화  
　C. 수동 관리 품목에는 "관리자 연동 품목에서만 사용 가능" 안내 표시  


**1.3.6.** 수량 조정  

**1.3.6.1.** 조정 모달  
1: 조정 타입 선택  
　A. 입고: 재고 추가 (+)  
　B. 출고: 재고 차감 (-)  
　C. 실사 조정: 현재 재고를 특정 값으로 설정  
　D. 폐기/손실: 재고 차감 (사유 필수)  
2: 입력 필드  
　A. 조정량 또는 조정 후 재고량  
　B. 사유/메모 (선택, 일부 타입에서 필수)  
3: 확인 버튼  
　A. [조정 완료] 클릭 시 즉시 반영  
　B. 조정 이력에 기록  


**1.3.7.** 품목 정보 수정  

**1.3.7.1.** 수정 가능 항목  
1: 기본 정보  
　A. 품목명  
　B. 카테고리  
　C. 단위  
　D. 품목 이미지  
2: 발주 관련 정보  
　A. 안전 재고량  
　B. 자동 발주 수량  
　C. 구매 단가 (원가 계산용)  
3: 기타  
　A. 메모/설명  
　B. 사용 여부 (활성화/비활성화)
