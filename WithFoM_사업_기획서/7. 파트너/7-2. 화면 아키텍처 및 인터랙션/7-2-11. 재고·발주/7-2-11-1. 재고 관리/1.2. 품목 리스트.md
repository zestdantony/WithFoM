**1.2.** 품목 리스트  


**1.2.1.** 리스트 구성  

**1.2.1.1.** 테이블 컬럼  
1: 기본 컬럼  
　A. 체크박스 (일괄 선택용)  
　B. 품목코드 (자동 생성)  
　C. 품목명  
　D. 카테고리  
　E. 현재 재고  
　F. 단위 (ea, 박스, 봉지, kg, L 등)  
　G. 안전 재고 (자동 발주 기준)  
　H. 재고 상태 (정상/주의/위험)  
　I. 자동 발주 상태 (활성화/비활성화)  
　J. 최근 입고일  
　K. 액션 버튼 (상세/발주/조정)  
2: 확장 컬럼 (토글로 표시/숨김)  
　A. 구매 단가  
　B. 공급업체  
　C. 평균 소진량 (일/주)  
　D. 예상 소진일  


**1.2.1.2.** 재고 상태 표시  
1: 정상 (녹색)  
　A. 현재 재고 > 안전 재고  
　B. 아이콘: 녹색 체크  
2: 주의 (노란색)  
　A. 현재 재고 ≤ 안전 재고 × 1.5 AND > 안전 재고  
　B. 아이콘: 노란색 느낌표  
　C. 자동 발주 임계치 접근 알림  
3: 위험 (빨간색)  
　A. 현재 재고 ≤ 안전 재고  
　B. 아이콘: 빨간색 경고  
　C. 자동 발주 트리거 (활성화 시)  
4: 품절 (검정)  
　A. 현재 재고 = 0  
　B. 아이콘: 검정색 X  
　C. 긴급 발주 필요 알림

3: 최근 검색어 표시

　B. 다중 카테고리 선택 가능  
2: 재고 상태 필터  
　A. 전체 / 정상 / 주의 / 위험 / 품절  
3: 자동 발주 상태 필터  
　A. 전체 / 활성화 / 비활성화  
4: 연동 상태 필터

**1.2.2.3.** 정렬

　D. 최근 입고일  
　E. 예상 소진일 (빠른순)  


**1.2.3.** 리스트 인터랙션  

**1.2.3.1.** 행 클릭  
1: 클릭 시 동작  
　A. 우측 또는 하단에 상세 정보 패널 표시  
　B. 또는 품목 상세 페이지로 이동  
2: 더블클릭 시 동작

**1.2.3.2.** 액션 버튼

3: 조정 버튼  
　A. 재고 수량 수동 조정 모달 표시  

**1.2.3.3.** 일괄 작업

**1.2.4.** 확인 요망 표시  

**1.2.4.1.** 자동 입고 확인 필요

　C. 버튼: [확인 완료] / [수량 조정] / [이의 신청]

　B. 입고 확인 이력 저장
