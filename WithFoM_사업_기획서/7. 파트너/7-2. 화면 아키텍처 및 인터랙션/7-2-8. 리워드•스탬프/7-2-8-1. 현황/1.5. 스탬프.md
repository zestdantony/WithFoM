**1.5.** 스탬프  


**1.5.1.** 스탬프 카드 TOP5(5개 카드)  

**1.5.1.1.** 카드 구성  
　1: 스탬프 카드 이름  
　2: 발행 수량  
　3: 사용 수량  
　4: 사용률  


**1.5.2.** 스탬프 카드 현황  

**1.5.2.1.** 상단 컨트롤  
　1: 왼쪽 위: 전체 또는 파트너가 설정한 스탬프 카드 목록 선택 버튼  
　2: 오른쪽 위: 3개월 / 6개월 / 1년 / 기간설정 검색 기능  

**1.5.2.2.** 현황 4카드  
　1: 발행 수량  
　2: 사용 수량  
　3: 사용률  
　4: 사용 금액  


**1.5.3.** 발행 스탬프 카드 리스트  

**1.5.3.1.** 최근 발행된 스탬프 카드 리스트(표기 항목)  
　1: 스탬프 카드 이름  
　2: 스탬프 카드 내용  
　3: 발행 기간  
　4: 발행 수량  
　5: 사용 수량  
　6: 사용률  
　7: 스탬프 카드 사용 금액  

**1.5.3.2.** 리스트 표시 규칙  
　1: 20개씩 리스트업  
　2: 5개 창까지 넘길 수 있게 해서 총 100개까지 확인 가능  


**1.5.4.** 수동 스탬프 적립 기능  

**1.5.4.1.** 수동 적립 버튼  
　1: 위치: 스탬프 현황 섹션 상단 우측  
　2: 버튼 텍스트: "수동 적립"  
　3: 버튼 스타일: 보조 버튼 (아웃라인)  

**1.5.4.2.** 적용 대상  
　1: 위드폼 오더를 통하지 않고 결제한 소비자  
　2: 소비자가 매장에서 구두로 스탬프 적립 요청 시 사용  
　3: 파트너가 결제 금액 확인 후 수동으로 적립 제공  

**1.5.4.3.** 수동 적립 프로세스  
　1: [수동 적립] 버튼 클릭 → 수동 적립 패널/모달 오픈  
　2: 상세 프로세스: `1.6. 스탬프 수동 적립` 참조
