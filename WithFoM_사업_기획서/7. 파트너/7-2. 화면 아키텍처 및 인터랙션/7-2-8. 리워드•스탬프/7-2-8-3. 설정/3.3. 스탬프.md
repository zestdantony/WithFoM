**3.3.** 스탬프  


**3.3.1.** 스탬프 설정  

**3.3.1.1.** 스탬프 카드 설정 박스  
　1: 스탬프 적립 기준 금액 입력창  
　　A. 설명: "실결제 금액 기준 스탬프 적립"  
　　B. 예시: 3,000원 이상 결제 시 스탬프 1개 적립  
　2: 다중 스탬프 적립 설정 (선택)  
　　A. 토글: "결제 금액에 따라 추가 스탬프 적립"  
　　B. 배수 적립: 기준 금액의 배수마다 추가 적립  
　　C. 구간별 적립: 금액 구간별로 적립 개수 직접 설정  
　3: 유효기간 입력창  
　　A. 유효기간 미설정 체크박스  
　　B. 체크박스 클릭 시 기간 설정 불가, 무기한 표시  
　　C. 기간 선택: 3개월, 6개월, 1년, 사용자 지정  
　　D. 기본값: 1년  
　4: 1일 적립 한도 입력창 (선택)  
　　A. 고객 1인당 1일 최대 적립 개수  
　　B. 권장값: 3개  

**3.3.1.2.** "N개 적립 시 혜택" 설정 박스  
　1: 스탬프 카드 박스 우측에 배치  
　2: 최대 3개의 혜택 설정 가능  
　3: 각 혜택: 적립 개수 + 혜택 내용 입력  
　4: 플레이스홀더: "예: 10, 20, 30" / "혜택 입력"  


**3.3.2.** 적립 제외 항목 설정  

**3.3.2.1.** 카테고리별 제외 설정  
　1: 체크박스 옵션:  
　　A. 행사 제품 제외 (1+1, 2+1 등)  
　　B. 이미 할인된 상품 제외  
　　C. MD 상품 제외 (머그컵, 텀블러 등)  
　2: 기본값: 모두 체크 해제 (제외 없음)  

**3.3.2.2.** 메뉴별 직접 제외 설정  
　1: 메뉴 검색 입력창  
　　A. 메뉴명 검색하여 제외할 메뉴 선택  
　2: 카테고리별 메뉴 리스트  
　　A. 카테고리 선택 → 해당 카테고리 메뉴 표시  
　　B. 각 메뉴 옆 체크박스로 제외 선택  
　3: 제외된 메뉴 리스트  
　　A. 현재 스탬프 적립 제외된 메뉴 목록 표시  
　　B. 각 메뉴 옆 X 버튼으로 제외 해제  
　4: 연동 안내:  
　　A. "`7-2-9. 메뉴 관리`에서도 메뉴별 스탬프 제외 설정 가능"  
　　B. "양쪽에서 설정한 내용은 자동으로 연동됩니다"  


**3.3.3.** 스탬프 요청/승인 흐름(대시보드 연동)  

**3.3.3.1.** 고객 스탬프 적립 가능 컨디션  
　1: 고객이 스탬프 적립 가능 컨디션일 경우 파트너 대시보드의 리워드 사용 보드에 박스 생성  
　2: 스탬프 적립 가능 컨디션:  
　　A. 결제 금액 충족  
　　B. 단골 스토어 지정 여부  
　　C. 제외 메뉴가 아닌 경우  
　3: 스탬프 적립 가능 컨디션일 경우 자동 적립  

**3.3.3.2.** 위드폼 오더 미사용 + 스탬프 기능 활성화 매장의 경우  
　1: 소비자가 스탬프 적립 조건을 만족하면, 앱/웹에서 "스탬프 적립 요청" 가능  
　2: 스탬프 적립 가능 컨디션일 경우 자동 적립  
　3: 파트너 대시보드의 리워드 사용 보드에 스탬프 박스 생성  

**3.3.3.3.** 결제 금액 충족 + 스탬프 시스템을 모르는 고객 안내 매뉴얼  
　1: 단골 스토어 지정 + 스탬프 적립 가능 여부 안내  

**3.3.3.4.** 첫 방문 고객 안내 매뉴얼  
　1: 위드폼에서 카페 검색 후 단골 스토어 지정 시 스탬프 카드 자동 생성됨을 안내
