**3.1.** 게임  


**3.1.1.** 게임 유형 선택  

**3.1.1.1.** 게임 유형  
　1: 룰렛  
　2: 뽑기  
　3: 사다리  
　4: (추가 게임 유형 확장 가능)  


**3.1.2.** 리워드 삽입 버튼  

**3.1.2.1.** 리워드 연동  
　1: 게임 유형 선택 후 리워드 삽입 버튼 생성  
　2: 리워드 삽입 버튼 클릭 시 리워드 설정에서 설정한 리워드와 게임 연동  
　3: 리워드 설정에서 설정한 리워드가 자동으로 게임에 적용됨  
　4: 파트너 스토어 게임의 리워드 당첨 확률은 광고주 게임과 동일하게 적용됨  


**3.1.3.** 게임 활성화/비활성화  

**3.1.3.1.** 리워드 연동  
　1: 리워드 활성화 시 게임도 자동 활성화 (연동)  
　2: 리워드 비활성화 시 게임도 자동 비활성화 (연동)  

**3.1.3.2.** 게임 활성화 조건  
　1: 리워드가 활성화되어 있어야 함  
　2: 게임 유형이 선택되어 있어야 함  
　3: 리워드가 최소 1개 이상 설정되어 있어야 함  


**3.1.4.** 게임 참여 통계 (선택 사항)  

**3.1.4.1.** 통계 항목  
　1: 파트너 스토어 게임 참여 횟수  
　2: 파트너 스토어 게임 리워드 발급 횟수  
　3: 파트너 스토어 게임 리워드 사용 횟수  
　4: 파트너 스토어 게임 참여율 (참여 횟수 / 게임 기회 획득 횟수 × 100%)  


**3.1.5.** 기획 배경  

**3.1.5.1.** 기능 목적  
　1: 이 기능은 컵홀더가 매칭되지 않은 파트너나 매칭을 기다리는 파트너, 테이크아웃이 아닌 매장을 이용하는 소비자들에게도 리워드를 제공하기 위해 추가되었습니다.  

**3.1.5.2.** 리워드 사용 위치  
　1: 파트너 리워드는 광고주 게임의 7등 슬롯과 파트너 스토어 게임 두 곳에서 모두 사용됩니다.
