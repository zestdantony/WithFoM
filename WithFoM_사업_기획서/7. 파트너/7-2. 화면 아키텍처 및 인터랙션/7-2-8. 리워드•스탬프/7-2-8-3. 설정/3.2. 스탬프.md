**3.2.** 스탬프  


**3.2.1.** 스탬프 설정  

**3.2.1.1.** 스탬프 카드 설정 박스  
　1: 스탬프 적립 기준 금액 입력창  
　　A. 설명: "실결제 금액 기준 스탬프 적립"  
　　B. 예시: 3,000원 이상 결제 시 스탬프 1개 적립  
　2: 다중 스탬프 적립 설정 (선택)  
　　A. 토글: "결제 금액에 따라 추가 스탬프 적립"  
　　B. 배수 적립: 기준 금액의 배수마다 추가 적립  
　　C. 구간별 적립: 금액 구간별로 적립 개수 직접 설정  
　3: 유효기간 입력창  
　　A. 유효기간 미설정 체크박스  
　　B. 체크박스 클릭 시 기간 설정 불가, 무기한 표시  
　　C. 기간 선택: 3개월, 6개월, 1년, 사용자 지정  
　　D. 기본값: 1년  
　4: 1일 적립 한도 입력창 (선택)  
　　A. 고객 1인당 1일 최대 적립 개수  
　　B. 권장값: 3개  


**3.2.2.** 스탬프 완성 혜택 설정  

**3.2.2.1.** 혜택 설정 박스  
　1: 위치: 스탬프 카드 박스 우측에 배치  
　2: 최대 3개의 혜택 설정 가능  
　3: 각 혜택: 적립 개수 + 혜택 유형 + 혜택 내용 입력  
　4: 플레이스홀더: "예: 10, 20, 30" / "혜택 유형 선택" / "혜택 내용 입력"  

**3.2.2.2.** 혜택 쿠폰 발급 설정  
　1: 혜택 유형 선택  
　　A. 무료 제공 쿠폰: 특정 메뉴 무료 제공  
　　B. 할인 쿠폰: 고정 금액 또는 비율 할인  
　2: 무료 제공 쿠폰 설정  
　　A. 대상 메뉴 선택: 메뉴 검색 또는 카테고리에서 선택  
　　B. 복수 메뉴 선택 가능 (소비자가 사용 시 1개 선택)  
　　C. 예시: "아메리카노, 카페라떼 중 택1 무료"  
　3: 할인 쿠폰 설정  
　　A. 할인 유형 선택: 고정 금액 / 비율 할인  
　　B. 고정 금액: 할인 금액 입력 (예: 1,000원)  
　　C. 비율 할인: 할인율 입력 (예: 10%)  
　　D. 최대 할인 한도 설정 (비율 할인 시, 선택)  
　4: 발급 쿠폰 유효기간 설정  
　　A. 발급일로부터 N일 (예: 30일)  
　　B. 무기한 선택 가능  

**3.2.2.3.** 혜택 발급 프로세스  
　1: 스탬프 완성 시 쿠폰 자동 발급  
　　A. 소비자가 설정된 개수(예: 10개) 스탬프 적립 완료 시  
　　B. 설정된 혜택이 쿠폰 형태로 소비자에게 자동 발급  
　　C. 발급 알림: 푸시 알림 및 앱 내 알림  
　2: 발급된 쿠폰 사용  
　　A. 소비자가 쿠폰함에서 발급된 쿠폰 확인  
　　B. 매장에서 쿠폰 사용 요청  
　　C. 파트너 대시보드 리워드 섹션에 쿠폰 사용 카드로 표시  
　　D. 일반 쿠폰과 동일한 승인/거절 프로세스 적용  
　3: 스탬프 차감 방식  
　　A. 혜택 발급 시 사용된 스탬프 개수만 차감 (전체 초기화 아님)  
　　B. 예시: 13개 보유 → 10개 혜택 사용 → 3개 잔여  
　　C. 잔여 스탬프는 그대로 유지되어 다음 혜택 적립에 합산  
　　D. 다중 혜택 설정 시에도 동일 로직 적용 (20개 혜택 사용 시 20개만 차감)  

**3.2.2.4.** 혜택 설정 예시  
　1: 기본 설정 예시  
　　A. 10개 적립 시: 아메리카노 무료 쿠폰 발급  
　　B. 20개 적립 시: 1,000원 할인 쿠폰 발급  
　　C. 30개 적립 시: 음료 50% 할인 쿠폰 발급 (최대 3,000원)  
　2: 차감 예시  
　　A. 25개 보유 → 20개 혜택 쿠폰 발급 → 5개 잔여  
　　B. 32개 보유 → 30개 혜택 쿠폰 발급 → 2개 잔여  


**3.2.3.** 적립 제외 항목 설정  

**3.2.3.1.** 카테고리별 제외 설정  
　1: 체크박스 옵션:  
　　A. 행사 제품 제외 (1+1, 2+1 등)  
　　B. 이미 할인된 상품 제외  
　　C. MD 상품 제외 (머그컵, 텀블러 등)  
　2: 기본값: 모두 체크 해제 (제외 없음)  

**3.2.3.2.** 메뉴별 직접 제외 설정  
　1: 메뉴 검색 입력창  
　　A. 메뉴명 검색하여 제외할 메뉴 선택  
　2: 카테고리별 메뉴 리스트  
　　A. 카테고리 선택 → 해당 카테고리 메뉴 표시  
　　B. 각 메뉴 옆 체크박스로 제외 선택  
　3: 제외된 메뉴 리스트  
　　A. 현재 스탬프 적립 제외된 메뉴 목록 표시  
　　B. 각 메뉴 옆 X 버튼으로 제외 해제  
　4: 연동 안내:  
　　A. "`7-2-10. 메뉴 관리`에서도 메뉴별 스탬프 제외 설정 가능"  
　　B. "양쪽에서 설정한 내용은 자동으로 연동됩니다"  


**3.2.4.** 스탬프 적립 방식  

**3.2.4.1.** 스탬프 기능 이용 조건  
　1: 필수 조건:  
　　A. 리워드 기능을 사용하는 파트너는 스탬프 기능 이용 가능  
　　B. 위드폼 오더 사용 여부에 따라 적립 방식이 달라짐  
　2: 서비스 타입별 적립 방식:  
　　A. 복합형 (오더 + 리워드): 자동 적립 + 수동 적립 모두 가능  
　　B. 리워드형 전용: 수동 적립만 가능 (위드폼 오더 미사용으로 자동 적립 불가)  
　　C. 오더형 전용: 리워드 기능 OFF로 스탬프 이용 불가  

**3.2.4.2.** 자동 적립 (위드폼 오더 결제 시)  
　1: 적립 조건:  
　　A. 소비자가 위드폼 오더를 통해 결제  
　　B. 결제 금액이 스탬프 적립 기준 충족  
　　C. 단골 스토어 지정 완료  
　　D. 제외 메뉴가 아닌 경우  
　2: 적립 처리:  
　　A. 조건 충족 시 시스템에서 자동 적립 (별도 승인 불필요)  
　　B. 파트너 대시보드에 적립 알림 미표시 (자동 처리됨)  
　　C. 소비자에게 적립 완료 알림 발송  

**3.2.4.3.** 수동 적립 (위드폼 오더 미사용 결제 시)  
　1: 적용 상황:  
　　A. 소비자가 위드폼 오더가 아닌 다른 방식으로 결제 (카드, 현금 등)  
　　B. 시스템에서 결제 금액 확인 불가로 자동 적립 불가  
　2: 수동 적립 요청:  
　　A. 소비자가 결제 후 파트너에게 구두로 스탬프 적립 요청  
　　B. 파트너가 결제 금액 확인 후 수동으로 스탬프 적립 제공  
　3: 수동 적립 방법:  
　　A. 파트너 포털에서 수동 스탬프 적립 기능 제공  
　　B. 상세 내용: `7-2-8-4. 수동 스탬프 적립` 참조  

**3.2.4.4.** 고객 안내 매뉴얼
