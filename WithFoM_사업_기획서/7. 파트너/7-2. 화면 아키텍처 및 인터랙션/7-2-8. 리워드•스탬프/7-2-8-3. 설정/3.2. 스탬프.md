**3.2.** 스탬프  


**3.2.1.** 스탬프 설정  

**3.2.1.1.** 스탬프 카드 설정 카드  
　1: 스탬프 적립 기준 금액 입력창  
　　A. 설명: "실결제 금액 기준 스탬프 적립"  
　　B. 예시: 3,000원 이상 결제 시 스탬프 1개 적립  
　2: 다중 스탬프 적립 설정 (선택)  
　　A. 토글: "결제 금액에 따라 추가 스탬프 적립"  
　　B. 배수 적립: 기준 금액의 배수마다 추가 적립  
　　C. 구간별 적립: 금액 구간별로 적립 개수 직접 설정  
　3: 유효기간 입력창  
　　A. 유효기간 미설정 체크박스  
　　B. 체크박스 클릭 시 기간 설정 불가, 무기한 표시  
　　C. 기간 선택: 3개월, 6개월, 1년, 사용자 지정  
　　D. 기본값: 1년  
　4: 1일 적립 한도 입력창 (선택)  
　　A. 고객 1인당 1일 최대 적립 개수  
　　B. 권장값: 3개  


**3.2.2.** 스탬프 완성 혜택 설정  

**3.2.2.1.** 혜택 설정 카드  
　1: 위치: 스탬프 카드 카드 우측에 배치  
　2: 최대 3개의 혜택 설정 가능  
　3: 각 혜택: 적립 개수 + 혜택 유형 + 혜택 내용 입력  
　4: 플레이스홀더: "예: 10, 20, 30" / "혜택 유형 선택" / "혜택 내용 입력"  

**3.2.2.2.** 혜택 쿠폰 발급 설정  
　1: 혜택 유형 선택  
　　A. 무료 제공 쿠폰: 특정 메뉴 무료 제공  
　　B. 할인 쿠폰: 고정 금액 또는 비율 할인  
　2: 무료 제공 쿠폰 설정  
　　A. 대상 메뉴 선택: 메뉴 검색 또는 카테고리에서 선택  
　　B. 복수 메뉴 선택 가능 (소비자가 사용 시 1개 선택)  
　　C. 예시: "아메리카노, 카페라떼 중 택1 무료"  
　3: 할인 쿠폰 설정  
　　A. 할인 유형 선택: 고정 금액 / 비율 할인  
　　B. 고정 금액: 할인 금액 입력 (예: 1,000원)  
　　C. 비율 할인: 할인율 입력 (예: 10%)  
　　D. 최대 할인 한도 설정 (비율 할인 시, 선택)  
　4: 발급 쿠폰 유효기간 설정  
　　A. 발급일로부터 N일 (예: 30일)  
　　B. 무기한 선택 가능  

**3.2.2.3.** 혜택 발급 프로세스  
　1: 스탬프 완성 시 쿠폰 자동 발급  
　　A. 소비자가 설정된 개수(예: 10개) 스탬프 적립 완료 시  
　　B. 설정된 혜택이 쿠폰 형태로 소비자에게 자동 발급  
　　C. 발급 알림: 푸시 알림 및 앱 내 알림  
　2: 발급된 쿠폰 사용  
　　A. 소비자가 쿠폰함에서 발급된 쿠폰 확인  
　　B. 매장에서 쿠폰 사용 요청  
　　C. 파트너 대시섹션 리워드 섹션에 쿠폰 사용 카드로 표시  
　　D. 일반 쿠폰과 동일한 승인/거절 프로세스 적용  
　3: 스탬프 차감 방식  
　　A. 혜택 발급 시 사용된 스탬프 개수만 차감 (전체 초기화 아님)  
　　B. 예시: 13개 보유 → 10개 혜택 사용 → 3개 잔여  
　　C. 잔여 스탬프는 그대로 유지되어 다음 혜택 적립에 합산  
　　D. 다중 혜택 설정 시에도 동일 로직 적용 (20개 혜택 사용 시 20개만 차감)  

**3.2.2.4.** 혜택 설정 예시  
　1: 기본 설정 예시  
　　A. 10개 적립 시: 아메리카노 무료 쿠폰 발급

**3.2.4.** 스탬프 적립 방식  

**3.2.4.1.** 스탬프 기능 이용 조건  
　1: 필수 조건:  
　　A. 리워드 기능을 사용하는 파트너는 스탬프 기능 이용 가능  
　　B. 위드폼 오더 사용 여부에 따라 적립 방식이 달라짐  
　2: 서비스 타입별 적립 방식:  
　　A. 복합형 (오더 + 리워드): 자동 적립 + 수동 적립 모두 가능  
　　B. 리워드형 전용: 수동 적립만 가능 (위드폼 오더 미사용으로 자동 적립 불가)  
　　C. 오더형 전용: 리워드 기능 OFF로 스탬프 이용 불가  

**3.2.4.2.** 자동 적립 (위드폼 오더 결제 시)  
　1: 적립 조건:  
　　A. 소비자가 위드폼 오더를 통해 결제  
　　B. 결제 금액이 스탬프 적립 기준 충족  
　　C. 단골 스토어 지정 완료  
　　D. 제외 메뉴가 아닌 경우  
　2: 적립 처리:  
　　A. 조건 충족 시 시스템에서 자동 적립 (별도 승인 불필요)  
　　B. 파트너 대시섹션에 적립 알림 미표시 (자동 처리됨)  
　　C. 소비자에게 적립 완료 알림 발송  

**3.2.4.3.** 수동 적립 (위드폼 오더 미사용 결제 시)  
　1: 적용 상황:  
　　A. 소비자가 위드폼 오더가 아닌 다른 방식으로 결제 (카드, 현금 등)  
　　B. 시스템에서 결제 금액 확인 불가로 자동 적립 불가  
　2: 수동 적립 요청:  
　　A. 소비자가 결제 후 파트너에게 구두로 스탬프 적립 요청  
　　B. 파트너가 결제 금액 확인 후 수동으로 스탬프 적립 제공  
　3: 수동 적립 방법:  
　　A. 파트너 포털에서 수동 스탬프 적립 기능 제공  
　　B. 상세 내용: `7-2-8-4. 수동 스탬프 적립` 참조  

**3.2.4.4.** 고객 안내 매뉴얼  
　1: 스탬프 시스템을 모르는 고객 안내

　3: 프리미엄 (템플릿 5종 + AI 디자인 커스텀)

　　B. 컨셉 4 — 프리미엄 엠보스: 금박/은박 효과 질감, 파트너 로고 엠보스 처리, 위드폼 워터마크  
　　C. 컨셉 5 — 시즌 에디션: 계절·이벤트에 따라 자동 변경 (봄꽃/여름바다/가을단풍/겨울눈 등), 파트너·위드폼 로고 통합 배치  
　　D. AI 디자인 커스텀 기능 제공 (3.2.6 참조)

**3.2.6.** 스탬프 AI 디자인 커스텀 (프리미엄 전용)  

**3.2.6.1.** 기능 개요  
　1: 프리미엄 멤버십 파트너 전용 기능  
　2: 5종 템플릿의 기본 레이아웃(로고 배치·스탬프 그리드 구조)을 유지한 채, AI와 이미지 업로드를 활용하여 비주얼 요소를 커스터마이징  
　3: 배경·스탬프 도장·그리드 영역 모두 이미지 삽입·변경 가능 (Option A 전부 허용)  

**3.2.6.2.** 진입 동선  
　1: 스탬프 설정 페이지에서 템플릿 선택 후 "AI 커스텀" 버튼 노출  
　　A. 프리미엄: 버튼 활성  
　　B. 플러스 이하: 버튼 잠금 + "프리미엄 전용 기능" 안내  
　2: AI 생성 카테고리에서 진입  
　　A. 사이드바 > AI 생성 > 스탬프 카드 AI 생성 (7-2-17-3)  
　　B. 리워드·스탬프 > 설정 > 스탬프와 **동일 화면·동일 데이터** 연동  
　3: 두 경로 모두 같은 AI 디자인 커스텀 화면으로 연결되며, 적용 시 스탬프 설정과 일체로 저장됨  

**3.2.6.3.** Step 1 — 베이스 템플릿 선택  
　1: 5종 템플릿 미리보기 카드 (가로 스크롤)  
　2: 각 카드에 컨셉명 + 미리보기 이미지 표시  
　3: 선택한 템플릿이 AI 커스텀의 베이스가 됨  

**3.2.6.4.** Step 2 — 커스텀 입력 (구분된 입력창)  
　1: 분위기·톤 (필수, 드롭다운 단일 선택)  
　　A. 따뜻한 / 차가운 / 모던 / 빈티지 / 감성 / 자연  

　2: 배경 설정 (필수, 탭 전환)  
　　A. 탭 1 — AI 생성: 배경 패턴 드롭다운 (단색/그라데이션/도트/스트라이프/일러스트)  
　　B. 탭 2 — 이미지 업로드: 파트너가 직접 배경 이미지 업로드  
　　　a. 규격: JPG/PNG, 최소 1024×640px, 5MB 이하  
　　　b. 업로드 시 카드 비율(16:10)에 맞춰 자동 크롭·리사이즈  
　　　c. 미리보기에서 크롭 영역 드래그 조정 가능  

　3: 컬러 톤 (필수, 컬러 팔레트)  
　　A. 매장 대표 컬러 자동 적용 (기본값)  
　　B. 프리셋: 파스텔 / 어스톤 / 비비드 / 다크 / 커스텀  
　　C. 커스텀 선택 시 컬러 피커 노출 (HEX 직접 입력 가능)  

　4: 스탬프 도장 아이콘 (선택, 탭 전환)  
　　A. 탭 1 — 프리셋: 기본 도장 스타일 10종 (원형/사각/하트/별/커피컵/잎사귀 등)  
　　B. 탭 2 — AI 생성: 분위기·톤에 맞는 도장 아이콘 3종 자동 생성, 택 1  
　　C. 탭 3 — 이미지 업로드: 파트너가 도장용 이미지 1종 업로드  
　　　a. 규격: PNG(투명 배경 권장), 최소 256×256px, 2MB 이하  
　　　b. 업로드 시 도장 프레임(원형/사각 중 선택)에 맞춰 자동 마스크·크롭  
　　　c. 마스크 적용 미리보기 제공  

　5: 그리드 영역 배경 (선택, 탭 전환)  
　　A. 탭 1 — 템플릿 기본: 템플릿에 포함된 그리드 디자인 그대로 사용 (기본값)  
　　B. 탭 2 — AI 생성: 분위기·컬러 톤에 맞는 그리드 영역 배경 자동 생성  
　　C. 탭 3 — 이미지 업로드: 그리드 영역 전체를 하나의 배경 이미지로 지정  
　　　a. 규격: JPG/PNG, 최소 800×400px, 3MB 이하  
　　　b. 그리드 셀 위치·크기는 고정, 배경 이미지만 깔림  
　　　c. 업로드 시 그리드 비율에 맞춰 자동 크롭·리사이즈  

　6: 시즌·테마 (선택, 드롭다운)  
　　A. 봄 / 여름 / 가을 / 겨울 / 크리스마스 / 발렌타인 / 없음  

　7: 추가 요청 (선택, 텍스트 입력창)  
　　A. 100자 이내 자유 텍스트  
　　B. 예시 플레이스홀더: "벚꽃 느낌으로 해주세요"  

**3.2.6.5.** Step 3 — AI 생성 및 선택  
　1: "커스텀 생성" 버튼 클릭 → AI가 3개 시안 생성 (10~15초, 로딩 애니메이션)  
　2: 3개 시안 카드 미리보기 (가로 나란히 배치)  
　3: 각 시안에 "선택" 버튼  
　4: "재생성" 버튼 (입력값 유지, 다른 변형 생성)  
　5: 월 재생성 횟수 제한: 2회 (표시: "이번 달 남은 재생성: N/2회")  

**3.2.6.6.** Step 4 — 미세 조정  
　1: 선택한 시안에서 미세 조정 가능  
　　A. 컬러 톤 미세 조정 (밝기/채도 슬라이더)  
　　B. 배경 투명도 조정 (슬라이더, 0~100%)  
　　C. 도장 아이콘 크기 미세 조정 (슬라이더, 80~120%)  
　2: 조정 시 실시간 미리보기 반영  

**3.2.6.7.** Step 5 — 확정 및 적용  
　1: "이 디자인 적용" 버튼 클릭  
　2: 확인 안내: "선택한 디자인으로 스탬프 카드가 변경됩니다. 기존 디자인은 보관됩니다."  
　3: 기존 디자인은 "이전 디자인 보관함"에 저장 (최대 5개)  
　4: 적용 즉시 소비자 앱에 반영  

**3.2.6.8.** 변경 불가 영역 (기본 레이아웃 보호)  
　1: 위드폼 로고 위치·크기 (브랜드 일관성)  
　2: 파트너 로고 위치·크기 (브랜드 일관성)  
　3: 스탬프 도장 그리드 셀 개수·배치·비율 (기능적 요소)  
　4: 혜택 텍스트 영역 위치·폰트 크기 (가독성 보장)  
　5: 전체 카드 비율·크기 (앱 레이아웃 호환)  

**3.2.6.9.** 이미지 업로드 공통 규칙  
　1: 지원 포맷: JPG, PNG (도장은 PNG 투명 배경 권장)  
　2: 업로드 시 자동 처리  
　　A. 비율 검사 → 부적합 시 크롭 가이드 노출  
　　B. 해상도 부족 시 경고: "이미지 해상도가 낮아 흐릿하게 표시될 수 있습니다"  
　　C. 용량 초과 시 자동 압축 시도 → 실패 시 재업로드 안내  
　3: 부적절 콘텐츠 자동 검사  
　　A. AI 이미지 검사 API를 통한 자동 검토  
　　B. 부적절 콘텐츠 감지 시 업로드 차단 + 안내: "업로드할 수 없는 이미지입니다"  
　4: 미리보기 필수 제공 (실제 카드에 적용된 모습 확인 후 확정)
