**3.1.** 리워드  


**3.1.1.** 리워드 참여 여부 선택  

**3.1.1.1.** 참여 여부 설정  
　1: 참여 여부 On/Off 버튼 생성  
　2: 참여 여부는 서비스 타입(리워드/오더)의 표시와 연동  
　3: 리워드 활성화 시 게임도 자동 활성화 (연동)  


**3.1.2.** 리워드 생성(최대 6개)  

**3.1.2.1.** 리워드 생성 인터페이스  
　1: 기본 입력카드 1개 노출  
　2: 오른쪽 상단 "리워드 추가" 버튼  
　3: 누를 때마다 1개씩 추가  
　4: 총 6개까지 생성 가능  
　5: 생성 카드는 등수별 정렬(1등~6등, 상단→하단), 위/아래 드래그로 등수 변경 가능  
　6: 우측 패널에 저장된 리워드 라이브러리(3.1.7.) 표시, 클릭 시 빈 슬롯에 자동 삽입(확률·수량 제외 프리필)  

**3.1.2.2.** 리워드 생성 카드 공통 구성  
　1: 왼쪽 상단: 등수 표시(위치 기반 자동 갱신, 드래그 이동 시 등수 번호 자동 변경)  
　2: 오른쪽 상단: 수정/삭제 버튼  
　3: 입력 항목  
　　A. 유형(쿠폰 or 포인트)  
　　B. 리워드명(예: 10% 할인 쿠폰, 1000원 할인 쿠폰, 포인트 500점, 포인트 3000점 등)  
　　C. 리워드 가격 or 가격대비 적용 퍼센트  
　　D. 총 발행 수량  
　　E. 유효기간  
　　　E-1. 상대적 유효기간: "발행 후 N일" 형식(리워드 발행 시점부터 카운트)  
　　　E-2. 절대적 유효기간: "YYYY-MM-DD까지" 형식(특정 날짜 만료)  
　　　E-3. 절대적 유효기간 리워드는 프리셋 저장 시 제외 대상(안내 표시)  
　　F. 당첨 확률(%) 입력  
　　G. 게임 적용 시 안내 문구(당첨 시 표시 메시지, 최대 30자)  

**3.1.2.3.** 리워드 조건 유형  
　1: 최소 결제금액 조건  
　2: 메뉴 조건 (카테고리/단일 메뉴)  
　3: 채널 조건 (매장/테이크아웃/오더)  
　4: 시간/요일 조건  


**3.1.3.** 꽝 및 확률 검증 섹션  

**3.1.3.1.** 상단 표시  
　1: 생성된 리워드 개수  
　2: 총 확률 합계  
　3: 꽝 포함 여부 ON/OFF 토글  

**3.1.3.2.** 리스트 표시  
　1: 리워드별 당첨 확률 입력  
　2: 리워드별 총 추정 예산  
　3: 리워드별 최소 총 추정 매출  

**3.1.3.3.** 리스트 우측 하단 표시  
　1: 토탈 예산 (추정치)  
　2: 토탈 최소 매출 (추정치)  

**3.1.3.4.** 꽝 설정(꽝 토글 ON 시 표시)  
　1: 꽝 확률(%) 입력  
　2: 꽝 메시지 설정(기본: "다음 기회에!", 커스텀 가능, 최대 20자)  
　3: 꽝 ON 시: 리워드 최대 5개 + 꽝 1개 = 6개 결과  
　4: 꽝 OFF 시: 리워드 최대 6개  

**3.1.3.5.** 확률 합계 검증  
　1: 모든 리워드 + 꽝(ON 시) 확률 합계 = 100% 필수  
　2: 100% 미달 시 [게임 적용] 버튼 비활성화 + 경고 메시지  
　3: 실시간 합계 계산 및 잔여 확률 표시  


**3.1.4.** 사용 제한 설정  

**3.1.4.1.** 쿠폰 제한  
　1: 1일 사용 횟수 제한  
　2: 1회 결제당 사용 개수 제한  

**3.1.4.2.** 포인트 제한  
　1: 1일 사용 가능 총액 제한  
　2: 1회 결제당 결제 금액 대비 비율 제한  


**3.1.5.** 적용 제외 항목 설정  

**3.1.5.1.** 카테고리별 제외 설정  
　1: 체크박스 옵션:  
　　A. 행사 제품 제외 (1+1, 2+1 등)  
　　B. 이미 할인된 상품 제외  
　　C. MD 상품 제외 (머그컵, 텀블러 등)  
　2: 기본값: 모두 체크 해제 (제외 없음)  

**3.1.5.2.** 메뉴별 직접 제외 설정  
　1: 메뉴 검색 입력창  
　　A. 메뉴명 검색하여 제외할 메뉴 선택  
　2: 카테고리별 메뉴 리스트  
　　A. 카테고리 선택 → 해당 카테고리 메뉴 표시  
　　B. 각 메뉴 옆 체크박스로 제외 선택  
　3: 제외된 메뉴 리스트  
　　A. 현재 리워드 적용 제외된 메뉴 목록 표시  
　　B. 각 메뉴 옆 X 버튼으로 제외 해제  
　4: 연동 안내:  
　　A. "`7-2-9. 메뉴 관리`에서도 메뉴별 리워드 제외 설정 가능"  
　　B. "양쪽에서 설정한 내용은 자동으로 연동됩니다"  


**3.1.6.** 게임 적용 버튼  

**3.1.6.1.** 버튼 활성화 조건  
　1: 리워드 최소 1개 이상 생성 완료  
　2: 확률 합계 = 100% 충족  
　3: 저장 공간 초과 시 저장 선택 완료 또는 "저장하지 않고 적용" 선택  

**3.1.6.2.** 클릭 시 동작  
　1: 기존 게임 구성이 있을 경우 교체 안내 모달 표시:  
　　A. "현재 게임에 적용된 리워드 구성이 교체됩니다. 기존 구성을 게임 설정에서 프리셋으로 저장한 후 적용하시겠습니까?"  
　　B. [게임 설정에서 저장 후 적용] → 게임 설정 페이지(7-2-7-1 > 1.3.)로 이동  
　　C. [바로 적용] → 즉시 교체 진행  
　2: 리워드 구성(등수+확률+수량+조건 포함)이 파트너 게임에 독립 복사본으로 적용  
　3: 리워드 설정 생성 폼 전체 초기화(빈 상태로 리셋)  
　4: 적용된 리워드가 확률·수량 제외한 템플릿으로 저장 라이브러리에 자동 보관  
　5: 적용 완료 모달 표시:  
　　A. "리워드가 게임에 적용되었습니다"  
　　B. [게임 설정으로 이동] 버튼 → 7-2-7-1 > 1.3. 페이지로 네비게이션  
　　C. [여기서 계속하기] 버튼 → 모달 닫기, 초기화된 리워드 설정 폼 유지  

**3.1.6.3.** 저장 공간 처리  
　1: 저장 공간 여유 ≥ 생성 리워드 수 → 전체 자동 저장 → [적용] 바로 활성화  
　2: 저장 공간 여유 < 생성 리워드 수 → 저장할 리워드 선택 모달 표시  
　　A. 생성된 리워드 목록 표시 + 체크박스  
　　B. 선택 가능 수량 = 남은 저장 공간 수  
　　C. 선택 완료 후 [적용] 활성화  
　3: 저장 공간 = 0 → "저장하지 않고 적용" 옵션 표시 → 선택 시 [적용] 활성화  

**3.1.6.4.** 복사본 원칙  
　1: 게임에 적용된 리워드는 저장 라이브러리와 별개의 독립 복사본  
　2: 라이브러리 원본 삭제·수정 → 게임 복사본 영향 없음  
　3: 게임 복사본은 적용 해제(게임 설정 1.3.에서 제거) 전까지 삭제 불가  
　4: 동일 리워드가 라이브러리와 게임에 동시 존재 가능 → 완전 별개 엔티티  


**3.1.7.** 리워드 저장 라이브러리  

**3.1.7.1.** 저장 용량  
　1: 베이직: 6개  
　2: 플러스: 12개  
　3: 프리미엄: 24개  
　4: 상위 플랜 슬롯은 자물쇠 아이콘 + "업그레이드 시 N개 추가 저장 가능" 안내  

**3.1.7.2.** 저장 항목 구성  
　1: 리워드 정의만 보관(유형, 이름, 가치, 조건, 안내 문구, 유효기간 유형)  
　2: 확률(%)과 총 발행 수량은 저장 대상에서 제외  
　3: 등수 정보는 저장 대상에서 제외(생성 폼에서 위치로 결정)  

**3.1.7.3.** 불러오기  
　1: 저장된 리워드 목록이 생성 폼 우측 패널에 표시  
　2: 저장 리워드 클릭 → 빈 생성 슬롯에 삽입(유형·이름·가치·조건·안내문구 프리필)  
　3: 확률·수량은 비어있음 → 파트너가 직접 입력 필요  
　4: 연속 클릭으로 빠른 리워드 세트 구성 가능  
　5: 절대 유효기간 리워드 불러오기 시: 만료일 경과 시 "유효기간이 만료된 리워드입니다. 유효기간을 재설정해주세요" 경고  

**3.1.7.4.** 삭제  
　1: 저장된 리워드 선택 → [삭제] 버튼 → 확인 모달 → 삭제  
　2: 삭제된 자리에 새 리워드 저장 가능  
　3: 삭제된 리워드가 프리셋에 포함된 경우: 프리셋에서 해당 항목 자동 제거  

**3.1.7.5.** 멤버십 다운그레이드 시  
　1: 모든 저장 리워드 및 프리셋 잠김 상태 전환  
　2: 새 플랜 한도에 맞게 유지할 항목 선택 안내 표시  
　3: 선택 완료 전까지 리워드 생성·적용 기능 제한  
　4: 선택 완료 후 미선택 항목 영구 삭제  


• 참조: 7-2-7-1. 게임 설정 > 1.3. 리워드 적용·프리셋 관리, 7-2-6-1. 멤버십 구독 > 1.2. 멤버십 상세 설명, 4-2-3. AI 게임 리워드 설계 보조
