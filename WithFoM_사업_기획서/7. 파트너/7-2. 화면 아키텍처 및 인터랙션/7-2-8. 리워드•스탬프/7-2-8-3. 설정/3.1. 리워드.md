**3.1.** 리워드  


**3.1.1.** 리워드 참여 여부 선택  

**3.1.1.1.** 참여 여부 설정  
　1: 참여 여부 On/Off 버튼 생성  
　2: 참여 여부는 서비스 타입(리워드/오더)의 표시와 연동  
　3: 리워드 활성화 시 게임도 자동 활성화 (연동)  


**3.1.2.** 리워드 생성(최대 6개)  

**3.1.2.1.** 리워드 생성 인터페이스  
　1: 기본 입력박스 1개 노출  
　2: 오른쪽 상단 "리워드 추가" 버튼  
　3: 누를 때마다 1개씩 추가  
　4: 총 6개까지 생성 가능  

**3.1.2.2.** 리워드 생성 박스 공통 구성  
　1: 왼쪽 상단: N번째 생성 숫자  
　2: 오른쪽 상단: 수정/삭제 버튼  
　3: 입력 항목  
　　A. 유형(쿠폰 or 포인트)  
　　B. 리워드명(예: 10% 할인 쿠폰, 1000원 할인 쿠폰, 포인트 500점, 포인트 3000점 등)  
　　C. 리워드 가격 or 가격대비 적용 퍼센트  
　　D. 총 발행 수량  
　　E. 유효기간  

**3.1.2.3.** 리워드 조건 유형  
　1: 최소 결제금액 조건  
　2: 메뉴 조건 (카테고리/단일 메뉴)  
　3: 채널 조건 (매장/테이크아웃/오더)  
　4: 시간/요일 조건  


**3.1.3.** 리워드 당첨 확률 보드  

**3.1.3.1.** 상단 표시  
　1: 생성된 리워드 개수  
　2: 총 확률 합계  

**3.1.3.2.** 리스트 표시  
　1: 리워드별 당첨 확률 입력  
　2: 리워드별 총 추정 예산  
　3: 리워드별 최소 총 추정 매출  

**3.1.3.3.** 리스트 우측 하단 표시  
　1: 토탈 예산 (추정치)  
　2: 토탈 최소 매출 (추정치)  


**3.1.4.** 사용 제한 설정  

**3.1.4.1.** 쿠폰 제한  
　1: 1일 사용 횟수 제한  
　2: 1회 결제당 사용 개수 제한  

**3.1.4.2.** 포인트 제한  
　1: 1일 사용 가능 총액 제한  
　2: 1회 결제당 결제 금액 대비 비율 제한  


**3.1.5.** 적용 제외 항목 설정  

**3.1.5.1.** 카테고리별 제외 설정  
　1: 체크박스 옵션:  
　　A. 행사 제품 제외 (1+1, 2+1 등)  
　　B. 이미 할인된 상품 제외  
　　C. MD 상품 제외 (머그컵, 텀블러 등)  
　2: 기본값: 모두 체크 해제 (제외 없음)  

**3.1.5.2.** 메뉴별 직접 제외 설정  
　1: 메뉴 검색 입력창  
　　A. 메뉴명 검색하여 제외할 메뉴 선택  
　2: 카테고리별 메뉴 리스트  
　　A. 카테고리 선택 → 해당 카테고리 메뉴 표시  
　　B. 각 메뉴 옆 체크박스로 제외 선택  
　3: 제외된 메뉴 리스트  
　　A. 현재 리워드 적용 제외된 메뉴 목록 표시  
　　B. 각 메뉴 옆 X 버튼으로 제외 해제  
　4: 연동 안내:  
　　A. "`7-2-9. 메뉴 관리`에서도 메뉴별 리워드 제외 설정 가능"  
　　B. "양쪽에서 설정한 내용은 자동으로 연동됩니다"  


**3.1.6.** 확인 버튼  

**3.1.6.1.** 버튼 기능  
　1: 누를 시 생성 완료  
　2: 완료 된 상태로 그대로 유지  
　3: 설정 완료 된 리워드가 6개일 경우, 리워드 생성 박스 생성이 불가능 하고 버튼을 비활성화  
　4: 버튼을 눌러도 더 이상 생성이 불가 합니다 팝업 메세지 출력
