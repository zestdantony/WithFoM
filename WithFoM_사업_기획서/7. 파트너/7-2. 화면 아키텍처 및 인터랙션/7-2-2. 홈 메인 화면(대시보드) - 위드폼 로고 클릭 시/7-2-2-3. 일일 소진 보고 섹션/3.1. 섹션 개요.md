**3.1.** 섹션 개요  


**3.1.1.** 섹션 정의  

**3.1.1.1.** 기본 정보  
1: 섹션 위치  
　A. 파트너 포털 홈 화면(대시보드)의 세 번째 섹션으로 배치  
　B. 결제 내역 섹션(7-2-2-1), 리워드 사용 내역 섹션(7-2-2-2) 다음에 위치  
　C. 오더/리워드 기능을 사용하지 않는 캠페인 전용 파트너의 경우, 첫 번째 섹션으로 노출  
2: 섹션 명칭  
　A. 섹션 제목: "일일 소진 보고"  
　B. 제목 우측에 오늘 보고 상태 뱃지 표시 (예: "보고 완료" 또는 "보고 필요")  


**3.1.2.** 섹션 목적  

**3.1.2.1.** 핵심 기능  
1: 매일 컵홀더 소진량을 입력하여 캠페인 진행 현황을 보고  
2: 대시보드에서 바로 소진량 입력 및 제출 가능  
3: 전일 소진량 확인 및 수정 기능 제공 (익일 영업 시작 후 2시간까지)  
4: 정확한 정산 및 캠페인 성과 측정을 위한 필수 데이터 수집  
5: 휴무일은 자동으로 0개 처리 (영업일 설정 연동)  
6: 당일 날짜와 요일을 명확히 표시하여 보고 대상 확인 용이  


**3.1.3.** 섹션 레이아웃  

**3.1.3.1.** 화면 구성  
1: 섹션 헤더  
　A. 좌측: 섹션 제목 "일일 소진 보고" + 보고 상태 뱃지  
　B. 우측: "상세 보기" 링크 (클릭 시 캠페인 소진 보고 페이지로 이동)  
2: 섹션 본문  
　A. 날짜 표시 영역 (당일 날짜 및 요일)  
　B. 전일 미보고 시 전일 날짜 카드 추가 표시  
　C. 전일 소진량 표시 영역  
　D. 오늘 소진량 입력 영역  
　E. 제출 버튼  


**3.1.4.** 보고 상태별 색상  

**3.1.4.1.** 상태 표시  
1: 보고 필요 (미완료)  
　A. 섹션 테두리: 빨간색  
　B. 상태 뱃지: 빨간색 배경, "보고 필요" 텍스트  
　C. 입력창 활성화 상태  
2: 보고 완료  
　A. 섹션 테두리: 초록색  
　B. 상태 뱃지: 초록색 배경, "보고 완료" 텍스트  
　C. 입력창 비활성화, 제출된 값 표시  
3: 휴무일  
　A. 섹션 테두리: 회색  
　B. 상태 뱃지: 회색 배경, "휴무일" 텍스트  
　C. 입력창 비활성화, "자동 0개 처리됨" 표시  
4: 캠페인 매칭 대기 (매칭 전에만 표시)  
　A. 섹션 테두리: 회색  
　B. 상태 뱃지: 회색 배경, "매칭 대기" 텍스트  
　C. 입력창 비활성화, 안내 문구 표시  
　D. 캠페인 매칭 확정 시 뱃지 제거, 일반 상태로 전환  
5: 입력 마감  
　A. 섹션 테두리: 빨간색  
　B. 상태 뱃지: 빨간색 배경, "입력 마감" 텍스트  
　C. 전일 입력 시간 초과로 인한 비활성화  


• 참조: 7-2-5-1. 캠페인 소진 보고, 7-1-5-3. 영업 종료 시점 일일 소진량 입력
