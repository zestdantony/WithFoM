**3.4.** 표시 정보  


**3.4.1.** 날짜 표시 정보  

**3.4.1.1.** 당일 날짜  
1: 표시 형식  
　A. "YYYY년 MM월 DD일 (요일)" 형식  
　B. 예: "2026년 1월 28일 (수)"  
2: 위치  
　A. 섹션 본문 상단에 배치  
　B. 캘린더 아이콘과 함께 표시  

**3.4.1.2.** 전일 미보고 시 표시  
1: 듀얼 날짜 표시 조건  
　A. 자정 경과 후 전일 소진량 미보고 상태  
　B. 익일 영업 시작 후 2시간까지만 표시  
2: 전일 날짜 카드 (좌측)  
　A. 빨간색 테두리 강조  
　B. 날짜 + "미보고" 라벨  
　C. 예: "1월 27일 (화) - 미보고"  
3: 당일 날짜 카드 (우측)  
　A. 일반 테두리  
　B. 날짜 + "오늘" 라벨  
　C. 예: "1월 28일 (수) - 오늘"  
　D. 전일 보고 완료 전까지 비활성화 (회색)  
4: 마감 임박 표시  
　A. 입력 마감 1시간 전부터 남은 시간 표시  
　B. 예: "1시간 23분 후 입력 마감"  
　C. 색상: 주황색 → 빨간색 (30분 이하)  

**3.4.1.3.** 24시간 영업장 기준  
1: 일일 마감 기준  
　A. 오전 11:00를 일일 마감 기준으로 적용  
　B. 전일 보고 마감: 다음 날 오전 11:00  


**3.4.2.** 캠페인 정보 표시  

**3.4.2.1.** 현재 캠페인 정보  
1: 캠페인명  
　A. 현재 참여 중인 캠페인의 이름  
　B. 예: "OO브랜드 신제품 런칭 캠페인"  
2: 캠페인 기간  
　A. 시작일 ~ 종료일 표시  
　B. 예: "2026.01.15 ~ 2026.02.14"  
3: N회차 표시  
　A. 현재 참여 중인 캠페인 회차  
　B. 예: "3회차"  


**3.4.3.** 소진량 표시 정보  

**3.4.3.1.** 전일 소진량  
1: 라벨  
　A. "어제 소진량" 텍스트  
2: 값 표시  
　A. 숫자 + "개" 단위 (예: "127개")  
　B. 미보고 시: "미보고" (빨간색)  
　C. 휴무일: "휴무" (회색)  
3: 수정 버튼  
　A. 값 우측에 [수정] 아이콘 버튼  
　B. 수정 가능 시간 내에만 표시  

**3.4.3.2.** 오늘 소진량 (입력 전)  
1: 입력 필드  
　A. 플레이스홀더: "소진 수량을 입력하세요"  
　B. 비어있는 상태  
2: 상태 표시  
　A. 입력창 좌측에 빨간 점 표시 (입력 필요 표시)  

**3.4.3.3.** 오늘 소진량 (입력 후)  
1: 입력 값 표시  
　A. 숫자 + "개" 단위 (예: "142개")  
　B. 배경색: 연한 초록색  
2: 보고 시간  
　A. 하단에 작은 글씨로 보고 시간 표시  
　B. 예: "22:35에 보고 완료"  


**3.4.4.** 상태 표시

**3.4.4.1.** 보고 상태 뱃지
1: 보고 필요  
　A. 뱃지 색상: 빨간색  
　B. 텍스트: "보고 필요"  
　C. 아이콘: 느낌표 (!)  
2: 보고 완료  
　A. 뱃지 색상: 초록색  
　B. 텍스트: "보고 완료"  
　C. 아이콘: 체크 (✓)  
3: 휴무일  
　A. 뱃지 색상: 회색  
　B. 텍스트: "휴무일"  
　C. 아이콘: 달력  
4: 캠페인 매칭 대기  
　A. 뱃지 색상: 회색  
　B. 텍스트: "매칭 대기"  
　C. 아이콘: 시계  
　D. 매칭 확정 시 뱃지 제거, 일반 상태(보고 필요/완료)로 전환  
5: 입력 마감  
　A. 뱃지 색상: 빨간색  
　B. 텍스트: "입력 마감"  
　C. 전일 소진량 입력 가능 시간 경과 시 표시
