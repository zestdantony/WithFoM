**1.2.** 노출 조건  


**1.2.1.** 활성화 조건  

**1.2.1.1.** 서비스 타입 조건  
1: 위드폼 오더 기능 이용 시 활성화  
　A. 서비스 타입이 "오더형" 또는 "복합형(리워드+오더)"인 경우에만 노출  
　B. 서비스 타입이 "리워드형"인 경우 이 섹션은 숨김 처리  
2: 서비스 타입 확인 경로  
　A. 파트너 포털 > 우측 상단 서비스 타입 표시 영역에서 확인  
　B. 리워드 ☐ / 오더 ☑️ → 오더형  
　C. 리워드 ☑️ / 오더 ☑️ → 복합형  


**1.2.2.** 비활성화 시 처리  

**1.2.2.1.** 리워드형 파트너  
1: 결제 내역 섹션 숨김  
　A. 대시보드에서 해당 섹션 자체가 노출되지 않음  
　B. 리워드 사용 내역 섹션이 첫 번째 섹션으로 자동 배치  
2: 서비스 타입 변경 안내  
　A. 오더 기능 활성화 시 자동으로 섹션 노출  
　B. 계정 설정 > 서비스 타입 설정에서 변경 가능  


**1.2.3.** 주문이 없는 경우  

**1.2.3.1.** 빈 상태 처리  
1: 섹션 자체는 노출 유지  
2: 카드 영역에 안내 문구 표시  
　A. "대기 중인 주문이 없습니다"  
　B. 문구 하단에 "주문이 들어오면 여기에 표시됩니다" 보조 텍스트  
3: 빈 상태 아이콘
