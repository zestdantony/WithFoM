**2.2.** 기능  


**2.2.1.** 포스터 생성  

**2.2.1.1.** 생성 방식  
1: AI 툴을 사용하여 매장 포스터 이미지 생성  
2: 매장 정보, 이벤트 내용, 브랜드 대표 컬러 등을 반영한 포스터 자동 디자인  
3: 구분된 입력창을 통해 AI가 이해하기 쉬운 프롬프트를 시스템이 자동 구성  
4: 파트너는 프롬프트를 직접 작성할 필요 없이 입력 필드에 내용만 채움  


**2.2.2.** 포스터 생성 프로세스  

**2.2.2.1.** Step 1 — 기본 정보 (자동 입력)  
　1: 매장명 (시스템 자동 반영)  
　2: 매장 로고 (자동 불러오기)  
　3: 매장 브랜드 대표 컬러 (7-2-15-1 매장 정보 수정 > 1.1.3에서 설정한 컬러 자동 반영)  
　4: 매장 주소/연락처 (선택 표시, 토글로 포함 여부 결정)  

**2.2.2.2.** Step 2 — 포스터 목적 선택 (드롭다운)  
　1: 이벤트 홍보 — 할인, 시즌 이벤트, 기념일  
　2: 신메뉴 소개 — 신규/시즌 메뉴 출시  
　3: 재방문 유도 — 스탬프·쿠폰 혜택 강조  
　4: 매장 안내 — 영업시간, 위치, Wi-Fi 등  
　5: 타임 프로모션 — 특정 시간대 프로모션  
　6: 자유 주제 — 직접 입력  

**2.2.2.3.** Step 3 — 내용 입력 (구분된 입력창)  
　1: 제목 (필수, 텍스트 입력)  
　　A. 포스터 메인 카피  
　　B. 예시 플레이스홀더: "여름 신메뉴 출시!"  
　2: 부제목 (선택, 텍스트 입력)  
　　A. 보조 문구  
　　B. 예시: "시원한 콜드브루 3종"  
　3: 상세 설명 (선택, 텍스트 입력)  
　　A. 이벤트/혜택 내용  
　　B. 예시: "7/1~7/31, 전 메뉴 10% 할인"  
　4: 강조 가격·혜택 (선택, 텍스트 입력)  
　　A. 눈에 띄는 숫자·혜택  
　　B. 예시: "₩3,500", "1+1"  
　5: 이미지 설명 (선택, 텍스트 입력)  
　　A. AI에게 원하는 비주얼 요청  
　　B. 예시: "차가운 아이스 커피가 놓인 여름 해변 느낌"  
　6: QR코드 연결 (선택, 토글 ON/OFF)  
　　A. 위드폼 QR 자동 삽입 여부  

**2.2.2.4.** Step 4 — 스타일 선택  
　1: 톤 (단일 선택)  
　　A. 기본 / 감성 / 직관 / 재미  
　2: 스타일 (단일 선택)  
　　A. 미니멀 / 빈티지 / 모던 / 캐주얼 (2.3 옵션과 동일)  
　3: 사이즈 (단일 선택)  
　　A. A4 (매장 게시)  
　　B. 정사각 (SNS용)  
　　C. 배너 (가로형)  
　　D. 테이블 텐트 (세로 소형)  
　4: 컬러 모드 (단일 선택)  
　　A. 매장 브랜드 컬러 자동 (기본값)  
　　B. 시즌 추천 컬러  
　　C. 직접 선택 (컬러 피커)  

**2.2.2.5.** Step 5 — 생성 및 선택  
　1: "포스터 생성" 버튼 클릭 → AI가 2~4개 시안 생성  
　2: 시안별 미리보기 → 선택  
　3: 미세 조정 (텍스트 수정, 레이아웃 미세 이동) → 확정  
　4: 다운로드: PNG (고화질) / JPG (웹용) / PDF (인쇄용)  
　5: 이전 생성 포스터 보관함 (최대 10개)  


• 참조: 7-2-6. 멤버십 혜택 > 프리미엄 - 정밀 분석 인사이트 및 고급 기능, 7-2-15-1. 매장 정보 수정 > 1.1.3. 매장 로고 및 브랜드 컬러
