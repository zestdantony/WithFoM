**3.8.** 액션 버튼 및 화면 전환  


**3.8.1.** 상단·공통 영역  


**3.8.1.1.** 기간 필터 변경  

1: 기간 선택  
　A. 오늘/이번 주/이번 달/최근 30일/기간 선택 중 선택  
　B. "기간 선택" 시 시작일·종료일 달력(날짜 피커) 노출  
　C. 적용 시 KPI·차트·벤치마크·인사이트 데이터 동기 갱신  
　D. 전년 동기 대비 옵션 선택 시 해당 비교 데이터 갱신  

**3.8.1.2.** KPI 카드 클릭  

1: 카드 클릭 시  
　A. 해당 지표 상세 섹션으로 스크롤 이동  
　B. 현황 탭 내 해당 지표 블록 포커스 또는 펼치기  


**3.8.1.3.** 탭 전환  

1: 탭 클릭  
　A. 현황 / 벤치마크 / 인사이트·실행 중 선택  
　B. 선택 탭 본문만 표시, 기간·플랜 조건 유지  


**3.8.1.4.** 이상치/주의 알림 클릭(베이직)  

1: 알림 클릭  
　A. 소진 수량 급변·리워드 발행 대비 사용 저조 등 알림 클릭 시 관련 상세 화면으로 이동  
　B. 이동 대상: 캠페인 상세, 리워드 설정, 소진 보고 입력 등 알림 유형에 따라 분기  


**3.8.2.** 캠페인 결과 리포트 다운로드  


**3.8.2.1.** 다운로드 버튼  

1: 현황 탭 > 캠페인 결과 섹션 내 [다운로드] 버튼  
　A. 클릭 시 형식 선택(PDF / Excel)  
　B. PDF: 요약 리포트 (전체 파트너)  
　C. Excel: 상세 데이터 (멤버십 플러스 이상, 미해당 시 잠금 또는 업그레이드 안내)  
　D. 선택 후 다운로드 실행 → 파일 저장  


**3.8.3.** 하단 CTA 영역  


**3.8.3.1.** 플랜별 CTA  

1: 베이직  
　A. "플러스 업그레이드" CTA 클릭 → 7-2-6-1. 멤버십 구독 페이지로 이동(플러스 강조)  

2: 플러스  
　A. "프리미엄 업그레이드" CTA 클릭 → 7-2-6-1. 멤버십 구독 페이지로 이동(프리미엄 강조)  

3: 프리미엄  
　A. "자동화 설정" 클릭 → 인사이트·실행 탭 내 타겟팅 알림톡 자동화 위젯으로 스크롤 또는 해당 설정 영역 포커스  
　B. "포스터 만들기" 클릭 → 7-2-17-2. 매장 포스터 AI 생성 페이지로 이동  


**3.8.4.** 플랜 잠금·업그레이드 CTA  

**3.8.4.1.** 벤치마크/인사이트·실행 잠금 영역  

1: 미리보기 + 잠금 + 업그레이드 CTA 버튼 노출  
2: 업그레이드 CTA 클릭 → 7-2-6-1. 멤버십 구독 페이지로 이동
