**2.3.** 액션 버튼 및 화면 전환  


**2.3.1.** 검색 섹션 내 동작  


**2.3.1.1.** 검색 실행  

1: 검색창  
　A. 닉네임 또는 회원번호 선택 후 검색어 입력  
　B. 검색 버튼 클릭(또는 엔터) → 2.2. 오늘 방문객 리스트·검색 결과 섹션에 검색 조건 반영하여 리스트 갱신  
　C. 검색 결과 없음 시 빈 상태 안내 표시  

**2.3.1.2.** 정렬 변경  

1: 정렬 카드 선택  
　A. 가입순(기본)/결제순/방문순/스탬프 사용순/쿠폰 사용순/포인트 사용순 중 선택  
　B. 선택 시 즉시 리스트 재조회·정렬 적용  
　C. 현재 선택된 정렬 옵션 시각적 강조  

**2.3.1.3.** 날짜(기간) 선택  

1: 기간 필터  
　A. 일/주/월/년/기간(직접 선택) 중 선택  
　B. 기간 선택 시 해당 기간 데이터로 리스트 갱신  
　C. 기간 직접 선택 시 시작일·종료일 달력(또는 날짜 피커) 노출  


**2.3.2.** 오늘 방문객 리스트·검색 결과 섹션 내 동작  


**2.3.2.1.** 리스트 행 클릭  

1: 행 클릭 시(정책 확정 시)  
　A. 해당 고객 상세 보기(드로어/모달/하위 페이지)로 이동 가능  
　B. 상세에서 방문 이력·스탬프·쿠폰·포인트 사용 내역 등 확인 (구현 범위는 별도 정의)  

**2.3.2.2.** 페이지네이션  

1: 페이지 전환  
　A. 페이지당 50명 기준 이전/다음 또는 페이지 번호 클릭  
　B. 전환 시 동일 검색·정렬·기간 조건 유지한 채 해당 페이지 데이터 로드  


**2.3.3.** 공통  


**2.3.3.1.** 검색·정렬·기간 변경 시  

1: 리스트 영역 로딩 표시 후 결과 반영  
2: URL 쿼리 동기화(선택 사항): 검색어·정렬·기간·상태·페이지를 쿼리로 유지 시 새로고침 후에도 동일 상태 복원  


**2.3.4.** 고객 상태 필터 동작  


**2.3.4.1.** 상태 필터 전환  

1: 전체/일반/휴면 탭 전환  
　A. 전환 시 page=1로 초기화 후 리스트 재조회  
　B. 기존 검색어·정렬·기간 조건은 유지  
　C. 현재 선택된 상태 탭 시각적 강조  
2: 상태별 리스트 표시  
　A. 전체: 일반+휴면 고객 함께 표시, 각 행에 상태 뱃지(일반: 회색, 휴면: 주황)  
　B. 일반: 활동 고객만 표시  
　C. 휴면: 휴면 고객만 표시  


**2.3.5.** 휴면 고객 케어 동작(플러스 이상)  


**2.3.5.1.** 휴면 세분화 필터(프리미엄)  

1: 상태 필터에서 '휴면' 선택 시 dormantDays 서브 필터 노출  
　A. 30일/60일/90일 선택 가능(프리미엄만)  
　B. 플러스: 서브 필터 미노출(30일 고정)  
　C. 변경 시 page=1 초기화 후 리스트 재조회  


**2.3.5.2.** 알림톡 발송 실행  

1: 필터링 결과 내 대상 고객 선택(전체/개별)  
2: 메시지 템플릿 선택 → 미리보기 → 발송 확인 → 발송 실행  
3: 발송 완료 시 결과 안내(발송 건수·잔여 횟수)
