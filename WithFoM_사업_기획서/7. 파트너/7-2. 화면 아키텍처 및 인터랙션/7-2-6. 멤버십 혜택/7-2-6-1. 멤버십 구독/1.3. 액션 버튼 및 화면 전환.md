**1.3.** 액션 버튼 및 화면 전환  


**1.3.1.** 위드폼 멤버십 섹션 내 버튼 동작  


**1.3.1.1.** 상세페이지 버튼  

1: 동작  
　A. 베이직 카드 내 "상세페이지 버튼" 클릭 → 베이직 상세 설명 페이지(또는 모달)로 이동  
　B. 플러스 카드 내 "상세페이지 버튼" 클릭 → 플러스 상세 설명 페이지(또는 모달)로 이동  
　C. 프리미엄 카드 내 "상세페이지 버튼" 클릭 → 프리미엄 상세 설명 페이지(또는 모달)로 이동  

2: 표시 내용  
　A. 해당 플랜의 혜택 상세 목록 (1.2. 멤버십 상세 설명 섹션 내용과 동일 체계)  
　B. 플랜 비교가 필요한 경우 간단 비교 테이블 노출 가능  


**1.3.1.2.** 플러스 가입 버튼 / 프리미엄 가입 버튼  

1: 표시 조건  
　A. 베이직 회원 화면: 플러스 카드·프리미엄 카드에 각각 "플러스 가입 버튼", "프리미엄 가입 버튼" 표시  
　B. 플러스 회원 화면: 프리미엄 카드에만 "프리미엄 가입 버튼" 표시  

2: 클릭 시 동작  
　A. 플러스 가입 버튼 클릭 → 결제 페이지로 이동 (플랜: 플러스, 요금 정보 반영)  
　B. 프리미엄 가입 버튼 클릭 → 결제 페이지로 이동 (플랜: 프리미엄, 요금·프로모션 반영)  
　C. 결제 완료 후 자동 갱신: 현재 플랜 변경, 카드 구성 전환(플러스 회원 화면 또는 프리미엄 회원 화면)  


**1.3.1.3.** 구독 취소 버튼  

1: 표시 조건  
　A. 플러스 회원 화면: 플러스 카드 하단에 "구독 취소 버튼" 표시  
　B. 프리미엄 회원 화면: 프리미엄 카드 하단에 "구독 취소 버튼" 표시  

2: 클릭 시 동작  
　A. 구독 취소 버튼 클릭 → 구독 취소 페이지로 이동  
　B. 취소 확인·유예 기간 등 정책에 따른 안내 후 취소 처리  
　C. 취소 완료 후 베이직 회원 화면으로 전환 (3개 카드 구성)  


**1.3.2.** 멤버십 상세 설명 섹션  

**1.3.2.1.** 동작  
1: 본 섹션은 플랜별 혜택 텍스트·리스트 제공이 주 목적  
2: 상세 설명 내 "상세 보기" 등 링크가 있는 경우 1.3.1.1. 상세페이지 버튼과 동일한 대상(플랜별 상세)으로 이동  
3: 구독/취소 액션은 위드폼 멤버십 섹션의 버튼으로만 수행  


• 참조: 7-2-6-1. 멤버십 구독 > 1.1. 위드폼 멤버십 섹션, 1.2. 멤버십 상세 설명 섹션, 7-2-1-1. 공통 프레임 > 1.4. 멤버십 플랜을 뱃지로 표시
