**1.0.** 페이지 개요  


**1.0.1.** 페이지 목적  


**1.0.1.1.** 페이지 정의  

1: 페이지 명칭  
　A. 멤버십 구독 페이지  
　B. 멤버십 혜택 카테고리 내 첫 번째 하위 메뉴  

2: 페이지 역할  
　A. 현재 구독 플랜 확인 (베이직/플러스/프리미엄)  
　B. 플랜별 혜택 비교 및 상세 설명 제공  
　C. 플러스·프리미엄 구독 신청(결제) 및 구독 취소 처리  


**1.0.1.2.** 핵심 기능  

1: 위드폼 멤버십 섹션  
　A. 현재 플랜 기준 카드 구성 (베이직 회원: 3카드, 플러스: 2카드, 프리미엄: 1카드)  
　B. 플랜별 요금·한 줄 설명·혜택 요약 표시 (한 줄 설명: 베이직 "기본 운영 기능 무료 제공", 플러스 "단골·재방문 관리 중심", 프리미엄 "정밀 분석 + 고급 기능")  
　C. 상세페이지 버튼으로 플랜별 상세 설명 이동  
　D. 플러스/프리미엄 가입 버튼으로 결제 페이지 이동  
　E. 구독 취소 버튼(플러스·프리미엄 회원)으로 구독 취소 페이지 이동  

2: 멤버십 상세 설명 섹션  
　A. 베이직: 등급별 컵홀더 혜택, 운영 현황, 스탬프(기본), 리워드·오더, 기본 리포트, 고객 리스트, 빠른 발주·재고 관리  
　B. 플러스: 베이직 전체 + 스탬프 카드 템플릿 3종(로고 믹스), AI 리뷰 답글, 지도 핀 커스터마이징, 휴면 케어, 고객 랭킹, 벤치마크  
　C. 프리미엄: 플러스 전체 + 스탬프 5종+AI 디자인 커스텀, AI 아이콘, 휴면·우수 고객 관리(알림톡 300회), 포스터 AI, 세분화 벤치마크, 랭킹·퍼센타일, 우선 지원  


**1.0.2.** 진입 경로  


**1.0.2.1.** 사이드바에서 진입  

1: 사이드바 네비게이션  
　A. 사이드바 > 멤버십 혜택 > 멤버십 구독  
　B. 멤버십 구독 페이지 전체 로드 (1.1. 위드폼 멤버십 섹션 + 1.2. 멤버십 상세 설명 섹션)  


**1.0.2.2.** 멤버십 뱃지 클릭 시 진입  

1: 공통 프레임 뱃지 경로  
　A. 좌측 상단 멤버십 플랜 뱃지 클릭 시 업그레이드 안내  
　B. "플러스로 업그레이드" 또는 "프리미엄으로 업그레이드" 버튼 클릭 시  
　C. 7-2-6-1. 멤버십 구독 페이지로 이동 (선택 플랜 강조)  
　D. URL 예: /partner/membership/subscribe?plan={선택플랜}&source=header_badge  
　E. 현재 프리미엄 플랜인 경우 업그레이드 버튼 미노출, "현재 최고 플랜입니다" 메시지 표시 (1.4.2. 컴포넌트 구현 참고 참조)
