**7.2.** 리워드 연동 요약  


**7.2.1.** 리워드 필터  

**7.2.1.1.** 필터 기능  
　1: 전체 메뉴 보기  
　2: 리워드 사용 가능 메뉴만 보기  
　3: 리워드 제외 메뉴만 보기  
　4: 스탬프 적립 가능 메뉴만 보기  
　5: 스탬프 제외 메뉴만 보기  


**7.2.2.** 일괄 리워드/스탬프 설정  

**7.2.2.1.** 일괄 적용 버튼  
　1: 선택한 메뉴에 일괄 적용  
　　A. 체크박스로 메뉴 다중 선택  
　　B. "선택 메뉴 리워드 제외" 버튼  
　　C. "선택 메뉴 스탬프 제외" 버튼  
　2: 카테고리별 일괄 적용  
　　A. 카테고리 선택 → 해당 카테고리 전체에 적용  
　　B. "카테고리 리워드 제외" 버튼  
　　C. "카테고리 스탬프 제외" 버튼  

**7.2.2.2.** 개별 메뉴 설정  
　1: 메뉴 리스트에서 각 메뉴 옆 토글 스위치  
　　A. 쿠폰 사용 가능 ON/OFF  
　　B. 포인트 사용 가능 ON/OFF  
　　C. 스탬프 적립 가능 ON/OFF  


**7.2.3.** 리워드·스탬프 설정과 연동  

**7.2.3.1.** 연동 구조  
　1: 설정 위치:  
　　A. `7-2-8-3. 설정 > 리워드`: 리워드 제외 메뉴 설정  
　　B. `7-2-8-3. 설정 > 스탬프`: 스탬프 제외 메뉴 설정  
　　C. `7-2-10. 메뉴 관리`: 메뉴별 개별 설정  
　2: 연동 방식:  
　　A. 어느 곳에서 설정하든 동일하게 적용  
　　B. 변경 시 실시간 자동 반영  
　3: 설정 충돌 방지:  
　　A. 카테고리 일괄 제외 + 개별 메뉴 허용 시 개별 설정 우선  
　　B. 변경 내역 이력 저장  

**7.2.3.2.** 제외된 메뉴 현황 요약  
　1: 리워드 제외 메뉴 수: N개  
　　A. 쿠폰 제외: N개  
　　B. 포인트 제외: N개  
　2: 스탬프 제외 메뉴 수: N개  
　3: 클릭 시 해당 메뉴 리스트 필터링  


**7.2.4.** 향후 확장  

**7.2.4.1.** 캠페인 연동  
　1: 향후 특정 리워드 캠페인과 연동되는 메뉴 일괄 설정 여지  

**7.2.4.2.** 시간대별 리워드 설정  
　1: 향후 특정 시간대에만 리워드 적용 가능하도록 설정 기능 확장 예정
