**8.2.** 레시피 등록 화면  


**8.2.1.** 화면 구성  

**8.2.1.1.** 레이아웃  
1: 상단 영역  
　A. 메뉴 기본 정보 표시 (메뉴명, 카테고리, 판매가)  
　B. 레시피 등록 상태 표시 (미등록 / 등록됨 / 검토 필요)  
2: 중앙 좌측  
　A. 재료 목록 입력 영역  
　B. 구성품 목록 입력 영역  
3: 중앙 우측  
　A. 옵션별 재료 변동 설정 영역  
　B. 재고 연동 상태 미리보기  
4: 하단  
　A. [임시 저장] / [저장] / [취소] 버튼  


**8.2.2.** 재료 입력 영역  

**8.2.2.1.** 입력 필드  
1: 재료 추가 버튼  
　A. [+ 재료 추가] 버튼 클릭 시 새 행 추가  
　B. 행 구성: 재료 선택 드롭다운 | 사용량 입력 | 단위 선택 | 필수 여부 | 삭제 버튼  
2: 재료 선택  
　A. 재고 관리에 등록된 품목 목록에서 선택  
　B. 검색 기능 제공 (자동완성)  
　C. 목록에 없는 경우 "신규 등록" 옵션 표시  
3: 사용량 입력  
　A. 숫자 입력 필드  
　B. 소수점 2자리까지 허용 (예: 18.5ml)  
4: 단위 선택  
　A. 드롭다운: ml, L, g, kg, ea(개), 장, 봉지 등  
　B. 재고 관리 품목의 단위와 자동 매칭  
5: 필수 여부  
　A. 토글 또는 체크박스  
　B. 필수 재료: 재고 부족 시 메뉴 판매 불가 경고  
　C. 선택 재료: 재고 부족 시 대체 재료 안내  


**8.2.2.2.** 재료 그룹핑  
1: 카테고리별 그룹  
　A. 원재료 (커피원두, 우유, 시럽 등)  
　B. 토핑 (휘핑, 드리즐, 파우더 등)  
　C. 얼음/물  
2: 그룹 접기/펼치기  
　A. 그룹 헤더 클릭 시 토글  
　B. 재료가 많은 레시피의 가독성 향상  


**8.2.3.** 구성품 입력 영역  

**8.2.3.1.** 입력 필드  
1: 구성품 추가 버튼  
　A. [+ 구성품 추가] 버튼 클릭 시 새 행 추가  
　B. 행 구성: 구성품 선택 | 사용량 | 조건 설정 | 삭제 버튼  
2: 구성품 선택  
　A. 재고 관리에 등록된 구성품 목록에서 선택  
　B. 기본 구성품: 컵, 뚜껑, 스트로우, 홀더, 비닐팩, 캐리어 등  
3: 조건 설정  
　A. 항상 사용: 매장/테이크 아웃 모두  
　B. 테이크 아웃 전용: 포장 시에만 사용  
　C. 매장 전용: 매장 이용 시에만 사용  
　D. 조건부: 특정 옵션 선택 시에만  


**8.2.3.2.** 컵 타입 연동  
1: 사이즈별 컵 자동 매칭  
　A. 스몰 → 12oz 컵  
　B. 레귤러 → 16oz 컵  
　C. 라지 → 20oz 컵  
2: 온도별 컵 자동 매칭  
　A. 아이스 → 플라스틱 컵  
　B. 핫 → 종이 컵  
3: 자동 매칭 규칙 편집 가능  


**8.2.4.** 옵션별 재료 변동  

**8.2.4.1.** 옵션 연동 설정  
1: 옵션 그룹 연동  
　A. 메뉴 편집에서 설정한 옵션 그룹 자동 불러오기  
　B. 각 옵션별 재료 변동량 설정  
2: 변동 타입  
　A. 추가: 기본 레시피에 재료 추가 (예: 샷 추가 → 에스프레소 +18ml)  
　B. 변경: 기존 재료량 변경 (예: 연하게 → 에스프레소 -9ml)  
　C. 대체: 재료 자체 변경 (예: 두유로 변경 → 우유 0ml, 두유 +150ml)  
3: 구성품 변동  
　A. 사이즈 업 → 컵/뚜껑 변경  
　B. 엑스트라 휘핑 → 휘핑크림 추가  


**8.2.5.** 저장 및 검증  

**8.2.5.1.** 저장 전 검증  
1: 필수 항목 확인  
　A. 최소 1개 이상의 재료 등록 필요  
　B. 재고 연동 품목 확인  
2: 경고 표시  
　A. 재고 미등록 품목 포함 시 경고  
　B. 단위 불일치 시 경고  
　C. 사용량 0 입력 시 확인 요청  

**8.2.5.2.** 저장 동작  
1: 임시 저장  
　A. 작성 중인 내용 저장  
　B. 재고 연동 미적용  
2: 저장  
　A. 레시피 확정 및 재고 연동 활성화  
　B. 이후 주문 발생 시 재고 자동 차감 시작
