**8.3.** 재고 연동 설정  


**8.3.1.** 연동 개요  

**8.3.1.1.** 연동 목적  
1: 레시피에 등록된 재료/구성품을 재고 관리 시스템과 자동 연동  
2: 메뉴 판매 시 재고 자동 차감으로 실시간 재고 현황 파악  
3: 재고 부족 예측 및 자동 발주 트리거 지원  


**8.3.2.** 품목 매칭  

**8.3.2.1.** 자동 매칭  
1: 매칭 조건  
　A. 레시피 재료명과 재고 관리 품목명이 동일할 경우 자동 매칭  
　B. 단위가 호환 가능할 경우 자동 변환 (예: ml ↔ L)  
2: 자동 매칭 상태 표시  
　A. 녹색 체크: 정상 매칭  
　B. 노란색 느낌표: 수동 확인 필요  
　C. 빨간색 X: 매칭 실패  

**8.3.2.2.** 수동 매칭  
1: 매칭 필요 상황  
　A. 레시피 재료명과 재고 품목명이 다른 경우  
　B. 예: 레시피 "에스프레소" ↔ 재고 "커피원두"  
2: 매칭 설정 화면  
　A. 미매칭 항목 리스트 표시  
　B. 각 항목별 재고 품목 선택 드롭다운  
　C. 변환 비율 입력 (예: 에스프레소 1샷 = 커피원두 7g)  
3: 매칭 저장  
　A. 한 번 매칭된 관계는 영구 저장  
　B. 다른 메뉴에서 동일 재료 사용 시 자동 적용  


**8.3.3.** 단위 변환  

**8.3.3.1.** 자동 변환  
1: 호환 단위  
　A. ml ↔ L (1L = 1000ml)  
　B. g ↔ kg (1kg = 1000g)  
　C. ea 동일  
2: 변환 적용  
　A. 레시피에서 ml 단위, 재고에서 L 단위일 경우 자동 변환  
　B. 변환 비율은 시스템에서 자동 계산  

**8.3.3.2.** 비호환 단위 처리  
1: 수동 변환 비율 설정  
　A. 예: 에스프레소 1샷 = 커피원두 7g  
　B. 예: 우유 1포션 = 우유 150ml  
2: 경고 표시  
　A. 단위 변환이 필요한 경우 경고 표시  
　B. 변환 비율 미설정 시 재고 차감 불가 알림  


**8.3.4.** 차감 로직  

**8.3.4.1.** 차감 트리거  
1: 위드폼 오더 연동 시  
　A. 주문 완료(결제 완료) 시점에 재고 차감  
　B. 주문 취소 시 재고 복구  
2: 위드폼 오더 미사용 시  
　A. 일일 판매 수량 수동 입력 시 재고 차감  
　B. 입력 시점: 영업 종료 후 또는 수시 입력  

**8.3.4.2.** 차감 계산  
1: 기본 차감  
　A. 레시피 기본 사용량 × 판매 수량  
　B. 예: 아메리카노 1잔 판매 → 에스프레소 2샷(36ml) 차감  
2: 옵션 적용 차감  
　A. 옵션별 재료 변동량 합산  
　B. 예: 샷 추가 옵션 → 에스프레소 +18ml 추가 차감  
3: 구성품 차감  
　A. 주문 유형(매장/테이크아웃)에 따른 조건부 차감  
　B. 테이크 아웃 시: 컵, 뚜껑, 스트로우, 홀더 차감  
　C. 매장 시: 머그컵 세척 카운트 (선택적)  


**8.3.5.** 연동 상태 모니터링  

**8.3.5.1.** 연동 현황 대시섹션  
1: 표시 항목  
　A. 총 메뉴 수  
　B. 레시피 등록 완료 메뉴 수  
　C. 재고 연동 완료 메뉴 수  
　D. 연동 오류 메뉴 수  
2: 알림 표시  
　A. 신규 메뉴 레시피 미등록 시 알림  
　B. 재고 품목 삭제로 인한 연동 해제 시 알림  

**8.3.5.2.** 연동 오류 처리  
1: 오류 유형  
　A. 재고 품목 삭제됨  
　B. 단위 변환 비율 미설정  
　C. 재고 부족으로 차감 불가  
2: 오류 해결  
　A. 재고 관리에서 품목 재등록 또는 대체 품목 매칭  
　B. 변환 비율 설정 완료  
　C. 재고 수량 수동 조정 또는 발주  


• 참조: 7-2-11-1. 재고 관리, 7-2-11-2. 발주
