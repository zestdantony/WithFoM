**72811.** 소비자 검색 섹션  
  
  
**728111.** 검색 입력 필드  
  
**7281111.** 위치  
72811111. 페이지 최상단, 탭 네비게이션 위, 항상 표시  
  
**7281112.** 입력 필드 구성  
72811121. 플레이스홀더: "닉네임 또는 회원번호로 검색"  
72811122. 우측: 검색 버튼 (돋보기 아이콘)  
72811123. 입력 시 실시간 필드 스타일 변경 (포커스 강조)  
  
**7281113.** 검색 실행 방식  
72811131. 닉네임 검색: 텍스트 입력, 부분 일치 검색  
72811132. 회원번호 검색: 숫자 4자리 이상 입력 시 회원번호 검색으로 자동 전환  
72811133. 엔터 키 또는 검색 버튼 클릭으로 검색 실행  
  
  
**728112.** 검색 결과 표시  
  
**7281121.** 소비자 정보 카드 (검색 성공 시)  
72811211. 닉네임  
72811212. 회원번호 뒷자리 (마스킹 처리, 예: ****1234)  
72811213. 리워드 보유 요약: 쿠폰 N개 · 포인트 N점 · 스탬프 N/M개  
72811214. 단골 스토어 지정 여부 (지정됨 / 미지정)  
72811215. 마지막 방문일  
  
**7281122.** 검색 결과 없음  
72811221. "검색 결과가 없습니다" 안내 메시지  
72811222. "닉네임 또는 회원번호를 다시 확인해 주세요" 부연  
  
  
**728113.** 모드 전환  
  
**7281131.** 검색 완료 시 동작  
72811311. 소비자 정보 카드가 검색 입력 필드 아래에 표시  
72811312. 탭 콘텐츠가 해당 소비자의 개인 리워드 현황으로 전환 (소비자 모드)  
72811313. 소비자 모드 진입 시 현재 선택된 탭 유지  
  
**7281132.** 검색 초기화  
72811321. 소비자 정보 카드 우측 상단 X 버튼 클릭  
72811322. 소비자 정보 카드 제거 + 탭 콘텐츠가 매장 전체 현황으로 복귀 (기본 모드)  
72811323. 검색 입력 필드 초기화  
  
**7281133.** 탭 전환 시 모드 유지  
72811331. 소비자 모드에서 탭 전환 → 같은 소비자의 다른 유형 현황 표시  
72811332. 기본 모드에서 탭 전환 → 매장 전체의 다른 유형 현황 표시
