**72810.** 페이지 개요  
  
  
**728101.** 페이지 목적  
  
**7281011.** 페이지 정의  
72810111. 리워드·스탬프 현황 페이지  
72810112. 매장의 리워드(쿠폰·포인트·스탬프) 운영 현황을 조회하고, 소비자별 리워드 보유·사용 상태를 확인하는 페이지  
  
**7281012.** 페이지 역할  
72810121. 매장 전체 리워드 통계 조회 (유형별 TOP5, 발행·사용 현황, 리스트)  
72810122. 특정 소비자의 리워드 보유·사용 내역 조회  
72810123. 스탬프 수동 적립 (소비자 앱 사용 불가 시 예외 처리)  
  
  
**728102.** 핵심 기능  
  
**7281021.** 소비자 검색  
72810211. 닉네임 또는 회원번호로 특정 소비자 조회  
  
**7281022.** 리워드 유형별 현황 조회  
72810221. 쿠폰·포인트·스탬프 탭 전환으로 유형별 현황 확인  
  
**7281023.** 스탬프 수동 적립  
72810231. 소비자 앱 사용 불가 시 파트너가 소비자를 검색하여 직접 스탬프 적립 (예외 처리)  
  
  
**728103.** 페이지 레이아웃 구조  
  
**7281031.** 전체 레이아웃  
72810311. 영역 1 (상단): 소비자 검색 섹션 — 항상 표시  
72810312. 영역 2 (검색 아래): 탭 네비게이션 — 쿠폰 | 포인트 | 스탬프  
72810313. 영역 3 (탭 아래): 탭 콘텐츠 — 선택된 탭에 따라 내용 변경

72810321. 검색바 하단에 가로 배치  
72810322. 3가지 탭: 쿠폰, 포인트, 스탬프  
72810323. 선택된 탭 강조 표시 (브랜드 컬러)  
72810324. 기본 선택: 쿠폰  
  
**7281033.** 탭 동작  
72810331. 탭 클릭 시 해당 리워드 유형의 현황 섹션으로 전환  
72810332. 현재 모드(기본/소비자) 유지 상태에서 탭 전환

**728106.** 서비스 타입별 리워드 처리 방식  
  
**7281061.** 오더형 (복합형) — 전자동  
72810611. 쿠폰·포인트: 소비자가 앱에서 주문 시 자동 적용  
72810612. 스탬프: 결제 금액 3,000원 이상 시 자동 적립  
72810613. 파트너 작업: 없음 (모든 처리가 시스템에서 자동)  
  
**7281062.** 리워드형 — 소비자 주도 (주요 방식)  
72810621. 쿠폰·포인트 사용: 소비자가 앱에서 [사용하기] → 1회용 토큰 생성 → 파트너 대시보드에 승인 요청 카드 노출 → 파트너 [승인] 또는 60초 자동 승인  
72810622. 스탬프 적립: 소비자가 매장에서 3,000원 이상 결제 → 앱에서 [스탬프 적립 요청] → 파트너 대시보드에 승인 요청 카드 노출 → 파트너 [승인] 또는 60초 자동 승인  
  
**7281063.** 리워드형 — 파트너 주도 (예외 방식, 스탬프만)  
72810631. 적용 조건: 소비자가 앱을 사용할 수 없는 상황 (시스템 오류, 기기 고장, 앱 미설치 등)  
72810632. 대상: 스탬프 적립만 (쿠폰·포인트 사용은 소비자 앱 필수)  
72810633. 방법: 파트너가 현황 페이지 스탬프 탭에서 [수동 적립] → 소비자 검색 → 결제 금액 확인 → 적립
