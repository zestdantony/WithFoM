　　　**3.3.** 스탬프  


　　　　**3.3.1.** 스탬프 설정  

　　　　　1: 스탬프 카드 설정 박스  
　　　　　　A. 스탬프 적립 기준 금액 입력창  
　　　　　　　　a. 설명: "실결제 금액 기준 스탬프 적립"  
　　　　　　　　b. 예시: 3,000원 이상 결제 시 스탬프 1개 적립  
　　　　　　B. 다중 스탬프 적립 설정 (선택)  
　　　　　　　　a. 토글: "결제 금액에 따라 추가 스탬프 적립"  
　　　　　　　　b. 배수 적립: 기준 금액의 배수마다 추가 적립  
　　　　　　　　c. 구간별 적립: 금액 구간별로 적립 개수 직접 설정  
　　　　　　C. 유효기간 입력창  
　　　　　　　　a. 유효기간 미설정 체크박스  
　　　　　　　　b. 체크박스 클릭 시 기간 설정 불가, 무기한 표시  
　　　　　　　　c. 기간 선택: 3개월, 6개월, 1년, 사용자 지정  
　　　　　　　　d. 기본값: 1년  
　　　　　　D. 1일 적립 한도 입력창 (선택)  
　　　　　　　　a. 고객 1인당 1일 최대 적립 개수  
　　　　　　　　b. 권장값: 3개  

　　　　　2: "N개 적립 시 혜택" 설정 박스  
　　　　　　A. 스탬프 카드 박스 우측에 배치  
　　　　　　B. 최대 3개의 혜택 설정 가능  
　　　　　　C. 각 혜택: 적립 개수 + 혜택 내용 입력  
　　　　　　D. 플레이스홀더: "예: 10, 20, 30" / "혜택 입력"  


　　　　**3.3.2.** 적립 제외 항목 설정  

　　　　　1: 카테고리별 제외 설정  
　　　　　　A. 체크박스 옵션:  
　　　　　　　　a. 행사 제품 제외 (1+1, 2+1 등)  
　　　　　　　　b. 이미 할인된 상품 제외  
　　　　　　　　c. MD 상품 제외 (머그컵, 텀블러 등)  
　　　　　　B. 기본값: 모두 체크 해제 (제외 없음)  

　　　　　2: 메뉴별 직접 제외 설정  
　　　　　　A. 메뉴 검색 입력창  
　　　　　　　　a. 메뉴명 검색하여 제외할 메뉴 선택  
　　　　　　B. 카테고리별 메뉴 리스트  
　　　　　　　　a. 카테고리 선택 → 해당 카테고리 메뉴 표시  
　　　　　　　　b. 각 메뉴 옆 체크박스로 제외 선택  
　　　　　　C. 제외된 메뉴 리스트  
　　　　　　　　a. 현재 스탬프 적립 제외된 메뉴 목록 표시  
　　　　　　　　b. 각 메뉴 옆 X 버튼으로 제외 해제  
　　　　　　D. 연동 안내:  
　　　　　　　　a. "`7-2-9. 메뉴 관리`에서도 메뉴별 스탬프 제외 설정 가능"  
　　　　　　　　b. "양쪽에서 설정한 내용은 자동으로 연동됩니다"  


　　　　**3.3.3.** 스탬프 요청/승인 흐름(대시보드 연동)  

　　　　　1: 고객 스탬프 적립 가능 컨디션  
　　　　　　A. 고객이 스탬프 적립 가능 컨디션일 경우 파트너 대시보드의 리워드 사용 보드에 박스 생성  

　　　　　　B. 스탬프 적립 가능 컨디션:  
　　　　　　　　a. 결제 금액 충족  
　　　　　　　　b. 단골 스토어 지정 여부  
　　　　　　　　c. 제외 메뉴가 아닌 경우  

　　　　　　C. 스탬프 적립 가능 컨디션일 경우 자동 적립  

　　　　　2: 위드폼 오더 미사용 + 스탬프 기능 활성화 매장의 경우  
　　　　　　A. 소비자가 스탬프 적립 조건을 만족하면, 앱/웹에서 "스탬프 적립 요청" 가능  
　　　　　　B. 스탬프 적립 가능 컨디션일 경우 자동 적립  
　　　　　　C. 파트너 대시보드의 리워드 사용 보드에 스탬프 박스 생성  

　　　　　3: 결제 금액 충족 + 스탬프 시스템을 모르는 고객 안내 매뉴얼  
　　　　　　A. 단골 스토어 지정 + 스탬프 적립 가능 여부 안내  

　　　　　4: 첫 방문 고객 안내 매뉴얼  
　　　　　　A. 위드폼에서 카페 검색 후 단골 스토어 지정 시 스탬프 카드 자동 생성됨을 안내  


　　　　**3.3.4.** 향후 확장 기능 안내  

　　　　　1: 스탬프-포인트 전환 기능  
　　　　　　A. 스탬프를 포인트로 전환하는 기능 추가 예정  
　　　　　　B. 자세한 내용: `10. 향후 확장 로드맵\10-2. 신규 기능 추가\6. 스탬프-포인트 통합 시스템.md`  
　　　　　　C. 예상 출시: 런칭 +12개월
