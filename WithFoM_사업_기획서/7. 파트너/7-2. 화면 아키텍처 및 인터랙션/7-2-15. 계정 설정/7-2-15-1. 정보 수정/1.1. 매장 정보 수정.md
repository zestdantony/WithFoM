**1.1.** 매장 정보 수정  


**1.1.1.** 기본 정보 관리  

**1.1.1.1.** 매장명  
1: 사업자 등록 상호명 또는 브랜드명  

**1.1.1.2.** 주소  
1: 도로명 주소 + 상세 주소  

**1.1.1.3.** 영업 시간  
1: 요일별 오픈/마감 시간  
2: 휴무일 설정  

**1.1.1.4.** 연락처  
1: 대표 전화번호  


**1.1.2.** 주소·지도 핀 설정  

**1.1.2.1.** 지도 핀 위치  
1: 입구 위치 기준으로 핀 지정 (드래그 조정 가능)  

**1.1.2.2.** 핀 위치 가이드  
1: 매장 건물 중앙이 아니라 실제 손님이 출입하는 '입구 위치'에 최대한 맞추도록 안내 툴팁 제공  

**1.1.2.3.** 소비자 지도 연동  
1: 설정된 핀이 소비자 홈 화면 파트너 지도에 자동 노출  

**1.1.2.4.** 캠페인 리스트 연동  
1: 해당 매장에서 진행 중인 캠페인이 핀 클릭 시 자동 표시  


**1.1.3.** 매장 로고 및 브랜드 컬러  

**1.1.3.1.** 매장 로고  
1: 로고 이미지 업로드  
　A. 지원 포맷: PNG/SVG/JPG, 최소 512px, RGB, 5MB 이하  
　B. 투명 배경 권장 (가이드 툴팁 제공)  
　C. 안전 여백: 로고 가장자리에서 10% 이상 여백 필요  
2: 업로드 시 미리보기 제공  
3: 로고 변경 시 기존 로고는 즉시 교체 (공통 프레임·스탬프 카드·포스터 등 연동)  

**1.1.3.2.** 브랜드 대표 컬러  
1: 자동 추출 방식  
　A. 로고 업로드 시 AI가 로고 이미지에서 대표 컬러 3색 자동 추출  
　B. 추출된 3색 중 1색을 "대표 컬러"로 자동 선택 (가장 면적이 넓은 색)  
　C. 파트너가 다른 색 선택 가능 (3색 중 택1)  
2: 수동 설정 방식  
　A. "직접 선택" 버튼 클릭 시 컬러 피커 노출  
　B. HEX 코드 직접 입력 가능  
　C. 프리셋 팔레트 제공 (카페 업종 추천 12색)  
3: 로고 미등록 시  
　A. 대표 컬러를 수동으로만 설정 가능  
　B. 미설정 시 위드폼 기본 컬러(#2563EB) 적용  
4: 용도 안내  
　A. "설정한 컬러는 스탬프 카드 디자인과 포스터 AI 생성 시 자동 반영됩니다"  
　B. 적용 범위: 스탬프 카드 템플릿 컬러 반영, 포스터 AI 생성 시 컬러 모드 기본값, 향후 커스터마이징 확장 시 활용  


**1.1.4.** 지도 아이콘 관리  

**1.1.4.1.** 기본 아이콘  
1: 등급별 기본 핀 아이콘 자동 적용  

**1.1.4.2.** 커스텀 아이콘 (멤버십 등급별)  
1: 플러스: 카페 로고 아이콘 생성 가능  
2: 프리미엄: 카페 로고 + 메인 이벤트 표시 아이콘 생성 가능  

**1.1.4.3.** 아이콘 스타일 선택  
1: 스타일이 다르게 표현된 아이콘 핀 3종 중 1개 선택 적용  

**1.1.4.4.** 적용 범위  
1: 소비자 홈 지도  
2: 캠페인 상세 페이지 지도
