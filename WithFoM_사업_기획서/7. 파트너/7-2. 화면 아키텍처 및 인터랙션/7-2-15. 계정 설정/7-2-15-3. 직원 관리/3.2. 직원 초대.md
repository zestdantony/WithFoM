**3.2.** 직원 초대  


**3.2.1.** 초대 절차  

**3.2.1.1.** 초대 방법  
1: 직원 초대 버튼 클릭  
2: 이메일/연락처 입력  
3: 권한 범위 설정  
4: 초대 발송  


**3.2.2.** 권한 범위 선택  

**3.2.2.1.** 부여 가능 권한  

| 권한 | 설명 |
|---|---|
| 리워드/결제 승인 | 리워드 사용 승인, 결제 승인 처리 |
| 일일 소진 보고 | 소진량 입력 및 수정 |
| 메뉴 관리 | 메뉴 등록/수정/삭제 |
| 발주 요청 | 컵홀더 발주 요청 |
| 매장 정보 수정 | 매장 정보 조회 및 수정 |


**3.2.3.** 초대 알림  

**3.2.3.1.** 알림 발송  
1: 이메일 또는 SMS로 초대 링크 발송  
2: 초대 메시지: "[매장명]에서 직원으로 초대했습니다. 링크를 클릭하여 가입을 완료하세요."  
3: 직원이 링크 클릭하여 가입 완료  


**3.2.4.** 초대 상태 관리  

**3.2.4.1.** 상태 표시  
1: 초대 대기: 초대 발송 완료, 직원 미가입  
2: 가입 완료: 직원 가입 완료  
3: 초대 만료: 7일 경과 시 초대 만료  
4: 재초대 가능  


• 참조: 7-5-2. 직원 계정
