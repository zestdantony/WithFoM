**3.3.** 권한 설정  


**3.3.1.** 부여 가능 권한  

**3.3.1.1.** 권한 테이블  

| 권한 | 설명 |
|---|---|
| 리워드/결제 승인 | 리워드 사용 승인, 결제 승인 처리 |
| 일일 소진 보고 | 소진량 입력 및 수정 |
| 메뉴 관리 | 메뉴 등록/수정/삭제 |
| 발주 요청 | 컵홀더 발주 요청 |
| 매장 정보 수정 | 매장 정보 조회 및 수정 |


**3.3.2.** 대표 전용 권한 (직원 부여 불가)  

**3.3.2.1.** 전용 권한  
1: 정산 조회  
2: 등급 확인  
3: 계정 관리 (직원 초대/권한 변경/비활성화)  
4: 탈퇴  


**3.3.3.** 권한 수정  

**3.3.3.1.** 수정 방법  
1: 직원 목록에서 해당 직원 선택  
2: 권한 수정 버튼 클릭  
3: 권한 체크박스 선택/해제  
4: 저장 버튼 클릭  
5: "권한이 수정되었습니다" 메시지 표시  


• 참조: 7-5-2-1. 직원 권한별 메뉴 노출
