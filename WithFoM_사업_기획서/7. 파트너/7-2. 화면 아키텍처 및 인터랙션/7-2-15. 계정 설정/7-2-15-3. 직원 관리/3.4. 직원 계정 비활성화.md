**3.4.** 직원 계정 비활성화  


**3.4.1.** 비활성화 절차  

**3.4.1.1.** 처리 방법  
1: 직원 목록에서 해당 직원 선택  
2: 비활성화 버튼 클릭  
3: 확인 팝업 표시  
　A. "해당 직원의 계정을 비활성화하시겠습니까?"  
　B. "비활성화 시 해당 직원은 즉시 로그인할 수 없습니다."  
4: 확인 시 즉시 비활성화  


**3.4.2.** 비활성화 효과  

**3.4.2.1.** 처리 결과  
1: 해당 직원 로그인 불가  
2: 모든 권한 회수  
3: 활성 세션 즉시 종료  


**3.4.3.** 재활성화  

**3.4.3.1.** 재활성화 방법  
1: 비활성화된 직원 선택  
2: 재활성화 버튼 클릭  
3: 확인 시 즉시 재활성화  
4: 이전에 부여된 권한 복원  


• 참조: 7-5-2. 직원 계정
