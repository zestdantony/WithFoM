**4.2.** 탈퇴 제한 조건 확인(섹션)  


**4.2.1.** 탈퇴 불가 조건  

**4.2.1.1.** 제한 조건  
1: 미처리 주문 확인  
　A. 진행 중인 주문이 있는 경우 탈퇴 불가  
　B. 주문 승인 대기 중인 건 확인  
　C. 주문 준비 중/준비 완료 상태인 건 확인  
　D. 주문 취소/환불 처리 대기 중인 건 확인  

2: 미정산 매출 확인  
　A. 미정산 매출이 있는 경우 탈퇴 불가  
　B. 정산 예정/정산 중 상태인 정산 건 확인  
　C. 입금/청구 대기 중인 정산 건 확인  

3: 미확정 청구 금액 확인  
　A. 미확정 청구 금액이 있는 경우 탈퇴 불가  
　B. 컵홀더 비용 미결제 건 확인  
　C. 멤버십 구독료 미결제 건 확인  


**4.2.2.** 탈퇴 제한 안내  

**4.2.2.1.** 안내 팝업  
1: 제한 조건 안내 팝업  
　A. 탈퇴 신청 시 제한 조건 자동 확인  
　B. 제한 조건 존재 시 안내 팝업 표시:  
　　a. '탈퇴를 진행할 수 없습니다.  
　　b. 다음 항목을 먼저 처리해 주세요:  
　　c. - 미처리 주문: N건  
　　d. - 미정산 매출: N원  
　　e. - 미확정 청구 금액: N원'  

2: 처리 안내  
　A. 각 항목별 처리 방법 안내  
　B. 관련 메뉴로 이동 버튼 제공  
　C. 모든 조건 해결 후 탈퇴 가능 안내  


**4.2.3.** 탈퇴 제한 조건 처리  

**4.2.3.1.** 미처리 주문 처리  
1: 처리 방법  
　A. 주문 관리 메뉴에서 미처리 주문 확인  
　B. 주문 승인/거절 처리  
　C. 주문 취소 처리 (해당 시)  
　D. 모든 주문 처리 완료 후 탈퇴 가능  

**4.2.3.2.** 미정산 매출 처리  
1: 처리 방법  
　A. 정산 관리 메뉴에서 미정산 매출 확인  
　B. 정산 확정 대기 중인 건 확인  
　C. 정산 확정 후 입금/청구 완료 대기  
　D. 모든 정산 완료 후 탈퇴 가능  

**4.2.3.3.** 미확정 청구 금액 처리  
1: 처리 방법  
　A. 결제 관리 메뉴에서 미확정 청구 금액 확인  
　B. 컵홀더 비용 미결제 건 결제  
　C. 멤버십 구독료 미결제 건 결제  
　D. 모든 청구 금액 결제 완료 후 탈퇴 가능  


• 참조: 7-1-9-2. 미처리 주문·미정산 여부 확인
