**2.1.1.** 구성 및 표시 정보  


**2.1.1.1.** 등급 정보 영역  

1: 이번달 등급 표시  
　A. 현재 등급 마크 표시 (대형 아이콘)  
　B. 등급명 표시 (브론즈/실버/골드/플래티넘)  
　C. 등급 유지 기간 표시 (예: "2025.07 ~")  
　D. 미매칭 상태 표시 (미매칭 시)  
　　a. 등급 변경 예정일 표시 (예: "2025.07.15 등급 변경 예정")  
　　b. 남은 유예 기간 표시 (예: "135일 남음")  
　　c. 경고 표시  
　　　aa. 3개월 경과 시: 노란색 경고 배지 "캠페인 매칭 필요"  
　　　bb. 5개월 3주차 이후: 빨간색 경고 배지 "1주 내 등급 변경"  
　　　cc. 매칭 완료 시: 경고 배지 숨김  

2: 등급 마크 스타일  
　A. 브론즈: 🥉 (#CD7F32)  
　B. 실버: 🥈 (#C0C0C0)  
　C. 골드: 🥇 (#FFD700)  
　D. 플래티넘: 💎 (#E5E4E2)  


**2.1.1.2.** 등급 산정 기준 표시 영역  

1: 산정 시작일  
　A. 현재 등급 산정 시작 일자 표시  

2: 경과 일수  
　A. 산정 시작일부터 현재까지의 경과 일수 표시  

3: 실 사용 기간 진행률  
　A. 실 사용 기간 / 필요 기간 진행률 프로그래스 바  
　B. 표시 형식: "N일 / M일" (예: "142일 / 180일")  
　C. 필요 기간: 브론즈~골드 180일, 플래티넘 365일  
　D. 프로그래스 바 색상: 등급별 색상 적용  

4: 등급 점수 표시  
　A. 현재 총점 (대형 폰트)  
　B. 다음 등급까지 필요 점수 gap 표시  
　C. 최고 등급(플래티넘) 도달 시: "최고 등급 달성" 표시  


**2.1.1.3.** 등급 혜택 안내 영역  

1: 적용 혜택 체크리스트  
↳ 현재 등급에 해당하는 혜택만 표시하며, 각 혜택의 적용/미적용 상태를 표시  
　A. 컵홀더 혜택  
　　a. 표시 내용: 브론즈 "50% 할인 구매" / 실버 이상 "무료 제공"  
　　b. 캠페인 참여 중: ✅ 적용 중  
　　c. 캠페인 미참여 중: ⚠️ "캠페인 미참여 중 — 매칭 시 자동 적용됩니다"  
　B. 멤버십 30% 할인 (골드/플래티넘만 표시)  
　　a. 구독 중: ✅ "{플랜명} 30% 할인 적용 중 ({원가} → {할인가})"  
　　b. 미구독: ⚠️ "플러스 구독 시 매월 ₩8,700, 프리미엄 구독 시 매월 ₩14,700의 할인 혜택을 놓치고 있습니다"  
　C. 인센티브 정산 (플래티넘만 표시)  
　　a. 캠페인 참여 중: ✅ "소진 컵홀더당 ₩10 인센티브 적용 중"  
　　b. 캠페인 미참여 중: ⚠️ "캠페인 미참여 중 — 매칭 시 자동 적용됩니다"  

2: 캠페인 미참여 시 멤버십 유도 문구 (미구독 시에만 표시)  
　A. 문구: "멤버십 구독 시 캠페인 우선 매칭 혜택을 받을 수 있습니다"  
　B. 텍스트 링크: "멤버십 확인하기" → 멤버십 구독 페이지  

3: 혜택 미적용 시 문의 안내  
　A. 안내 문구: "적용되지 않은 혜택이 있나요?"  
　B. 문의 링크: "1:1 문의하기" → 도움말·문의 > 1대1 문의 (카테고리: 등급/혜택 자동 선택)
