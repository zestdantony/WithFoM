**2.2.3.1.** 컵홀더 보고 현황 조회 API  

1: 엔드포인트  
　A. GET /api/v1/partner/grade/report-status  

2: 요청 파라미터  
　A. period: weekly / 6months (기본: weekly)

**2.2.3.2.** 보고 내역 조회 API  

1: 엔드포인트  
　A. GET /api/v1/partner/grade/report-history  

2: 요청 파라미터  
　A. startDate: 시작일 (YYYY-MM-DD)  
　B. endDate: 종료일 (YYYY-MM-DD)  
　C. page: 페이지 번호 (기본: 1)  
　D. limit: 페이지당 개수 (기본: 20)  


**2.2.3.3.** 컴포넌트 상태 관리  

1: 드롭다운 상태 (expandedState)  
　A. weekly: boolean  
　B. sixMonths: boolean  

2: 모달 상태 (modalState)  
　A. reportHistory: boolean
