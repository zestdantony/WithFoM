**2.4.3.** 개발자 인터랙션 명세  


**2.4.3.1.** 위드폼 오더 이용 현황 조회 API  

1: 엔드포인트  
　A. GET /api/v1/partner/grade/order-status  

2: 응답 구조 (이용 중)  

```json
{
  "isUsing": true,
  "startDate": "2025-03-15T00:00:00Z",
  "usageDays": 320,
  "statistics": {
    "totalOrders": 2450,
    "monthlyAverage": 245,
    "recentMonth": 268
  },
  "gradeImpact": {
    "scoreContribution": 10,
    "enablesPlatinum": true,
    "currentGradeWithout": "silver",
    "currentGradeWith": "gold"
  }
}
```

3: 응답 구조 (미이용)  

```json
{
  "isUsing": false,
  "startDate": null,
  "usageDays": 0,
  "statistics": null,
  "gradeImpact": {
    "scoreContribution": 0,
    "enablesPlatinum": false,
    "potentialGradeWith": "gold",
    "potentialScoreGain": 10
  },
  "signup": {
    "url": "/partner/order/signup",

    ]
  }
}
```


**2.4.3.2.** 컴포넌트 상태 관리  

1: 모달 상태 (modalState)  
　A. orderSignup: boolean  

2: 이용 상태 (orderStatus)  
　A. isUsing: boolean  
　B. usageDays: number  


**2.4.3.3.** 접근성 요구사항  

1: 섹션 ARIA  
　A. role="region"  
　B. aria-labelledby="order-status-title"  

2: 이용 상태 ARIA  
　A. 이용 중: aria-label="위드폼 오더 이용 중"  
　B. 미이용: aria-label="위드폼 오더 미이용"
