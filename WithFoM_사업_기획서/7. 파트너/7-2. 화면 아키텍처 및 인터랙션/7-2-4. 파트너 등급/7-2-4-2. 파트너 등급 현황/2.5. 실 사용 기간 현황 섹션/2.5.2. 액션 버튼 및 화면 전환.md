**2.5.2.** 액션 버튼 및 화면 전환  


**2.5.2.1.** 진행률 프로그래스 바 클릭  

1: 클릭 대상  
　A. 등급 산정 기간 프로그래스 바  

2: 동작  
　A. 기간 상세 정보 드롭다운 확장  
　B. 캠페인별 참여 기간 타임라인 표시  
　C. 확장 애니메이션: 부드러운 높이 변경 (250ms)  


**2.5.2.2.** 캠페인 참여 내역 보기  

1: 클릭 대상  
　A. "캠페인 참여 내역" 링크  

2: 이동 목적지  
　A. 캠페인 관리 > 캠페인 발주 페이지로 이동  
　B. 캠페인 현황 섹션으로 자동 스크롤  

3: 화면 전환 애니메이션  
　A. 좌측에서 우측으로 슬라이드 전환 (300ms)  


**2.5.2.3.** 등급 부여 예정일 확인  

1: 클릭 대상  
　A. "등급 부여 예정일" 영역  

2: 동작  
　A. 등급 부여 조건 충족 현황 모달 표시  


**2.5.2.4.** 등급 부여 조건 모달  

1: 표시 조건  
　A. 실 사용 기간 섹션에서 등급 부여 예정일 클릭 시  

2: 모달 구성  
　A. 모달 제목: "등급 부여 조건 현황"  
　B. 조건별 충족 여부 체크리스트  
　　a. 보고율 조건: ✓ 충족 / ✗ 미충족  
　　b. 스캔율 조건: ✓ 충족 / ✗ 미충족  
　　c. 오더 이용 조건: ✓ 충족 / ✗ 미충족  
　　d. 사용 기간 조건: ✓ 충족 / ✗ 미충족  
　C. 예상 등급 부여일  
　D. 미충족 조건 개선 안내  
　E. 버튼: [닫기]  

3: 모달 닫기 동작  
　A. 외부 클릭 시 닫힘  
　B. X 버튼 클릭 시 닫힘  
　C. ESC 키 입력 시 닫힘  


**2.5.2.5.** 타임라인 상호작용  

1: 세그먼트 호버  
　A. 해당 기간 상세 정보 툴팁 표시  
　B. 캠페인명, 기간, 참여 일수 표시  

2: 세그먼트 클릭  
　A. 해당 캠페인 상세 정보 모달 표시
