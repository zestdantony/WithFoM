**1.1.3.2.** 화면 전환 및 토스트  

1: 화면 전환 애니메이션  
　A. 전환 방향: 좌측에서 우측으로 슬라이드  
　B. duration: 300ms  
　C. easing: ease-in-out  

2: 토스트 메시지  
　A. 표시 조건: 현재 등급 카드 접근 시  
　B. 스타일: 파란색 배경, 흰색 텍스트  
　C. 위치: 화면 하단 중앙  
　D. 자동 사라짐: 3000ms  
　E. 내용: "현재 등급: {gradeName}"  


**1.1.3.3.** 컴포넌트 상태 관리  

1: 현재 등급 (currentGrade)  
　A. grade: string  
　B. isHighlighted: boolean  
　C. showCurrentLabel: boolean ("현재 등급" 라벨 표시 여부)  

2: 토스트 상태 (toastState)  
　A. visible: boolean  
　B. message: string  
　C. autoHideMs: 3000  


**1.1.3.4.** 실시간 데이터 연동  

1: WebSocket 이벤트  
　A. grade.updated: 등급 변경 발생 시 현재 등급 카드 하이라이트 갱신  
　B. grade.benefit_applied: 혜택 적용 알림  

2: 연결 정보  
　A. 엔드포인트: wss://api.withfom.com/partner/grade/realtime  
　B. 인증: JWT 토큰 기반 연결  


**1.1.3.5.** 접근성 요구사항  

1: 카드 ARIA  
　A. role="listitem" (정보 표시용)  
　B. aria-label="{등급명} 등급 혜택 정보"  
　C. 현재 등급: aria-current="true"  

2: 키보드 내비게이션  
　A. Tab: 카드 간 이동 (정보 확인용)  
　B. "나의 등급 확인하기" 버튼: Enter/Space로 클릭  


**1.1.3.6.** 에러 처리  

1: 에러 코드  
　A. GRD_002: 혜택 조회 실패  

2: 사용자 메시지  
　A. "혜택 정보를 불러오지 못했습니다"  
　B. 재시도 버튼 표시  


**1.1.3.7.** 성능 최적화  

1: 캐싱  
　A. 로컬 스토리지에 7일 캐싱  
　B. 혜택 정책 변경 시 캐시 무효화
