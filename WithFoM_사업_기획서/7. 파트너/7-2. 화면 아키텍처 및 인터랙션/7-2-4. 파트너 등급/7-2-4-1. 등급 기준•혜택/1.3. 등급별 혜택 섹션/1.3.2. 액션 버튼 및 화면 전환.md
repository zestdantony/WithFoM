**1.3.2.** 액션 버튼 및 화면 전환  


**1.3.2.1.** 등급 카드 클릭  

1: 클릭 대상  
　A. 브론즈 / 실버 / 골드 / 플래티넘 혜택 카드  

2: 동작  
　A. 해당 등급 혜택 상세 모달 표시  
　B. 컵홀더 혜택, 멤버십 혜택, 인센티브 정보 상세 표시  


**1.3.2.2.** 등급 상세 모달  

1: 표시 조건  
　A. 등급 카드 클릭 시  

2: 모달 구성  
　A. 모달 제목: "{등급명} 등급 혜택"  
　B. 등급 마크 아이콘 및 색상  
　C. 혜택 상세 리스트  
　D. 닫기 버튼  

3: 모달 닫기 동작  
　A. 모달 외부 클릭 시 닫힘  
　B. X 버튼 클릭 시 닫힘  
　C. ESC 키 입력 시 닫힘  

4: 애니메이션  
　A. 모달 등장: 페이드인 + 스케일업 (200ms)  
　B. 배경 딤 처리 (opacity: 0.5)  


**1.3.2.3.** 현재 등급 확인 버튼  

1: 클릭 대상  
　A. "나의 등급 확인하기" 버튼  

2: 이동 목적지  
　A. 파트너 등급 > 파트너 등급 현황 페이지로 이동  
　B. 나의 등급 섹션으로 자동 스크롤  

3: 화면 전환 애니메이션  
　A. 좌측에서 우측으로 슬라이드 전환 (300ms)  


**1.3.2.4.** 혜택 적용 확인 버튼  

1: 클릭 대상  
　A. "혜택 적용 확인" 버튼  

2: 동작  
　A. 현재 적용 중인 혜택 확인 모달 표시  
　B. 컵홀더 발주 시 할인율 표시  
　C. 멤버십 할인 현황 표시  
　D. 인센티브 정산 현황 표시 (플래티넘)  


**1.3.2.5.** 토스트 메시지  

1: 표시 조건  
　A. 현재 등급 카드 접근 시  

2: 토스트 내용  
　A. 알림 토스트 (파란색): "현재 등급: {등급명}"  
　B. 표시 위치: 화면 하단 중앙  
　C. 자동 사라짐: 3초  


• 참조: 7-2-4-2. 파트너 등급 현황
