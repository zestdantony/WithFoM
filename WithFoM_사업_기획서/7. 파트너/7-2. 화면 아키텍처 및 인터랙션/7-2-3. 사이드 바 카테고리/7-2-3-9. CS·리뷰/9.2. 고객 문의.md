**9.2.** 고객 문의  


**9.2.1.** 표시 항목  

**9.2.1.1.** 섹션 구성  
1: 문의 목록(섹션)  
2: 문의 상세(섹션)  
3: 답변 작성(섹션)  


**9.2.2.** 문의 목록  

**9.2.2.1.** 검색 및 필터  
1: 기간 필터 (오늘/1주일/1개월/전체)  
2: 문의 유형 필터 (전체/리워드 관련/주문 관련)  
3: 답변 상태 필터 (전체/답변 완료/답변 대기)  
4: 닉네임/내용 검색  

**9.2.2.2.** 문의 카드 표시 정보  
1: 소비자 닉네임  
2: 문의 유형 (리워드/주문)  
3: 문의 내용 (미리보기)  
4: 접수일시  
5: 답변 상태 (완료/대기)  

**9.2.2.3.** 문의 카드 클릭 시  
1: 문의 상세 섹션에 상세 내용 표시  
2: 답변 작성 영역 활성화  


**9.2.3.** 문의 상세  

**9.2.3.1.** 표시 정보  
1: 소비자 닉네임  
2: 문의 유형  
3: 문의 전체 내용  
4: 첨부 파일 (있는 경우)  
5: 접수일시  
6: 관련 정보 (주문번호, 리워드명 등)  


**9.2.4.** 답변 작성  

**9.2.4.1.** 답변 입력  
1: 답변 내용 입력 (최대 1000자)  
2: 등록 버튼  
3: 임시저장 버튼  
4: 취소 버튼  

**9.2.4.2.** 답변 등록 완료  
1: "답변이 등록되었습니다" 메시지 표시  
2: 문의 목록에서 해당 문의 답변 상태 업데이트  
3: 소비자에게 답변 등록 알림 발송  


**9.2.5.** 문의 유형별 분류  

**9.2.5.1.** 리워드 관련 문의  
1: 리워드 사용 거절  
2: 리워드 미적용  
3: 리워드 유효기간  
4: 기타 리워드 문의  

**9.2.5.2.** 주문 관련 문의 (오더형/복합형만)  
1: 주문 취소 요청  
2: 주문 변경 요청  
3: 준비 시간 문의  
4: 주문 누락  
5: 기타 주문 문의  


• 참조: 7-4-3. 소비자 → 파트너 문의
