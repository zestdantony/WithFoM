**9.1.** 리뷰 관리  


**9.1.1.** 표시 항목  

**9.1.1.1.** 보드 구성  
1: 리뷰 목록(보드)  
2: 리뷰 상세(보드)  
3: 답글 작성(보드)  


**9.1.2.** 리뷰 목록  

**9.1.2.1.** 검색 및 필터  
1: 기간 필터 (오늘/1주일/1개월/전체)  
2: 평점 필터 (전체/5점/4점/3점/2점/1점)  
3: 답글 상태 필터 (전체/답글 완료/답글 미작성)  
4: 닉네임/내용 검색  

**9.1.2.2.** 리뷰 카드 표시 정보  
1: 소비자 닉네임  
2: 평점 (별점)  
3: 리뷰 내용 (미리보기)  
4: 작성일시  
5: 답글 상태 (완료/미작성)  

**9.1.2.3.** 리뷰 카드 클릭 시  
1: 리뷰 상세 보드에 상세 내용 표시  
2: 답글 작성 영역 활성화  


**9.1.3.** 리뷰 상세  

**9.1.3.1.** 표시 정보  
1: 소비자 닉네임  
2: 평점 (별점)  
3: 리뷰 전체 내용  
4: 첨부 이미지 (있는 경우)  
5: 작성일시  
6: 관련 주문 정보 (주문번호, 메뉴 등)  


**9.1.4.** 답글 작성  

**9.1.4.1.** 답글 입력  
1: 답글 내용 입력 (최대 500자)  
2: 등록 버튼  
3: 취소 버튼  

**9.1.4.2.** 답글 등록 완료  
1: "답글이 등록되었습니다" 메시지 표시  
2: 리뷰 목록에서 해당 리뷰 답글 상태 업데이트  
3: 소비자에게 답글 등록 알림 발송  


**9.1.5.** 리뷰 신고  

**9.1.5.1.** 신고 버튼 클릭 시  
1: 신고 사유 선택 모달 표시  
2: 신고 사유: 욕설/비방, 허위 사실, 광고성 내용, 개인정보 노출, 기타  
3: 기타 선택 시 상세 내용 입력 필수  

**9.1.5.2.** 신고 접수 완료  
1: "신고가 접수되었습니다" 메시지 표시  
2: 관리자에게 신고 내역 전달  
3: 검토 후 결과 알림 발송  


• 참조: 7-4-4. 리뷰 관리
