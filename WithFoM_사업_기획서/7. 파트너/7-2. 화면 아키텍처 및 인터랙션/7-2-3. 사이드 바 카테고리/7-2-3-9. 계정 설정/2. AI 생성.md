　　**3-9.2.** AI 생성  

　A. 지도 아이콘 AI 생성  
　　a. 대상  
　　aa. 플러스 및 프리미엄 멤버십 파트너만 사용 가능  
　　b. 멤버십 등급별 생성 옵션  
　　aa. 플러스  
　　• 카페 로고만 아이콘으로 생성  
　　• 월 1회 재생성 가능  
　　bb. 프리미엄  
　　• 카페 로고 + 메인 이벤트 표시 아이콘 생성  
　　• 월 3회 재생성 가능  
　　c. 생성 방식  
　　aa. AI가 매장 로고·키워드·상권 정보 기반으로 스타일이 다르게 표현된 아이콘 핀 3종 자동 생성  
　　bb. 3종 중 1개 선택 적용  
　　d. 재생성 제한 표시  
　　aa. 화면에 "이번 달 남은 재생성 횟수: N회" 표시  
　B. 매장 포스터 AI 생성 (프리미엄 전용)  
　　a. 대상  
　　aa. 프리미엄 멤버십 파트너만 사용 가능  
　　b. 기능  
　　aa. AI 툴을 사용하여 매장 포스터 이미지 생성  
　　bb. 매장 정보, 이벤트 내용, 브랜드 컬러 등을 반영한 포스터 자동 디자인  
　　c. 활용  
　　aa. 매장 내 게시용  
　　bb. SNS 홍보용
