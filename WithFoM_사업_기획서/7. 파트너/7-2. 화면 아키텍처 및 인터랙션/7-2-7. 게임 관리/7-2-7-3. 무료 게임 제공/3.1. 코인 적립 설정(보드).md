**3.1.** 코인 적립 설정(보드)  


**3.1.1.** 적립 설정  

**3.1.1.1.** 적립 수량 입력  
　1: 게임에 적립할 폼 코인 수량 입력  
　2: 보유 코인 범위 내에서만 입력 가능  
　3: 예: "500개 적립 → 500회 무료 게임 제공 가능"  

**3.1.1.2.** 적립 버튼  
　1: [적립하기] 버튼 클릭  
　2: 확인 모달: "폼 코인 [N]개를 게임에 적립하시겠습니까?"  


**3.1.2.** 적립 현황  

**3.1.2.1.** 표시 항목  
　1: 현재 적립된 코인 수량  
　2: 남은 제공 가능 게임 횟수  
　3: 오늘 소모된 코인 수량  

**3.1.2.2.** 부족 알림  
　1: 적립 코인 100개 이하 시 경고 표시  
　2: "코인이 부족합니다. 충전하세요!" + [충전] 버튼  


**3.1.3.** 자동 충전 설정 (선택)  

**3.1.3.1.** 자동 충전 ON/OFF  
　1: 적립 코인이 N개 이하로 떨어지면 자동 구매  
　2: 자동 구매 수량 설정 (1,000개 단위)  

**3.1.3.2.** 결제 수단 지정  
　1: 자동 결제에 사용할 카드 선택  


• 참조: 위드폼 코인 시스템 정책 정의서
