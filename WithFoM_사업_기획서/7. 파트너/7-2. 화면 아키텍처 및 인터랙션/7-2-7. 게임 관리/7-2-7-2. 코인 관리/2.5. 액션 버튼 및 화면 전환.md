**2.5.** 액션 버튼 및 화면 전환  


**2.5.1.** 코인 현황 섹션 액션  


**2.5.1.1.** 코인 관리 버튼  

1: [코인 구매] 버튼  
　A. 동작: 코인 구매 모달(2.2.) 오픈  

2: [게임에 적립] 버튼  
　A. 동작: 무료 게임 제공 섹션(2.4.)으로 스크롤 이동  


**2.5.2.** 코인 구매 모달 액션  


**2.5.2.1.** 모달 내 버튼  

1: 수량 선택 버튼(1,000 / 2,000 / 5,000 / 10,000)  
　A. 동작: 해당 수량 선택 및 결제 금액 갱신  

2: [결제하기] 버튼  
　A. 동작: PG사 결제 프로세스 호출  
　B. 성공: 코인 충전 완료 메시지 + 모달 닫기 + 현황 갱신  
　C. 실패: 오류 메시지 표시  

3: [닫기] 버튼  
　A. 동작: 모달 닫기  


**2.5.3.** 무료 게임 제공 섹션 액션  


**2.5.3.1.** 적립 관련 버튼  

1: [적립하기] 버튼  
　A. 동작: 적립 확인 모달 표시  
　B. 확인 시: 코인 적립 처리 + 현황 갱신  

2: [충전] 버튼(부족 시)  
　A. 동작: 코인 구매 모달(2.2.) 오픈  


**2.5.3.2.** 설정 저장 버튼  

1: [저장] 버튼  
　A. 동작: 대상·한도·시간 설정 저장 후 확인 메시지 표시  


**2.5.4.** 코인 내역 섹션 액션  


**2.5.4.1.** 내역 상세 버튼  

1: 구매 건 클릭 시  
　A. [증빙 보기] 버튼: 결제 증빙 조회  
　B. [환불 요청] 버튼: 환불 확인 모달 표시(조건 충족 시)  


**2.5.5.** 화면 전환 흐름  


**2.5.5.1.** 주요 전환 경로  

1: 사이드바 > 게임 관리 > 코인 관리  
　A. 기본: 코인 현황(2.1.) 표시  

2: 코인 현황 → 코인 구매  
　A. [코인 구매] 클릭 → 모달 오픈  

3: 코인 현황 → 코인 적립  
　A. [게임에 적립] 클릭 → 2.4. 무료 게임 제공 섹션으로 스크롤  

4: 부족 알림 → 코인 구매  
　A. [충전] 클릭 → 모달 오픈  


• 참조: 2.1. 코인 현황 섹션, 2.2. 코인 구매 모달, 2.3. 코인 내역 섹션, 2.4. 무료 게임 제공 섹션
