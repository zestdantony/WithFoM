**2.4.** 무료 게임 제공 섹션  


**2.4.1.** 코인 적립 설정  


**2.4.1.1.** 적립 수량 입력  

1: 입력 방법  
　A. 게임에 적립할 폼 코인 수량 입력  
　B. 보유 코인 범위 내에서만 입력 가능  
　C. 예: "500개 적립 → 500회 무료 게임 제공 가능"  


**2.4.1.2.** 적립 버튼  

1: 적립 처리  
　A. [적립하기] 버튼 클릭  
　B. 확인 모달: "폼 코인 [N]개를 게임에 적립하시겠습니까?"  


**2.4.2.** 적립 현황  


**2.4.2.1.** 표시 항목  

1: 현황 정보  
　A. 현재 적립된 코인 수량  
　B. 남은 제공 가능 게임 횟수(= 적립 코인 ÷ 1)  
　C. 오늘 소모된 코인 수량  


**2.4.2.2.** 부족 알림  

1: 경고 표시  
　A. 적립 코인 100개 이하 시 경고 표시  
　B. "코인이 부족합니다. 충전하세요!" + [충전] 버튼(클릭 시 2.2. 구매 모달 오픈)  


**2.4.3.** 자동 충전 설정(선택)  


**2.4.3.1.** 자동 충전 ON/OFF  

1: 자동 충전 조건  
　A. 적립 코인이 N개 이하로 떨어지면 자동 구매  
　B. 자동 구매 수량 설정(1,000개 단위)  


**2.4.3.2.** 결제 수단 지정  

1: 자동 결제 카드  
　A. 자동 결제에 사용할 카드 선택  


**2.4.3.3.** 자동 충전 안전장치  

1: 자동 충전 실패 시 동작  
　A. 1차 실패: 1시간 후 재시도  
　B. 2차 실패: 24시간 후 재시도 + 파트너 알림(푸시+이메일)  
　C. 3차 실패: 자동 충전 일시 중지 + "결제 수단을 확인하세요" 경고 배너  
　D. 중지 후 파트너가 결제 수단 변경 또는 수동 충전 시 자동 충전 재활성화  

2: 카드 만료 예정 알림  
　A. 자동 충전 등록 카드 유효기간 30일 전 알림: "자동 충전 카드가 곧 만료됩니다"  

3: 월 자동 충전 한도  
　A. 월 최대 자동 충전 횟수: 10회  
　B. 10회 초과 시 자동 충전 일시 중지 + 알림  


**2.4.4.** 제공 대상 설정  


**2.4.4.1.** 대상 고객 선택  

1: 체크박스 옵션  
　A. □ 모든 고객  
　B. □ 신규 고객(첫 방문)  
　C. □ 단골 고객(단골 스토어 등록)  
　D. □ VIP 고객(N회 이상 방문 또는 N원 이상 결제)  
　E. □ 휴면 고객(30일 이상 미방문)  


**2.4.4.2.** 복수 선택  

1: 선택 규칙  
　A. 여러 대상 동시 선택 가능  
　B. "모든 고객" 선택 시 다른 옵션 비활성화  


**2.4.5.** VIP 기준 설정  


**2.4.5.1.** VIP 조건 입력  

1: 조건 설정  
　A. 방문 횟수 기준: N회 이상  
　B. 결제 금액 기준: N원 이상  
　C. 둘 중 하나 충족 시 VIP  


**2.4.6.** 일일 제공 한도  


**2.4.6.1.** 전체 한도  

1: 한도 설정  
　A. 하루 최대 제공 횟수 설정  
　B. 예: "하루 최대 100회"  
　C. 한도 도달 시 무료 게임 중단  


**2.4.6.2.** 고객 1인당 한도  

1: 1인 한도  
　A. 고정: 하루 1회(변경 불가)  
　B. 1회 초과 이용 시 고객 본인 코인 사용  


**2.4.7.** 제공 시간 설정  


**2.4.7.1.** 시간 옵션  

1: 시간 설정  
　A. 24시간(기본)  
　B. 특정 시간대(예: 14:00~18:00)  
　C. 영업 시간 연동  


**2.4.7.2.** 시간 외 동작  

1: 시간 외 처리  
　A. 설정 시간 외에는 무료 게임 미제공  
　B. 고객 본인 코인으로는 이용 가능  


• 참조: 2.1. 코인 현황 섹션, 2.2. 코인 구매 모달, 7-2-7-1. 게임 설정
