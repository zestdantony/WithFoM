**2.1.** 코인 현황 섹션  


**2.1.1.** 보유 코인 표시  


**2.1.1.1.** 코인 잔액  

1: 잔액 표시  
　A. 총 보유 폼 코인 수량  
　B. 원화 환산 가치(예: "1,500개 = 15,000원 상당")  


**2.1.1.2.** 코인 분류 표시  

1: 분류별 수량  
　A. 게임 적립 코인: 파트너 자체 게임(매장당 1개)에 적립되어 고객 제공용으로 사용 중인 코인  
　B. 사용 가능 코인: 아직 적립/사용되지 않은 가용 코인  
　C. 계산식: 사용 가능 = 보유 − 적립  


**2.1.1.3.** 이번 달 증정 코인  

1: 멤버십 플랜별 증정  
　A. 플러스: 200개 / 프리미엄: 500개  
　B. 베이직: "플러스 이상 가입 시 매달 코인 증정!" 안내 + 업그레이드 링크  


**2.1.2.** 버튼 영역  


**2.1.2.1.** 코인 구매 버튼  

1: [코인 구매]  
　A. 클릭 시 구매 모달(2.2.) 오픈  


**2.1.2.2.** 게임 적립 버튼  

1: [게임에 적립]  
　A. 클릭 시 무료 게임 제공 섹션(2.4.)으로 스크롤 이동  


**2.1.3.** 유효기간 안내  


**2.1.3.1.** 만료 예정 코인 알림  

1: 경고 표시  
　A. 30일 내 만료 예정 코인이 있으면 경고 표시  
　B. 예: "⚠️ 7일 내 만료 예정: 100개"  

2: 유효기간 정책  
　A. 구매한 폼 코인의 유효기간은 구매일로부터 1년  
　B. 유효기간 만료 전 알림 발송(30일 전, 7일 전)  


**2.1.3.2.** 코인 유효기간과 게임 적립 관계  

1: 게임에 적립된 코인의 유효기간  
　A. 구매 시점의 유효기간(1년) 기준 적용  
　B. 게임에 적립 = 사용 예약 상태이므로 만료 대상에서 제외하지 않음  
　C. 적립 코인 만료 30일 전: "게임에 적립된 코인 N개가 곧 만료됩니다" 경고  

2: 적립 코인 만료 시 동작  
　A. 만료된 코인만큼 게임 적립 잔량 감소  
　B. 잔량 0 도달 시 무료 게임 자동 중단 + 파트너 알림  
　C. 새로 구매한 코인으로 재적립 필요  

3: 멤버십 다운그레이드 시 코인 처리  
　A. 보유 코인은 유지(멤버십 변경과 무관)  
　B. 월 증정 코인은 변경된 플랜 기준 적용(프리미엄→플러스: 500→200)  
　C. 베이직 다운그레이드 시 월 증정 중단, 기존 잔여 코인은 유효기간까지 사용 가능
