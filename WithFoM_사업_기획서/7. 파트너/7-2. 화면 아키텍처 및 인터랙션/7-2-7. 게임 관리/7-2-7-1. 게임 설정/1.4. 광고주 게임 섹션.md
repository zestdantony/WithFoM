**1.4.** 광고주 게임 섹션  


**1.4.1.** 섹션 개요  


**1.4.1.1.** 섹션 역할  

1: 광고주 캠페인 게임의 7등 슬롯에 파트너 리워드 참여 여부 관리  
2: 7등 리워드 풀(최대 6개) 구성 및 확률 설정  
3: 광고주 게임용 프리셋 저장·불러오기  
4: 연결된 광고주 캠페인 정보 읽기 전용 표시  


**1.4.2.** 7등 리워드 참여 토글  


**1.4.2.1.** 토글 기능  

1: ON/OFF 스위치  
　A. ON(기본값): 파트너 리워드가 광고주 게임 7등 슬롯에 삽입  
　B. OFF: 파트너 리워드 미삽입 → 광고주 예비 7등 리워드 또는 꽝으로 대체  
2: 토글 변경 시 즉시 서버 반영(별도 저장 버튼 불필요)  
3: OFF 전환 시 확인 모달  
　A. "7등 리워드 참여를 해제하면 광고주 게임에서 파트너 리워드가 제공되지 않습니다. 해제하시겠습니까?"  
　B. 해제 후에도 7등 리워드 풀·프리셋은 유지(재활성화 시 즉시 복원)  

**1.4.2.2.** 토글과 캠페인 관계  

1: 토글은 모든 연결 캠페인에 일괄 적용  
2: 캠페인 개별 ON/OFF는 지원하지 않음  
3: 광고주 캠페인 자체의 노출·진행은 파트너 토글과 무관(QR 스캔 시 항상 운영)  


**1.4.3.** 연결 캠페인 목록  


**1.4.3.1.** 읽기 전용 캠페인 카드  

1: 현재 진행 중인 광고주 캠페인 목록 표시  
　A. 캠페인명  
　B. 광고주명  
　C. 진행 기간  
　D. 상태(진행 중/예정/종료)  
2: 상세 정보는 표시하지 않음(광고주 게임 설정은 광고주 플랫폼에서 관리)  
3: 캠페인이 없을 경우: "현재 연결된 광고주 캠페인이 없습니다" 안내  


**1.4.4.** 7등 리워드 풀 관리  


**1.4.4.1.** 리워드 풀 구성  

1: 파트너가 최대 6개 리워드로 7등 리워드 풀 구성  
2: 저장 라이브러리(7-2-8-3 > 3.1.7.)에서 리워드 선택하여 추가  
　A. 추가 시 독립 복사본 생성(라이브러리 원본과 별개)  
3: 각 리워드에 독립 확률(%) 설정  
4: 꽝 없음(7등 당첨 시 100% 파트너 리워드 당첨)  
5: 확률 합계 = 100% 필수  

**1.4.4.2.** 7등 당첨 메커니즘  

1: 소비자가 광고주 게임에서 7등 당첨 시  
2: 서브 추첨 진행: 파트너 리워드 풀(최대 6개) 중 확률 기반 결정  
3: 당첨 결과 소비자에게 표시(파트너 리워드 안내 문구)  
4: 참여 OFF 시: 서브 추첨 미진행, 광고주 예비 리워드/꽝 표시  

**1.4.4.3.** 리워드 편집  

1: 등수 개념 없음(7등 리워드 풀 내에서 확률만 적용)  
2: 각 리워드 카드 표시 항목  
　A. 리워드명  
　B. 유형(쿠폰/포인트) 아이콘  
　C. 당첨 확률(%) — 인라인 수정  
　D. 유효기간 표시  
　E. [제거] 버튼  
3: [반영] 버튼 활성화 조건: 리워드 1개 이상 + 확률 합계 100% (미달 시 비활성화)  
4: [반영] 버튼 클릭 시 7등 리워드 풀 즉시 서버 저장  


**1.4.5.** 광고주 게임 프리셋 관리  


**1.4.5.1.** 프리셋 저장  

1: [현재 7등 구성을 프리셋으로 저장] 버튼  
2: 저장 항목  
　A. 각 리워드의 sourceRewardId  
　B. 확률(%)  
3: 절대 유효기간 리워드 프리셋 제외  
　A. 절대 유효기간 리워드는 프리셋 저장에서 자동 제외 + 안내 표시  
4: 저장 한도: 베이직 1개, 플러스 2개, 프리미엄 4개 (파트너 게임 프리셋과 별도 관리)  

**1.4.5.2.** 프리셋 불러오기  

1: 저장된 프리셋 목록 표시  
2: [적용] 클릭 시 sourceRewardId 기반으로 라이브러리 확인 → 새 복사본 생성 → 풀에 적용  
3: 불완전 프리셋 처리(삭제된 리워드 포함 시 경고 + 유효 항목만 적용)  

**1.4.5.3.** 프리셋 삭제  

1: [삭제] 버튼 → 확인 모달 → 삭제  


**1.4.6.** 멤버십 다운그레이드 시  


**1.4.6.1.** 광고주 게임 프리셋 잠금  

1: 모든 광고주 게임 프리셋 잠김 상태 전환  
2: 새 플랜 한도에 맞게 유지할 프리셋 선택 안내  
3: 선택 완료 후 미선택 프리셋 영구 삭제  


• 참조: 1.0. 페이지 개요 > 1.0.1.2. 게임 유형 구분, 6-3-1. 10단계 캠페인 생성 위자드 > 8단계 7등 예비 리워드 설정, 7-2-8-3 > 3.1.7. 리워드 저장 라이브러리
