**1.5.** 액션 버튼 및 화면 전환  


**1.5.1.** 파트너 게임 탭 액션  


**1.5.1.1.** 게임 생성 섹션(1.1.) 액션  

1: [게임 만들기] 버튼  
　A. 표시 조건: 게임 미생성 상태  
　B. 활성화 조건: 저장 라이브러리에 리워드 1개 이상 존재  
　C. 비활성화 시 안내: "먼저 리워드를 생성해 주세요" + [리워드 설정으로 이동] 링크  
　D. 동작: 게임 생성·수정 화면(1.2.) Step 1로 이동  

2: [유형 변경] 버튼  
　A. 표시 조건: 게임 생성 완료 상태  
　B. 활성화 조건: 잔여 변경 횟수가 1회 이상 남아있을 때 활성화, 0회일 때 비활성화  
　C. 비활성화 시 툴팁: "이번 달 유형 변경 횟수를 모두 사용했습니다"  
　D. 동작: 확인 모달("유형이 변경됩니다. 리워드 구성은 유지됩니다.") → 1.2. Step 1로 이동  
　E. 기존 리워드 복사본은 새 게임에 자동 이관  

3: [설정 수정] 버튼  
　A. 표시 조건: 게임 생성 완료 상태  
　B. 동작: 1.2. Step 2(게임 이름 설정)로 이동  

4: [삭제] 버튼  
　A. 표시 조건: 게임 생성 완료 상태  
　B. 동작: 삭제 확인 모달 → 삭제 → 1.1.1. 미생성 상태로 전환  
　C. 저장된 프리셋·라이브러리 리워드는 유지  


**1.5.1.2.** 게임 생성·수정 화면(1.2.) 액션  

1: Step 1 [다음] 버튼  
　A. 활성화 조건: 게임 유형 선택 완료  
　B. 동작: Step 2로 이동  

2: Step 2 [다음] 버튼  
　A. 활성화 조건: 게임 이름 입력 완료  
　B. 동작: Step 3로 이동  

3: Step 3 [게임 저장] 버튼  
　A. 활성화 조건: 유형 선택 + 리워드+꽝 합계 최소 2개 이상 적용 + 확률 합계 100%  
　B. 동작: 게임 생성/수정 완료 → 1.1. 섹션으로 복귀 + 성공 토스트  

4: [취소] 버튼(전 Step 공통)  
　A. 동작: 변경 사항 확인 모달 → 미저장 시 1.1.로 복귀  

5: [뒤로] 버튼(Step 2, 3)  
　A. 동작: 이전 Step으로 이동(입력 데이터 유지)  


**1.5.1.3.** 리워드 적용·프리셋 관리 섹션(1.3.) 액션  

1: [리워드 설정으로 이동] 링크 버튼  
　A. 표시 조건: 적용 리워드 0개일 때  
　B. 동작: 7-2-8-3 리워드 설정 페이지로 네비게이션  

2: 저장 라이브러리 패널 리워드 [추가]  
　A. 동작: 선택 리워드의 독립 복사본 생성 → 적용 목록 하단 추가  
　B. 비활성화 조건: 적용 리워드 6개 도달 시  

3: 적용 리워드 [제거] 버튼  
　A. 동작: 확인 모달 → 해당 복사본 삭제  
　B. 최소 수량 조건 미달 시 제거 차단  

4: [게임에 반영] 버튼  
　A. 활성화 조건: 리워드+꽝 합계 최소 2개 이상 + 확률 합계 100%  
　B. 동작: 현재 편집 상태를 라이브 게임에 즉시 반영 + 성공 토스트  

5: [리워드 보충] 버튼  
　A. 동작: 보충 총량 입력 → 확률 비례 분배 미리보기 → [보충 확정]  

6: [현재 구성을 프리셋으로 저장] 버튼  
　A. 동작: 프리셋명 입력 모달 → 저장  
　B. 비활성화 조건: 프리셋 한도 초과 시  

7: 프리셋 [적용] 버튼  
　A. 동작: 프리셋 리워드 구성으로 편집 영역 교체  
　B. 불완전 프리셋: 경고 표시 + 유효 항목만 적용  

8: 프리셋 [삭제] 버튼  
　A. 동작: 확인 모달 → 프리셋 삭제  


**1.5.2.** 광고주 게임 탭 액션  


**1.5.2.1.** 7등 참여 토글(1.4.) 액션  

1: 토글 ON→OFF 전환  
　A. 동작: 확인 모달 → 서버 즉시 반영  
　B. 7등 리워드 풀·프리셋 데이터 유지(재활성화 시 복원)  

2: 토글 OFF→ON 전환  
　A. 동작: 즉시 서버 반영(모달 없음)  
　B. 기존 7등 리워드 풀 구성 즉시 활성화  


**1.5.2.2.** 7등 리워드 풀(1.4.) 액션  

1: 저장 라이브러리에서 리워드 [추가]  
　A. 동작: 독립 복사본 생성 → 풀에 추가  
　B. 비활성화 조건: 풀 6개 도달 시  

2: 리워드 [제거] 버튼  
　A. 동작: 확인 모달 → 풀에서 삭제  

3: [반영] 버튼  
　A. 활성화 조건: 리워드 1개 이상 + 확률 합계 100%  
　B. 동작: 7등 리워드 풀 서버 저장 + 성공 토스트  

4: [현재 7등 구성을 프리셋으로 저장] 버튼  
　A. 동작: 프리셋명 입력 모달 → 저장  

5: 프리셋 [적용] / [삭제] 버튼  
　A. 파트너 게임 프리셋과 동일한 동작 방식  


**1.5.3.** 탭 전환 액션  


**1.5.3.1.** 탭 전환  

1: "파트너 게임" 탭 클릭 → 1.1.~1.3. 섹션 표시  
2: "광고주 게임" 탭 클릭 → 1.4. 섹션 표시  
3: 탭 전환 시 미저장 변경사항이 있으면 확인 모달 표시  
　A. "저장하지 않은 변경사항이 있습니다. 탭을 전환하시겠습니까?"  
　B. [저장 후 전환] / [저장하지 않고 전환] / [취소]  


**1.5.4.** 외부 연결  


**1.5.4.1.** 리워드 설정에서의 진입  

1: 7-2-8-3 > 3.1.6. [게임 적용] 완료 모달  
　A. [게임 설정으로 이동] 버튼 → 본 페이지 파트너 게임 탭 > 1.3. 섹션으로 포커스 이동  
　B. [여기서 계속하기] 버튼 → 리워드 설정 페이지 유지
