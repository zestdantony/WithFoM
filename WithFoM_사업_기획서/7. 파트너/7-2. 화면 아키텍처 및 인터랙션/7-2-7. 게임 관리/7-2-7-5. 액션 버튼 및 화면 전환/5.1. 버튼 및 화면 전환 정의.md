**5.1.** 버튼 및 화면 전환 정의  


**5.1.1.** 게임 설정 화면 액션  

**5.1.1.1.** 게임 목록 섹션  
　1: [+ 새 게임 만들기] 버튼  
　　A. 동작: 게임 생성 화면으로 이동  
　　B. 조건: 게임 3개 미만일 때만 활성화  
　2: [설정] 버튼  
　　A. 동작: 해당 게임의 상세 설정 화면으로 이동  
　3: [삭제] 버튼  
　　A. 동작: 삭제 확인 모달 표시  
　　B. 확인 시: 게임 삭제 및 목록 갱신  
　4: 활성화 토글  
　　A. ON: 게임 활성화 (리워드 활성화 필수)  
　　B. OFF: 게임 비활성화  

**5.1.1.2.** 게임 생성 화면  
　1: 게임 유형 카드 클릭  
　　A. 동작: 해당 유형 선택 상태로 변경  
　2: [다음] 버튼  
　　A. 동작: 상세 설정 화면으로 이동  
　　B. 조건: 게임 유형 선택 시 활성화  
　3: [저장] 버튼  
　　A. 동작: 게임 생성 완료 후 목록으로 이동  
　　B. 조건: 필수 설정 완료 시 활성화  
　4: [취소] 버튼  
　　A. 동작: 게임 목록으로 이동 (저장 없음)  


**5.1.2.** 코인 관리 화면 액션  

**5.1.2.1.** 코인 현황 섹션  
　1: [코인 구매] 버튼  
　　A. 동작: 코인 구매 모달 오픈  
　2: [게임에 적립] 버튼  
　　A. 동작: 무료 게임 제공 > 코인 적립 설정 화면으로 이동  

**5.1.2.2.** 코인 구매 모달  
　1: 수량 선택 버튼 (1,000 / 2,000 / 5,000 / 10,000)  
　　A. 동작: 해당 수량 선택 및 결제 금액 갱신  
　2: [결제하기] 버튼  
　　A. 동작: PG사 결제 프로세스 호출  
　　B. 성공: 코인 충전 완료 메시지 + 모달 닫기  
　　C. 실패: 오류 메시지 표시  
　3: [닫기] 버튼  
　　A. 동작: 모달 닫기  


**5.1.3.** 무료 게임 제공 화면 액션  

**5.1.3.1.** 코인 적립 설정 섹션  
　1: [적립하기] 버튼  
　　A. 동작: 적립 확인 모달 표시  
　　B. 확인 시: 코인 적립 처리  
　2: [충전] 버튼 (부족 시)  
　　A. 동작: 코인 구매 모달 오픈  

**5.1.3.2.** 제공 대상 설정 섹션  
　1: 대상 체크박스  
　　A. 동작: 선택 상태 토글  
　2: [저장] 버튼  
　　A. 동작: 설정 저장 후 확인 메시지 표시  


**5.1.4.** 화면 전환 흐름  

**5.1.4.1.** 주요 화면 전환  
　1: 사이드바 > 게임 관리  
　　A. 기본: 게임 설정 > 게임 목록 화면  
　2: 게임 목록 → 게임 생성  
　　A. [+ 새 게임 만들기] 클릭  
　3: 게임 목록 → 게임 수정  
　　A. 게임 카드의 [설정] 클릭  
　4: 코인 현황 → 코인 구매  
　　A. [코인 구매] 클릭 → 모달 오픈  
　5: 코인 현황 → 코인 적립  
　　A. [게임에 적립] 클릭 → 무료 게임 제공 화면 이동
