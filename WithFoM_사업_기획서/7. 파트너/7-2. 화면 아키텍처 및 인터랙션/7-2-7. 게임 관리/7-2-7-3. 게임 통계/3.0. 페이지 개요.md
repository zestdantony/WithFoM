**3.0.** 페이지 개요  


**3.0.1.** 페이지 목적  


**3.0.1.1.** 페이지 정의  

1: 페이지 명칭  
　A. 게임 통계 페이지  
　B. 게임 관리 카테고리 내 세 번째 하위 메뉴  

2: 페이지 역할  
　A. 게임 참여 현황 모니터링 (오늘/기간별 이용 횟수, 유료·무료 구분)  
　B. 리워드 당첨 분석 (설정 확률 대비 실제 당첨 비율 검증)  
　C. 리워드 소진 현황 관리 (잔여 수량, 예상 소진 시점, 보충 이력)  
　D. 광고주 캠페인 게임 참여 현황 표시 (QR 참여, 7등 파트너 리워드)  


**3.0.2.** 핵심 기능  


**3.0.2.1.** 게임 참여 현황  

1: 오늘 요약 카드(총 참여·유료·무료·당첨) + 기간별 참여 추이 차트  


**3.0.2.2.** 리워드 당첨·소진 현황  

1: 리워드별 당첨 분석(도넛 차트 + 확률 비교 테이블) + 소진 프로그레스 바 + 예상 소진 시점  


**3.0.2.3.** 광고주 게임 현황  

1: QR 참여 횟수 + 7등 파트너 리워드 당첨·사용 건수 (광고주 게임 ON 시에만 표시)  


**3.0.3.** 진입 경로  


**3.0.3.1.** 사이드바에서 진입  

1: 사이드바 네비게이션  
　A. 사이드바 > 게임 관리 > 게임 통계  
　B. 게임 참여 현황 섹션(3.1.)부터 표시  


• 참조: 7-2-3-4. 게임 관리 > 4.3. 게임 통계, 7-2-7-1. 게임 설정 > 1.3. 리워드 적용·프리셋 관리, 7-2-8-1. 현황
