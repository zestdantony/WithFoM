**3.3.** 액션 버튼 및 화면 전환  


**3.3.1.** 게임 통계 페이지 액션  


**3.3.1.1.** 기간 필터  

1: 기간 선택 변경  
　A. 동작: 오늘/이번 주/이번 달/직접 선택 전환 시 페이지 전체 통계 데이터 갱신  
　B. 기간 변경 시 3.1. 참여 현황 + 3.2. 리워드 현황 모두 동시 갱신  


**3.3.1.2.** 리워드 보충 바로가기  

1: [리워드 보충] 버튼  
　A. 위치: 3.2.2. 소진 현황 영역 하단  
　B. 동작: 게임 설정(7-2-7-1) > 1.3. 리워드 적용·프리셋 관리 섹션으로 이동  
　C. 리워드 보충이 필요한 상황에서 바로 조치 가능  


**3.3.1.3.** 게임 설정 바로가기  

1: [게임 설정] 링크  
　A. 동작: 게임 설정(7-2-7-1) 게임 생성 섹션으로 이동  
　B. 확률 조정·게임 수정 등 운영 조치 유도  


**3.3.1.4.** 리워드 현황 바로가기  

1: [리워드 현황] 링크  
　A. 동작: 리워드·스탬프 현황(7-2-8-1)으로 이동  
　B. 게임을 통해 발행된 리워드의 사용 현황·사용률·사용 금액 등 상세 정보 확인 유도  
　C. 게임 통계에서는 당첨·소진 중심으로 표시하고, 사용 현황은 리워드 현황 페이지에서 확인  


**3.3.1.5.** 매장 관리 바로가기  

1: [상세 분석 보기] 링크  
　A. 동작: 매장 관리(7-2-6-3) 현황 탭으로 이동  
　B. 게임 관련 경영 분석 (참여율 추이·재방문 효과·고객 분포) 확인 유도  


**3.3.2.** 화면 전환 흐름  


**3.3.2.1.** 주요 전환 경로  

1: 사이드바 > 게임 관리 > 게임 통계  
　A. 기본: 게임 참여 현황(3.1.) 표시  

2: 게임 통계 → 게임 설정  
　A. [게임 설정] 클릭 → 7-2-7-1 게임 생성 섹션  

3: 게임 통계 → 리워드 보충  
　A. [리워드 보충] 클릭 → 7-2-7-1 > 1.3. 리워드 적용·프리셋 관리 섹션  

4: 게임 통계 → 리워드 현황  
　A. [리워드 현황] 클릭 → 7-2-8-1 현황 페이지  

5: 게임 통계 → 매장 관리  
　A. [상세 분석 보기] 클릭 → 7-2-6-3 매장 관리 현황 탭  


• 참조: 3.0. 페이지 개요, 3.1. 게임 참여 현황, 3.2. 리워드 당첨·소진 현황, 7-2-7-1. 게임 설정, 7-2-8-1. 현황, 7-2-6-3. 매장 관리
