**2.3.** 상품 연동(보드)  


**2.3.1.** 리워드 연동  

**2.3.1.1.** 연동 가능 리워드  
　1: 할인 쿠폰 (기존 생성된 쿠폰)  
　2: 포인트 쿠폰 (500원, 1,000원 등)  
　3: 사이즈업/추가 증정 쿠폰  
　4: 스탬프 추가 적립 (+1개, +2개)  

**2.3.1.2.** 리워드 선택 UI  
　1: 기존 리워드 목록에서 선택  
　2: [+ 새 리워드 만들기] 버튼으로 즉시 생성  


**2.3.2.** 직접 입력 상품  

**2.3.2.1.** 커스텀 상품  
　1: 상품명 직접 입력 (최대 15자)  
　2: 아이콘/이미지 선택 또는 업로드  
　3: 설명 텍스트 (선택)  

**2.3.2.2.** 폼 코인 상품  
　1: 파트너 전용 폼 코인을 상품으로 설정  
　2: 예: "폼 코인 5개 당첨!"  
　3: 해당 매장에서만 사용 가능한 코인 지급  


**2.3.3.** 꽝 설정  

**2.3.3.1.** 꽝 메시지  
　1: 기본: "다음 기회에!"  
　2: 커스텀 메시지 입력 가능  

**2.3.3.2.** 꽝 대체 옵션  
　1: 꽝 대신 소액 보상 (예: 폼 코인 1개)  
　2: 100% 당첨 모드 선택 시 꽝 없음  


• 참조: 위드폼 코인 시스템 정책 정의서
