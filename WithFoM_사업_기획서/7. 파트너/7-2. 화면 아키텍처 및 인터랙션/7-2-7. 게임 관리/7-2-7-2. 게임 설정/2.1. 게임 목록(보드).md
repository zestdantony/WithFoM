**2.1.** 게임 목록(보드)  


**2.1.1.** 보드 구성  

**2.1.1.1.** 생성된 게임 리스트  
　1: 게임 이름  
　2: 게임 유형 (룰렛 / 뽑기 / 스크래치)  
　3: 활성화 상태 (ON/OFF 토글)  
　4: 적립된 코인 수량  
　5: 오늘 이용 횟수  

**2.1.1.2.** 게임 카드  
　1: 각 게임을 카드 형태로 표시  
　2: 미리보기 썸네일  
　3: [설정] / [삭제] 버튼  


**2.1.2.** 게임 추가  

**2.1.2.1.** 추가 버튼  
　1: [+ 새 게임 만들기] 버튼  
　2: 최대 3개까지 생성 가능  
　3: 3개 초과 시 버튼 비활성화 + 안내 메시지  


**2.1.3.** 활성화 토글  

**2.1.3.1.** ON/OFF 동작  
　1: ON: 고객이 해당 게임 이용 가능  
　2: OFF: 고객에게 노출되지 않음  
　3: 적립된 코인은 유지됨  


• 참조: 위드폼 코인 시스템 정책 정의서
