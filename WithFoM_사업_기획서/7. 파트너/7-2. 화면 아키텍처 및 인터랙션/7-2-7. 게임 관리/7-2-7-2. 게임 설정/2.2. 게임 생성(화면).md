**2.2.** 게임 생성(화면)  


**2.2.1.** 게임 유형 선택  

**2.2.1.1.** 유형 카드  
　1: 룰렛: 회전 룰렛 미리보기 + 설명  
　2: 뽑기: 캡슐 뽑기 미리보기 + 설명  
　3: 스크래치: 긁기 복권 미리보기 + 설명  

**2.2.1.2.** 선택 시 동작  
　1: 카드 클릭 시 선택 상태 표시  
　2: [다음] 버튼으로 상세 설정 진행  


**2.2.2.** 룰렛 설정  

**2.2.2.1.** 칸 개수 선택  
　1: 6칸 / 8칸 / 12칸 중 선택  

**2.2.2.2.** 각 칸 설정  
　1: 상품 선택 (리워드 연동 또는 직접 입력)  
　2: 당첨 확률 (%) 입력  
　3: 색상/아이콘 설정  

**2.2.2.3.** 실시간 미리보기  
　1: 설정 변경 시 우측에 미리보기 표시  
　2: 테스트 스핀 기능  


**2.2.3.** 뽑기 설정  

**2.2.3.1.** 캡슐 개수 선택  
　1: 9개 (3×3) / 16개 (4×4) / 25개 (5×5) 중 선택  

**2.2.3.2.** 상품 풀 설정  
　1: 상품 목록 추가  
　2: 각 상품별 등장 확률 설정  
　3: 캡슐 색상/디자인 선택  


**2.2.4.** 스크래치 설정  

**2.2.4.1.** 당첨 조건  
　1: 같은 그림 3개 일치 시 당첨  

**2.2.4.2.** 상품 설정  
　1: 당첨 상품 목록 추가  
　2: 각 상품별 당첨 확률 설정  


**2.2.5.** 확률 검증  

**2.2.5.1.** 자동 검증  
　1: 모든 확률 합계 = 100% 검증  
　2: 불일치 시 경고 및 조정 안내  

**2.2.5.2.** 저장  
　1: [저장] 버튼으로 게임 생성 완료  
　2: 생성 후 게임 목록에 추가
