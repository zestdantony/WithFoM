**2.2.** 게임 목록(섹션)  


**2.2.1.** 섹션 구성  

**2.2.1.1.** 생성된 게임 리스트  
　1: 게임 카드 표시 항목  
　　A. 게임 이름  
　　B. 게임 유형 (룰렛 / 뽑기 / 스크래치 / 사다리)  
　　C. 활성화 상태 (ON/OFF 토글)  
　　D. 적립된 코인 수량  
　　E. 오늘 이용 횟수  

**2.2.1.2.** 게임 카드 UI  
　1: 카드 형태 표시  
　　A. 각 게임을 카드 형태로 표시  
　　B. 미리보기 썸네일 포함  
　　C. [설정] / [삭제] 버튼  


**2.2.2.** 게임 추가  

**2.2.2.1.** 추가 버튼  
　1: 신규 게임 생성  
　　A. [+ 새 게임 만들기] 버튼  
　　B. 최대 3개까지 생성 가능  
　　C. 3개 초과 시 버튼 비활성화 + "최대 3개까지 생성 가능합니다" 안내  


**2.2.3.** 활성화 토글  

**2.2.3.1.** ON/OFF 동작  
　1: ON 상태  
　　A. 고객이 해당 게임 이용 가능  
　　B. 리워드 활성화 상태에서만 ON 가능  
　2: OFF 상태  
　　A. 고객에게 노출되지 않음  
　　B. 적립된 코인은 유지됨  

**2.2.3.2.** 리워드 연동 안내  
　1: 안내 문구 표시  
　　A. "게임 활성화는 리워드 활성화 시 자동 적용됩니다"  
　　B. 리워드 설정 페이지로 이동 링크 제공  


• 참조: 7-1-2-2-3. 게임 설정
