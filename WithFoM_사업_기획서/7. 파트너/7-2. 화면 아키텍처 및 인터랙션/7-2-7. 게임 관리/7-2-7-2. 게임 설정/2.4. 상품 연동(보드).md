**2.4.** 상품 연동(보드)  


**2.4.1.** 리워드 연동  

**2.4.1.1.** 연동 가능 리워드  
　1: 연동 가능 항목  
　　A. 할인 쿠폰 (기존 생성된 쿠폰)  
　　B. 포인트 쿠폰 (500원, 1,000원 등)  
　　C. 사이즈업/추가 증정 쿠폰  
　　D. 스탬프 추가 적립 (+1개, +2개)  

**2.4.1.2.** 리워드 선택 UI  
　1: 선택 방법  
　　A. 기존 리워드 목록에서 선택  
　　B. [+ 새 리워드 만들기] 버튼으로 즉시 생성  


**2.4.2.** 직접 입력 상품  

**2.4.2.1.** 커스텀 상품  
　1: 입력 항목  
　　A. 상품명 직접 입력 (최대 15자)  
　　B. 아이콘/이미지 선택 또는 업로드 (권장: 100×100px)  
　　C. 설명 텍스트 (선택)  

**2.4.2.2.** 폼 코인 상품  
　1: 코인 상품 설정  
　　A. 파트너 전용 폼 코인을 상품으로 설정  
　　B. 예: "폼 코인 5개 당첨!"  
　　C. 해당 매장에서만 사용 가능한 코인 지급  


**2.4.3.** 꽝 설정  

**2.4.3.1.** 꽝 메시지  
　1: 메시지 설정  
　　A. 기본: "다음 기회에!"  
　　B. 커스텀 메시지 입력 가능  

**2.4.3.2.** 꽝 대체 옵션  
　1: 대체 옵션  
　　A. 꽝 대신 소액 보상 (예: 폼 코인 1개)  
　　B. 100% 당첨 모드 선택 시 꽝 없음  


**2.4.4.** 확률 설정  

**2.4.4.1.** 확률 입력 방식  
　1: 입력 방법  
　　A. 각 상품별 당첨 확률(%) 직접 입력  
　　B. 전체 합계가 100%가 되도록 자동 검증  
　　C. 예: 대박 5%, 중박 15%, 소박 30%, 꽝 50%  

**2.4.4.2.** 확률 프리셋  
　1: 프리셋 옵션  
　　A. "100% 당첨" 프리셋 (꽝 없음)  
　　B. "일반" 프리셋 (꽝 포함)  
　　C. "고가치" 프리셋 (높은 당첨 확률)  

**2.4.4.3.** 수량 제한 (선택)  
　1: 제한 설정  
　　A. 특정 상품의 총 당첨 수량 제한 설정  
　　B. 예: "아메리카노 무료 쿠폰" 1일 최대 10개  
　　C. 수량 소진 시 해당 칸은 꽝으로 대체  


• 참조: 7-1-2-2-3. 게임 설정
