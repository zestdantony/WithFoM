**2.3.** 게임 생성 화면  


**2.3.1.** 게임 유형 선택  

**2.3.1.1.** 유형 카드  
　1: 룰렛  
　　A. 회전 룰렛 미리보기 + 설명  
　　B. 회전하는 룰렛을 돌려 당첨 칸에 멈추면 상품 획득  
　2: 뽑기 (가챠)  
　　A. 캡슐 뽑기 미리보기 + 설명  
　　B. 여러 개의 캡슐/카드 중 하나를 선택하여 상품 획득  
　3: 스크래치  
　　A. 긁기 복권 미리보기 + 설명  
　　B. 3×3 그리드에서 같은 그림 3개가 나오면 당첨  
　4: 사다리  
　　A. 사다리 게임 미리보기 + 설명  
　　B. 선택한 시작점에서 사다리를 타고 내려가 도착점의 상품 획득  

**2.3.1.2.** 선택 시 동작  
　1: 카드 클릭  
　　A. 클릭 시 선택 상태 표시 (테두리 강조)  
　　B. [다음] 버튼으로 상세 설정 진행  


**2.3.2.** 룰렛 설정  

**2.3.2.1.** 칸 개수 선택  
　1: 선택 옵션  
　　A. 6칸 / 8칸 / 12칸 중 선택  

**2.3.2.2.** 각 칸 설정  
　1: 설정 항목  
　　A. 상품 선택 (리워드 연동 또는 직접 입력)  
　　B. 당첨 확률 (%) 입력  
　　C. 색상/아이콘 설정  

**2.3.2.3.** 실시간 미리보기  
　1: 미리보기 기능  
　　A. 설정 변경 시 우측에 미리보기 표시  
　　B. 테스트 스핀 기능 제공  


**2.3.3.** 뽑기 설정  

**2.3.3.1.** 캡슐 개수 선택  
　1: 선택 옵션  
　　A. 9개 (3×3) / 16개 (4×4) / 25개 (5×5) 중 선택  

**2.3.3.2.** 상품 풀 설정  
　1: 설정 항목  
　　A. 상품 목록 추가  
　　B. 각 상품별 등장 확률 설정  
　　C. 캡슐 색상/디자인 선택  


**2.3.4.** 스크래치 설정  

**2.3.4.1.** 당첨 조건  
　1: 조건  
　　A. 같은 그림 3개 일치 시 당첨  

**2.3.4.2.** 상품 설정  
　1: 설정 항목  
　　A. 당첨 상품 목록 추가  
　　B. 각 상품별 당첨 확률 설정  


**2.3.5.** 사다리 설정  

**2.3.5.1.** 사다리 구성  
　1: 시작점 개수  
　　A. 4개 / 6개 / 8개 중 선택  
　2: 도착점 상품 설정  
　　A. 각 도착점에 상품 또는 꽝 설정  
　　B. 사다리 경로는 랜덤 생성  

**2.3.5.2.** 상품 배치  
　1: 배치 방식  
　　A. 각 도착점에 상품 직접 배치  
　　B. 확률 기반 랜덤 배치 옵션  


**2.3.6.** 확률 검증  

**2.3.6.1.** 자동 검증  
　1: 검증 조건  
　　A. 모든 확률 합계 = 100% 검증  
　　B. 불일치 시 경고 및 조정 안내  

**2.3.6.2.** 저장  
　1: 저장 처리  
　　A. [저장] 버튼으로 게임 생성 완료  
　　B. 생성 후 게임 목록에 추가  
　　C. 리워드 미활성화 시 저장만 되고 활성화는 보류
