**3.3.** 코인 내역(보드)  


**3.3.1.** 내역 리스트  

**3.3.1.1.** 표시 항목  
　1: 리스트 컬럼  
　　A. 날짜/시간  
　　B. 구분 (구매 / 증정 / 적립 / 소모 / 만료)  
　　C. 수량 (+/- 표시)  
　　D. 잔액  
　　E. 상세 내용 (예: "멤버십 증정", "게임 적립", "고객 이용")  

**3.3.1.2.** 필터  
　1: 필터 옵션  
　　A. 기간: 오늘 / 이번 주 / 이번 달 / 기간 직접 선택  
　　B. 유형: 전체 / 구매 / 증정 / 적립 / 소모 / 만료  


**3.3.2.** 통계 요약  

**3.3.2.1.** 월별 통계  
　1: 이번 달 요약  
　　A. 이번 달 구매: N개  
　　B. 이번 달 증정: N개  
　　C. 이번 달 소모: N개  
　　D. 이번 달 만료: N개
