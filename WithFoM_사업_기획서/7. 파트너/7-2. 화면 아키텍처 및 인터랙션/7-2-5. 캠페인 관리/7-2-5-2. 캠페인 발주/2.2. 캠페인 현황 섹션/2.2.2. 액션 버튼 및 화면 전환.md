**2.2.2.** 액션 버튼 및 화면 전환  


**2.2.2.1.** 드롭다운 선택 동작  

1: 캠페인 선택 시  
　A. 드롭다운에서 특정 회차 선택 시 해당 회차의 캠페인 내용으로 전환  
　B. 기본 정보, 입고율, 재고율, 평균 소진량 모두 선택 회차 기준으로 갱신  
　C. 선택 카드 라벨에 선택한 회차 표시  


**2.2.2.2.** 검색 버튼 동작  

1: 검색 실행 시  
　A. 년/월 입력 후 검색 버튼 클릭 시 해당 기간의 캠페인 결과 필터링  
　B. 검색 결과가 년, 월 선택 카드 아래에 목록으로 표시  
　C. 검색 결과에서 캠페인 선택 시 해당 회차 내용 표시  
　D. 검색 결과 없을 시 "해당 기간의 캠페인이 없습니다" 안내 표시  


• 참조: 7-1-5-2. 컵홀더 입고 확인
