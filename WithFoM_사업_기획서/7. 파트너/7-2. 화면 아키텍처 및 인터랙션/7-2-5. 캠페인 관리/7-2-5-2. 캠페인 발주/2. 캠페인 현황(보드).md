# 2: 캠페인 현황(보드)  

　A. 좌측 위에 드롭다운 되는 선택 박스  
　　a. 선택 박스에 캠페인이라고 표시  
　　b. 클릭 시 최근 회차부터 모든 회차가 드롭다운 되어 표시  
　　c. 드롭다운 시 보여지는 것은 10개  
　　d. 더 아래로 스크롤 하여 이전 회차 검색 가능  
　　e. 선택 시 선택 회차 내용 표시  
　B. 최근 회차 표시(검색 시 검색된 회차 표시)  
　C. 선택 박스 우측으로 년, 월 선택 박스와 검색 버튼  
　　a. 캠페인을 년 또는 년, 월로 검색 가능  
　D. 년, 월 선택 박스 아래로 검색된 캠페인 표시  
　　a. 선택 시 선택 회차 내용 표시  
　E. N회차 표시 우측으로 캠페인당 총 컵홀더 수량 표시  
　F. 컵홀더 수 우측으로 캠페인 기간 표시  
　G. 기간 우측으로 컵홀더 유형 표시  
　H. 입고율을 프로그래스 바로 표시  
　　a. 총 컵홀더 수량 표시  
　　b. 입고 횟수 표시  
　　c. 입고당 수량 표시  
　　d. 예상 캠페인 종료일 표시  
　I. N회차 입고분의 재고율을 프로그래스 바로 표시  
　　a. 입고된 수량 표시  
　　b. 입력한 소진량 표시  
　　c. 재고 수량 표시  
　　d. 예상 사용 기간 표시  
　J. 하루 평균 소진량 표시(결과치)  
　　a. 입고 일과 추가 발주를 한 날을 계산  
　　b. 입고 수량과 추가 발주 시 재고량 계산  
　　↳ 재고량은 임계치를 얼마로 설정 했느냐로 판단  
　　c. 이 조건들로 계산해서 하루 소진량을 계산  
　　d. 추가 발주 할 때마다 하루 평균 소진량 업데이트.  
　　↳ 관련 모듈에 자동 반영  
　　e. 소진량 업데이트에 대한 설명 표시  
　　↳ '입고·임계치·발주 시점을 분석해 AI가 추정한 소진량입니다.'
