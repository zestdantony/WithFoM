**2.1.1.** 구성 및 표시 정보  


**2.1.1.1.** 캠페인 매칭 규칙  

1: 단일 캠페인 원칙  

　1: 동시 참여 제한  
　　A. 파트너 계정당 동시에 하나의 캠페인만 참여 가능  
　　B. 복수 캠페인 동시 참여 불가  
　　C. 현재 캠페인 종료 전까지 새로운 캠페인 매칭 불가  

　2: 캠페인 연속 매칭  
　　A. 현재 캠페인 종료 1주일 전부터 다음 캠페인 매칭 가능  
　　B. 다음 캠페인 확정 시 현재 캠페인 종료 익일부터 자동 전환  
　　C. 캠페인 간 공백 없이 연속 진행 가능  

2: 매칭 대기 상태  

　1: 매칭 대기 중 표시  
　　A. 현재 캠페인 종료 후 ~ 다음 캠페인 매칭 전까지  
　　B. "캠페인 매칭 대기 중입니다" 안내 표시  
　　C. 다음 캠페인 예정일이 있으면 함께 표시  

　2: 매칭 확정 시  
　　A. 매칭 대기 상태 종료  
　　B. 일반 캠페인 진행 상태로 전환  


**2.1.1.2.** 매칭 시 표시 항목  

1: 매칭 알림  
　A. 매칭 시, '캠페인이 매칭 되었습니다.  
지정된 일정에 맞춰 컵홀더 배포를 시작해 주세요.' 메세지 표시  

2: 캠페인 정보  
　A. 파트너의 N회차 캠페인 넘버 표시  
　B. 매칭 된 컵홀더 갯수 표시  
　C. 캠페인 기간 표시  
　D. 컵홀더 유형 표시  

3: 확인/포기 버튼 표시  
　A. 확인/포기 버튼 생성  

4: 응답 기한 및 자동 처리  
　A. 응답 기한: 매칭 알림 수신 후 익일 15시까지  
　　a. 응답 기한 카운트다운 타이머 표시 (남은 시간 실시간 표시)  
　B. 리마인드 알림: 기한 2시간 전 자동 발송 (필수)  
　　a. 알림 메시지: '캠페인 매칭 응답 기한이 2시간 남았습니다.  
참여를 확정하시려면 지금 확인해 주세요.  
미응답 시 자동 포기 처리됩니다.'  
　　b. 발송 채널: 푸시 알림 + 앱 내 알림 (필수)  
　C. 미응답 시 자동 포기 처리  
　　a. 응답 기한(익일 15시) 경과 시 시스템이 자동으로 포기 처리  
　　b. 자동 포기 사유: "TIMEOUT" (응답 기한 초과)으로 기록  
　　c. 자동 포기 알림: '캠페인 매칭 응답 기한이 초과되어  
자동으로 포기 처리되었습니다.  
다음 캠페인 매칭을 기다려 주세요.'  
　　d. 해당 캠페인 재매칭 불가  
　　e. 자동 포기도 포기 횟수에 포함  

5: 캠페인 포기 정책
　A. 캠페인 참여는 필수이며, 포기 시 포기 횟수가 차감됨
　B. 포기 가능 횟수: 총 3회 (매칭 전 포기 + 참여 중 포기 + 자동 포기 모두 포함)
　C. 포기 횟수별 조치
　　a. 1~2회: 포기 처리 + 남은 포기 횟수 알림 발송
　　　aa. 알림 메시지: '캠페인 포기가 처리되었습니다. 남은 포기 횟수: N회'
　　b. 3회(최종): 포기 처리 + 계약 정지 (기존 이용제한 체계의 "일시 정지"와 동일)
　　　aa. 정지 알림: '캠페인 포기가 3회에 도달하여 계약이 정지되었습니다.
상세 내용은 고객센터로 문의해 주세요.'
　D. 관리자 사유 검토
　　a. 포기 사유는 관리자가 사후 검토
　　b. 합당한 사유로 판단 시 포기 횟수 복원 + 파트너에게 복원 알림
　　c. 자동 포기(TIMEOUT)는 즉시 1회 차감, 파트너 사후 문의 시 관리자 검토 가능
　E. 포기 시 매칭 감점
　　a. 매칭 전 포기 / 자동 포기(TIMEOUT): 매칭 점수 -0.5점 감점
　　b. 참여 중 포기(중도 포기): 매칭 점수 -1.0점 감점
　　c. 감점은 다음 캠페인 매칭 우선순위에 반영
　F. 초기화 조건
　　a. 계약 정지 해제 시 포기 횟수 + 매칭 감점 모두 초기화
　　b. 포기 이력(사유, 일시, 관리자 검토 결과)은 영구 기록
　G. 포기 횟수 표시: 캠페인 참여 섹션 하단에 "포기 N/3회" 표시


**2.1.1.4.** 캠페인 참여 중 중도 포기 정책

1: 중도 포기 절차 (관리자 승인 필요)
　A. 참여 확정 후 캠페인 진행 중에도 중도 포기(참여 중지) 요청 가능
　B. "참여 중지" 버튼 클릭 시 사유 직접 입력 필수 (선택지 없음, 텍스트 직접 입력만 가능)
　C. 사유 입력 후 제출 시 관리자에게 중도 포기 요청이 전달됨 (즉시 포기 아님)
　D. 관리자가 요청을 검토하여 승인 또는 반려
　　a. 승인: 중도 포기 확정 → 후속 처리(감점, 보상 등) 진행
　　b. 반려: 요청 취소 → 파트너는 캠페인 계속 참여

2: 요청 중 상태
　A. 요청 제출 후 ~ 관리자 승인/반려 전까지 "참여 중지 요청중" 상태
　B. 요청 중에도 파트너는 캠페인에 계속 참여해야 함 (컵홀더 배포 및 소진 보고 유지)
　C. 요청 중에는 추가 참여 중지 요청 불가
　D. 요청 상태 표시: 캠페인 참여 섹션에 "참여 중지 요청중" 뱃지 표시

3: 승인 후 영향
　A. 포기 전까지 참여 기간은 등급 산정에 포함
　B. 승인 시점부터 미매칭 기간 카운트 시작
　C. 해당 캠페인 재참여 불가
　D. 포기 횟수 1회 차감 + 매칭 점수 -1.0점 감점 (매칭 전 포기 -0.5점보다 가중)
　E. 관리자가 승인 시 합당한 사유로 판단한 경우 포기 횟수 미차감 처리 가능

4: 남은 컵홀더 보상
　A. 승인 확정 시 남은 컵홀더에 대한 비용을 파트너가 위드폼에 지불
　B. 남은 컵홀더 수량 = 입고된 수량 - 파트너가 보고한 소진량
　C. 개당 가격 = 승인 시점 위드폼 협력사 기준가
　D. 보상 금액 = 남은 컵홀더 수량 × 개당 기준가
　E. 정산 시점: 관리자 승인 확정 즉시 정산

5: 중도 포기 사유
　A. 직접 입력만 가능 (매칭 전 포기의 선택지 카테고리와 다름)
　B. 텍스트 입력 필수 (미입력 시 요청 버튼 비활성화)


**2.1.1.5.** 현재 적용 혜택 표시  

1: 등급별 혜택  
　A. 브론즈: "컵홀더 50% 할인가 적용" 표시  
　B. 실버: "컵홀더 무료 제공" 표시  
　C. 골드: "컵홀더 무료 제공 + 멤버십 30% 할인" 표시  
　D. 플래티넘: "컵홀더 무료 제공 + 멤버십 30% 할인 + 개당 10원 인센티브" 표시  
　E. 현재 등급 마크 아이콘과 함께 표시
