**2.1.2.** 액션 버튼 및 화면 전환  


**2.1.2.1.** 확인 버튼 동작  

1: 확인 버튼 클릭 시  
　A. 확인 버튼을 누르면, '축하합니다. 컵홀더 캠페인 매칭 완료!  
지정된 일정에 맞춰 컵홀더 배포를 시작하시고,  
소진 현황도 함께 등록해 주세요.' 메세지와 취소/확인 버튼이 팝업으로 생성  

2: 등급별 확정 처리  
　A. 실버 이상: [확인] 클릭 시 즉시 참여 확정  
　B. 브론즈: 결제 화면 이동 → 결제 완료 시 참여 확정  
　C. 브론즈 결제 기한: 익일 15시까지(응답 기한과 동일)  
　D. 미결제 시 자동 포기 처리, 재매칭 불가  


**2.1.2.2.** 참여 확정 완료 후 UI 전환  

1: 매칭 시점 UI → 참여 중 UI 전환  
　A. 확인/포기 버튼 영역이 사라지고, 참여 중 상태 UI로 전환  
　B. 전환 시 "캠페인 참여가 확정되었습니다!" 토스트 알림 표시 (3초간)  

2: 참여 중 상태 표시  
　A. 상태 뱃지: "참여중" (녹색 뱃지, 캠페인 정보 영역 우측 상단)  
　B. 캠페인 기본 정보 유지 표시 (회차, 수량, 기간, 유형)  
　C. 등급별 혜택 정보 유지 표시  

3: 참여 중지 버튼  
　A. 참여 중 상태에서 "참여 중지" 버튼 표시  
　B. 위치: 캠페인 정보 영역 하단 우측  
　C. 디자인: 빨간색 텍스트 버튼 (#FF3B30), 아웃라인 스타일  
　D. 동작: 2.1.2.4 참여 중지 버튼 동작 참조  

4: 후속 노출/활성화  
　A. 캠페인 현황 섹션에 해당 캠페인 정보 노출  
　B. 첫 입고 예정일/회차별 수량 등 입고 일정 안내(해당 시)  
　C. 캠페인 관련 알림 활성화(입고, 소진 보고, 재고 부족 등)  


**2.1.2.3.** 포기 버튼 동작

1: 포기 버튼 클릭 시
　A. 포기 버튼을 누르면 포기 사유 선택 팝업 생성
　B. 팝업 상단에 "포기 N/3회 (남은 N회)" 현황 표시
　C. 안내 문구: "캠페인 포기 시 포기 횟수가 차감됩니다. 총 3회 포기 시 계약이 정지됩니다."
　D. 포기 2회 사용 상태(남은 1회)인 경우 추가 경고: "이번 포기가 마지막입니다. 추가 포기 시 계약이 즉시 정지됩니다."

2: 포기 사유 선택
　A. 체크박스와 함께 포기 이유가 보기로 표시 (1개 필수 선택)
　　a. 컵홀더 규격(사이즈) 변경으로 호환이 불가능 합니다.
　　b. 임시 휴무/단축 운영(리뉴얼, 공사, 이전) 예정입니다.
　　　aa. 선택 시 아래에 입력창 생성
　　　bb. 입력창 안에 '재오픈 날짜를 입력해 주세요' 표시
　　c. 시즌/행사 운영(대관, 단체 주문)으로 캠페인 진행이 어렵습니다.
　　　aa. 선택 시 아래에 입력창 생성
　　　bb. 입력창 안에 '재오픈 날짜를 입력해 주세요' 표시
　　d. 평보다 방문객이 적어 소진 일정 충족이 어렵습니다.
　　e. 매장 운영 정책과 맞지 않습니다.(컵홀더 광고/QR 유도 등)
　　f. 기타(직접 입력)
　　　aa. 선택 시 아래에 입력창 생성
　　　bb. 입력창 안에 '직접 입력' 표시

3: 포기 사유 확인
　A. 하단에 취소/확인 버튼 생성
　B. 사유 미선택 시 확인 버튼 비활성화

4: 포기 완료 처리
　A. [확인] 클릭 시 캠페인 상태를 "포기됨"으로 변경
　B. 포기 사유(및 추가 입력 값)를 시스템에 기록
　C. 해당 캠페인 재매칭 불가
　D. 포기 횟수 +1 차감 + 매칭 점수 -0.5점 감점
　E. 포기 완료 알림: '캠페인 포기가 처리되었습니다.
남은 포기 횟수: N회. 포기 사유는 관리자 검토 후 최종 처리됩니다.'
　F. 3회째 포기인 경우 계약 정지 처리 + 정지 알림 발송


**2.1.2.4.** 참여 중지 버튼 동작 (캠페인 진행 중)

1: 참여 중지 버튼 표시 조건
　A. 캠페인 참여 확정 후 진행 중 상태에서만 표시
　B. 빨간색 텍스트 버튼: "참여 중지"
　C. "참여 중지 요청중" 상태에서는 버튼 비활성화 (회색 처리, "요청 처리중" 텍스트)

2: 참여 중지 클릭 시
　A. 확인 팝업 생성: '캠페인 참여 중지를 요청하시겠습니까?
요청은 관리자 검토 후 승인 시 확정됩니다.
승인 시 포기 횟수가 차감되고, 남은 컵홀더에 대한 보상금이 정산됩니다.
총 3회 포기 시 계약이 정지됩니다.'
　B. 팝업에 "포기 N/3회 (남은 N회)" 현황 표시
　C. 남은 컵홀더 보상 예상 안내: "남은 컵홀더 N개 × 개당 ₩OOO = 예상 보상금 ₩OOO (승인 시 정산)"
　D. 포기 2회 사용 상태인 경우 추가 경고: "이번 포기가 마지막입니다. 승인 시 계약이 정지됩니다."
　E. 취소/요청하기 버튼

3: 요청 확인 시 — 사유 입력
　A. 포기 사유 직접 입력 팝업 표시 (매칭 전 포기와 다름 — 선택지 없음)
　B. 텍스트 입력 필드: 플레이스홀더 "참여 중지 사유를 입력해 주세요"
　C. 사유 미입력 시 요청하기 버튼 비활성화

4: 요청 제출 완료
　A. 캠페인 상태를 "참여 중지 요청중"으로 변경
　B. 요청 제출 알림: '참여 중지 요청이 접수되었습니다.
관리자 검토 후 결과가 안내됩니다. 승인 전까지 캠페인 참여를 유지해 주세요.'
　C. 상태 뱃지: "참여중" → "참여 중지 요청중" (주황색 뱃지)으로 전환
　D. "참여 중지" 버튼 비활성화

5: 관리자 승인 시 — 중지 확정 처리
　A. 캠페인 상태를 "중도 포기"로 변경
　B. 승인 전까지 참여 기간은 등급 산정에 포함
　C. 승인 시점부터 미매칭 기간 카운트 시작
　D. 해당 캠페인 재참여 불가
　E. 포기 횟수 +1 차감 + 매칭 점수 -1.0점 감점 (매칭 전 포기 -0.5점보다 가중)
　　a. 관리자가 합당한 사유로 판단 시 포기 횟수 미차감 처리 가능
　F. 남은 컵홀더 보상금 정산 처리
　　a. 남은 수량 = 입고 수량 - 파트너 보고 소진량
　　b. 보상금 = 남은 수량 × 승인 시점 위드폼 협력사 기준가
　　c. 파트너가 위드폼에 지불
　G. 승인 완료 알림: '캠페인 참여 중지가 승인되었습니다.
남은 포기 횟수: N회. 보상금 ₩OOO이 정산됩니다.'
　H. 3회째 포기인 경우 계약 정지 처리 + 정지 알림 발송

6: 관리자 반려 시
　A. 캠페인 상태를 "참여중"으로 복원
　B. 반려 알림: '참여 중지 요청이 반려되었습니다.
사유: {반려 사유}. 캠페인 참여를 계속해 주세요.'
　C. "참여 중지" 버튼 재활성화
　D. 파트너는 캠페인 계속 참여
