**2.1.** 캠페인 참여(보드)  


↳ 광고주와 매칭 시 활성화  


**2.1.1.** 매칭 시 표시 항목  

**2.1.1.1.** 매칭 정보  

　1: 매칭 알림  
　　A. 매칭 시, '캠페인이 매칭 되었습니다.  
지정된 일정에 맞춰 컵홀더 배포를 시작해 주세요.' 메세지 표시  

　2: 캠페인 정보  
　　A. 파트너의 N회차 캠페인 넘버 표시  
　　B. 매칭 된 컵홀더 갯수 표시  
　　C. 캠페인 기간 표시  
　　D. 컵홀더 유형 표시  

　3: 확인/포기 버튼  
　　A. 확인/포기 버튼 생성  

　4: 응답 기한 및 자동 처리  
　　A. 응답 기한: 매칭 알림 수신 후 익일 15시까지  
　　B. 리마인드 알림: 기한 2시간 전 발송(선택 사항)  
　　C. 미응답 시 자동 포기 처리  


**2.1.2.** 확인 버튼 동작  

**2.1.2.1.** 확인 처리  

　1: 확인 버튼 클릭 시  
　　A. 확인 버튼을 누르면, '축하합니다. 컵홀더 캠페인 매칭 완료!  
지정된 일정에 맞춰 컵홀더 배포를 시작하시고,  
소진 현황도 함께 등록해 주세요.' 메세지와 취소/확인 버튼이 팝업으로 생성  

　2: 등급별 확정 처리  
　　A. 실버 이상: [확인] 클릭 시 즉시 참여 확정  
　　B. 브론즈: 결제 화면 이동 → 결제 완료 시 참여 확정  
　　C. 브론즈 결제 기한: 익일 15시까지(응답 기한과 동일)  
　　D. 미결제 시 자동 포기 처리, 재매칭 불가  


**2.1.2.2.** 참여 확정 완료 안내  
　1: 상태 변경  
　　A. 확인/포기 버튼 영역을 "참여 확정됨" 상태로 변경  
　2: 후속 노출/활성화  
　　A. 캠페인 현황(보드)에 해당 캠페인 정보 노출  
　　B. 첫 입고 예정일/회차별 수량 등 입고 일정 안내(해당 시)  
　　C. 캠페인 관련 알림 활성화(입고, 소진 보고, 재고 부족 등)  


**2.1.3.** 포기 버튼 동작  

**2.1.3.1.** 포기 처리  

　1: 포기 버튼 클릭 시  
　　A. 포기 버튼을 누르면, 포기 이유를 선택 하는 창 팝업으로 생성  

　2: 포기 사유 선택  
　　A. 체크박스와 함께 포기 이유가 보기로 표시  
　　　a. 컵홀더 규격(사이즈) 변경으로 호환이 불가능 합니다.  
　　　b. 임시 휴무/단축 운영(리뉴얼, 공사, 이전) 예정입니다.  
　　　　aa. 선택 시 아래에 입력창 생성  
　　　　bb. 입력창 안에 '재오픈 날짜를 입력해 주세요' 표시  
　　　c. 시즌/행사 운영(대관, 단체 주문)으로 캠페인 진행이 어렵습니다.  
　　　　aa. 선택 시 아래에 입력창 생성  
　　　　bb. 입력창 안에 '재오픈 날짜를 입력해 주세요' 표시  
　　　d. 평보다 방문객이 적어 소진 일정 충족이 어렵습니다.  
　　　e. 매장 운영 정과 맞지 않습니다.(컵홀더 광고/QR 유도 등)  
　　　f. 기타(직접 입력)  
　　　　aa. 선택 시 아래에 입력창 생성  
　　　　bb. 입력창 안에 '직접 입력' 표시  

　3: 포기 사유 확인  
　　A. 취소/확인 버튼 생성  

　4: 포기 완료 처리  
　　A. [확인] 클릭 시 캠페인 상태를 "포기됨"으로 변경  
　　B. 포기 사유(및 추가 입력 값)를 시스템에 기록  
　　C. 해당 캠페인 재매칭 불가  


• 참조: 7-1-4. 캠페인 매칭 확인 및 참여 확정(캠페인 발주)
