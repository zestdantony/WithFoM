**2.0.3.** 공통 노출 조건  


**2.0.3.1.** 기본 노출 조건  

1: 노출 대상  
　A. 캠페인 매칭 중이거나 참여 중인 파트너에게 노출  
　B. 매칭 대기 상태인 경우 "캠페인 매칭 대기 중입니다" 안내 표시  
　C. 캠페인 이력이 없는 신규 파트너는 빈 상태 안내 표시  


**2.0.3.2.** 데이터 갱신 조건  

1: 실시간 갱신  
　A. 캠페인 매칭 알림 수신 시 즉시 반영  
　B. 입고 완료 시 즉시 반영  
　C. 배포 시작/종료 상태 변경 시 즉시 반영  

2: 주기적 갱신  
　A. 재고율: 소진 보고 입력 시마다 업데이트  
　B. 하루 평균 소진량: 추가 발주 시마다 업데이트  
　C. 입고율: 입고 완료 처리 시 업데이트  


**2.0.3.3.** 반응형 레이아웃  

1: 디바이스별 레이아웃  
　A. 데스크톱(1024px 이상): 전체 레이아웃, 좌측 사이드바 + 우측 콘텐츠  
　B. 태블릿(768px~1023px): 사이드바 축소, 콘텐츠 영역 확장  
　C. 모바일(767px 이하): 단일 컬럼, 하단 네비게이션
