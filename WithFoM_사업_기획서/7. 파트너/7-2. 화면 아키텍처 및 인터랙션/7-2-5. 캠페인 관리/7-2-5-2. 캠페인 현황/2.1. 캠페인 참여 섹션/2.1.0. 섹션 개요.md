**2.1.0.** 섹션 개요  


**2.1.0.1.** 섹션 정의  

1: 섹션 명칭  
　A. 캠페인 참여 섹션  
　B. 캠페인 현황 페이지 내 첫 번째 섹션  

2: 섹션 목적  
　A. 광고주와 매칭된 캠페인의 참여 확정 또는 포기 처리  
　B. 캠페인 매칭 규칙 및 매칭 정보 안내  
　C. 참여 중인 캠페인의 상태 표시 및 중도 포기(참여 중지) 처리  
　D. 컵홀더 배송·입고 진행 상태 확인 및 배송완료 처리  

↳ 매칭 시점에는 승인/포기 선택 UI, 참여 확정 후에는 참여중 표시 + 배송·입고 상태 + 참여 중지 UI로 전환  


**2.1.0.2.** 섹션 구성  

1: 매칭 규칙 영역  
　A. 단일 캠페인 원칙 안내  
　B. 캠페인 연속 매칭 안내  
　C. 매칭 대기 상태 안내  

2: 매칭 정보 표시 영역
　A. 캠페인 참여 필수 라벨
　B. 매칭 알림 메시지
　C. 캠페인 기본 정보(회차, 수량, 기간, 유형)
　D. 응답 기한 및 자동 처리 안내
　E. 포기 횟수 현황 표시 (N/3회)

3: 매칭 시점 — 확인/포기 버튼 영역  
　A. 확인(승인) 버튼 및 확정 처리 팝업  
　B. 포기 버튼 및 사유 선택 팝업  

4: 참여 확정 후 — 참여중 상태 영역  
　A. "참여중" 상태 뱃지 표시  
　B. 참여 중지 버튼 및 중도 포기 처리  

5: 컵홀더 배송·입고 진행 상태 영역 (참여 확정 후 노출)  
　A. 현재 배송 단계 상태 버튼 표시 (주문접수 / 제작중 / 배송준비 / 배송중)  
　B. 배송중 단계에서 배송완료 버튼 표시 (파트너 수동 배송완료 처리)  
　C. 입고 완료 상태 표시 (배송완료 후 입고 → 배포 자동 시작)  
　D. 배송 문의 버튼 (배송 문제 시 7-2-11-2 발주 주문 상세로 이동)  


• 참조: 7-1-4. 캠페인 매칭 확인 및 참여 확정(캠페인 발주)
