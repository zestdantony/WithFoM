**3.1.** 배포 시작 일정 및 절차  


**3.1.1.** 캠페인 배포 일정 안내  

**3.1.1.1.** 배포 일정 정보  

　1: 배포 시작일 표시  
　　A. 캠페인 참여 확정 후 배포 시작일 안내  
　　B. 입고 예정일 및 배포 시작일 표시  
　　C. 캠페인 기간과 배포 일정 연동 표시  

　2: 배포 일정 확인 방법  
　　A. 캠페인 발주 화면에서 배포 일정 확인  
　　B. 캠페인 현황 섹션에서 입고 및 배포 일정 확인  
　　C. 알림을 통한 배포 시작일 리마인드  


**3.1.2.** 배포 시작 전 확인 사항  

**3.1.2.1.** 준비 사항 확인  

　1: 컵홀더 입고 확인  
　　A. 첫 입고 예정일 확인  
　　B. 입고 수량 확인  
　　C. 입고 완료 알림 수신 여부 확인  

　2: 매장 준비 사항  
　　A. 컵홀더 보관 장소 준비  
　　B. 배포 담당자 안내  
　　C. 소진 보고 방법 안내  


**3.1.3.** 배포 시작 절차  

**3.1.3.1.** 배포 시작 처리  

　1: 배포 시작 확인  
　　A. 입고 완료 확인 후 배포 시작 가능  
　　B. 배포 시작일 도달 시 배포 시작 버튼 활성화  
　　C. 배포 시작 버튼 클릭 시 배포 상태 변경  

　2: 배포 시작 안내 메시지  
　　A. 메시지:  
　　　a. '컵홀더 배포를 시작하시겠습니까?  
　　　b. 배포 시작 후 일일 소진 보고를 시작해 주세요.'  
　　B. 확인 버튼: 배포 시작 처리  


**3.1.4.** 배포 시작 후 안내  

**3.1.4.1.** 안내 사항  

　1: 배포 시작 완료 안내  
　　A. "배포가 시작되었습니다" 메시지 표시  
　　B. 일일 소진 보고 시작 안내  
　　C. 소진 보고 방법 가이드 링크 제공  

　2: 일일 소진 보고 활성화  
　　A. 배포 시작 후 일일 소진 보고 기능 활성화  
　　B. 매일 영업 종료 시점 소진량 입력 안내  
　　C. 소진 보고 미완료 시 알림 발송
