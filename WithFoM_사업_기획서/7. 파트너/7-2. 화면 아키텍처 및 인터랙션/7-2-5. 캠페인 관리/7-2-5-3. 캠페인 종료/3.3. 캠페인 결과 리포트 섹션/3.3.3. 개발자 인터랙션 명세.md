**3.3.3.2.** 리포트 다운로드 API  

1: 엔드포인트  
　A. GET /api/v1/partner/campaign/report/{campaignId}/download?format=pdf|excel  

2: 파라미터  
　A. format: pdf (요약 리포트, 전체 파트너) / excel (상세 데이터, 멤버십 플러스 이상)  


**3.3.3.3.** 성과 지표 색상 분기  

```javascript
const getConsumptionRateColor = (rate) => {
  if (rate >= 100) return '#22C55E';
  if (rate >= 80) return '#3B82F6';
  if (rate >= 50) return '#F59E0B';
  return '#EF4444';
};

const getScanRateColor = (rate) => {
  if (rate >= 115) return '#22C55E';
  if (rate >= 100) return '#3B82F6';
  if (rate >= 80) return '#F59E0B';
  return '#EF4444';
};
```


**3.3.3.4.** 권한별 렌더링 분기  

1: 상세 리포트 접근 제어  
　A. detailed.available === false 시 상세 리포트 영역에 "멤버십 플러스 이상에서 이용 가능합니다." 안내 표시  
　B. detailed.available === true 시 상세 리포트 전체 렌더링  


**3.3.3.5.** 접근성 요구사항  

1: ARIA 속성  
　A. 다운로드 버튼: aria-label="캠페인 결과 리포트 다운로드"  
　B. 차트 영역: 대체 텍스트로 주요 수치 요약 제공  


**3.3.3.6.** 에러 처리  

1: 에러 코드  
　A. RPT_001: 리포트 조회 실패  
　B. RPT_002: 다운로드 실패
