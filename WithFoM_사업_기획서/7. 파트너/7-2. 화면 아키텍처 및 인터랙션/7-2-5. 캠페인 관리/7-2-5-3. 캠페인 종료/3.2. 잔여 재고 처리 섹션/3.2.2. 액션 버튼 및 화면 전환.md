**3.2.2.** 액션 버튼 및 화면 전환  


**3.2.2.1.** 배포 지속 안내 (배포 종료 기한 없는 캠페인)  

1: 화면 표시  
　A. "잔여 컵홀더 N개 — 소진 시까지 배포를 계속해 주세요" 안내 메시지 표시  
　B. 일일 소진 보고 입력 상태 유지 (활성화)  
　C. 별도 처리 버튼 없음  

2: 소진 완료 시  
　A. 전량 소진 보고 완료 → 자동으로 "소진 완료" 상태 전환  
　B. 3.1 소진 완료 확인 섹션의 조기 소진 종료 흐름으로 전환  


**3.2.2.2.** 처리 방법 선택 (배포 종료 기한 도래 캠페인)  

1: 처리 방법 선택  
　A. 라디오 버튼 그룹으로 처리 방법 표시  
　　a. 폐기 처리 (기본 선택)  
　　b. 반품 처리 (광고주 요청 시에만 노출)  
　B. [처리 방법 확정] 버튼  

2: 처리 방법 확정 시 확인 다이얼로그  
　A. "선택한 처리 방법: {방법명}. 확정하시겠습니까?"  
　B. [확인] / [취소] 버튼  


**3.2.2.3.** 폐기 처리 확정 후  

1: 폐기 확인 다이얼로그  
　A. "잔여 컵홀더 N개 폐기를 진행합니다"  
　B. [폐기 완료] / [취소] 버튼  

2: 완료 후 화면  
　A. 처리 상태: "폐기 완료" 표시  
　B. "처리가 완료되었습니다. 감사합니다" 안내  


**3.2.2.4.** 반품 처리 확정 후  

1: 반송 정보 표시  
　A. 반송 주소: 위드폼 지정 창고 주소 표시  
　B. 배송비 안내: "배송비는 위드폼이 부담합니다"  
　C. 운송장 번호 입력 필드 표시  

2: 반품 발송 완료 처리  
　A. 운송장 번호 입력 후 [반품 발송 완료] 버튼 활성화  
　B. 클릭 시 확인 다이얼로그: "운송장 번호 {번호}로 반품 발송을 완료하시겠습니까?"  
　C. [확인] 클릭 → 처리 상태 "반품 발송 완료"로 변경  

3: 완료 후 화면  
　A. 처리 상태: "반품 발송 완료" 표시  
　B. 입력한 운송장 번호 표시  


**3.2.2.5.** 처리 완료 공통 동작  

1: 관리자 알림  
　A. 처리 방법 및 완료 시점 관리자에게 자동 알림 전송  

2: 처리 완료 후 변경  
　A. 처리 완료 후 변경이 필요하시면 고객센터로 문의해 주세요
