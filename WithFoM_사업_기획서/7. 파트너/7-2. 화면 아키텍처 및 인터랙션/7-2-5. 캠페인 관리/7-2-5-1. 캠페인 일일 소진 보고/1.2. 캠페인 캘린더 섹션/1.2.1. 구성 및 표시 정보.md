**1.2.1.** 구성 및 표시 정보  


**1.2.1.1.** 캘린더 기본 기능  

1: 캘린더 생성  
　A. 캠페인 기간 동안의 캘린더 생성  
　B. 월 단위로 캘린더 표시  
　C. 달력 넘겨가며 확인 가능  

2: 소진량 표시  
　A. 매일 입력한 컵홀더 소진량을 캘린더에 표시  
　B. 날짜별 소진량 숫자로 표시  
　C. 보고 완료/미완료 상태 색상으로 구분  


**1.2.1.2.** 캠페인 관련 내용 표시  

1: 캠페인 일정 표시  
　A. N번차 캠페인 매칭일 표시  
　B. N번차 캠페인 페이일 표시  
　C. N번차 캠페인의 N회차 입고량 표시  
　D. N번차 캠페인 종료일 표시  
　E. 플래티넘: N번차 캠페인의 인센티브 결산일 표시  

2: 종료일 표시 방식  
　A. 매칭시 정해진 종료시기 표시  
　B. 실제 컵홀더 소진일 표시  
　C. 두 종료일이 다를 경우 모두 표시

　A. 보고 완료 날짜: 그린 컬러로 표시
