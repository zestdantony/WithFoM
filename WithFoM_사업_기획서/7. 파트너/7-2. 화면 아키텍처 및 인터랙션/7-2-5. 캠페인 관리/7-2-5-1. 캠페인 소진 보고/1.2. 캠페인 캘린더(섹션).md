**1.2.** 캠페인 캘린더(섹션)  


**1.2.1.** 캘린더 기본 기능  

**1.2.1.1.** 기본 기능  

　1: 캘린더 생성  
　　A. 캠페인 기간 동안의 캘린더 생성  
　　B. 월 단위로 캘린더 표시  
　　C. 달력 넘겨가며 확인 가능  

　2: 소진량 표시  
　　A. 매일 입력한 컵홀더 소진량을 캘린더에 표시  
　　B. 날짜별 소진량 숫자로 표시  
　　C. 보고 완료/미완료 상태 색상으로 구분  


**1.2.2.** 캠페인 관련 내용 표시  

**1.2.2.1.** 캠페인 정보 표시  

　1: 캠페인 일정 표시  
　　A. N번차 캠페인 매칭일 표시  
　　B. N번차 캠페인 페이일 표시  
　　C. N번차 캠페인의 N회차 입고량 표시  
　　D. N번차 캠페인 종료일 표시  

　2: 종료일 표시 방식  
　　A. 매칭시 정해진 종료시기 표시  
　　B. 실제 컵홀더 소진일 표시  
　　C. 두 종료일이 다를 경우 모두 표시  


**1.2.3.** 상태 색상 표시  

**1.2.3.1.** 색상 구분  

　1: 보고 상태 색상  
　　A. 보고 완료 날짜: 그린 컬러로 표시  
　　B. 미보고 날짜: 빨간색으로 표시  
　　C. 대시섹션 연동 표시  


**1.2.4.** 캘린더에서 수정 기능  

**1.2.4.1.** 수정 기능  

　1: 수정 가능 조건  
　　A. 캘린더에서도 소진량 수정 가능  
　　B. 수정 가능 시간은 익일 오픈 후 2시간까지  
　　C. 수정 불가 시간에는 읽기 전용  

　2: 수정 방법  
　　A. 캘린더의 날짜 클릭 시 해당 날짜 소진량 수정 화면 표시  
　　B. 수정 가능 시간 내에만 수정 버튼 활성화  
　　C. 수정 결과는 대시섹션와 연동
