　　**5-1.1.** 컵홀더 소진 보고(보드)  

　A. 보드 제목 오른쪽 옆에 작은 박스로 완료/미완료 표시  
　B. 보드 색상  
　　a. 미완료: 빨간색  
　　b. 완료: 그린 컬러  
　C. 전일 소진량 표시  
　　a. 전일 소진량 옆에 수정 버튼 생성  
　　　aa. 수정 버튼 위에 '실수 입력/누락 시 수정 가능(익일 오픈 후 2시간까지)' 문구 표시  
　　　bb. 수정 가능 시간까지만 활성화  
　　　　　• 수정 가능 시간  
　　　　　• "익일 오픈 후 2시간"이 기준  
　　　　　• 기준은 매장별 설정값  
　　　　　• 수정 불가능 시간엔 팝업으로 이유 안내  
　　　　　• 팝업 멘트  
　　　　'소진량 수정 시간 종료  
　　　　소진량 수정은 익일 오픈 후 2시간까지만 수정 가능하며,  
　　　　이후에는 정산 정확도를 위해 변경할 수 없습니다.'  
　　　　cc. 수정 결과는 캘린더나 대시보드 등에 모두 연동  
　D. 오늘 소진량 입력창  
　　a. 입력창 안에 '오늘 소진량을 입력 하세요' 표시  
　　b. 입력창 우측에 제출 버튼  
　　　aa. 제출 후 입력값이 평균 대비 1.5배 이상 차이 나면 경고 메세지 팝업  
　　　　　• 팝업 멘트  
　　　'입력값이 평균값을 벗어났습니다.  
　　　맞는지 확인해 주세요.'  
　　　　　• 팝업창에 취소/확인 버튼 생성  
　　　bb. 입력값 제출 시 관련 모든 시스템에 연동
