**1.1.2.** 액션 버튼 및 화면 전환  


**1.1.2.1.** 수정 버튼 동작  

1: 수정 버튼 활성화 조건  
　A. 수정 가능 시간: 익일 오픈 후 2시간까지  
　B. 기준은 매장별 설정값 (영업 시작 시간 기준)  
　C. 24시간 매장 예외: 다음 날 오전 11:00까지 수정 가능  
　D. 수정 가능 시간까지만 버튼 활성화  

2: 수정 불가 시 팝업  
　A. 수정 불가능 시간엔 팝업으로 이유 안내  
　B. 팝업 멘트: '소진량 수정 시간 종료  
소진량 수정은 익일 오픈 후 2시간까지만 수정 가능하며,  
이후에는 정산 정확도를 위해 변경할 수 없습니다.'  

3: 수정 결과 연동  
　A. 수정 결과는 캘린더나 대시섹션 등에 모두 연동  
　B. 수정 이력 기록 (수정 전/후 값 모두 저장)  


**1.1.2.2.** 제출 버튼 동작  

1: 제출 버튼  
　A. 입력창 우측에 제출 버튼  
　B. 클릭 시 입력값 제출 처리  

2: 경고 팝업  
　A. 제출 후 입력값이 평균 대비 1.5배 이상 차이 나면 경고 메세지 팝업  
　B. 팝업 멘트: '입력값이 평균값을 벗어났습니다.  
맞는지 확인해 주세요.'  
　C. 팝업창에 취소/확인 버튼 생성  

3: 팝업 분기  
　A. 취소 선택: 입력창으로 돌아가 수정 가능  
　B. 확인 선택: 입력값 그대로 제출 처리  

4: 제출 연동  
　A. 입력값 제출 시 관련 모든 시스템에 연동  


**1.1.2.3.** 섹션 클릭 시 화면 전환  

1: 섹션 클릭 시  
　A. 사이드바의 일일 소진 보고 페이지로 이동  
　B. 상세 소진 보고 화면으로 이동  


**1.1.2.4.** 미보고 알림 바로가기  

1: 바로가기 버튼  
　A. 알림 메시지 내 소진 보고 바로가기 버튼 제공  
　B. 클릭 시 캠페인 소진 보고 페이지로 이동  
　C. 소진량 입력창으로 자동 포커스  


• 참조: 7-1-5-3. 영업 종료 시점 일일 소진량 입력, 7-1-5-4. 미보고 알림 수신
