　　　**1.2.** 소진량 수정 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 수정 버튼 클릭 | 전일 소진량 수정 화면 진입 |
| 수정값 입력 | 올바른 소진량 입력 |
| 제출 버튼 클릭 | 수정 데이터 전송 |

## 수정 가능 시간

| 조건 | 수정 가능 시간 |
|---|---|
| 일반 매장 | 익일 영업 시작 후 2시간 이내 |
| 24시간 매장 | 다음 날 오전 11:00까지 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 관리자 | 수정 이력 저장 | 수정 전/후 값 모두 기록 |
| 관리자 | 등급 산정 재반영 | 수정된 값으로 등급 점수 재계산 |
| 관리자 | FDS 모니터링 | 수정 빈도 및 패턴 모니터링 |
| 협력사 | 재고 현황 재계산 | 수정된 소진량 기준 재고 상태 업데이트 |

## 시스템 흐름

```
파트너 수정 버튼 → 수정 가능 시간 확인 → 수정값 입력 → 수정 이력 저장 → 등급 산정 재반영 → FDS 모니터링
```
