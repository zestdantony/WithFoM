　　**3-2.1.** 미보고 알림 연동

## 시스템 자동 동작

| 조건 | 동작 |
|---|---|
| 영업 종료 후 2시간 | 소진 보고 미완료 시 알림 발송 |
| 익일 오전 | 연속 미보고 시 추가 알림 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 파트너 | 푸시 알림 | "오늘 소진 보고가 아직 완료되지 않았습니다" |
| 파트너 | 대시보드 경고 | 미보고 경고 위젯 표시 |
| 관리자 | 미보고 로그 | 연속 미보고 파트너 리스트 관리 |
| 관리자 | 등급 산정 | 미보고 시 보고율 점수 하락 |

## 시스템 흐름

```
영업 종료 후 2시간 → 소진 보고 미완료 확인 → 파트너 푸시 알림 → 관리자 미보고 로그 → 등급 산정 반영
```

## 미보고 패널티

| 연속 미보고 | 영향 |
|---|---|
| 3일 | 경고 메시지 발송 |
| 7일 | 등급 하락 경고 |
| 14일 | 관리자 수동 연락 |
