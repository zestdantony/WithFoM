**2.1.** 미보고 알림 연동

## 시스템 자동 동작

| 조건 | 동작 |
|---|---|
| 영업 종료 후 1시간 경과 | 1차 알림 발송 |
| 익일 오전 10시 미보고 시 | 2차 알림 발송 |
| 연속 3일 미보고 | 경고 알림 발송 |

## 알림 메시지 상세

| 알림 유형 | 메시지 내용 |
|---|---|
| 1차/2차 알림 | "어제 소진 보고가 완료되지 않았습니다. 소진 보고를 완료해 주세요." |
| 연속 미보고 경고 | "소진 보고가 연속으로 누락되고 있습니다. 정확한 정산을 위해 소진 보고를 꼭 완료해 주세요." |

## 알림 구성 요소

| 요소 | 내용 |
|---|---|
| 소진 보고 바로가기 | 버튼 제공 |
| 소진 보고 방법 안내 | 링크 제공 |
| 파트너 등급 영향 | 안내 문구 포함 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 파트너 | 푸시 알림 | "오늘 소진 보고가 아직 완료되지 않았습니다" |
| 파트너 | 대시보드 경고 | 미보고 경고 위젯 표시 |
| 관리자 | 미보고 로그 | 연속 미보고 파트너 리스트 관리 |
| 관리자 | 등급 산정 | 미보고 시 보고율 점수 하락 (45% 가중치) |

## 시스템 흐름

```
영업 종료 후 2시간 → 소진 보고 미완료 확인 → 파트너 푸시 알림 → 관리자 미보고 로그 → 등급 산정 반영
```

## 미보고 패널티

| 연속 미보고 | 영향 |
|---|---|
| 3일 | 경고 메시지 발송 |
| 7일 | 등급 하락 경고 |
| 14일 | 관리자 수동 연락 |
