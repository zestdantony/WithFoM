　　**9-2.1.** 알림 설정 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 알림 종류별 ON/OFF | 알림 유형별 수신 여부 설정 |
| 저장 버튼 클릭 | 설정 완료 |

## 알림 설정 항목

| 알림 종류 | 기본값 |
|---|---|
| 리워드 승인 요청 | ON (필수) |
| 결제 승인 요청 | ON (필수) |
| 일일 소진 보고 알림 | ON |
| 재고 부족 경고 | ON |
| 등급 변동 알림 | ON |
| 정산 완료 알림 | ON |
| 캠페인 신규 매칭 | ON |
| 마케팅/이벤트 알림 | OFF |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 관리자 | 알림 설정 저장 | 파트너별 알림 설정 상태 저장 |
| 관리자 | 푸시 발송 제어 | 설정에 따라 푸시 알림 발송 여부 결정 |

## 시스템 흐름

```
파트너 알림 설정 변경 → 서버 저장 → 알림 발송 시 설정 확인 → 설정된 알림만 발송
```
