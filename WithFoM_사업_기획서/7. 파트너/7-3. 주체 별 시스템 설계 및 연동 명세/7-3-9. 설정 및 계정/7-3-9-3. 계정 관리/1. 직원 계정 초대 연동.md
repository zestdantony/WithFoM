**3.1.** 직원 계정 초대 연동

## 파트너(대표) 동작

| 동작 | 내용 |
|---|---|
| 직원 초대 버튼 클릭 | 직원 초대 화면 진입 |
| 이메일/연락처 입력 | 초대할 직원 정보 입력 |
| 권한 설정 | 직원 권한 범위 설정 |
| 초대 발송 | 초대 링크/SMS 발송 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 직원 | 초대 알림 | 이메일/SMS로 초대 링크 발송 |
| 직원 | 가입 진행 | 초대 링크 통해 직원 계정 가입 |
| 관리자 | 계정 생성 로그 | 파트너별 직원 계정 생성 이력 기록 |

## 시스템 흐름

```
대표 직원 초대 → 초대 링크 생성 → 직원 알림 발송 → 직원 가입 완료 → 관리자 로그 기록
```

## 직원 권한 설정 항목

| 권한 | 부여 가능 |
|---|---|
| 리워드/결제 승인 | ✅ |
| 일일 소진 보고 | ✅ |
| 메뉴 관리 | ✅ |
| 정산 조회 | 대표만 가능 |
| 계정 관리 | 대표만 가능 |
