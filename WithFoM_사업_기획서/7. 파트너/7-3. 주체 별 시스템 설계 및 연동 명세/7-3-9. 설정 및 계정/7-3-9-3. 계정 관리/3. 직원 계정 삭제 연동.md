# 3: 직원 계정 삭제 연동

## 파트너(대표) 동작

| 동작 | 내용 |
|---|---|
| 직원 목록에서 선택 | 삭제할 직원 선택 |
| 삭제 버튼 클릭 | 직원 계정 삭제 확인 |
| 확인 버튼 클릭 | 계정 삭제 완료 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 직원 | 계정 비활성화 | 즉시 로그아웃 및 접근 불가 |
| 직원 | 알림 발송 | "계정이 삭제되었습니다." |
| 관리자 | 계정 삭제 로그 | 파트너별 직원 계정 삭제 이력 기록 |

## 시스템 흐름

```
대표 계정 삭제 → 서버 상태 변경 → 직원 즉시 로그아웃 → 직원 알림 → 관리자 로그 기록
```
