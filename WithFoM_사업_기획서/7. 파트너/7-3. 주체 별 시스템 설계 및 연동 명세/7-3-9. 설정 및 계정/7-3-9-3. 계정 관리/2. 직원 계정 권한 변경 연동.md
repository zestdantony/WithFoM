　　**9-3.2.** 직원 계정 권한 변경 연동

## 파트너(대표) 동작

| 동작 | 내용 |
|---|---|
| 직원 목록에서 선택 | 권한 변경할 직원 선택 |
| 권한 수정 | 권한 범위 변경 |
| 저장 버튼 클릭 | 권한 변경 완료 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 직원 | 권한 즉시 반영 | 변경된 권한으로 앱 메뉴/기능 접근 제한 |
| 직원 | 알림 발송 | "계정 권한이 변경되었습니다." |
| 관리자 | 권한 변경 로그 | 파트너별 직원 권한 변경 이력 기록 |

## 시스템 흐름

```
대표 권한 변경 → 서버 저장 → 직원 앱 권한 즉시 반영 → 직원 알림 → 관리자 로그 기록
```
