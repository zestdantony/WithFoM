# 2: 지도 핀 설정 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 지도 핀 위치 설정 | 지도에서 매장 정확한 위치 핀 설정 |
| 저장 버튼 클릭 | 핀 위치 저장 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 소비자 | 지도 핀 표시 | 소비자 앱 지도에 설정된 위치로 핀 표시 |
| 관리자 | 위치 정보 저장 | 파트너 위치 좌표 저장 |
| 관리자 | 캠페인 매칭 | 위치 기반 캠페인 매칭 계산 |

## 시스템 흐름

```
파트너 지도 핀 설정 → 서버 좌표 저장 → 소비자 앱 지도 핀 반영 → 캠페인 매칭 재계산
```
