　　**7-1.1.** 매장 정보 수정 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 매장 정보 수정 | 매장명, 주소, 연락처, 영업시간 등 수정 |
| 저장 버튼 클릭 | 수정 완료 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 소비자 | 매장 정보 갱신 | 소비자 앱 지도/매장 상세에 수정 내용 반영 |
| 관리자 | 정보 변경 로그 | 매장 정보 변경 이력 기록 |
| 관리자 | 캠페인 매칭 | 지역 변경 시 캠페인 매칭 재계산 |

## 시스템 흐름

```
파트너 매장 정보 수정 → 서버 저장 → 소비자 앱 정보 갱신 → 관리자 로그 기록 → (지역 변경 시) 캠페인 매칭 재계산
```
