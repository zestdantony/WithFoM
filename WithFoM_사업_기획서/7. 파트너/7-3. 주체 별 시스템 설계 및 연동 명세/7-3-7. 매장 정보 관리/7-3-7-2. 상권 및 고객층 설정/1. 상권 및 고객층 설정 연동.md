# 1: 상권 및 고객층 설정 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 상권 유형 선택 | 오피스/주거/학교/역세권 등 선택 |
| 고객층 선택 | 주요 고객층 최대 5개 선택 |
| 저장 버튼 클릭 | 설정 완료 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 관리자 | 매장 특성 저장 | 파트너 상권/고객층 정보 저장 |
| 관리자 | 캠페인 매칭 | 광고주 타깃과 매칭하여 적합한 캠페인 배정 |
| 관리자 | 광고주 리포트 | 캠페인 배포 매장 특성 리포트에 반영 |

## 시스템 흐름

```
파트너 상권/고객층 설정 → 서버 저장 → 캠페인 매칭 알고리즘에 반영 → 광고주 리포트 갱신
```

## 상권 유형 예시

| 유형 |
|---|
| 오피스/비즈니스 |
| 주거/생활권 |
| 대학/학교 |
| 역세권/교통 허브 |
| 상업/쇼핑 |
