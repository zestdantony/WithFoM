# 2: 메뉴 수정 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 메뉴 선택 | 수정할 메뉴 선택 |
| 정보 수정 | 메뉴명, 가격, 옵션, 이미지 수정 |
| 저장 버튼 클릭 | 수정 완료 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 소비자 | 메뉴 정보 갱신 | 소비자 앱에 수정된 정보 실시간 반영 |
| 관리자 | 수정 이력 로그 | 메뉴 수정 이력 기록 (가격 변경 등) |

## 시스템 흐름

```
파트너 메뉴 수정 → 서버 저장 → 소비자 앱 메뉴 정보 갱신 → 관리자 수정 이력 기록
```
