# 3: 메뉴 품절 설정 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 품절 버튼 클릭 | 해당 메뉴 품절 처리 |
| 품절 해제 버튼 클릭 | 품절 상태 해제 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 소비자 | 품절 표시 | 소비자 앱에서 해당 메뉴 "품절" 표시, 주문 불가 |
| 관리자 | 품절 현황 로그 | 파트너별 품절 메뉴 현황 기록 |

## 시스템 흐름

```
파트너 품절 설정 → 서버 상태 변경 → 소비자 앱 품절 표시 → 관리자 로그 기록
```
