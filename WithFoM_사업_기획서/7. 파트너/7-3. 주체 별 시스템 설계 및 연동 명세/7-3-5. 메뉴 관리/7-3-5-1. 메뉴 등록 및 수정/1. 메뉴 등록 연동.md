　　**5-1.1.** 메뉴 등록 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 메뉴 추가 버튼 클릭 | 새 메뉴 등록 화면 진입 |
| 메뉴 정보 입력 | 메뉴명, 가격, 옵션, 이미지 입력 |
| 저장 버튼 클릭 | 메뉴 등록 완료 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 소비자 | 메뉴 노출 | 소비자 앱 메뉴 목록에 실시간 반영 |
| 관리자 | 메뉴 등록 로그 | 파트너별 메뉴 등록 현황 기록 |
| 관리자 | 메뉴 검수 | (필요 시) 부적절 메뉴 검수 대상 |

## 시스템 흐름

```
파트너 메뉴 등록 → 서버 저장 → 소비자 앱 메뉴 목록 반영 → 관리자 로그 기록
```

> ⚠️ 오더형/복합형 파트너만 해당 (리워드형은 메뉴 관리 미노출)
