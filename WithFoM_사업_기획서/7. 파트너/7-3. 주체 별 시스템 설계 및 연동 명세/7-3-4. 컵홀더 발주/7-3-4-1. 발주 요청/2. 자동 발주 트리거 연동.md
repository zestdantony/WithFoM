# 2: 자동 발주 트리거 연동

## 시스템 자동 동작

| 조건 | 동작 |
|---|---|
| 재고 30% 이하 | 자동 발주 권장 알림 |
| 재고 임계치 도달 | 설정된 자동 발주 수량으로 자동 요청 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 파트너 | 재고 부족 알림 | "컵홀더 재고가 N개 남았습니다. 발주를 권장합니다" |
| 파트너 | 자동 발주 알림 | (자동 발주 설정 시) "자동 발주가 요청되었습니다" |
| 관리자 | 자동 발주 로그 | 자동 발주 기록 |
| 관리자 + 협력사 | 발주 알림 | 관리자 및 협력사에 동시 알림 |
| 협력사 | 출고 준비 | 자동 발주 건 출고 준비 |

## 시스템 흐름

```
재고 임계치 도달 → 자동 발주 요청 → 관리자 + 협력사 동시 알림 → 협력사 출고 준비
```

## 자동 발주 설정 항목

| 항목 | 설정 |
|---|---|
| 임계치 | 재고가 N% 이하일 때 자동 발주 (20%~40%, 기본 30%) |
| 발주 수량 | 매장 수용량을 기본 발주량으로 자동 계산 |
| 자동 발주 ON/OFF | 기능 사용 여부 |

## 자동 발주 수량 계산

| 항목 | 내용 |
|---|---|
| 기본 발주량 | 매장 수용량 |
| 계산 방식 | 매장 수용량 - 현재 재고 = 발주 수량 |
| 발주 단위 | 박스당 최소단위 |
