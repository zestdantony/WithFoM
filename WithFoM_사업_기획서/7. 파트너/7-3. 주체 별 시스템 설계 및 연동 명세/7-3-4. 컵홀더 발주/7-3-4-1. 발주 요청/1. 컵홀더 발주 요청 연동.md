　　**4-1.1.** 컵홀더 발주 요청 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 발주하기 버튼 클릭 | 컵홀더 추가 발주 요청 |
| 발주 수량 확인 | 기본 수량 또는 수량 변경 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 관리자 | 발주 요청 접수 | 발주 요청 리스트에 추가 |
| 관리자 | 발주 승인/반려 | 발주 적정성 검토 후 처리 |
| 관리자 + 협력사 | 발주 알림 | 관리자 및 협력사에 동시 알림 |
| 협력사 | 출고 준비 | 승인 시 물류사 출고 준비 시작 |

## 시스템 흐름

```
파트너 발주 요청 → 관리자 + 협력사 동시 알림 → 관리자 발주 승인 → 협력사 출고 준비 → 배송 시작
```

## 알림

| 대상 | 알림 내용 |
|---|---|
| 파트너 | "발주 요청이 접수되었습니다." |
| 관리자 | "새로운 컵홀더 발주 요청이 있습니다. [파트너명]" |
| 협력사 | "발주 요청이 접수되었습니다. [파트너명], [수량]" |
