　　　**1.3.** 수용량 초과 발주 확인 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 150%/200% 발주 선택 | 수용량 초과 발주 요청 |
| 확인 팝업에서 확인 클릭 | 발주 진행 |

## 확인 팝업 내용

| 항목 | 내용 |
|---|---|
| 제목 | 수용량 초과 발주 안내 |
| 본문 | 설정하신 매장 수용량([N]개)을 초과하는 발주입니다.<br>선택하신 수량: [M]개 ([X]%)<br>보관 공간이 충분한지 확인해 주세요. |
| 버튼 | 취소 / 발주 신청 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 파트너 | 확인 팝업 표시 | 수용량 초과 안내 후 파트너 선택 |
| 관리자 | 발주 로그 | 수용량 초과 발주 건 별도 표시 (제한 없음) |
| 협력사 | 발주 접수 | 초과 발주 건 정상 처리 |

## 시스템 흐름

```
파트너 150%/200% 발주 선택 → 확인 팝업 표시 → 파트너 확인 → 발주 정상 진행 → 관리자/협력사 알림
```
