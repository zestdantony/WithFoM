　　　**3.** 캠페인 결과 리포트 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 리포트 확인 | 리포트 > 캠페인 결과 탭에서 조회 |
| 성과 지표 분석 | 소진율, 스캔율, 재방문율 등 확인 |
| 리포트 다운로드 | PDF/Excel 형식으로 다운로드 |

## 리포트 구성

| 리포트 유형 | 제공 대상 | 포함 항목 |
|---|---|---|
| 요약 리포트 | 전체 파트너 | 배정 수량, 소진 수량, 소진율, QR 스캔 수, 스캔율 |
| 상세 리포트 | 멤버십 플러스 이상 | 일별 소진 추이, 시간대별 스캔 분포, 요일별 패턴, 재방문율 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 파트너 | 리포트 조회 | 캠페인 종료 후 성과 리포트 자동 생성 및 제공 |
| 파트너 | 리포트 다운로드 | PDF/Excel 형식 다운로드 가능 |
| 관리자 | 리포트 생성 | 캠페인 종료 시 자동 리포트 생성 |
| 관리자 | 등급 산정 반영 | 소진율, 스캔율 등 등급 점수에 반영 |

## 시스템 흐름

```
캠페인 종료 → 리포트 자동 생성 → 파트너 리포트 탭 노출 → 파트너 조회 및 다운로드 → 등급 산정 반영
```

## 등급 산정 연계

| 지표 | 반영 항목 | 비중 |
|---|---|---|
| 보고율 | 일일 소진 보고 완료율 | 45% |
| 스캔율 | QR 스캔 / 소진 비율 | 45% |
| 결제 기능 이용 | 위드폼 오더 사용 여부 | 10% |

## 다운로드 권한

| 형식 | 내용 | 권한 |
|---|---|---|
| PDF | 요약 리포트 | 전체 파트너 |
| Excel | 상세 데이터 | 멤버십 플러스 이상 |
