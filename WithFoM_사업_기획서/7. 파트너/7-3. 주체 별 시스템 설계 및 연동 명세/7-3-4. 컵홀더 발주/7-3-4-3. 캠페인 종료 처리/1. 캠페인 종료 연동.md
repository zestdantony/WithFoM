　　　**1.** 캠페인 종료 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 캠페인 종료 확인 | 종료된 캠페인 탭에서 캠페인 상태 확인 |
| 결과 리포트 확인 | 캠페인 성과 리포트 조회 |
| 다음 캠페인 준비 | 개선 포인트 파악 후 다음 캠페인 대비 |

## 캠페인 종료 조건

| 종료 유형 | 조건 | 시스템 동작 |
|---|---|---|
| 기간 만료 종료 | 캠페인 설정 기간 도래 | 자동으로 "종료" 상태 전환 |
| 조기 소진 종료 | 배정 수량 100% 소진 | 자동으로 "소진 완료" 상태 전환 |
| 중도 종료 | 광고주 또는 관리자 요청 | 관리자 승인 후 "중도 종료" 상태 전환 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 파트너 | 종료 알림 발송 | 캠페인 종료 및 결과 확인 안내 |
| 관리자 | 캠페인 상태 변경 | 종료/소진완료/중도종료 상태로 전환 |
| 관리자 | 리포트 생성 | 캠페인 결과 리포트 자동 생성 |
| 관리자 | 정산 연동 | 캠페인별 정산 프로세스 시작 |
| 광고주 | 성과 리포트 공유 | 캠페인 성과 데이터 광고주 대시보드 반영 |

## 시스템 흐름

```
캠페인 종료 조건 도달 → 상태 자동 전환 → 파트너 종료 알림 → 리포트 생성 → 정산 프로세스 시작
```

## 종료 알림 유형

| 알림 유형 | 발송 시점 | 내용 |
|---|---|---|
| 소진 완료 알림 | 마지막 컵홀더 소진 보고 시 | "축하합니다! 캠페인 컵홀더가 모두 소진되었습니다." |
| 기간 만료 알림 | 캠페인 종료일 익일 | "캠페인이 종료되었습니다. 결과를 확인해 주세요." |
| 잔여 재고 알림 | 기간 만료 + 잔여 재고 존재 시 | "캠페인이 종료되었지만 남은 컵홀더를 모두 소진해주세요." |

• 참조: 7-1-6-1. 캠페인 소진 완료 확인
