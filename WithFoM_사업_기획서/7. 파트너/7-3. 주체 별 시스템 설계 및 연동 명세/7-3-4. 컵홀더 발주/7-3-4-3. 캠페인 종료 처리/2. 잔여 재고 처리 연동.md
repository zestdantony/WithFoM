　　　**2.** 잔여 재고 처리 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 잔여 재고 확인 | 재고 관리 > 잔여 재고 탭에서 확인 |
| 처리 방법 선택 | 이월/반품/폐기 중 선택 |
| 처리 완료 버튼 클릭 | 선택한 처리 방법에 따라 완료 처리 |

## 잔여 재고 발생 조건

| 상황 | 설명 |
|---|---|
| 기간 만료 | 캠페인 종료일까지 컵홀더를 모두 소진하지 못함 |
| 예상 대비 저조 | 방문객 수가 예상보다 적어 소진 속도가 느림 |
| 휴무/이벤트 | 예상치 못한 휴무 또는 특별 이벤트로 인한 소진 지연 |

## 처리 방법

| 처리 방법 | 설명 | 파트너 조치 |
|---|---|---|
| 보관 유지 (이월) | 다음 캠페인까지 매장에서 보관 | 별도 조치 불필요 |
| 지정 주소로 반송 | 위드폼 지정 창고로 반송 후 다음 캠페인 시 재입고 | 지정 주소로 반송 (배송비 위드폼 부담) |
| 폐기 처리 | 광고주가 폐기를 요청한 경우에만 진행 | 폐기 후 폐기 확인 버튼 클릭 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 파트너 | 잔여 재고 표시 | 잔여 수량 및 처리 방법 표시 |
| 관리자 | 재고 현황 업데이트 | 잔여 재고 상태 추적 |
| 관리자 | 처리 완료 확인 | 이월/반품/폐기 처리 상태 기록 |
| 협력사 | 반품 입고 처리 | 반품 시 협력사 물류 시스템에 반영 |

## 시스템 흐름

```
캠페인 종료 + 잔여 재고 확인 → 파트너 처리 방법 선택 → 처리 완료 버튼 → 관리자 처리 상태 업데이트
```

## 처리 기한 및 미처리 시 조치

| 조건 | 조치 |
|---|---|
| 캠페인 종료 후 14일 이내 | 처리 완료 권장 |
| 처리 기한 경과 | 관리자에게 자동 에스컬레이션 |
