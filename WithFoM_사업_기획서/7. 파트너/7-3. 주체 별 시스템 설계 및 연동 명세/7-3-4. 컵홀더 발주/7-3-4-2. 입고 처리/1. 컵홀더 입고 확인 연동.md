# 1: 컵홀더 입고 확인 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 입고 확인 버튼 클릭 | 배송 완료된 컵홀더 수량 확인 |
| 수량 확인 | 배송 수량과 실제 수량 일치 확인 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 관리자 | 입고 완료 처리 | 발주 상태 "입고 완료"로 변경 |
| 관리자 | 재고 현황 업데이트 | 파트너 현재 재고량 갱신 |
| 협력사 | 배송 완료 확인 | 물류 추적 상태 "배송 완료" 처리 |

## 시스템 흐름

```
파트너 입고 확인 → 발주 상태 "입고 완료" → 재고 현황 업데이트 → 협력사 배송 완료 확인
```
