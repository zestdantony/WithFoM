# 2: 불량 및 반품 요청 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 불량 신고 버튼 클릭 | 불량 컵홀더 사진 첨부 및 설명 입력 |
| 반품 요청 버튼 클릭 | 반품 수량 및 사유 입력 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 관리자 | 불량/반품 접수 | 관리자 CS 대시보드에 접수 건 표시 |
| 관리자 | 알림 발송 | "불량/반품 요청이 접수되었습니다. [파트너명]" |
| 관리자 | 승인/반려 처리 | 불량/반품 적정성 검토 후 처리 |
| 협력사 | 알림 발송 | "불량/반품 요청이 접수되었습니다. [파트너명]" (관리자와 동시 알림) |
| 협력사 | 반품 수거 | 승인 시 반품 수거 진행 |
| 협력사 | 재발송 | 불량 승인 시 교환품 발송 |

## 시스템 흐름

```
파트너 불량/반품 요청 → 관리자 + 협력사 동시 알림 → 관리자 승인/반려 → (승인 시) 협력사 반품 수거/교환 발송
```

## 알림

| 대상 | 알림 내용 |
|---|---|
| 파트너 | "불량/반품 요청이 접수되었습니다. 처리 결과를 안내드리겠습니다." |
| 관리자 | "불량/반품 요청이 접수되었습니다. [파트너명]" |
| 협력사 | "불량/반품 요청이 접수되었습니다. [파트너명], [수량], [사유]" |
