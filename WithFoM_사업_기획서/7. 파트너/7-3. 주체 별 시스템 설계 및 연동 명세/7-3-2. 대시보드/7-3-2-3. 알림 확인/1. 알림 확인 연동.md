**3.1.** 알림 확인 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 알림(벨) 아이콘 클릭 | 알림 목록 조회 |
| 개별 알림 클릭 | 해당 페이지로 이동 |

## 알림 유형 및 연동

| 알림 유형 | 우선순위 | 클릭 시 이동 |
|---|---|---|
| 리워드 승인 대기 | 높음 | 리워드 사용 페이지 |
| 결제 승인 대기 | 높음 | 결제 페이지 |
| 일일 소진 보고 미완료 | 높음 | 일일 소진 보고 페이지 |
| 재고 부족 경고 | 중간 | 컵홀더 발주 페이지 |
| 이상치 경고 | 중간 | 소진 보고 상세 페이지 |
| 등급 변동 알림 | 낮음 | 등급/멤버십 페이지 |
| 정산 완료 알림 | 낮음 | 정산 내역 페이지 |
| 캠페인 신규 매칭 | 낮음 | 캠페인 상세 페이지 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 관리자 | 알림 읽음 상태 | 파트너별 알림 읽음 상태 기록 |
| 관리자 | 알림 응답 시간 | 중요 알림 응답 시간 통계 |

## 시스템 흐름

```
파트너 알림 클릭 → 알림 읽음 처리 → 해당 페이지 이동 → 관리자 읽음 상태 저장
```
