　　**2-1.2.** 리워드 승인 처리 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 승인 버튼 클릭 | 리워드 사용 승인 |
| 또는 60초 경과 | 자동 승인 처리 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 소비자 | 승인 완료 화면 | 리워드 사용 성공 화면 표시 |
| 소비자 | 리워드 차감 | 사용된 쿠폰/포인트/스탬프 차감 |
| 소비자 | 사용 이력 추가 | 마이페이지 리워드 사용 이력에 추가 |
| 관리자 | 사용 완료 로그 | 리워드 사용 완료 기록 |
| 관리자 | 통계 업데이트 | 파트너 재방문율, 리워드 사용률 통계 반영 |

## 시스템 흐름

```
파트너 승인 (또는 자동 승인) → 소비자 리워드 차감 → 소비자 승인 완료 알림 → 관리자 통계 업데이트
```

| 대상 | 알림 내용 |
