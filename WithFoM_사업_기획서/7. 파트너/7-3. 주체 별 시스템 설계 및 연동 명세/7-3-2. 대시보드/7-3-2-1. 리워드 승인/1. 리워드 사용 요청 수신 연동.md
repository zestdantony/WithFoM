**1.1.** 리워드 사용 요청 수신 연동

## 소비자 동작

| 동작 | 내용 |
|---|---|
| 리워드 선택 | 사용할 쿠폰/포인트/스탬프 선택 |
| 사용하기 버튼 클릭 | 파트너에게 승인 요청 전송 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 파트너 | 대시보드 카드 생성 | 리워드 사용 요청 카드 표시 (최대 5개) |
| 파트너 | 푸시 알림 | "N건의 리워드 사용 요청이 대기 중입니다" |
| 파트너 | 60초 타이머 | 미응답 시 자동 승인을 위한 카운트다운 |
| 관리자 | 사용 요청 로그 | 모든 리워드 사용 요청 기록 |
| 관리자 | 실시간 모니터링 | 전체 리워드 사용 현황 대시보드 반영 |

## 시스템 흐름

```
소비자 리워드 사용 요청 → 파트너 대시보드 카드 생성 → 파트너 푸시 알림 → 60초 타이머 시작 → 관리자 로그 기록
```
