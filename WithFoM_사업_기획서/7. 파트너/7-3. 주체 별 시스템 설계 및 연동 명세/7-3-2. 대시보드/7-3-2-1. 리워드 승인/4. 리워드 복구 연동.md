　　**2-1.4.** 리워드 복구 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 복구 버튼 클릭 | 잘못 승인한 리워드 복구 요청 |
| 복구 사유 입력 | 복구 사유 선택/입력 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 소비자 | 리워드 복구 | 사용된 쿠폰/포인트/스탬프 되살리기 |
| 소비자 | 알림 발송 | "리워드가 복구되었습니다. 다시 사용하실 수 있습니다." |
| 관리자 | 복구 로그 | 복구 요청 및 사유 기록 |
| 관리자 | FDS 연동 | 과도한 복구 시 이상 탐지 |

## 시스템 흐름

```
파트너 복구 버튼 클릭 → 복구 사유 입력 → 소비자 리워드 복구 → 소비자 알림 발송 → 관리자 복구 로그 기록 → FDS 모니터링
```

## FDS 경고 조건

| 조건 | 경고 |
|---|---|
| 일일 복구 5회 초과 | 경고 레벨 1 |
| 월간 복구 20회 초과 | 경고 레벨 2 |
| 연속 복구 패턴 감지 | 관리자 수동 검토 |
