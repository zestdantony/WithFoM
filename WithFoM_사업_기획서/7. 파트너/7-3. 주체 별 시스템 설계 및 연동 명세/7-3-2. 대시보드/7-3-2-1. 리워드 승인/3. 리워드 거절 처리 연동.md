# 3: 리워드 거절 처리 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 거절 버튼 클릭 | 리워드 사용 거절 |
| 거절 사유 선택 | (선택) 거절 사유 입력 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 소비자 | 거절 알림 | "리워드 사용이 거절되었습니다. 매장에 문의해주세요." |
| 소비자 | 리워드 유지 | 사용하려던 쿠폰/포인트/스탬프 그대로 유지 |
| 관리자 | 거절 로그 | 리워드 사용 거절 기록 + 사유 |
| 관리자 | FDS 모니터링 | 과도한 거절 시 이상 탐지 |

## 시스템 흐름

```
파트너 거절 버튼 클릭 → 소비자 거절 알림 발송 → 대시보드 카드 제거 → 관리자 거절 로그 기록 → FDS 모니터링
```

## 거절 사유 예시

| 사유 |
|---|
| 조건 불충족 (리워드 사용 조건 미충족) |
| 재고 소진 (해당 메뉴/상품 품절) |
| 운영 불가 (매장 사정으로 사용 불가) |
