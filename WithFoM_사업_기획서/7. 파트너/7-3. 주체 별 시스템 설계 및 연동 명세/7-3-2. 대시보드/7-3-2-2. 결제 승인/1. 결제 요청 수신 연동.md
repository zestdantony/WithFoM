　　　**2.1.** 결제 요청 수신 연동

## 소비자 동작

| 동작 | 내용 |
|---|---|
| 메뉴 선택 | 주문할 메뉴 및 옵션 선택 |
| 결제 요청 버튼 클릭 | 파트너에게 결제 승인 요청 전송 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 파트너 | 대시보드 카드 생성 | 결제 승인 요청 카드 표시 (최대 5개) |
| 파트너 | 푸시 알림 | "N건의 결제 승인 요청이 있습니다" |
| 파트너 | 60초 타이머 | 미응답 시 자동 승인을 위한 카운트다운 |
| 관리자 | 결제 요청 로그 | 모든 결제 요청 기록 |

## 시스템 흐름

```
소비자 결제 요청 → 파트너 대시보드 카드 생성 → 파트너 푸시 알림 → 60초 타이머 시작 → 관리자 로그 기록
```

> ⚠️ 오더형/복합형 파트너만 해당 (리워드형은 결제 기능 미사용)
