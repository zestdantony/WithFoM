**2.3.** 결제 거절 처리 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 거절 버튼 클릭 | 결제 거절 |
| 거절 사유 선택 | (선택) 거절 사유 입력 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 소비자 | 거절 알림 | "결제가 거절되었습니다. 매장에 문의해주세요." |
| 소비자 | 결제 취소 | 결제된 금액 자동 환불 처리 |
| 관리자 | 거절 로그 | 결제 거절 기록 + 사유 |
| 관리자 | FDS 모니터링 | 과도한 거절 시 이상 탐지 |

## 시스템 흐름

```
파트너 거절 버튼 클릭 → 소비자 거절 알림 → 결제 자동 환불 → 관리자 거절 로그 기록
```

## 거절 사유 예시

| 사유 |
|---|
| 재고 소진 (해당 메뉴 품절) |
| 영업 종료 (영업 시간 외 주문) |
| 배달 불가 (배달 가능 지역 외) |
