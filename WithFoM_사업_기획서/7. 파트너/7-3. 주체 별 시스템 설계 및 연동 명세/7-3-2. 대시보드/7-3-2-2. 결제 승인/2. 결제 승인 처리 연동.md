# 2: 결제 승인 처리 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 승인 버튼 클릭 | 결제 승인 |
| 또는 60초 경과 | 자동 승인 처리 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 소비자 | 결제 승인 알림 | "결제가 승인되었습니다. 음료를 준비 중입니다." |
| 소비자 | 주문 상태 변경 | "준비 중" 상태로 변경 |
| 관리자 | 결제 완료 로그 | 결제 완료 기록 |
| 관리자 | 매출 통계 반영 | 실시간 매출 통계 업데이트 |
| 관리자 | 정산 데이터 | 월말 정산 데이터에 반영 |

## 시스템 흐름

```
파트너 승인 (또는 자동 승인) → 소비자 결제 승인 알림 → 주문 상태 "준비 중" 변경 → 관리자 매출 통계 반영
```
