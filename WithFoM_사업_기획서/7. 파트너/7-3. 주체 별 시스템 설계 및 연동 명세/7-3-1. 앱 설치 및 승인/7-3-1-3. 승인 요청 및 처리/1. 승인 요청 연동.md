**3.1.** 승인 요청 연동  


**3.1.1.** 파트너 동작  


| 동작 | 내용 |
|---|---|
| 서류 확인 | 업로드한 계약서, 사업자등록증 확인 |
| 승인 요청 버튼 클릭 | 관리자에게 승인 요청 전송 |


**3.1.2.** 연동 요약  


| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 관리자 | 심사 상태 변경 | "심사 대기" 상태로 변경 |
| 관리자 | 심사 담당자 배정 | 자동/수동으로 심사 담당자 배정 |
| 관리자 | 심사 기한 설정 | 영업일 기준 1~3일 심사 기한 설정 |
| 관리자 | 알림 발송 | 새 심사 건 알림 (푸시/이메일) |


**3.1.3.** 시스템 흐름  


```
파트너 승인 요청 → 심사 상태 "심사 대기" 변경 → 관리자 알림 발송 → 심사 담당자 배정
```


**3.1.4.** 알림  


| 대상 | 알림 내용 |
|---|---|
| 파트너 | "승인 요청이 접수되었습니다. 영업일 1~3일 내 결과를 안내드립니다." |
| 관리자 | "새로운 파트너 승인 요청이 접수되었습니다." |
