　　**1-3.2.** 승인 완료 연동

## 관리자 동작

| 동작 | 내용 |
|---|---|
| 서류 검토 | 계약서, 사업자등록증 검토 |
| 승인 버튼 클릭 | 파트너 승인 처리 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 파트너 | 상태 변경 | 승인 상태로 변경, 가입 절차 진행 가능 |
| 파트너 | 알림 발송 | 승인 완료 푸시/SMS/이메일 알림 |
| 파트너 | 가입 버튼 활성화 | 앱에서 가입하기 버튼 활성화 |

## 시스템 흐름

```
관리자 승인 처리 → 파트너 상태 "승인 완료" 변경 → 파트너 알림 발송 → 앱에서 가입 버튼 활성화
```

## 알림

| 대상 | 알림 내용 |
|---|---|
| 파트너 | "파트너 승인이 완료되었습니다! 가입을 진행해주세요." |
