　　**1-2.1.** 계약서 다운로드 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 계약서 메뉴 접근 | 앱/웹에서 계약서 다운로드 선택 |
| PDF 다운로드 | 계약서 파일 다운로드 완료 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 관리자 | 다운로드 로그 | 다운로드 시점, 기기 정보 기록 |
| 관리자 | 진행 상태 업데이트 | 가입 진행 상태를 "계약서 다운로드 완료"로 변경 |

## 시스템 흐름

```
파트너 계약서 다운로드 요청 → 서버 API → PDF 파일 전송 → 관리자 DB 진행 상태 업데이트
```
