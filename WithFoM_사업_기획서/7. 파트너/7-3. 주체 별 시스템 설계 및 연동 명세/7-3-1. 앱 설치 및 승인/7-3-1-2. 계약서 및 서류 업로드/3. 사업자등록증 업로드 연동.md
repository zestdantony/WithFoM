　　**1-2.3.** 사업자등록증 업로드 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 사업자등록증 촬영/선택 | 카메라 촬영 또는 갤러리에서 선택 |
| 업로드 버튼 클릭 | 서버에 파일 전송 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 관리자 | 파일 저장 | 업로드된 사업자등록증 서버 저장 |
| 관리자 | 사업자 정보 검증 | 국세청 API 통한 사업자등록번호 유효성 검증 |
| 관리자 | AI 문서 인식 | OCR로 사업자등록증 정보 자동 추출 |
| 관리자 | 심사 대기 목록 추가 | 관리자 심사 대기 목록에 자동 추가 |

## 시스템 흐름

```
파트너 사업자등록증 업로드 → 파일 서버 저장 → 국세청 API 검증 → AI OCR 정보 추출 → 관리자 심사 대기 목록 추가
```
