# 2: 계약서 업로드 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 계약서 촬영/선택 | 카메라 촬영 또는 갤러리에서 선택 |
| 업로드 버튼 클릭 | 서버에 파일 전송 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 관리자 | 파일 저장 | 업로드된 계약서 파일 서버 저장 |
| 관리자 | 심사 대기 목록 추가 | 관리자 심사 대기 목록에 자동 추가 |
| 관리자 | OCR 검증 | AI가 계약서 내용 자동 인식 및 검증 |
| 관리자 | 알림 | 새로운 심사 대기 건 알림 발송 |

## 시스템 흐름

```
파트너 계약서 업로드 → 파일 서버 저장 → AI OCR 분석 → 관리자 심사 대기 목록 추가 → 관리자 알림
```
