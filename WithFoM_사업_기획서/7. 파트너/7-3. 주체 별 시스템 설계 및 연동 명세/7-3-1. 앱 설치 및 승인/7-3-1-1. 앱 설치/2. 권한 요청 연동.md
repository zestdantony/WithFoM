# 2: 권한 요청 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 카메라 권한 허용 | 사업자등록증/계약서 촬영용 |
| 위치 권한 허용 | 매장 위치 확인, 캠페인 매칭용 (필수) |
| 알림 권한 허용 | 푸시 알림 수신용 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 관리자 | 권한 상태 기록 | 파트너별 권한 허용/거부 상태 저장 |
| 관리자 | 서비스 제한 판단 | 위치 권한 거부 시 서비스 이용 불가 처리 |
| 소비자 | 매장 지도 노출 | 위치 권한 허용 시 소비자 지도에 매장 노출 가능 |

## 시스템 흐름

```
파트너 권한 허용/거부 → 앱 권한 API → 관리자 DB 저장 → 서비스 이용 가능 여부 판단 → 소비자 지도 노출 결정
```

> ⚠️ **필수:** 위치 권한 거부 시 캠페인 매칭 불가로 위드폼 서비스 이용 불가
