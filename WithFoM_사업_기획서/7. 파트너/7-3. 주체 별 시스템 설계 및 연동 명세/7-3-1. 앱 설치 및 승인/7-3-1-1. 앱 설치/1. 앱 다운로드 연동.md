# 1: 앱 다운로드 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 앱 스토어 접속 | 위드폼 파트너 앱 검색 |
| 앱 다운로드 | 설치 버튼 클릭 후 설치 완료 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 관리자 | 다운로드 통계 | 일별/월별 앱 다운로드 수 집계 |
| 관리자 | 마케팅 분석 | 유입 경로별 다운로드 추적 |
| 관리자 | 앱 버전 관리 | 설치된 앱 버전 분포 확인 |

## 시스템 흐름

```
파트너 앱 다운로드 → App Store/Play Store API → 다운로드 완료 이벤트 → 관리자 통계 시스템 반영
```
