**1.2.** 쿠폰 설정 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 쿠폰 생성 | 쿠폰 종류, 할인율, 유효기간 설정 |
| 쿠폰 수정/삭제 | 기존 쿠폰 정보 변경 또는 삭제 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 소비자 | 쿠폰 노출 | 7등 리워드 또는 결제 시 사용 가능 쿠폰 표시 |
| 관리자 | 쿠폰 발급 현황 | 파트너별 쿠폰 발급량/사용률 통계 |

## 시스템 흐름

```
파트너 쿠폰 설정 → 서버 저장 → 소비자 쿠폰 목록 반영 → 관리자 발급 현황 갱신
```

## 쿠폰 설정 항목

| 항목 | 설정 |
|---|---|
| 쿠폰 종류 | 고정 할인 / 정률 할인 / 무료 음료 |
| 할인 금액/율 | 1,000원 / 10% 등 |
| 유효기간 | 발급일로부터 N일 |
| 사용 조건 | 최소 주문 금액 등 |
