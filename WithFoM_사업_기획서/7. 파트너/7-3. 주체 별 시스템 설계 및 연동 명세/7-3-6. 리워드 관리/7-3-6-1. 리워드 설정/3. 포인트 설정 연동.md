　　　**1.3.** 포인트 설정 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 적립률 설정 | 결제 금액 대비 포인트 적립률 설정 |
| 사용 조건 설정 | 포인트 사용 가능 조건 설정 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 소비자 | 포인트 적립 | 결제 시 설정된 적립률로 포인트 적립 |
| 소비자 | 포인트 사용 | 결제 시 포인트 차감 사용 |
| 관리자 | 포인트 현황 | 파트너별 포인트 발급/사용 통계 |

## 시스템 흐름

```
파트너 포인트 설정 → 서버 저장 → 소비자 결제 시 포인트 적립/사용 적용 → 관리자 현황 갱신
```

## 포인트 설정 항목

| 항목 | 설정 |
|---|---|
| 적립률 | 결제 금액의 N% 적립 |
| 최소 사용 포인트 | 100포인트 이상부터 사용 가능 등 |
| 최대 사용 비율 | 결제 금액의 최대 N%까지 사용 가능 |
