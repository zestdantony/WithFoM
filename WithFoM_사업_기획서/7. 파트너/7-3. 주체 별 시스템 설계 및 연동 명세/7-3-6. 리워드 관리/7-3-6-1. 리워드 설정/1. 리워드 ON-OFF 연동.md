# 1: 리워드 ON/OFF 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 리워드 ON 설정 | 소비자에게 파트너 리워드 제공 활성화 |
| 리워드 OFF 설정 | 파트너 리워드 비활성화 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 소비자 | 7등 리워드 변경 | ON: 파트너 쿠폰/포인트 제공 / OFF: 광고주 꼴등 상품으로 대체 |
| 관리자 | 리워드 상태 로그 | 파트너별 리워드 ON/OFF 상태 기록 |
| 관리자 | 등급 산정 반영 | 리워드 OFF 시 결제 기능 점수 영향 |

## 시스템 흐름

```
파트너 리워드 ON/OFF 설정 → 서버 상태 변경 → 소비자 7등 리워드 표시 변경 → 관리자 로그 기록
```

## 7등(꼴등) 리워드 규칙

| 파트너 리워드 상태 | 7등 리워드 |
|---|---|
| ON | 파트너가 설정한 매장 쿠폰/포인트 제공 |
| OFF | 광고주가 설정한 꼴등 상품으로 대체 |
