**2.1.** 등급 산정 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 등급 확인 | 파트너 포털 > 파트너 등급 > 나의 등급에서 확인 |
| 등급 기준 확인 | 등급 혜택•기준 메뉴에서 산정 기준 확인 |
| 등급 변동 알림 수신 | 등급 변경 예정 및 확정 알림 수신 |

## 시스템 자동 동작

| 조건 | 동작 |
|---|---|
| 캠페인 종료 시 | 등급 산정 점수 자동 계산 및 재산정 |
| 익월 1일 | 새 등급 적용 |
| 6개월 미매칭 도달 시 | 한 단계 다운그레이드 자동 적용 |
| 매칭 확정 시 | 미매칭 기간 카운트 리셋, 등급 유지 |

## 등급 산정 기준 (점수제)

| 항목 | 가중치 | 점수 범위 |
|---|---|---|
| 보고율 | 45% | 20점(최하) ~ 80점(상) |
| 스캔율 | 45% | 20점(최하) ~ 80점(상) |
| 위드폼 오더 이용 | 10% | 0점(미이용) / 100점(이용) |

## 등급 점수 기준

| 등급 | 점수 | 산정 기간 | 혜택 |
|---|---|---|---|
| 브론즈 | 36점 미만 | 6개월 | 컵홀더 단가 50% 자부담 |
| 실버 | 36점 이상 | 6개월 | 컵홀더 무료 제공 |
| 골드 | 55점 이상 | 6개월 | 컵홀더 무료 + 멤버십 30% 할인 |
| 플래티넘 | 73점 이상 | 1년 | 컵홀더 무료 + 개당 10원 인센티브 |

※ 플래티넘 달성을 위해 위드폼 오더 이용 필수

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 파트너 | 등급 변동 알림 | "다음 달 등급이 [등급]으로 변경될 예정입니다" |
| 파트너 | 등급 배지 변경 | 앱 상단 등급 배지 갱신 |
| 파트너 | 혜택 자동 적용 | 등급별 혜택 발주/결제 시 자동 적용 |
| 관리자 | 등급 산정 로그 | 파트너별 등급 산정 이력 기록 |
| 관리자 | 캠페인 매칭 | 등급별 캠페인 우선 매칭 적용 |

## 시스템 흐름

```
[캠페인 종료 시]
캠페인 종료 → 데이터 집계 → 이전 6개월(1년) 기준 등급 재산정 → 파트너 알림 → 익월 등급 적용 → 혜택 자동 적용

[미매칭 다운그레이드]
마지막 캠페인 종료 → 미매칭 기간 카운트 시작 → 3개월 경과(1차 알림) → 5개월 3주차(2차 알림) → 6개월 도달 → 한 단계 다운 적용

[매칭 시 등급 유지]
미매칭 대기 중 → 캠페인 매칭 확정 → 미매칭 카운트 리셋 → 등급 유지 → 캠페인 종료 후 유예 기간 재시작

[다운그레이드 후 재매칭]
다운그레이드 → 새 캠페인 매칭 → 캠페인 참여 및 성과 → 종료 시 이전 6개월(1년) 재산정 → 조건 충족 시 즉시 업그레이드

[중도 포기]
캠페인 참여 중 → 중도 포기 → 참여 기간은 산정 포함 → 포기 시점부터 미매칭 카운트 시작
```

## 실 사용 기간 계산

| 조건 | 처리 방식 |
|---|---|
| 캠페인 참여 기간 | 등급 산정 기간에 카운팅 |
| 미매칭 대기 기간 | 등급 산정 기간에서 제외 |
| 휴업 기간 (사유 보고 시) | 등급 산정 기간에서 제외, 미매칭 기간에서도 제외 |

※ 등급 산정 기간은 달력상 경과 기간이 아닌 실제 캠페인 참여 기간 기준

## 등급 유지 및 다운그레이드 규칙

| 규칙 | 조건 | 결과 |
|---|---|---|
| 등급 유지 | 6개월 이내 새 캠페인 매칭 완료 | 현재 등급 유지, 유예 기간 재시작 |
| 다운그레이드 | 마지막 캠페인 종료 후 6개월 연속 미매칭 | 한 단계 다운 (플래티넘→골드→실버→브론즈) |
| 브론즈 유지 | 브론즈에서 6개월 미매칭 | 브론즈 유지 (최하 등급) |
| 휴업 예외 | 휴업 신청 기간 | 미매칭 기간 카운트에서 제외 |
| 혜택 유지 | 등급 변동 전까지 | 현재 등급 혜택 유지 (미매칭 중에도 적용) |

## 다운그레이드 후 재산정

| 상황 | 처리 |
|---|---|
| 다운그레이드 후 캠페인 매칭 | 해당 캠페인 포함 이전 6개월(플래티넘 1년) 재산정 |
| 캠페인 종료 시 성과 충족 | 즉시 업그레이드 적용 |

## 중도 포기 처리

| 상황 | 처리 |
|---|---|
| 포기 전까지 참여 기간 | 등급 산정에 포함 |
| 포기 시점 이후 | 미매칭 기간 카운트 시작 |
