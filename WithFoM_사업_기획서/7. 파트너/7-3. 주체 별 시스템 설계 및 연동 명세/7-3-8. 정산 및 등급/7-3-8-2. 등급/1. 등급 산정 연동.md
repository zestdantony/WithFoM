　　**8-2.1.** 등급 산정 연동

## 시스템 자동 동작

| 조건 | 동작 |
|---|---|
| 매월 말일 | 등급 산정 점수 자동 계산 |
| 익월 1일 | 새 등급 적용 |

## 등급 산정 기준

| 항목 | 가중치 |
|---|---|
| 보고율 | 45% |
| 스캔율 | 45% |
| 결제 기능 점수 | 10% |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 파트너 | 등급 변동 알림 | "다음 달 등급이 [등급]으로 변경될 예정입니다" |
| 파트너 | 등급 배지 변경 | 앱 상단 등급 배지 갱신 |
| 관리자 | 등급 산정 로그 | 파트너별 등급 산정 이력 기록 |
| 관리자 | 캠페인 매칭 | 등급별 캠페인 우선 매칭 적용 |

## 시스템 흐름

```
월말 데이터 집계 → 보고율+스캔율+결제 점수 계산 → 등급 결정 → 파트너 알림 → 익월 등급 적용
```

## 등급 혜택

| 등급 | 혜택 |
|---|---|
| 플래티넘 | 최우선 매칭, 인센티브 최대 |
| 골드 | 우선 매칭, 멤버십 10% 할인, 인센티브 |
| 실버 | 일반 매칭 |
| 브론즈 | 컵홀더 비용 일부 부담 |
