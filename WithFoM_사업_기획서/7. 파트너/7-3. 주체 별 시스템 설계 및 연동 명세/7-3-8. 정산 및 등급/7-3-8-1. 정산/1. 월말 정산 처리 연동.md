　　　**1.1.** 월말 정산 처리 연동

## 시스템 자동 동작

| 조건 | 동작 |
|---|---|
| 매월 말일 | 월간 정산 데이터 자동 집계 |
| 익월 초 | 정산 완료 후 알림 발송 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 파트너 | 정산 완료 알림 | "이번 달 정산이 완료되었습니다" |
| 파트너 | 정산 내역 조회 | 앱에서 정산 상세 내역 확인 가능 |
| 관리자 | 정산 데이터 생성 | 파트너별 월간 정산 데이터 생성 |
| 관리자 | 인센티브/비용 계산 | 등급별 인센티브 또는 비용 산정 |
| 관리자 | 입금 처리 | 정산금 입금 처리 |

## 시스템 흐름

```
월말 정산 집계 → 등급별 인센티브/비용 계산 → 관리자 검토 → 파트너 알림 → 정산금 입금
```

## 등급별 정산 규칙

| 등급 | 정산 내용 |
|---|---|
| 플래티넘 | 인센티브 지급 |
| 골드 | 인센티브 지급 (플래티넘보다 낮음) |
| 실버 | 비용 없음 |
| 브론즈 | 컵홀더 비용 일부 부담 |
