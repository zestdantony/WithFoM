　　　**1.1.** 월말 정산 처리 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 정산 내역 확인 | 정산 관리 > 정산 메뉴에서 확인 |
| 월별/캠페인별 선택 | 정산 기간 선택하여 조회 |
| 정산 확정 확인 | 정산 확정 알림 수신 및 내역 확인 |

## 시스템 자동 동작

| 조건 | 동작 |
|---|---|
| 매월 말일 | 월간 정산 데이터 자동 집계 |
| 익월 초 | 정산 확정 후 알림 발송 |
| 익월 15일 이내 | 입금/청구 처리 |

## 정산 항목

| 항목 | 내용 | 정산 방향 |
|---|---|---|
| 컵홀더 정산 | 소진 수량 기준 정산 | – (파트너 납부) |
| 멤버십 정산 | 구독료 정산 | – (파트너 납부) |
| 인센티브 정산 | 플래티넘 등급 인센티브 | + (파트너 수령) |
| 리워드/프로모션 | 프로모션 참여 정산 | ± |
| 폼 코인 구매 | 폼 코인 구매 정산 | – (파트너 납부) |
| 오더 수수료 | 위드폼 오더 수수료 | – (파트너 납부) |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 파트너 | 정산 확정 알림 | "이번 달 정산이 확정되었습니다. 내역을 확인해주세요." |
| 파트너 | 정산 내역 조회 | 앱에서 정산 상세 내역 확인 가능 |
| 파트너 | 정산 증빙 다운로드 | 세금계산서, 정산서 PDF 다운로드 가능 |
| 관리자 | 정산 데이터 생성 | 파트너별 월간 정산 데이터 생성 |
| 관리자 | 인센티브/비용 계산 | 등급별 인센티브 또는 비용 산정 |
| 관리자 | 입금/청구 처리 | 정산금 입금 또는 청구 처리 |

## 시스템 흐름

```
월말 정산 집계 → 항목별 계산 → 등급별 혜택 반영 → 관리자 검토 → 정산 확정 → 파트너 알림 → 입금/청구 처리 → 증빙 발행
```

## 등급별 정산 규칙

| 등급 | 컵홀더 비용 | 멤버십 할인 | 인센티브 |
|---|---|---|---|
| 플래티넘 | 무료 | 30% 할인 | 개당 10원 |
| 골드 | 무료 | 30% 할인 | - |
| 실버 | 무료 | - | - |
| 브론즈 | 50% 부담 | - | - |

## 오더 정산 (위드폼 오더 이용 시)

| 항목 | 내용 |
|---|---|
| 수수료율 | 기본 6%, 오픈 이벤트 가입 시 평생 4% |
| 정산 주기 | 결제일 기준 영업일 3일 후 입금 |

• 참조: 7-1-8. 월말 정산 및 이의신청
