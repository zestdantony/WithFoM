　　　**1.2.** 정산 이의신청 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 이의신청 화면 접속 | 정산 관리 > 정산 상세 리스트 > [이의신청] 버튼 클릭 |
| 이의신청 항목 선택 | 컵홀더/멤버십/인센티브/기타 중 선택 |
| 사유 및 증빙 입력 | 이의신청 사유 작성 및 증빙자료 첨부 (선택) |
| 제출 버튼 클릭 | 이의신청 접수 완료 |

## 이의신청 가능 조건

| 조건 | 내용 |
|---|---|
| 신청 시점 | 정산 확정 후 7일 이내 |
| 신청 대상 | 정산 금액 오류, 소진 수량 오류, 등급 혜택 미반영, 인센티브 계산 오류 등 |
| 권한 | 파트너 대표 계정만 가능 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 파트너 | 이의신청 접수 | 접수 번호 발급, 접수 완료 알림 발송 |
| 파트너 | 이의신청 상태 확인 | 접수완료/검토중/처리완료/반려 상태 확인 |
| 관리자 | 이의신청 접수 | 관리자 정산 이의 목록에 추가 |
| 관리자 | 알림 발송 | "정산 이의신청이 접수되었습니다. [파트너명]" |
| 관리자 | 검토 및 처리 | 이의신청 내용 검토 후 승인/반려 (3영업일 이내) |
| 관리자 | 정산 수정 | 승인 시 정산 내역 자동 수정 |

## 시스템 흐름

```
파트너 이의신청 제출 → 접수 번호 발급 → 관리자 알림 → 관리자 검토 (3영업일) → 승인/반려 처리 → 파트너 결과 알림 → (승인 시) 정산 내역 수정 → 증빙 재발행
```

## 이의신청 상태

| 상태 | 설명 |
|---|---|
| 접수 완료 | 이의신청 접수 완료, 검토 대기 중 |
| 검토 중 | 이의신청 내용 검토 진행 중 |
| 처리 완료 | 이의신청 처리 완료, 정산 내역 수정됨 |
| 반려 | 이의신청 반려, 사유 안내 |

## 알림

| 대상 | 시점 | 알림 내용 |
|---|---|---|
| 파트너 | 접수 시 | "이의신청이 접수되었습니다. 검토 후 결과를 안내드리겠습니다." |
| 파트너 | 처리 완료 시 | "이의신청이 승인되었습니다. 정산 내역이 수정되었습니다." |
| 파트너 | 반려 시 | "이의신청이 반려되었습니다. [반려 사유]" |
| 관리자 | 접수 시 | "정산 이의신청이 접수되었습니다. [파트너명]" |
