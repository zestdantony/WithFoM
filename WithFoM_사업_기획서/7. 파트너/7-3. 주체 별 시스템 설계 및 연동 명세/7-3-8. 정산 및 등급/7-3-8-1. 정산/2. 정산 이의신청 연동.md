　　　**1.2.** 정산 이의신청 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 이의신청 버튼 클릭 | 정산 내역 이의신청 화면 진입 |
| 사유 및 증빙 입력 | 이의신청 사유 작성 및 증빙자료 첨부 |
| 제출 버튼 클릭 | 이의신청 접수 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 관리자 | 이의신청 접수 | 관리자 정산 이의 목록에 추가 |
| 관리자 | 알림 발송 | "정산 이의신청이 접수되었습니다. [파트너명]" |
| 관리자 | 검토 및 처리 | 이의신청 내용 검토 후 승인/반려 |

## 시스템 흐름

```
파트너 이의신청 제출 → 관리자 알림 → 관리자 검토 → 승인/반려 처리 → 파트너 결과 알림
```

## 알림

| 대상 | 알림 내용 |
|---|---|
| 파트너 | "이의신청이 접수되었습니다. 검토 후 결과를 안내드리겠습니다." |
| 파트너 | (처리 완료 시) "이의신청이 [승인/반려]되었습니다." |
| 관리자 | "정산 이의신청이 접수되었습니다. [파트너명]" |
