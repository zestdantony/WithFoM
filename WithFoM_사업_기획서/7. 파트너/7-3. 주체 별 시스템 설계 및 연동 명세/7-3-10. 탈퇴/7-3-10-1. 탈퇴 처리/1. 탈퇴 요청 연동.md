　　　**1.1.** 탈퇴 요청 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 탈퇴 메뉴 접근 | 파트너 포털 > 계정 설정 > 탈퇴 |
| 탈퇴 사유 선택 | 탈퇴 사유 선택/입력 (서비스 불만족, 비용 부담, 매장 운영 중단, 다른 서비스 이용, 기타) |
| 비밀번호 재입력 | 탈퇴 확인을 위한 비밀번호 재입력 |
| 탈퇴 확인 체크박스 | 탈퇴 확인 체크박스 선택 필수 |
| 탈퇴 요청 버튼 클릭 | 탈퇴 요청 제출 |

## 탈퇴 불가 조건

| 조건 | 메시지 | 처리 방법 |
|---|---|---|
| 미처리 주문 존재 | "처리되지 않은 주문이 있습니다." | 주문 관리에서 모든 주문 처리 완료 |
| 미정산 금액 존재 | "정산되지 않은 금액이 있습니다." | 정산 관리에서 정산 완료 대기 |
| 미확정 청구 금액 존재 | "미결제 금액이 있습니다." | 결제 관리에서 미결제 건 결제 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 파트너 | 탈퇴 제한 조건 확인 | 탈퇴 불가 조건 자동 확인 및 안내 |
| 파트너 | 탈퇴 완료 알림 | 탈퇴 처리 완료 및 유예 기간 안내 |
| 관리자 | 탈퇴 요청 접수 | 관리자 탈퇴 요청 목록에 추가 |
| 관리자 | 알림 발송 | "파트너 탈퇴 요청이 접수되었습니다. [파트너명]" |
| 관리자 | 검토 및 처리 | 탈퇴 조건 확인 후 승인/반려 |
| 소비자 | 매장 비노출 | (탈퇴 승인 시) 소비자 앱에서 매장 비노출 |

## 시스템 흐름

```
파트너 탈퇴 요청 → 탈퇴 조건 확인 → 조건 불충족 시 안내 → 조건 충족 시 탈퇴 처리 → 계정 비활성화 → 30일 유예 기간 시작 → 유예 기간 종료 → 데이터 영구 삭제
```

## 30일 유예 기간

| 기간 | 상태 | 처리 |
|---|---|---|
| 탈퇴 신청 즉시 | 로그인 차단, 서비스 이용 중단 | 정산 조회만 제한적 가능 |
| 0~30일 | 유예 기간 | 정산 관련 데이터 보관, 분쟁 대응 가능 |
| 30일 경과 | 데이터 영구 삭제 | Hard Delete 방식, 복구 불가 |

## 보관 데이터

| 구분 | 데이터 | 처리 |
|---|---|---|
| 삭제 대상 | 계정 정보, 매장 정보, 사업자 정보, 프로필 | 영구 삭제 |
| 보존 대상 | 주문/결제 이력, 정산 증빙, 계약 문서 | 비식별화 처리 후 법정 보관 기간 동안 보관 |

## 재가입 안내

| 항목 | 내용 |
|---|---|
| 재가입 가능 여부 | 탈퇴 후 재가입 가능 |
| 이전 데이터 연동 | 동일 사업자등록번호로 재가입 시 이전 데이터 연동 없음 |
| 처리 방식 | 신규 가입으로 처리 |
