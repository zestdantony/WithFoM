　　**10-1.1.** 탈퇴 요청 연동

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 탈퇴 메뉴 접근 | 설정 > 계정 관리 > 탈퇴 |
| 탈퇴 사유 선택 | 탈퇴 사유 선택/입력 |
| 탈퇴 요청 버튼 클릭 | 탈퇴 요청 제출 |

## 탈퇴 불가 조건

| 조건 | 메시지 |
|---|---|
| 미처리 주문 존재 | "처리되지 않은 주문이 있습니다." |
| 미정산 금액 존재 | "정산되지 않은 금액이 있습니다." |
| 계약 기간 미만 | "최소 계약 기간(2년) 전에는 탈퇴 시 인센티브 반환이 필요합니다." |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 관리자 | 탈퇴 요청 접수 | 관리자 탈퇴 요청 목록에 추가 |
| 관리자 | 알림 발송 | "파트너 탈퇴 요청이 접수되었습니다. [파트너명]" |
| 관리자 | 검토 및 처리 | 탈퇴 조건 확인 후 승인/반려 |
| 소비자 | 매장 비노출 | (탈퇴 승인 시) 소비자 앱에서 매장 비노출 |

## 시스템 흐름

```
파트너 탈퇴 요청 → 탈퇴 조건 확인 → 관리자 알림 → 관리자 검토 → 승인/반려 → (승인 시) 30일 유예 기간 시작
```

## 30일 유예 기간

| 기간 | 상태 |
|---|---|
| 0~30일 | 탈퇴 철회 가능, 매장 비노출 |
| 30일 경과 | 데이터 영구 삭제 |
