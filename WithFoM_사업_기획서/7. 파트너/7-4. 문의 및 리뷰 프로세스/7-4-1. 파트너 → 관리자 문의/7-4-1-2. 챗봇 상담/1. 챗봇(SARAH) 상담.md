**2.1.** 챗봇(SARAH) 상담

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 도움말·문의 메뉴 접근 | 사이드바에서 도움말·문의 선택 |
| 챗봇 상담 선택 | SARAH 챗봇 화면 진입 |
| 질문 입력 | 궁금한 내용 텍스트 입력 |

## 챗봇 처리 흐름

| 단계 | 내용 |
|---|---|
| 1단계 | FAQ 기반 자동 응답 시도 |
| 2단계 | AI 분석 기반 응답 생성 |
| 3단계 | 해결 불가 시 1:1 문의 전환 안내 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 관리자 | 상담 로그 | 챗봇 상담 내역 기록 (AI 학습용) |
| 관리자 | 1:1 문의 전환 | 챗봇 해결 불가 시 자동 1:1 문의 연결 |

## 시스템 흐름

```
파트너 질문 입력 → AI 분석 → FAQ/AI 응답 → (해결 불가 시) 1:1 문의 전환 → 관리자 로그 기록
```

## 1:1 문의 전환 조건

| 조건 |
|---|
| 파트너가 "상담원 연결" 요청 |
| 동일 질문 3회 반복 |
| 챗봇이 "해결 불가" 판단 |
