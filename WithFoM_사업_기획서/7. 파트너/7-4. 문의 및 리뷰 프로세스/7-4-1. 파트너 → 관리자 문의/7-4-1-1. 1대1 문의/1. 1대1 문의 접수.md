　　**1-1.1.** 1:1 문의 접수

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 도움말·문의 메뉴 접근 | 사이드바에서 도움말·문의 선택 |
| 1:1 문의하기 선택 | 문의 작성 화면 진입 |
| 문의 내용 작성 | 카테고리 선택, 제목, 내용, 첨부파일 입력 |
| 제출 버튼 클릭 | 문의 접수 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 관리자 | 문의 접수 | CS 대시보드에 새 문의 추가 |
| 관리자 | 알림 발송 | "새로운 1:1 문의가 접수되었습니다. [파트너명]" |
| 관리자 | 담당자 배정 | 카테고리별 담당자 자동/수동 배정 |

## 시스템 흐름

```
파트너 문의 작성 → 서버 접수 → 관리자 알림 → 담당자 배정 → 답변 대기
```
