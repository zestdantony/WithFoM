　　**1-1.3.** 추가 문의 및 재문의

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 답변 확인 후 추가 문의 | 동일 건에 대해 추가 질문 작성 |
| 재문의 버튼 클릭 | 추가 문의 전송 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 관리자 | 추가 문의 알림 | "추가 문의가 등록되었습니다. [파트너명]" |
| 관리자 | 문의 상태 변경 | "답변 완료" → "추가 문의" 상태로 변경 |

## 시스템 흐름

```
파트너 추가 문의 작성 → 서버 접수 → 문의 상태 변경 → 관리자 알림 → 추가 답변 대기
```
