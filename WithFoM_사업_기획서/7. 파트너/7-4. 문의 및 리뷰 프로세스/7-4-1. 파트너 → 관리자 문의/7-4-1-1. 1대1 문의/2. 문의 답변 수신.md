# 2: 문의 답변 수신

## 관리자 동작

| 동작 | 내용 |
|---|---|
| 문의 확인 | CS 대시보드에서 문의 내용 확인 |
| 답변 작성 | 답변 내용 입력 |
| 답변 발송 | 파트너에게 답변 전송 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 파트너 | 답변 알림 | "문의에 대한 답변이 등록되었습니다." |
| 파트너 | 답변 확인 | 앱 내 문의 내역에서 답변 확인 가능 |

## 시스템 흐름

```
관리자 답변 작성 → 서버 저장 → 파트너 알림 발송 → 파트너 답변 확인
```

## 알림

| 대상 | 알림 내용 |
|---|---|
| 파트너 | "문의에 대한 답변이 등록되었습니다. 확인해주세요." (푸시/앱 내 알림) |
