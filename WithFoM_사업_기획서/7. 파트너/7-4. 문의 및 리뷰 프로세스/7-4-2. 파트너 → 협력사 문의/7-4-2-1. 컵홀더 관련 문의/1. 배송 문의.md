**1.1.** 배송 문의

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 컵홀더 발주 메뉴 접근 | 발주 내역 확인 |
| 배송 문의 버튼 클릭 | 배송 관련 문의 화면 진입 |
| 문의 내용 작성 | 배송 지연, 오배송, 누락 등 문의 |
| 제출 버튼 클릭 | 문의 접수 |

## 문의 유형

| 유형 |
|---|
| 배송 지연 |
| 오배송 |
| 수량 누락 |
| 배송 일정 변경 요청 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 협력사 | 문의 접수 | 협력사 CS 시스템에 문의 전달 |
| 협력사 | 알림 발송 | "배송 문의가 접수되었습니다. [파트너명]" |
| 관리자 | 문의 로그 | 파트너-협력사 문의 내역 모니터링 |

## 시스템 흐름

```
파트너 배송 문의 → 서버 접수 → 협력사 알림 → 협력사 답변 → 파트너 답변 수신
```

## 알림

| 대상 | 알림 내용 |
|---|---|
| 파트너 | "배송 문의가 접수되었습니다." |
| 협력사 | "배송 문의가 접수되었습니다. [파트너명], [발주번호]" |
