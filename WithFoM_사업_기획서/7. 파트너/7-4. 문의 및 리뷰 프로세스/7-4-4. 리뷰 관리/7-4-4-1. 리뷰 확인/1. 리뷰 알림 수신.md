　　**4-1.1.** 리뷰 알림 수신

## 소비자 동작

| 동작 | 내용 |
|---|---|
| 주문 완료 후 | 리뷰 작성 화면 안내 |
| 별점 및 리뷰 작성 | 평점, 텍스트 리뷰, 사진 입력 |
| 리뷰 등록 | 리뷰 제출 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 파트너 | 리뷰 알림 | "새로운 리뷰가 등록되었습니다." |
| 파트너 | 리뷰 목록 | 리워드 > 리뷰 관리에서 확인 |
| 관리자 | AI 검수 | 부적절한 리뷰 AI 자동 필터링 |
| 관리자 | 리뷰 통계 | 파트너별 평균 평점, 리뷰 수 통계 |

## 시스템 흐름

```
소비자 리뷰 등록 → AI 검수 → 리뷰 공개 → 파트너 알림 → 파트너 리뷰 확인
```
