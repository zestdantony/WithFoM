　　**4-1.2.** 리뷰 답글 작성

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 리뷰 관리 메뉴 접근 | 리워드 > 리뷰 관리 선택 |
| 리뷰 확인 | 소비자 리뷰 내용 확인 |
| 답글 작성 | 답글 내용 입력 |
| 답글 등록 | 답글 게시 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 소비자 | 답글 알림 | "리뷰에 매장 답글이 등록되었습니다." |
| 소비자 | 답글 확인 | 앱 내 리뷰 상세에서 답글 확인 |
| 관리자 | 답글 로그 | 파트너 답글 활동 통계 |

## 시스템 흐름

```
파트너 답글 작성 → 서버 저장 → 소비자 알림 → 리뷰 상세에 답글 표시
```
