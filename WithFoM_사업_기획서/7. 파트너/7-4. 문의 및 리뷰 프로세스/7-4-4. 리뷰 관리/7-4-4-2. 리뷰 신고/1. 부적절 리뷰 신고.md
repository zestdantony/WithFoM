**2.1.** 부적절 리뷰 신고

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 리뷰 관리 메뉴 접근 | 리워드 > 리뷰 관리 선택 |
| 신고 버튼 클릭 | 부적절 리뷰 신고 |
| 신고 사유 선택 | 신고 사유 선택 및 상세 설명 입력 |
| 신고 제출 | 신고 접수 |

## 신고 사유

| 사유 |
|---|
| 허위 사실 |
| 비방/욕설 |
| 광고/스팸 |
| 관련 없는 내용 |
| 기타 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 관리자 | 신고 접수 | 관리자 리뷰 신고 목록에 추가 |
| 관리자 | 알림 발송 | "리뷰 신고가 접수되었습니다." |
| 관리자 | 검토 및 처리 | 신고 내용 검토 후 리뷰 블라인드/유지 결정 |

## 시스템 흐름

```
파트너 리뷰 신고 → 서버 접수 → 관리자 알림 → 관리자 검토 → 블라인드/유지 처리 → 파트너 결과 알림
```

## 알림

| 대상 | 알림 내용 |
|---|---|
| 파트너 | "리뷰 신고가 접수되었습니다. 검토 후 결과를 안내드리겠습니다." |
| 파트너 | (처리 완료 시) "신고하신 리뷰가 [블라인드/유지] 처리되었습니다." |
| 관리자 | "리뷰 신고가 접수되었습니다. [파트너명]" |
