　　　**1.2.** 리워드 문의 답변

## 파트너 동작

| 동작 | 내용 |
|---|---|
| 고객 문의 메뉴 접근 | 리워드 > 고객 문의 선택 |
| 문의 확인 | 소비자 문의 내용 확인 |
| 답변 작성 | 답변 내용 입력 |
| 답변 발송 | 소비자에게 답변 전송 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 소비자 | 답변 알림 | "문의에 대한 답변이 등록되었습니다." |
| 소비자 | 답변 확인 | 앱 내 문의 내역에서 답변 확인 |
| 관리자 | 답변 로그 | 답변 시간, 내용 모니터링 |

## 시스템 흐름

```
파트너 답변 작성 → 서버 저장 → 소비자 알림 → 소비자 답변 확인
```
