# 1: 주문 문의 수신

## 소비자 동작

| 동작 | 내용 |
|---|---|
| 주문 내역 접근 | 소비자 앱에서 주문 내역 선택 |
| 문의하기 선택 | 해당 주문 문의 화면 진입 |
| 문의 내용 작성 | 주문 관련 문의 |
| 제출 버튼 클릭 | 문의 접수 |

## 문의 유형

| 유형 |
|---|
| 주문 취소 요청 |
| 주문 변경 요청 |
| 준비 시간 문의 |
| 주문 누락 |
| 기타 주문 문의 |

## 연동 요약

| 연동 대상 | 연동 항목 | 영향 |
|---|---|---|
| 파트너 | 문의 알림 | "새로운 주문 문의가 있습니다." (푸시/앱 내 알림) |
| 파트너 | 문의 목록 | 메뉴 오더 > 고객 문의에서 확인 |
| 관리자 | 문의 로그 | 소비자-파트너 문의 내역 모니터링 |

## 시스템 흐름

```
소비자 문의 작성 → 서버 접수 → 파트너 알림 → 파트너 답변 → 소비자 답변 수신
```

## 알림

| 대상 | 알림 내용 |
|---|---|
| 파트너 | "새로운 주문 문의가 있습니다. [주문번호]" |
| 소비자 | "문의가 접수되었습니다." |
