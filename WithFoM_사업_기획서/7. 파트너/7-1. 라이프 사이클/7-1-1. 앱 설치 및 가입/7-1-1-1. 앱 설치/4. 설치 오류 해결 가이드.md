　　**1.4.** 설치 오류 해결 가이드  

　　**1.4.1.** 개요  

앱 설치 중 발생할 수 있는 일반적인 오류와 해결 방법을 안내합니다.  

　　**1.4.2.** 일반적인 설치 오류  

　　**1.4.2.1.** 저장 공간 부족  

**증상:**  
• "저장 공간이 부족합니다" 메시지 표시  
• 설치가 중간에 중단됨  


**해결 방법:**  

| 단계 | 조치 |  
|---|---|  
| 1 | 불필요한 앱 삭제 또는 사진/동영상 백업 후 삭제 |  
| 2 | 캐시 데이터 정리 (설정 > 저장 공간 > 캐시 정리) |  
| 3 | 최소 200MB 이상 여유 공간 확보 |  
| 4 | 앱 재설치 시도 |  


　　**1.4.2.2.** 네트워크 오류  

**증상:**  
• "네트워크 연결 오류" 메시지  
• 다운로드가 진행되지 않음  


**해결 방법:**  

| 단계 | 조치 |  
|---|---|  
| 1 | Wi-Fi 연결 확인 및 재연결 |  
| 2 | 모바일 데이터 사용 중인 경우 데이터 잔량 확인 |  
| 3 | 비행기 모드 켜기 → 10초 대기 → 비행기 모드 끄기 |  
| 4 | 기기 재시작 후 재시도 |  


　　**1.4.2.3.** 앱 스토어에서 검색되지 않음  

**증상:**  
• "위드폼 파트너" 검색 시 결과 없음  
• 앱 페이지 접근 불가  


**해결 방법:**  

| 단계 | 조치 |  
|---|---|  
| 1 | 정확한 앱 이름 확인 ("위드폼 파트너" 정확히 입력) |  
| 2 | 앱 스토어 업데이트 확인 |  
| 3 | 앱 스토어 캐시 삭제 후 재검색 |  
| 4 | 웹사이트에서 직접 다운로드 링크 사용 |  


　　**1.4.2.4.** 설치 중 중단  

**증상:**  
• 설치 진행 중 갑자기 중단  
• 설치 아이콘만 표시되고 진행되지 않음  


**해결 방법:**  

| 단계 | 조치 |  
|---|---|  
| 1 | 기기 재시작 |  
| 2 | 앱 스토어 업데이트 확인 |  
| 3 | 설치 중인 앱 취소 후 재시도 |  
| 4 | 기기 시간/날짜 설정 확인 (자동 설정 권장) |  


　　**1.4.3.** iOS 특정 오류  

　　**1.4.3.1.** "신뢰할 수 없는 엔터프라이즈 개발자" 오류  

**증상:**  
• 설치 후 실행 시 위 메시지 표시  


**해결 방법:**  
1: 설정 > 일반 > VPN 및 기기 관리  
2: 개발자 인증서 확인  
3: "신뢰" 버튼 클릭  


　　**1.4.3.2.** App Store 로그인 필요  

**증상:**  
• "로그인 필요" 메시지  


**해결 방법:**  
1: App Store에 Apple ID로 로그인  
2: Settings > App Store > 계정 확인  


　　**1.4.4.** Android 특정 오류  

　　**1.4.4.1.** "앱을 설치할 수 없습니다"  

**증상:**  
• 설치 버튼 클릭 시 위 메시지 표시  


**해결 방법:**  

| 단계 | 조치 |  
|---|---|  
| 1 | 설정 > 보안 > 알 수 없는 소스 설치 허용 (Android 8.0 이하) |  
| 2 | 설정 > 앱 > 특수 앱 액세스 > 알 수 없는 앱 설치 (Android 8.0 이상) |  
| 3 | Google Play Protect 확인 (설정 > Google > Play Protect) |  


　　**1.4.4.2.** "이 앱은 기기와 호환되지 않습니다"  

**증상:**  
• Google Play에서 호환성 오류 메시지  


**해결 방법:**  
1: Android 버전 확인 (설정 > 휴대전화 정보)  
2: 최소 요구사항 (Android 10.0 이상) 확인  
3: 기기 업데이트 필요 시 OS 업그레이드  


　　**1.4.5.** 추가 조치  

　　**1.4.5.1.** 모든 방법을 시도해도 해결되지 않는 경우  

1: **고객센터 문의**  
　　a. 앱 내 문의: 설정 > 고객센터 > 1:1 문의  
　　b. 이메일: partner@withfom.com  
　　c. 전화: 1588-0000 (평일 09:00~18:00)  


2: **문의 시 제공할 정보**  
　　a. 기기 모델명 및 OS 버전  
　　b. 오류 메시지 스크린샷  
　　c. 오류 발생 시점 및 재현 방법  


　　**1.4.6.** 참고  

• 앱 설치 전 기기 재시작을 권장합니다  
• 정기적인 OS 업데이트로 호환성 문제를 예방할 수 있습니다  
• Wi-Fi 환경에서 설치하면 데이터 요금을 절감할 수 있습니다  


• 참조: 경쟁사(배달의민족, 요기요) 설치 오류 가이드 참고
