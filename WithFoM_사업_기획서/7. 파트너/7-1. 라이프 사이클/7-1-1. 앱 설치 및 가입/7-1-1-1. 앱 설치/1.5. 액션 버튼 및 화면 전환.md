**1.5.** 액션 버튼 및 화면 전환


**1.5.1.** 앱 다운로드 화면 전환

1: 앱 스토어 이동
　A. 트리거: QR 코드 스캔 또는 다운로드 링크 클릭
　B. 이동 경로: App Store (iOS) 또는 Google Play Store (Android)
　C. 이동 방식: 외부 앱 실행 (앱 스토어)
　D. 딥링크: 위드폼 파트너 앱 상세 페이지로 직접 이동

2: 웹사이트에서 앱 스토어 이동
　A. 트리거: 위드폼 파트너 홈페이지 > "앱 다운로드" 버튼 클릭
　B. 기기 감지: User-Agent를 통해 iOS/Android 자동 감지
　C. 이동 경로: 감지된 기기에 맞는 앱 스토어로 자동 이동
　D. 수동 선택: iOS/Android 선택 버튼 제공


**1.5.2.** 설치 완료 후 화면 전환

1: 앱 최초 실행
　A. 트리거: 설치된 위드폼 파트너 앱 아이콘 탭
　B. 이동 경로: 앱 스플래시 화면 → 온보딩 화면 또는 로그인 화면
　C. 신규 사용자: 온보딩 화면 → 이용 약관 동의 화면
　D. 기존 사용자: 로그인 화면

2: 온보딩 화면 구성
　A. 스와이프 또는 [다음] 버튼으로 페이지 이동
　B. 페이지 1: 위드폼 파트너 서비스 소개
　C. 페이지 2: 캠페인 매칭 및 수익 안내
　D. 페이지 3: 리워드/오더 기능 안내
　E. [건너뛰기] 버튼: 온보딩 스킵 → 이용 약관 동의 화면으로 이동
　F. [시작하기] 버튼: 이용 약관 동의 화면으로 이동


**1.5.3.** 권한 요청 화면 전환

1: 권한 요청 팝업 표시
　A. 트리거: 앱 최초 실행 시 또는 특정 기능 사용 시
　B. 권한 요청 순서: 알림 → 위치 정보 → 카메라 (필요 시)

2: 권한 응답에 따른 화면 전환

| 권한 | 허용 시 | 거부 시 |
|---|---|---|
| 알림 | 다음 권한 요청 또는 다음 화면으로 이동 | 안내 메시지 표시 후 다음 권한 요청 |
| 위치 정보 | 다음 권한 요청 또는 다음 화면으로 이동 | 서비스 이용 불가 안내 팝업 표시 |
| 카메라 | 해당 기능 사용 가능 | 해당 기능 사용 시 재요청 안내 |

3: 위치 정보 권한 거부 시 처리
　A. 안내 팝업: "위치 정보 권한이 필요합니다"
　B. [설정으로 이동] 버튼: 기기 설정 앱으로 이동
　C. [나중에] 버튼: 제한된 기능으로 앱 사용 (캠페인 매칭 불가)


**1.5.4.** 설치 오류 시 화면 전환

1: 설치 실패 안내
　A. 앱 스토어에서 오류 메시지 표시
　B. [다시 시도] 버튼: 설치 재시도
　C. [취소] 버튼: 앱 스토어 메인 화면으로 이동

2: 호환성 오류 안내
　A. "이 앱은 기기와 호환되지 않습니다" 메시지 표시
　B. 최소 요구사항 안내 표시
　C. [확인] 버튼: 앱 스토어 메인 화면으로 이동


**1.5.5.** 다음 단계로의 화면 전환

1: 앱 설치 → 이용 약관 동의
　A. 트리거: 온보딩 완료 또는 [시작하기] 버튼 클릭
　B. 이동 경로: 7-1-1-2. 이용 약관 및 개인정보 처리방침 동의 화면
　C. 전환 애니메이션: 슬라이드 (좌 → 우)

2: 진행 상태 표시
　A. 상단에 진행 단계 인디케이터 표시
　B. 현재 단계: 1/9 (앱 설치)
　C. 다음 단계 예고: "다음: 이용 약관 동의"


**1.5.6.** 뒤로가기 동작

1: 온보딩 화면에서 뒤로가기
　A. 이전 온보딩 페이지로 이동
　B. 첫 페이지에서 뒤로가기: 앱 종료 확인 팝업

2: 권한 요청 팝업에서 뒤로가기
　A. 팝업 닫힘 (권한 거부와 동일하게 처리)
　B. 다음 권한 요청으로 진행


**1.5.7.** 모바일 환경 고려사항

1: 터치 인터랙션
　A. 버튼 탭: 해당 동작 실행
　B. 스와이프: 온보딩 페이지 전환
　C. 롱 프레스: 없음

2: 화면 전환 애니메이션
　A. 온보딩 페이지 전환: 슬라이드 (좌 → 우)
　B. 권한 팝업: 바텀 시트 형태로 표시
　C. 다음 단계 이동: 슬라이드 (좌 → 우)


• 참조: 7-1-1-2. 이용 약관 및 개인정보 처리방침 동의
