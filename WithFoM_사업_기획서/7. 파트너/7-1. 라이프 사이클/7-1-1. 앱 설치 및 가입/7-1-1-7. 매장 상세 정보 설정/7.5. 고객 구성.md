**7.5.** 고객 구성  


↳ 매장 상세 정보 설정 단계 중 다섯 번째 화면  

**진행 단계 표시:** 8단계 중 5단계  


**7.5.1.** 상단 영역  

**7.5.1.1.** 제목: "고객 구성 입력"  

**7.5.1.2.** 설명 텍스트: "매장 고객의 성별 및 연령대 비율을 입력해 주세요"  

**7.5.1.3.** 진행 단계 표시: "8단계 중 5단계" (우측 상단)  


**7.5.2.** 메인 입력 영역  

**7.5.2.1.** 연령대별 입력 섹션  
　1: 연령대: 10대, 20대, 30대, 40대, 50대+  
　2: 각 연령대별 입력 섹션 구성  
　　A. 연령대명 표시 (예: "10대")  
　　B. 성별 비율 슬라이더 (0%~100%)  
　　　a. 슬라이더 왼쪽: 남성  
　　　b. 슬라이더 오른쪽: 여성  
　　　c. 슬라이더 조정으로 남/여 비율 입력  
　　　d. 슬라이더를 왼쪽으로 이동하면 남성 비율 증가, 오른쪽으로 이동하면 여성 비율 증가  

　　C. 현재 비율 표시: "남성: N%, 여성: M%" (슬라이더 하단)  

　3: 슬라이더 동작  
　　A. 각 연령대별 슬라이더는 독립적으로 조작 가능  
　　B. 슬라이더 조작 시 실시간으로 비율 변경 및 표시  
　　C. 슬라이더 위치에 따라 남/여 비율이 자동 계산되어 표시됨  

**7.5.2.2.** 총 남/여 비율 표시 영역  
　1: 위치: 모든 연령대 입력 섹션 하단  
　2: 표시 형식: "총 남성 비율: N%, 총 여성 비율: M%"  
　3: 표시 조건: 모든 연령대의 남여 비율 입력 완료 시 자동 계산 및 표시  


**7.5.3.** 하단 영역  

**7.5.3.1.** "다음 단계" 버튼  
　1: 버튼 상태  
　　A. 비활성화 시: 회색 (#CCCCCC), 클릭 불가  
　　B. 활성화 시: 파란색 (#0066FF), 클릭 가능  
　　C. 조건: 모든 연령대의 남여 비율 입력 완료 및 총 남/여 비율 표시 시 활성화  

　2: 버튼 클릭 시  
　　A. 다음 화면(음료 3종 가격 입력)으로 이동  
　　B. 입력한 정보는 자동 저장됨  


**7.5.4.** 동작 흐름  

**7.5.4.1.** 화면 진입  
　1: 각 연령대별 입력 섹션 표시 (10대, 20대, 30대, 40대, 50대+)  
　2: 모든 슬라이더 중앙 위치 (50%:50%)  
　3: 총 남/여 비율 표시 영역은 아직 표시되지 않음  
　4: "다음 단계" 버튼 비활성화 상태  

**7.5.4.2.** 연령대별 남여 비율 입력  
　1: 슬라이더 조작으로 남/여 비율 입력  
　　A. 슬라이더를 왼쪽으로 이동: 남성 비율 증가, 여성 비율 감소  
　　B. 슬라이더를 오른쪽으로 이동: 여성 비율 증가, 남성 비율 감소  
　　C. 슬라이더 위치에 따라 남/여 비율이 자동 계산되어 표시됨  

　2: 슬라이더 조작 시 실시간으로 비율 변경 및 표시  
　　A. 현재 비율 표시 업데이트: "남성: N%, 여성: M%"  

　3: 모든 연령대의 남여 비율 입력 완료 시  
　　A. 총 남/여 비율 자동 계산  
　　B. 총 남성 비율 = (10대 남성 비율 × 10대 전체 비중) + (20대 남성 비율 × 20대 전체 비중) + ...  
　　C. 총 여성 비율 = (10대 여성 비율 × 10대 전체 비중) + (20대 여성 비율 × 20대 전체 비중) + ...  
　　D. 계산된 총 남/여 비율이 하단에 표시됨  
　　E. "다음 단계" 버튼 활성화  

**7.5.4.3.** 다음 단계 진행  
　1: 모든 연령대 입력 완료 및 총 남/여 비율 표시 시 "다음 단계" 버튼 클릭 가능  
　2: 버튼 클릭 시 다음 화면(음료 3종 가격 입력)으로 이동  
　3: 입력한 정보는 자동 저장됨  


**7.5.5.** 입력 사항 활용 (파트너용 설명)  

**7.5.5.1.** 입력한 고객 구성 정보는 매장에 맞는 캠페인 매칭에 활용되며, 광고주가 타깃 고객층을 설정할 때 이를 바탕으로 적합한 캠페인이 매칭됩니다  

**7.5.5.2.** 고객 구성 정보는 매장 리포트에도 활용되어 고객 분석에 도움이 됩니다  


**7.5.6.** 입력 사항 활용 (개발자용)  

**7.5.6.1.** 캠페인 매칭  
　1: 광고주 타깃 연령대/성별과 매칭하여 적합한 캠페인 추천  
　2: 각 연령대별 남/여 비율 정보를 활용하여 정밀 매칭  
　3: 총 남/여 비율 정보를 활용하여 전체적인 매칭 우선순위 결정  

**7.5.6.2.** 데이터 분석  
　1: 고객 구성별 이용 패턴 및 선호도 분석  
　2: 연령대별 및 성별별 벤치마크 비교  
　3: 매장 리포트 생성 시 고객 구성 정보 활용
