**7.2.** 리워드/오더 선택

**진행 단계 표시:** 8단계 중 2단계

**7.2.1.1.** 제목: "서비스 타입 선택"  

**7.2.1.2.** 설명 텍스트: "매장에서 제공할 서비스 타입을 선택해 주세요 (선택 사항)"  

**7.2.1.3.** 진행 단계 표시: "8단계 중 2단계" (우측 상단)  


**7.2.2.** 메인 선택 영역  

**7.2.2.1.** 서비스 타입 카드 (2개)  
　1: 카드 1: 리워드형  
　　A. 카드 구성  
　　　a. 서비스 타입명: "리워드형" (큰 글씨, 상단 중앙)  
　　　b. 기능 상태: "리워드 기능 (☑️): 사용 가능"  
　　　c. 설명 텍스트: "QR 스캔 후 게임/리워드만 제공, 앱 주문 불가, 메뉴 오더/관리 미노출"  
　　　d. 체크박스: 좌측 상단에 체크박스 표시  

　　B. 카드 선택 시  
　　　a. 체크박스 선택 표시 (체크마크)  
　　　b. 테두리 색상 변경: 회색 → 파란색 (#0066FF)  
　　　c. 배경색 약간 변경: 밝은 파란색 배경 (#F0F7FF)  

　2: 카드 2: 오더형  
　　A. 카드 구성  
　　　a. 서비스 타입명: "오더형" (큰 글씨, 상단 중앙)  
　　　b. 기능 상태: "오더 기능 (☑️): 사용 가능"  
　　　c. 설명 텍스트: "앱 주문·결제만 제공, 게임/리워드 없음"  
　　　d. 등급 산정 안내: "오더형 사용 시 파트너 등급 산정에 도움이 됩니다." (작은 글씨, 파란색, 별표 아이콘)  
　　　e. 체크박스: 좌측 상단에 체크박스 표시  

　　B. 카드 선택 시  
　　　a. 체크박스 선택 표시 (체크마크)  
　　　b. 테두리 색상 변경: 회색 → 파란색 (#0066FF)  
　　　c. 배경색 약간 변경: 밝은 파란색 배경 (#F0F7FF)  

**7.2.2.2.** 선택 방식  
　1: 각 카드에 체크박스 표시  
　2: 체크박스 클릭으로 선택/해제 가능  
　3: 하나만 선택 시: 각각의 기능만 적용  
　　A. 리워드형만 선택: 리워드 기능만 활성화, 오더 기능 비활성화  
　　B. 오더형만 선택: 오더 기능만 활성화, 리워드 기능 비활성화  

　4: 둘 다 선택 시: 복합형으로 적용  
　　A. 리워드와 오더 기능 모두 활성화  
　　B. 쿠폰/포인트 결제 시 자동 적용  

　5: 둘 다 미선택 시: 선택 없이 다음 단계로 진행 가능  
　　A. 리워드 기능은 리워드 설정 페이지에서 나중에 활성화 가능  
　　B. 오더 기능은 주문 관리 페이지에서 나중에 활성화 가능  
　　C. 미선택 상태에서도 캠페인(컵홀더) 기능은 정상 이용 가능  


**7.2.3.** 하단 영역  

**7.2.3.1.** "다음 단계" 버튼  
　1: 버튼 상태  
　　A. 항상 활성화: 파란색 (#0066FF), 클릭 가능  
　　B. 선택 여부와 관계없이 다음 단계로 진행 가능  

　2: 버튼 클릭 시  
　　A. 다음 화면(상권 입력)으로 이동  
　　B. 선택한 정보는 자동 저장되며, 해당 기능들이 활성화됨  
　　C. 미선택 시: 리워드/오더 기능 모두 비활성화 상태로 저장  

**7.2.3.2.** 미선택 시 안내 문구  
　1: 표시 조건  
　　A. 두 카드 모두 미선택 상태에서 "다음 단계" 버튼 클릭 시 하단에 안내 문구 표시  
　2: 안내 문구 내용  
　　A. '리워드 기능은 리워드 설정 페이지에서,  
오더 기능은 주문 관리 페이지에서 나중에 활성화할 수 있습니다.'  
　3: 안내 문구 디자인  
　　A. 배경: 연한 파란색 (#F0F7FF)  
　　B. 아이콘: 정보(ℹ️) 아이콘  
　　C. 글자색: 진한 회색 (#666666)  


**7.2.4.** 동작 흐름  

**7.2.4.1.** 화면 진입  
　1: 두 가지 서비스 타입 카드 표시 (리워드형, 오더형)  
　2: 각 카드에 체크박스 표시 (미선택 상태)  
　3: "다음 단계" 버튼 활성화 상태 (미선택으로도 진행 가능)  

**7.2.4.2.** 서비스 타입 선택  
　1: 카드 내 체크박스 클릭으로 선택/해제  
　2: 리워드형만 선택 시: 리워드 관련 메뉴 및 화면 표시  
　3: 오더형만 선택 시: 오더 관련 메뉴 및 화면 표시  
　　A. 등급 산정 안내 문구 강조 표시  

　4: 둘 다 선택 시: 리워드와 오더 관련 메뉴 및 화면 모두 표시  
　5: 미선택 시에도 "다음 단계" 버튼 활성화 (진행 가능)  
　　A. 안내 문구 표시: 리워드 설정 페이지 / 주문 관리 페이지에서 나중에 선택 가능  

**7.2.4.3.** 다음 단계 진행  
　1: "다음 단계" 버튼 클릭  
　2: 선택한 정보 저장 및 해당 기능 활성화 (미선택 시 비활성화 상태로 저장)  
　3: 다음 화면(상권 입력)으로 이동  
　4: 미선택 시 안내 문구 노출 후 이동  


**7.2.5.** 입력 사항 활용 (파트너용 설명)  

**7.2.5.1.** 리워드형 선택 시  
　1: 고객에게 리워드를 제공하여 고객 만족도를 높이고 재방문을 유도할 수 있습니다  
　2: QR 스캔 후 게임을 통해 고객 참여를 높이고 매장 브랜딩 효과를 얻을 수 있습니다  

**7.2.5.2.** 오더형 선택 시  
　1: 편리하고 간편한 결제 시스템으로 주문 처리를 효율화할 수 있습니다  
　　A. 앱을 통한 주문·결제로 대기 시간을 줄이고 주문 오류를 최소화합니다  
　　B. 무인 주문 시스템으로 인력 부담을 줄일 수 있습니다  

　2: 파트너 등급 산정에 도움이 됩니다  
　　A. 위드폼 오더 이용 여부가 등급 산정 기준 중 하나로 포함됩니다  
　　B. 오더형 사용 시 등급 산정 점수(10% 가중치)에 반영되어 더 높은 등급을 받을 수 있습니다  

**7.2.5.3.** 복합형 선택 시  
　1: 리워드와 오더 기능을 모두 사용할 수 있어 더 다양한 서비스를 제공할 수 있습니다  
　2: 고객에게 리워드를 제공하면서 동시에 편리한 주문·결제 서비스를 제공할 수 있습니다  
　3: 쿠폰/포인트 결제 시 자동 적용되어 고객 편의성을 높일 수 있습니다  


**7.2.6.** 입력 사항 활용 (개발자용)  

**7.2.6.1.** 서비스 기능 활성화

　　B. 오더형 선택 시: 오더 관련 API 및 화면 활성화  
　　C. 복합형 선택 시: 리워드와 오더 관련 API 및 화면 모두 활성화  
　　D. 미선택 시: 리워드/오더 기능 모두 비활성화 상태로 저장 (service_mode: 'none')  
　　　a. 리워드 기능: 리워드 설정 페이지에서 추후 활성화 가능  
　　　b. 오더 기능: 주문 관리 페이지에서 추후 활성화 가능  

**7.2.6.2.** 메뉴 관리  
　1: 오더형/복합형 선택 시: 메뉴 등록 및 관리 기능 활성화  
　2: 리워드형만 선택 시: 메뉴 관리 기능 비활성화  

**7.2.6.3.** 정산

　2: 리워드 사용 내역, 오더 결제 내역 등 타입별 정산 데이터 수집  

**7.2.6.4.** 화면 구성

　　A. 리워드형 선택 시: 리워드 사용 내역 섹션, 리워드 승인 기능 등 표시  
　　B. 오더형 선택 시: 결제 내역 섹션, 주문 승인 기능 등 표시  
　　C. 복합형 선택 시: 리워드와 오더 관련 섹션 및 기능 모두 표시

　　B. 등급 산정 시 10% 가중치로 반영
