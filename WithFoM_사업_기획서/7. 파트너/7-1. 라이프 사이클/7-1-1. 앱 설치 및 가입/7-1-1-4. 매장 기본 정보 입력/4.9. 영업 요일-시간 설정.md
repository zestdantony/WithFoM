　　**4.9.** 영업 요일/시간 설정  


　　　　　1: 영업 요일/시간 설정 보드  
　　　　　　A. 월요일부터 일요일까지 가로로 월/화/수/목/금/토/일의 클릭 가능한 작은 버튼이 있는 보드  
　　　　　　B. 각 요일 버튼을 클릭하면 해당 버튼이 파란색으로 변함  
　　　　　　C. 요일 버튼 클릭 시 입력창 생성  
　　　　　　　　a. 오픈 시간 입력창  
　　　　　　　　b. 마감 시간 입력창  
　　　　　　　　c. 쉬는 시간 시작 시간 입력창  
　　　　　　　　d. 쉬는 시간 끝나는 시간 입력창  
　　　　　　　　e. 총 4개의 입력창이 보드 내의 월~금 버튼 아래 생성  

　　　　　2: 영업 시간 입력 및 확인  
　　　　　　A. 각 입력창의 시간을 입력  
　　　　　　B. 입력창 아래에 확인 버튼 클릭  
　　　　　　C. 확인 버튼 클릭 시 해당 요일의 영업 시간이 생성되어 입력 보드 아래 새로운 보드(영업 요일/시간 표시 보드) 안에 표시  

　　　　　3: 요일별 일괄 설정  
　　　　　　A. 월, 화, 수, 목, 금 버튼을 눌러도 입력창은 요일별로 5개가 뜨는 것이 아니라 똑같이 하나의 입력창만 표시  
　　　　　　B. 입력 시 누른 요일 모두가 한 번에 세팅됨  

　　　　　4: 개별 요일 설정  
　　　　　　A. 토요일 버튼을 누르고 다른 시간대를 입력 후 확인을 누르면 토요일만 다른 시간대로 세팅  
　　　　　　B. 같은 시간대를 입력하면 월~토요일이 같은 시간대로 세팅  

　　　　　5: 휴무일 설정  
　　　　　　A. 일요일은 입력하지 않고 넘어가면 일요일은 휴무일로 세팅  

　　　　　6: 영업 요일/시간 표시 보드  
　　　　　　A. 세로로 월~일까지 정렬  
　　　　　　B. 영업 요일/시간 표시 보드에서는 수정 불가하고 표시만 됨  

　　　　　7: 영업 시간 수정  
　　　　　　A. 수정 시에는 영업 요일/시간 설정 보드에서 원하는 요일을 다시 누르고 입력 후 확인을 누름  
　　　　　　B. 확인 버튼 클릭 시 영업 요일/시간 표시 보드에 변경되어 표시됨  

　　　　　8: 설정 완료  
　　　　　　A. 영업 요일/시간 표시 보드에 표시됨과 동시에 영업 요일/시간은 설정됨  

　　　　　9: 24시간 매장 설정  
　　　　　　A. 24시간 매장의 경우 마감 시간을 00:00으로 고정
