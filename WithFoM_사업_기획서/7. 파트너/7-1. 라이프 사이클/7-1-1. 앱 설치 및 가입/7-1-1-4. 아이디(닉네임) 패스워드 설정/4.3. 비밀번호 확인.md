**4.3.** 비밀번호 확인  

↳ 아이디(닉네임) 패스워드 설정 단계 중 세 번째 입력 영역  


**4.3.1.** 입력 필드 구성  

**4.3.1.1.** 제목: "비밀번호 확인"  

**4.3.1.2.** 입력창: 보안 텍스트 입력 (기본 숨김 처리 ●●●●●●●●)  

**4.3.1.3.** 플레이스홀더: "비밀번호를 한 번 더 입력해 주세요"  

**4.3.1.4.** 비밀번호 보기/숨기기 토글: 눈 아이콘 (입력창 우측)  


**4.3.2.** 비밀번호 재입력  

**4.3.2.1.** 설정한 비밀번호와 동일하게 재입력  

**4.3.2.2.** 실시간 일치 여부 검사: 비밀번호 필드와 문자 단위 비교  


**4.3.3.** 일치 여부 표시 (실시간)  

**4.3.3.1.** 입력 중 실시간으로 비밀번호 필드와 비교  

**4.3.3.2.** 표시 상태  
　1: 일치: "비밀번호가 일치합니다" (초록색 체크 아이콘)  
　2: 불일치: "비밀번호가 일치하지 않습니다" (빨간색 X 아이콘)  
　3: 미입력: 표시 없음  


**4.3.3.3.** 일치 확인 후에만 [다음] 버튼 활성화에 기여  


**4.3.4.** 비밀번호 설정 필드 변경 시  

**4.3.4.1.** 비밀번호 설정 (4.2) 필드를 수정하면 비밀번호 확인 필드 초기화  

**4.3.4.2.** 일치 상태도 초기화 (재입력 필요)  


**4.3.5.** 키보드 인터랙션  

**4.3.5.1.** 키보드 타입: 보안 키보드  

**4.3.5.2.** Return 키: 키보드 숨김, [다음] 버튼 포커스  

**4.3.5.3.** 자동완성 비활성화 (보안)  


**4.3.6.** [다음] 버튼 활성화 기여 조건  

**4.3.6.1.** 아이디 중복 확인 완료 (사용 가능) — 4.1 영역  

**4.3.6.2.** 비밀번호 강도 "강함" 이상 — 4.2 영역  

**4.3.6.3.** 비밀번호 확인 일치 — 4.3 영역  

**4.3.6.4.** 세 가지 조건 모두 충족 시 [다음] 버튼 활성화
