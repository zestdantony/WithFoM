**4.2.** 비밀번호 설정  

↳ 아이디(닉네임) 패스워드 설정 단계 중 두 번째 입력 영역  


**4.2.1.** 입력 필드 구성  

**4.2.1.1.** 제목: "비밀번호 입력"  

**4.2.1.2.** 입력창: 보안 텍스트 입력 (기본 숨김 처리 ●●●●●●●●)  

**4.2.1.3.** 플레이스홀더: "영문 대/소문자, 숫자, 특수문자 중 3가지 이상 조합"  

**4.2.1.4.** 비밀번호 보기/숨기기 토글: 눈 아이콘 (입력창 우측)  

**4.2.1.5.** 비밀번호 강도 표시 바: 입력창 하단  


**4.2.2.** 비밀번호 규칙  

**4.2.2.1.** 최소 8자 이상  

**4.2.2.2.** 영문 대문자, 영문 소문자, 숫자, 특수문자 중 3가지 이상 조합 필수  

**4.2.2.3.** 허용 특수문자: !@#$%^&*(),.?":{}|<>  

**4.2.2.4.** 공백 사용 불가  


**4.2.3.** 비밀번호 강도 표시 (실시간)  

**4.2.3.1.** 입력 중 실시간으로 강도 측정 및 표시  

**4.2.3.2.** 강도 기준  
　1: 약함: 8자 미만 또는 1가지 종류만 사용  
　　A. UI: 빨간색 바 (1/3)  
　　B. 메시지: "8자 이상 입력해주세요" 또는 "2가지 이상 조합해주세요"  
　2: 보통: 8자 이상, 2가지 종류 조합  
　　A. UI: 노란색 바 (2/3)  
　　B. 메시지: "보통 강도의 비밀번호입니다"  
　3: 강함: 8자 이상, 3가지 이상 종류 조합  
　　A. UI: 초록색 바 (3/3)  
　　B. 메시지: "강력한 비밀번호입니다"  


**4.2.3.3.** "강함" 이상일 때만 다음 단계 진행 가능  


**4.2.4.** 비밀번호 보기/숨기기 토글  

**4.2.4.1.** 눈 아이콘 탭: 비밀번호 표시/숨김 전환  

**4.2.4.2.** 기본 상태: 숨김 (●●●●●●●●)  

**4.2.4.3.** 보기 상태: 입력한 텍스트 그대로 표시  


**4.2.5.** 보안 가이드  

**4.2.5.1.** 다른 사이트와 동일한 비밀번호 사용 자제 권장  

**4.2.5.2.** 주기적 비밀번호 변경 권장 안내  

**4.2.5.3.** 안내 텍스트: 입력 필드 하단에 작은 글씨로 표시 (회색)  


**4.2.6.** 키보드 인터랙션  

**4.2.6.1.** 키보드 타입: 보안 키보드  

**4.2.6.2.** Return 키: 비밀번호 확인 필드로 포커스 이동  

**4.2.6.3.** 자동완성 비활성화 (보안)
