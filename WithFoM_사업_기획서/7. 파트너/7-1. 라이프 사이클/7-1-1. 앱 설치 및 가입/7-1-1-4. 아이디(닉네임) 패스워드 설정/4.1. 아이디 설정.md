**4.1.** 아이디 설정  

↳ 아이디(닉네임) 패스워드 설정 단계 중 첫 번째 입력 영역  
**진행 단계 표시:** 4/9 단계  


**4.1.1.** 입력 필드 구성  

**4.1.1.1.** 제목: "아이디 입력"  

**4.1.1.2.** 입력창: 텍스트 입력 (단일 필드)  

**4.1.1.3.** 플레이스홀더: "영문, 숫자 조합 4~20자"  

**4.1.1.4.** [중복 확인] 버튼: 입력창 우측에 배치  


**4.1.2.** 아이디 형식 규칙  

**4.1.2.1.** 최소 4자 이상, 최대 20자 이하  

**4.1.2.2.** 영문(대/소문자), 숫자 조합만 허용  

**4.1.2.3.** 특수문자 사용 불가  

**4.1.2.4.** 한글 사용 불가  

**4.1.2.5.** 공백 사용 불가  


**4.1.3.** 실시간 유효성 검사  

**4.1.3.1.** 입력 중 실시간으로 형식 검증  

**4.1.3.2.** 형식 미달 시 입력창 하단에 안내 메시지 표시  
　1: 4자 미만: "아이디는 4자 이상이어야 합니다"  
　2: 20자 초과: "아이디는 20자 이하이어야 합니다"  
　3: 특수문자/한글 입력: "영문, 숫자만 사용 가능합니다"  


**4.1.3.3.** 안내 메시지 색상: 빨간색  


**4.1.4.** 중복 확인  

**4.1.4.1.** [중복 확인] 버튼 활성화 조건  
　1: 아이디 형식 유효 시에만 활성화  


**4.1.4.2.** 버튼 클릭 시 서버에 아이디 중복 여부 조회  

**4.1.4.3.** 결과 표시  
　1: 사용 가능: "사용 가능한 아이디입니다" (초록색 체크 아이콘)  
　2: 중복: "이미 사용 중인 아이디입니다" (빨간색 X 아이콘)  
　3: 형식 오류: "영문, 숫자 조합으로 입력해주세요" (빨간색 경고 아이콘)  


**4.1.4.4.** 중복 확인 완료 후 아이디 수정 시 중복 확인 상태 초기화 (재확인 필요)  


**4.1.5.** 아이디 활용 안내  

**4.1.5.1.** 아이디는 로그인에 사용  

**4.1.5.2.** 관리자가 파트너를 검색할 때 사용  

**4.1.5.3.** 소비자에게 노출되는 것은 매장명이며, 매장명은 별도로 입력  


**4.1.6.** 아이디 수정 정책  

**4.1.6.1.** 아이디는 가입 후 수정 불가  

**4.1.6.2.** 가입 전 화면에서는 자유롭게 변경 가능  


**4.1.7.** 키보드 인터랙션  

**4.1.7.1.** 키보드 타입: 기본 (영문/숫자)  

**4.1.7.2.** Return 키: 비밀번호 입력 필드로 포커스 이동
