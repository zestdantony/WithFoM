# 3: 계약서 다운로드 및 이메일 수신

## 개요

계약서 서명이 완료되면 서명된 계약서를 다운로드하거나 이메일로 받을 수 있습니다.

## 계약서 다운로드

### 다운로드 방법

| 방법 | 설명 |
|---|---|
| 앱 내 다운로드 | 앱 내에서 직접 PDF 파일 다운로드 |
| 다운로드 버튼 | "계약서 다운로드" 버튼 클릭 |
| 파일 저장 | 기기 저장 공간에 PDF 파일 저장 |

### 다운로드 파일 정보

| 항목 | 내용 |
|---|---|
| 파일 형식 | PDF (Portable Document Format) |
| 파일명 | WithFoM_파트너_계약서_[매장명]_[날짜].pdf |
| 파일 크기 | 약 500KB ~ 1MB |
| 포함 내용 | 계약서 전체 내용 및 서명 |

### 다운로드 절차

1. 서명 완료 후 "계약서 다운로드" 버튼 클릭
2. 다운로드 진행 상태 확인
3. 다운로드 완료 확인
4. 파일 저장 위치 확인

## 이메일 수신

### 이메일 자동 발송

| 항목 | 내용 |
|---|---|
| 발송 시점 | 계약서 서명 완료 즉시 |
| 수신 주소 | 가입 시 입력한 이메일 주소 |
| 발송 내용 | 서명된 계약서 PDF 파일 첨부 |

### 이메일 내용

| 항목 | 내용 |
|---|---|
| 제목 | [위드폼 파트너] 파트너 계약서 |
| 본문 | 계약서 서명 완료 안내 및 계약서 첨부 |
| 첨부 파일 | 서명된 계약서 PDF 파일 |

### 이메일 수신 확인

| 확인 방법 | 내용 |
|---|---|
| 이메일 수신함 확인 | 등록한 이메일 주소의 수신함 확인 |
| 스팸 폴더 확인 | 스팸 폴더에 분류되었을 수 있음 |
| 이메일 재발송 | 미수신 시 "이메일 재발송" 버튼 클릭 |

## 계약서 보관

### 보관 방법

| 방법 | 설명 |
|---|---|
| 앱 내 보관 | 앱 내에서 계약서 확인 및 보관 |
| 다운로드 보관 | 기기에 다운로드하여 보관 |
| 이메일 보관 | 이메일로 받은 계약서 보관 |
| 클라우드 보관 | 클라우드 스토리지에 업로드하여 보관 (선택) |

### 보관 권장사항

| 항목 | 내용 |
|---|---|
| 다중 보관 | 여러 방법으로 보관 권장 |
| 백업 | 중요한 문서이므로 백업 권장 |
| 접근 가능성 | 필요 시 쉽게 접근할 수 있는 위치에 보관 |

## 계약서 재다운로드

### 재다운로드 방법

| 방법 | 설명 |
|---|---|
| 앱 내 메뉴 | 설정 > 계약서 > 계약서 다운로드 |
| 이메일 재발송 | 설정 > 계약서 > 이메일 재발송 |

### 재다운로드 시점

| 시점 | 내용 |
|---|---|
| 계약서 분실 시 | 계약서를 분실한 경우 |
| 백업 필요 시 | 추가 백업이 필요한 경우 |
| 이메일 미수신 시 | 이메일을 받지 못한 경우 |

## 다운로드 및 이메일 수신 오류 해결

### 다운로드 오류

| 오류 | 해결 방법 |
|---|---|
| 다운로드 실패 | 네트워크 연결 확인 후 재시도 |
| 저장 공간 부족 | 저장 공간 확보 후 재시도 |
| 파일 손상 | 재다운로드 시도 |

### 이메일 미수신

| 원인 | 해결 방법 |
|---|---|
| 이메일 주소 오류 | 이메일 주소 확인 및 수정 |
| 스팸 분류 | 스팸 폴더 확인 |
| 이메일 서버 오류 | 이메일 재발송 요청 |

## 계약서 확인

### 다운로드한 계약서 확인

| 확인 항목 | 내용 |
|---|---|
| 파일 열기 | PDF 뷰어로 파일이 정상적으로 열리는지 확인 |
| 내용 확인 | 계약서 내용이 정상적으로 표시되는지 확인 |
| 서명 확인 | 서명이 정상적으로 표시되는지 확인 |

### 이메일로 받은 계약서 확인

| 확인 항목 | 내용 |
|---|---|
| 첨부 파일 확인 | 이메일에 PDF 파일이 첨부되었는지 확인 |
| 파일 열기 | 첨부 파일을 열어 내용 확인 |
| 서명 확인 | 서명이 정상적으로 표시되는지 확인 |

## 참고

- 계약서는 법적 구속력을 가지는 중요한 문서입니다
- 계약서를 안전하게 보관하시기 바랍니다
- 계약서 분실 시 재다운로드 또는 이메일 재발송을 요청할 수 있습니다
