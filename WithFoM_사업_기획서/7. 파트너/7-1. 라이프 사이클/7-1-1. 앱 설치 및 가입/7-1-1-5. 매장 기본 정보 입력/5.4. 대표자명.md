**5.4.** 대표자명  

↳ 매장 기본 정보 입력 단계 중 네 번째 표시 영역 (자동 기입)  


**5.4.1.** 표시 필드 구성  

**5.4.1.1.** 제목: "대표자명"  

**5.4.1.2.** 표시 방식: 텍스트 (읽기 전용)  

**5.4.1.3.** 배경색: 연한 회색 (자동 기입 표시)  

**5.4.1.4.** 잠금 아이콘 표시 (수정 불가)  


**5.4.2.** 자동 기입 조건  

**5.4.2.1.** 본인 인증(7-1-1-3) 단계에서 확인된 이름이 자동 기입  

**5.4.2.2.** 자동 기입된 정보는 수정 불가 (본인 인증 정보 고정)  


**5.4.3.** 자동 기입 정보  

| 항목 | 출처 | 수정 가능 여부 |
|---|---|---|
| 대표자명 | 7-1-1-3. 본인 인증 → 인증된 이름 | 수정 불가 |


**5.4.4.** 대표자명 활용  

**5.4.4.1.** 계약서 서명 시 대표자명으로 사용  

**5.4.4.2.** 정산 관련 서류에 표시  


**5.4.5.** 참고사항  

**5.4.5.1.** 대표자명 변경 필요 시 가입 완료 후 고객센터를 통해 변경 요청  

**5.4.5.2.** 사업자등록증의 대표자명과 본인 인증 이름이 다를 경우 사업자 인증 단계에서 확인
