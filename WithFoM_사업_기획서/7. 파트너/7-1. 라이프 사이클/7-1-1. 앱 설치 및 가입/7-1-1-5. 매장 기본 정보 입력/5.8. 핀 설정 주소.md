**5.8.** 핀 설정 주소  

↳ 매장 기본 정보 입력 단계 중 여덟 번째 입력 영역  


**5.8.1.** 핀 설정 구성  

**5.8.1.1.** 제목: "매장 위치 핀 설정"  

**5.8.1.2.** [핀 설정] 버튼: 매장 주소(5.7) 입력 완료 후 활성화  

**5.8.1.3.** 지도 미리보기: 설정된 핀 위치 표시  


**5.8.2.** 핀 설정 방법  

**5.8.2.1.** [핀 설정] 버튼 클릭 시 지도 화면 표시  

**5.8.2.2.** 기본 위치: 매장 주소 기반 좌표로 자동 설정  

**5.8.2.3.** 핀 드래그: 정확한 매장 위치로 미세 조정 가능  

**5.8.2.4.** [확인] 버튼 클릭 시 핀 위치 저장  


**5.8.3.** 핀 위치 표시  

**5.8.3.1.** 설정 완료 후 지도 미리보기에 핀 표시  

**5.8.3.2.** 위도/경도 좌표 저장 (내부 데이터)  


**5.8.4.** 핀 설정 활용  

**5.8.4.1.** 소비자 앱에서 매장 위치 확인에 활용  

**5.8.4.2.** 배송 경로 산정에 활용  


**5.8.5.** 핀 수정  

**5.8.5.1.** [핀 재설정] 버튼으로 언제든 재설정 가능  

**5.8.5.2.** 매장 주소 변경 시 핀 위치 초기화 후 재설정 필요
