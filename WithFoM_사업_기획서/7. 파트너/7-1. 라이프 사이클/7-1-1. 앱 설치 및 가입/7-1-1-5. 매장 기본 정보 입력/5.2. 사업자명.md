**5.2.** 사업자명  

↳ 매장 기본 정보 입력 단계 중 두 번째 표시 영역 (자동 기입)  


**5.2.1.** 표시 필드 구성  

**5.2.1.1.** 제목: "사업자명"  

**5.2.1.2.** 표시 방식: 텍스트 (읽기 전용)  

**5.2.1.3.** 기본 상태: 비어 있음 (사업자 인증 전)  

**5.2.1.4.** 배경색: 연한 회색 (자동 기입 표시)  

**5.2.1.5.** 잠금 아이콘 표시 (수정 불가)  


**5.2.2.** 자동 표시 조건  

**5.2.2.1.** 사업자 인증 완료 시 국세청 API 연동을 통해 자동 조회  

**5.2.2.2.** 사업자등록번호에 해당하는 사업자명이 자동 표시  

**5.2.2.3.** 수동 입력 불가 (자동 표시 전용 필드)  


**5.2.3.** 필드 상태 표시  

**5.2.3.1.** 인증 전: "사업자 인증 후 자동 표시됩니다" 안내 문구 (회색)  

**5.2.3.2.** 인증 완료: 사업자명 텍스트 표시 + 잠금 아이콘  

**5.2.3.3.** 수정 불가 필드로 비활성화 처리
