**5.7.** 매장 주소  

↳ 매장 기본 정보 입력 단계 중 일곱 번째 입력 영역  


**5.7.1.** 입력 필드 구성  

**5.7.1.1.** 제목: "매장 주소"  

**5.7.1.2.** 주소 표시창: 텍스트 (읽기 전용, 주소 검색으로만 입력)  

**5.7.1.3.** [주소 검색] 버튼: 입력창 우측에 배치  

**5.7.1.4.** 상세 주소 입력창: 주소 검색 완료 후 활성화  

**5.7.1.5.** 필수 입력 항목  


**5.7.2.** 주소 검색 방식  

**5.7.2.1.** [주소 검색] 버튼 클릭 시 주소 검색 모달 표시  

**5.7.2.2.** 검색 방식: 도로명 주소 검색 (카카오/다음 주소 API)  

**5.7.2.3.** 검색 결과 선택 시 주소 필드에 자동 입력  

**5.7.2.4.** 모달 형태: 풀스크린 또는 대형 바텀 시트  

**5.7.2.5.** 검색 결과 스크롤 가능  


**5.7.3.** 상세 주소 입력  

**5.7.3.1.** 주소 검색 완료 후 상세 주소 입력 필드 활성화  

**5.7.3.2.** 플레이스홀더: "상세 주소를 입력해주세요 (예: 1층)"  

**5.7.3.3.** 선택 입력 항목  


**5.7.4.** 매장 주소 활용  

**5.7.4.1.** 컵홀더 배송 주소로 사용  

**5.7.4.2.** 캠페인 매칭 시 지역 기반 매칭에 활용  

**5.7.4.3.** 소비자 앱에서 매장 위치 표시  


**5.7.5.** 키보드 인터랙션  

**5.7.5.1.** 상세 주소 입력 시 키보드 타입: 기본 (한글/영문)  

**5.7.5.2.** Return 키: 키보드 숨김, 핀 설정으로 포커스 이동
