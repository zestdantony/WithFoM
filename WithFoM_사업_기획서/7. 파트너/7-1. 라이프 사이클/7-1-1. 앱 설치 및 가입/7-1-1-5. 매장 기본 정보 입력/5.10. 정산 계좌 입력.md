**5.10.** 정산 계좌 입력  

↳ 매장 기본 정보 입력 단계 중 열 번째 입력 영역  


**5.10.1.** 은행 선택  

**5.10.1.1.** 제목: "은행 선택"  

**5.10.1.2.** 선택 방식: 드롭다운  

**5.10.1.3.** 주요 은행 상단 표시 (KB국민, 신한, 우리, 하나, 기업, 농협 등)  

**5.10.1.4.** 필수 선택 항목  


**5.10.2.** 계좌번호 입력  

**5.10.2.1.** 제목: "계좌번호"  

**5.10.2.2.** 입력창: 숫자 입력 (단일 필드)  

**5.10.2.3.** 플레이스홀더: "계좌번호를 입력해주세요"  

**5.10.2.4.** 숫자만 입력 가능  

**5.10.2.5.** 은행별 계좌번호 형식 검사  


**5.10.3.** 예금주 확인  

**5.10.3.1.** 예금주명 입력 또는 자동 조회  

**5.10.3.2.** 대표자명과 일치 여부 확인  

**5.10.3.3.** 1원 인증을 통해 계좌 실명 확인 진행  

**5.10.3.4.** 확인 결과 표시  
　1: 일치: "계좌가 확인되었습니다" (초록색 체크 아이콘)  
　2: 불일치: "예금주 정보가 일치하지 않습니다" (빨간색 X 아이콘)  


**5.10.4.** 정산 계좌 활용  

**5.10.4.1.** 캠페인 정산 금액 입금 계좌  

**5.10.4.2.** 인센티브 지급 계좌  


**5.10.5.** 정산 계좌 수정 정책  

**5.10.5.1.** 가입 후에도 정산 관리에서 변경 가능  

**5.10.5.2.** 변경 시 본인 인증 재확인 필요  


**5.10.6.** 키보드 인터랙션  

**5.10.6.1.** 키보드 타입: 숫자 전용  

**5.10.6.2.** Return 키: 키보드 숨김
