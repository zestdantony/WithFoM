**5.9.** 영업 요일-시간 설정  

↳ 매장 기본 정보 입력 단계 중 아홉 번째 입력 영역  


**5.9.1.** 영업 요일 선택  

**5.9.1.1.** 제목: "영업 요일"  

**5.9.1.2.** 선택 방식: 월~일 체크박스 형태  

**5.9.1.3.** 복수 선택 가능  

**5.9.1.4.** 빠른 선택 버튼  
　1: [매일] 버튼: 전체 요일 일괄 선택  
　2: [평일만] 버튼: 월~금 선택  


**5.9.2.** 영업 시간 설정  

**5.9.2.1.** 시작 시간 선택: 드롭다운 또는 타임피커  

**5.9.2.2.** 종료 시간 선택: 드롭다운 또는 타임피커  

**5.9.2.3.** [24시간 영업] 토글: 활성화 시 시간 선택 비활성화  

**5.9.2.4.** 요일별 다른 영업 시간 설정 가능  
　1: [요일별 설정] 토글 활성화 시 각 요일마다 개별 시간 입력 필드 표시  


**5.9.3.** 브레이크 타임 설정 (선택)  

**5.9.3.1.** [브레이크 타임 추가] 토글  

**5.9.3.2.** 활성화 시 시작/종료 시간 선택 필드 추가  

**5.9.3.3.** 브레이크 타임 중 위드폼 오더 주문 불가 처리  


**5.9.4.** 영업 시간 활용  

**5.9.4.1.** 소비자 앱에서 영업 중/영업 종료 표시  

**5.9.4.2.** 위드폼 오더 이용 시 주문 가능 시간 제한  

**5.9.4.3.** 컵홀더 배송 일정 산정에 참고  


**5.9.5.** 영업 시간 수정 정책  

**5.9.5.1.** 가입 후에도 매장 정보 수정에서 변경 가능  

**5.9.5.2.** 변경 시 소비자 앱에 실시간 반영
