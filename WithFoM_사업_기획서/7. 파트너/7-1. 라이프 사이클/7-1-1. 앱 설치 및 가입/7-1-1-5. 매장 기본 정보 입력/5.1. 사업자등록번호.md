**5.1.** 사업자등록번호  

↳ 매장 기본 정보 입력 단계 중 첫 번째 입력 영역  
**진행 단계 표시:** 5/9 단계  


**5.1.1.** 입력 필드 구성  

**5.1.1.1.** 제목: "사업자등록번호"  

**5.1.1.2.** 입력창: 숫자 입력 (단일 필드)  

**5.1.1.3.** 플레이스홀더: "사업자등록번호 10자리 입력"  

**5.1.1.4.** 하이픈 자동 포맷팅: 입력 시 123-45-67890 형식으로 자동 변환  


**5.1.2.** 입력 형식 규칙  

**5.1.2.1.** 숫자 10자리  

**5.1.2.2.** 하이픈 포함 입력 가능 (자동 제거 후 포맷팅)  
　1: 하이픈 포함: 123-45-67890  
　2: 하이픈 미포함: 1234567890  


**5.1.2.3.** 두 형식 모두 입력 가능하며, 시스템이 자동으로 형식 통일  


**5.1.3.** 실시간 유효성 검사  

**5.1.3.1.** 10자리 입력 시 형식 유효성 자동 확인  

**5.1.3.2.** 사업자등록번호 체크섬 검증  

**5.1.3.3.** 형식 오류 시 안내 메시지 표시  
　1: 10자리 미달: "사업자등록번호 10자리를 입력해주세요" (빨간색)  
　2: 체크섬 오류: "올바른 사업자등록번호를 입력해주세요" (빨간색)  


**5.1.4.** 사업자 인증 단계 연동  

**5.1.4.1.** 입력한 번호가 7-1-1-6. 사업자 인증 화면에 자동 반영  

**5.1.4.2.** 중복 입력 없이 바로 인증 절차 진행 가능  

**5.1.4.3.** 사업자 인증 단계에서 번호 수정 가능  

**5.1.4.4.** 사업자 인증 완료 시 사업자명(5.2) 자동 표시  


**5.1.5.** 키보드 인터랙션  

**5.1.5.1.** 키보드 타입: 숫자 전용  

**5.1.5.2.** Return 키: 다음 필드(매장명)로 포커스 이동
