**5.3.** 매장명  

↳ 매장 기본 정보 입력 단계 중 세 번째 입력 영역  


**5.3.1.** 입력 필드 구성  

**5.3.1.1.** 제목: "매장명"  

**5.3.1.2.** 입력창: 텍스트 입력 (단일 필드)  

**5.3.1.3.** 플레이스홀더: "소비자에게 표시될 매장명을 입력해주세요"  

**5.3.1.4.** 필수 입력 항목  


**5.3.2.** 매장명 규칙  

**5.3.2.1.** 한글, 영문, 숫자, 일부 특수문자 허용  

**5.3.2.2.** 최소 1자 이상, 최대 30자 이하  

**5.3.2.3.** 사업자명과 다를 수 있음  


**5.3.3.** 매장명 활용  

**5.3.3.1.** 소비자 앱에서 매장 검색 및 표시에 사용  

**5.3.3.2.** 컵홀더 캠페인 매칭 시 표시되는 매장명  

**5.3.3.3.** 파트너 포털에서도 매장명으로 표시  


**5.3.4.** 매장명 수정 정책  

**5.3.4.1.** 가입 후에도 매장 정보 수정에서 변경 가능  

**5.3.4.2.** 변경 시 소비자 앱에 실시간 반영  


**5.3.5.** 키보드 인터랙션  

**5.3.5.1.** 키보드 타입: 기본 (한글/영문)  

**5.3.5.2.** Return 키: 다음 필드(이메일)로 포커스 이동
