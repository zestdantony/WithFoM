**5.6.** 연락처  

↳ 매장 기본 정보 입력 단계 중 여섯 번째 입력 영역 (자동 기입, 수정 가능)  


**5.6.1.** 입력 필드 구성  

**5.6.1.1.** 제목: "연락처"  

**5.6.1.2.** 입력창: 전화번호 입력 (단일 필드)  

**5.6.1.3.** 플레이스홀더: "010-0000-0000"  

**5.6.1.4.** 배경색: 연한 회색 (자동 기입 표시) → 수정 시 기본 배경색으로 변경  

**5.6.1.5.** 편집 아이콘 표시 (수정 가능)  


**5.6.2.** 자동 기입 조건  

**5.6.2.1.** 본인 인증(7-1-1-3) 단계에서 확인된 휴대폰 번호가 자동 기입  

**5.6.2.2.** 매장 대표 연락처로 사용  


**5.6.3.** 자동 기입 정보  

| 항목 | 출처 | 수정 가능 여부 |
|---|---|---|
| 연락처 | 7-1-1-3. 본인 인증 → 인증된 휴대폰 번호 | 수정 가능 |


**5.6.4.** 연락처 수정  

**5.6.4.1.** 매장 대표 연락처가 본인 인증 번호와 다를 경우 수정 가능  

**5.6.4.2.** 입력창 클릭 후 새로운 연락처 입력  

**5.6.4.3.** 하이픈 자동 포맷팅: 010-1234-5678  

**5.6.4.4.** 실시간 형식 검사  
　1: 형식 오류 시: "올바른 연락처를 입력해주세요" (빨간색)  


**5.6.5.** 연락처 활용  

**5.6.5.1.** 캠페인 관련 연락 및 배송 안내에 사용  

**5.6.5.2.** 긴급 연락 시 사용  

**5.6.5.3.** 수정된 연락처로 캠페인 관련 연락 발송  


**5.6.6.** 키보드 인터랙션  

**5.6.6.1.** 키보드 타입: 숫자 전용  

**5.6.6.2.** Return 키: 다음 필드(매장 주소)로 포커스 이동
