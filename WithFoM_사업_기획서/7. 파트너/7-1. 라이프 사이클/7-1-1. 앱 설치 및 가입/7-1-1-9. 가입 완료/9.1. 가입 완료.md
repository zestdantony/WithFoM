**9.1.** 가입 완료

↳ 모든 가입 절차 완료 및 서비스 이용 시작 안내  
**진행 단계 표시:** 9/9 단계 (완료)


**9.1.1.** 완료 화면 표시

**9.1.1.1.** 축하 메시지
　1: "가입이 완료되었습니다!" 헤더
　2: 축하 아이콘 또는 애니메이션 (Lottie)
　3: 환영 메시지: "위드폼 파트너가 되신 것을 환영합니다"

**9.1.1.2.** 가입 정보 요약

| 표시 항목 | 내용 |
|---|---|
| 매장명 | 입력한 매장명 |
| 아이디 | 설정한 아이디 (닉네임) |
| 서비스 타입 | 리워드형 / 오더형 / 복합형 |
| 등록일 | 가입 완료 일자 |


**9.1.2.** 승인 대기 안내

**9.1.2.1.** 승인 절차
　1: 가입 완료 → 관리자 승인 대기
　2: 예상 소요 시간: 영업일 기준 1~2일
　3: 승인 완료 알림: 푸시 알림 + 이메일

**9.1.2.2.** 승인 전 이용 가능 기능
　1: 대시보드 확인
　2: 매장 정보 수정
　3: 앱 설정
　4: 고객센터 문의

**9.1.2.3.** 승인 전 이용 불가 기능
　1: 캠페인 매칭
　2: 주문 관리 (오더형)
　3: 정산 관리


**9.1.3.** 서비스 이용 시작

**9.1.3.1.** 대시보드 접속
　1: [대시보드로 이동] 버튼 클릭
　2: 파트너 홈 화면(대시보드)으로 이동
　3: 네비게이션 스택 초기화 (가입 플로우로 돌아가기 불가)

**9.1.3.2.** 대시보드 첫 진입 상태
　1: 상단에 "승인 대기 중" 배너 표시
　2: 일부 기능 비활성화 상태
　3: 승인 완료 시 "승인 대기 중" 배너 자동 제거 및 모든 기능 활성화


**9.1.4.** 알림 설정 확인

**9.1.4.1.** 푸시 알림 권한 확인
　1: 푸시 알림 권한이 비활성화된 경우 안내 팝업 표시
　2: "승인 알림을 받으려면 알림을 허용해주세요"

**9.1.4.2.** 알림 설정 유도
　1: [설정으로 이동] 버튼: 기기 앱 설정 화면으로 이동
　2: [나중에] 버튼: 팝업 닫힘


**9.1.5.** 승인 완료 후 변경사항

**9.1.5.1.** 승인 완료 알림
　1: 푸시 알림: "가입이 승인되었습니다! 지금 바로 서비스를 이용해보세요"
　2: 이메일: 승인 완료 안내 및 시작 가이드

**9.1.5.2.** 대시보드 상태 변경
　1: "승인 대기 중" 배너 제거
　2: 모든 기능 활성화
　3: 캠페인 매칭 시작
