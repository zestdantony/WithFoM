**9.2.** 액션 버튼 및 화면 전환


**9.2.1.** 가입 완료 화면 구성

**9.2.1.1.** 화면 진입
　1: 트리거: 파트너 계약서 서명 완료 후 [다음] 버튼 클릭
　2: 이전 단계: 7-1-1-8. 파트너 계약서 작성
　3: 진행 상태: 9/9 단계 (완료)

**9.2.1.2.** 화면 요소
　1: 완료 축하 메시지 및 아이콘
　2: 가입 정보 요약
　3: 다음 단계 안내
　4: [대시보드로 이동] 버튼


**9.2.2.** 완료 화면 표시 내용

**9.2.2.1.** 축하 메시지
　1: "가입이 완료되었습니다!" 헤더
　2: 축하 아이콘 또는 애니메이션
　3: 환영 메시지: "위드폼 파트너가 되신 것을 환영합니다"

**9.2.2.2.** 가입 정보 요약

| 표시 항목 | 내용 |
|---|---|
| 매장명 | 입력한 매장명 |
| 아이디 | 설정한 아이디 |
| 서비스 타입 | 리워드형 / 오더형 / 복합형 |
| 등록일 | 가입 완료 일자 |


**9.2.3.** 다음 단계 안내

**9.2.3.1.** 승인 대기 안내
　1: "관리자 승인 후 서비스를 이용하실 수 있습니다"
　2: 예상 소요 시간: "영업일 기준 1~2일"
　3: 승인 완료 알림 방법: "푸시 알림 및 이메일로 알려드립니다"

**9.2.3.2.** 승인 전 이용 가능 기능
　1: 대시보드 확인
　2: 매장 정보 수정
　3: 앱 설정


**9.2.4.** [대시보드로 이동] 버튼 동작

**9.2.4.1.** 버튼 클릭 시 동작
　1: 파트너 대시보드(홈) 화면으로 이동
　2: 이동 경로: /partner/dashboard
　3: 네비게이션 스택 초기화 (뒤로가기 시 가입 플로우로 돌아가지 않음)

**9.2.4.2.** 화면 전환
　1: 전환 애니메이션: 페이드 또는 슬라이드
　2: 대시보드 첫 진입 상태로 표시


**9.2.5.** 대시보드 첫 진입 화면

**9.2.5.1.** 승인 대기 상태 표시
　1: 상단에 "승인 대기 중" 배너 표시
　2: 일부 기능 비활성화 상태

**9.2.5.2.** 사용 가능 기능
　1: 매장 정보 확인 및 수정
　2: 알림 설정
　3: 고객센터 문의

**9.2.5.3.** 사용 불가 기능 (승인 전)
　1: 캠페인 매칭
　2: 주문 관리 (오더형)
　3: 정산 관리


**9.2.6.** 추가 버튼 동작

**9.2.6.1.** [매장 정보 확인] 버튼 (선택적)
　1: 동작: 매장 정보 확인 화면으로 이동
　2: 입력한 정보 최종 확인 가능

**9.2.6.2.** [고객센터 문의] 버튼 (선택적)
　1: 동작: 고객센터 화면으로 이동
　2: 승인 관련 문의 가능


**9.2.7.** 뒤로가기 동작

**9.2.7.1.** 가입 완료 화면에서 뒤로가기
　1: 안내 팝업: "가입이 완료되었습니다. 대시보드로 이동하시겠습니까?"
　2: [대시보드로 이동] 버튼: 대시보드 화면으로 이동
　3: 가입 플로우로 돌아가기 불가


**9.2.8.** 푸시 알림 설정 확인

**9.2.8.1.** 알림 권한 확인
　1: 푸시 알림 권한이 비활성화된 경우 안내 팝업
　2: "승인 알림을 받으려면 알림을 허용해주세요"

**9.2.8.2.** 알림 설정 유도
　1: [설정으로 이동] 버튼: 앱 설정 화면으로 이동
　2: [나중에] 버튼: 팝업 닫힘


**9.2.9.** 승인 완료 후 화면 변경

**9.2.9.1.** 승인 완료 알림 수신
　1: 푸시 알림: "가입이 승인되었습니다! 지금 바로 서비스를 이용해보세요"
　2: 이메일: 승인 완료 안내 및 시작 가이드

**9.2.9.2.** 대시보드 상태 변경
　1: "승인 대기 중" 배너 제거
　2: 모든 기능 활성화
　3: 캠페인 매칭 시작


**9.2.10.** 모바일 환경 고려사항

**9.2.10.1.** 완료 화면 레이아웃
　1: 중앙 정렬 축하 메시지 및 아이콘
　2: 스크롤 없이 한 화면에 모든 정보 표시

**9.2.10.2.** 버튼 배치
　1: [대시보드로 이동] 버튼: 화면 하단 고정
　2: 터치하기 쉬운 크기 및 위치


• 참조: 7-2-2. 홈 메인 화면(대시보드)
