# 1: 동의 절차 안내

## 개요

파트너 가입 시 위드폼 서비스 이용을 위해 각종 약관 및 정책에 동의해야 합니다. 본 문서에서는 동의 항목, 동의 절차, 동의 철회 방법 등을 안내합니다.

---

## 동의 항목 구분

### 필수 동의 항목

서비스 이용을 위해 반드시 동의해야 하는 항목입니다.

| 동의 항목 | 설명 | 필수 여부 | 상세 문서 |
|---|---|---|---|
| 서비스 이용 약관 | 위드폼 파트너 서비스 이용 조건 및 규정 | **필수** | 2: 서비스 이용 약관 |
| 개인정보 처리방침 | 개인정보 수집, 이용, 보관, 삭제에 관한 정책 | **필수** | 3: 개인정보 처리방침 |
| 위치 정보 이용 동의 | 매장 위치 기반 서비스 제공을 위한 위치정보 수집 동의 | **필수** | 4: 위치 정보 이용 동의 |
| 제3자 정보 제공 동의 (광고주) | 광고주에게 매장 정보 및 운영 데이터 제공 | **필수** | 6: 제3자 정보 제공 동의 |

### 선택 동의 항목

동의하지 않아도 서비스 이용에 제한이 없는 항목입니다.

| 동의 항목 | 설명 | 필수 여부 | 상세 문서 |
|---|---|---|---|
| 마케팅 정보 수신 동의 | 이메일/SMS/푸시를 통한 마케팅 정보 수신 | 선택 | 5: 마케팅 정보 수신 동의 |
| 제3자 정보 제공 동의 | 광고주(필수), 제휴사(선택) 등 제3자에게 정보 제공 | **광고주 제공 필수** | 6: 제3자 정보 제공 동의 |
| 행태정보 수집 및 이용 동의 | 온라인 활동 정보 수집 및 맞춤형 서비스 제공 | 선택 | 7. 행태정보 수집 및 이용 동의 |
| 쿠키 사용 동의 | 웹 포털 이용 시 쿠키 사용 (선택 쿠키) | 선택 | 8. 쿠키 사용 동의 |
| 앱 권한 동의 | 앱 기능 사용을 위한 기기 권한 허용 | 안내 | 9. 앱 권한 동의 |

## 동의 절차

### 1단계: 약관 확인

| 순서 | 내용 |
|---|---|
| 1 | 각 약관 제목을 클릭하여 상세 내용 확인 |
| 2 | 약관 전문을 끝까지 스크롤하여 읽기 |
| 3 | 약관 내용 이해 후 개별 동의 체크박스 선택 |

※ 약관 내용을 확인하지 않으면 동의 체크박스가 비활성화됩니다.

### 2단계: 동의 선택

| 동의 방법 | 설명 |
|---|---|
| 전체 동의 | 상단의 "전체 동의" 체크박스 클릭 시 모든 항목에 일괄 동의 |
| 개별 동의 | 각 약관별로 개별 동의 체크박스 선택 |
| 필수 동의만 | 필수 동의 항목만 선택 (선택 항목은 미동의) |

### 3단계: 동의 확인 및 진행

| 순서 | 내용 |
|---|---|
| 1 | 동의한 항목 최종 확인 |
| 2 | 필수 동의 항목 모두 체크 확인 (미체크 시 진행 불가) |
| 3 | "동의하고 계속하기" 버튼 클릭 |
| 4 | 다음 단계(아이디 설정)로 이동 |

## 약관 내용 확인 방법

### 앱 내에서 확인

| 경로 | 내용 |
|---|---|
| 가입 화면 | 각 약관명 클릭 시 전문 확인 가능 |
| 설정 > 약관 및 정책 | 가입 후에도 언제든지 약관 확인 가능 |
| 설정 > 개인정보 관리 | 개인정보 관련 정책 상세 확인 |

### 웹사이트에서 확인

| 경로 | URL |
|---|---|
| 서비스 이용 약관 | withfom.com/terms |
| 개인정보 처리방침 | withfom.com/privacy |
| 위치정보 이용약관 | withfom.com/location |

## 동의 철회

### 철회 가능 여부

| 동의 항목 | 철회 가능 여부 | 철회 시 영향 |
|---|---|---|
| 서비스 이용 약관 | ❌ 철회 불가 | 철회 시 서비스 이용 불가 (탈퇴 필요) |
| 개인정보 처리방침 | ❌ 철회 불가 | 철회 시 서비스 이용 불가 (탈퇴 필요) |
| 위치 정보 이용 동의 | ❌ 철회 불가 | 철회 시 서비스 이용 불가 (탈퇴 필요) |
| 제3자 정보 제공 동의 (광고주) | ❌ 철회 불가 | 철회 시 서비스 이용 불가 (탈퇴 필요) |
| 마케팅 정보 수신 동의 | ✅ 철회 가능 | 마케팅 정보 수신 중단 (서비스 이용 가능) |
| 제3자 정보 제공 동의 (제휴사/금융기관) | ✅ 철회 가능 | 제휴 혜택 이용 불가 (서비스 이용 가능) |
| 행태정보 수집 및 이용 동의 | ✅ 철회 가능 | 맞춤형 서비스 제한 (서비스 이용 가능) |
| 쿠키 사용 동의 | ✅ 철회 가능 | 선택 쿠키 사용 중단 (서비스 이용 가능) |
| 앱 권한 동의 | ✅ 변경 가능 | 해당 기능 사용 불가 (기본 서비스 이용 가능) |

### 철회 방법

| 동의 항목 | 철회 경로 |
|---|---|
| 마케팅 정보 수신 | 앱 > 설정 > 알림 설정 > 마케팅 수신 OFF |
| 제3자 정보 제공 | 앱 > 설정 > 개인정보 > 제3자 제공 동의 해지 |
| 행태정보 수집 및 이용 | 앱 > 설정 > 개인정보 > 행태정보 수집 동의 해지 |
| 쿠키 사용 | 웹 포털 > 설정 > 개인정보 > 쿠키 설정 |
| 앱 권한 | 앱 > 설정 > 권한 관리 또는 기기 설정 > 앱 > 위드폼 파트너 > 권한 |
