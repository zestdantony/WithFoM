　　**2.9.** 앱 권한 동의  

　　**2.9.1.** 개요  

위드폼 파트너 앱의 정상적인 서비스 제공을 위한 앱 권한 사용에 관한 안내입니다. **필수 권한**과 **선택 권한**으로 구분됩니다.  


---

## 제1조 (앱 권한의 정의)

앱 권한이란 모바일 앱이 기기의 특정 기능이나 데이터에 접근하기 위해 필요한 사용자 허가를 말합니다.

---

## 제2조 (필수 권한)

서비스 이용을 위해 반드시 필요한 권한입니다.

| 권한 | 사용 목적 | 필수 여부 |
|---|---|---|
| 인터넷 | 서버와 통신하여 데이터 송수신 | **필수** |
| 알림 | 캠페인 매칭, 리워드 승인 요청, 정산 안내 등 푸시 알림 수신 | **필수** |
| 저장 공간 | 앱 데이터 저장, 캐시 저장 | **필수** |

※ 필수 권한은 서비스 이용을 위해 반드시 허용해야 합니다.

---

## 제3조 (선택 권한)

서비스 이용에 필수는 아니지만, 특정 기능 사용을 위해 필요한 권한입니다.

| 권한 | 사용 목적 | 필수 여부 |
|---|---|---|
| 위치 정보 | 매장 위치 확인, 지도에서 정확한 위치 설정 | 선택 |
| 카메라 | 사업자등록증 촬영, 매장 사진 업로드 | 선택 |
| 사진/미디어 | 매장 사진, 계약서 이미지 업로드 | 선택 |
| 전화 | 고객센터 전화 연결 (선택적) | 선택 |

※ 선택 권한은 해당 기능 사용 시에만 요청되며, 거부해도 기본 서비스 이용에는 제한이 없습니다.

---

## 제4조 (권한별 상세 설명)

### 4-1. 알림 권한 (필수)

| 알림 유형 | 내용 |
|---|---|
| 캠페인 매칭 알림 | 새로운 캠페인 매칭 시 알림 |
| 리워드 승인 요청 | 소비자의 리워드 사용 요청 시 알림 |
| 정산 안내 | 월말 정산 내역 안내 |
| 소진 보고 알림 | 소진 보고 미완료 시 알림 |
| 시스템 공지 | 서비스 점검, 약관 변경 등 공지 |

※ 알림 권한을 거부할 경우, 중요한 서비스 안내를 받지 못할 수 있습니다.

### 4-2. 위치 정보 권한 (선택)

| 사용 목적 | 내용 |
|---|---|
| 매장 위치 설정 | 지도에서 정확한 매장 위치 핀 설정 |
| 캠페인 매칭 | 위치 기반 캠페인 매칭 |
| 배송 서비스 | 컵홀더 배송을 위한 배송지 확인 |

※ 위치 정보 권한은 "위치 정보 이용 동의"와 별개로 앱 권한에서도 허용해야 합니다.

### 4-3. 카메라 권한 (선택)

| 사용 목적 | 내용 |
|---|---|
| 사업자등록증 촬영 | 사업자 인증 시 사업자등록증 촬영 |
| 매장 사진 업로드 | 매장 프로필 사진 촬영 및 업로드 |
| 계약서 촬영 | 계약서 이미지 촬영 및 업로드 |

### 4-4. 사진/미디어 권한 (선택)

| 사용 목적 | 내용 |
|---|---|
| 갤러리에서 선택 | 사업자등록증, 매장 사진, 계약서 이미지 선택 |
| 이미지 업로드 | 프로필 사진, 문서 이미지 업로드 |

---

## 제5조 (권한 허용 방법)

### 5-1. 앱 설치 시
- 앱 설치 후 첫 실행 시 필수 권한 요청 팝업 표시
- 각 권한별 허용/거부 선택 가능

### 5-2. 기능 사용 시
- 선택 권한은 해당 기능 사용 시점에 요청
- 예: 사업자등록증 촬영 시 카메라 권한 요청

### 5-3. 설정에서 변경
- 앱 > 설정 > 권한 관리에서 언제든지 변경 가능
- 기기 설정 > 앱 > 위드폼 파트너 > 권한에서도 변경 가능

---

## 제6조 (권한 거부 시 영향)

| 권한 | 거부 시 영향 |
|---|---|
| 인터넷 | 서비스 이용 불가 |
| 알림 | 푸시 알림 수신 불가 (서비스 이용 가능) |
| 저장 공간 | 앱 데이터 저장 불가 (서비스 이용 불가) |
| 위치 정보 | 위치 기반 기능 사용 불가 (기본 서비스 이용 가능) |
| 카메라 | 카메라 촬영 기능 사용 불가 (갤러리에서 선택 가능) |
| 사진/미디어 | 갤러리에서 이미지 선택 불가 (카메라 촬영 가능) |
| 전화 | 앱 내 전화 연결 불가 (수동으로 전화 가능) |

---

## 제7조 (권한 재요청)

| 상황 | 처리 방법 |
|---|---|
| 권한 거부 후 재요청 | 해당 기능 사용 시 다시 권한 요청 팝업 표시 |
| 권한 설정 변경 | 설정 > 권한 관리에서 변경 후 즉시 적용 |

---

## 제8조 (개인정보와의 관계)

앱 권한은 기기 기능 접근 권한이며, 개인정보 수집 및 이용과는 별개입니다. 개인정보 수집 및 이용에 대해서는 "개인정보 처리방침"을 참고하시기 바랍니다.

---

## 제9조 (권한 관리 권장사항)

| 권한 | 권장 설정 |
|---|---|
| 알림 | ✅ 허용 (중요한 서비스 안내 수신) |
| 위치 정보 | ✅ 허용 (캠페인 매칭 최적화) |
| 카메라 | 선택 (사업자등록증 촬영 시 필요) |
| 사진/미디어 | 선택 (이미지 업로드 시 필요) |
