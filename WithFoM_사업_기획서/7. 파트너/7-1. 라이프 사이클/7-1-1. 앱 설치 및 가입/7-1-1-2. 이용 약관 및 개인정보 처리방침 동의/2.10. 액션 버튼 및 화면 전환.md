**2.10.** 액션 버튼 및 화면 전환


**2.10.1.** 약관 동의 화면 구성

1: 화면 진입
　A. 트리거: 온보딩 완료 후 또는 [시작하기] 버튼 클릭
　B. 이전 단계: 7-1-1-1. 앱 설치
　C. 진행 상태: 2/9 단계

2: 약관 목록 표시
　A. 필수 약관: 빨간 별표(*) 표시
　B. 선택 약관: 별도 표시 없음
　C. 각 약관 제목 옆 [보기] 버튼: 약관 전문 팝업 표시


**2.10.2.** 약관 확인 버튼 동작

1: [보기] 버튼 클릭 시
　A. 트리거: 약관 항목 옆 [보기] 버튼 탭
　B. 동작: 해당 약관 전문 팝업/모달 표시
　C. 모달 타입: 풀스크린 모달 또는 바텀 시트

2: 약관 전문 모달 내 동작
　A. 스크롤: 약관 내용 확인
　B. [닫기] 버튼: 모달 닫힘, 동의 화면으로 복귀
　C. [동의] 버튼: 해당 약관 체크박스 자동 선택 + 모달 닫힘

3: 약관 전문 확인 의무
　A. 필수 약관: 전문 확인 없이도 체크 가능 (확인 권장)
　B. 선택 약관: 전문 확인 없이도 체크 가능


**2.10.3.** 동의 체크박스 동작

1: [전체 동의] 체크박스
　A. 체크 시: 모든 약관 (필수 + 선택) 일괄 체크
　B. 체크 해제 시: 모든 약관 일괄 체크 해제
　C. 개별 항목 체크 해제 시: [전체 동의] 자동 해제

2: 개별 약관 체크박스
　A. 체크 시: 해당 약관만 동의
　B. 체크 해제 시: 해당 약관만 동의 해제
　C. 모든 개별 항목 체크 시: [전체 동의] 자동 체크

3: 필수 약관 미동의 시
　A. [다음] 버튼: 비활성화 상태 유지
　B. 안내 메시지: "필수 약관에 모두 동의해주세요"


**2.10.4.** [다음] 버튼 동작

1: 버튼 활성화 조건
　A. 모든 필수 약관 체크 완료
　B. 선택 약관은 체크하지 않아도 버튼 활성화

2: 버튼 클릭 시 동작
　A. 동의 정보 서버 전송
　B. 다음 단계로 화면 전환: 7-1-1-3. 본인 인증
　C. 전환 애니메이션: 슬라이드 (좌 → 우)

3: 버튼 비활성화 시
　A. 버튼 색상: 회색 (비활성화 상태)
　B. 탭 시: 토스트 메시지 "필수 약관에 모두 동의해주세요"


**2.10.5.** 뒤로가기 동작

1: 동의 화면에서 뒤로가기
　A. 동작: 앱 종료 확인 팝업 표시
　B. 이유: 가입 첫 단계이므로 이전 화면 없음

2: 앱 종료 확인 팝업
　A. 메시지: "가입을 취소하고 앱을 종료하시겠습니까?"
　B. [취소] 버튼: 팝업 닫힘, 동의 화면 유지
　C. [종료] 버튼: 앱 종료

3: 약관 전문 모달에서 뒤로가기
　A. 동작: 모달 닫힘
　B. 동의 화면으로 복귀


**2.10.6.** 다음 단계로의 화면 전환

1: 이용 약관 동의 → 본인 인증
　A. 트리거: [다음] 버튼 클릭 (필수 약관 모두 동의 시)
　B. 이동 경로: 7-1-1-3. 본인 인증 화면
　C. 전환 애니메이션: 슬라이드 (좌 → 우)

2: 진행 상태 표시 업데이트
　A. 현재 단계: 2/9 → 3/9
　B. 다음 단계 예고: "다음: 본인 인증"


**2.10.7.** 모바일 환경 고려사항

1: 터치 인터랙션
　A. 체크박스 탭: 동의/해제 토글
　B. 약관 제목 탭: 약관 전문 모달 표시 (선택적)
　C. [보기] 버튼 탭: 약관 전문 모달 표시

2: 화면 스크롤
　A. 약관 항목이 많을 경우 스크롤 가능
　B. [다음] 버튼: 화면 하단 고정 (floating)

3: 약관 전문 모달
　A. 풀스크린 모달 또는 바텀 시트 형태
　B. 스와이프 다운: 모달 닫기 (바텀 시트의 경우)


• 참조: 7-1-1-3. 본인 인증
