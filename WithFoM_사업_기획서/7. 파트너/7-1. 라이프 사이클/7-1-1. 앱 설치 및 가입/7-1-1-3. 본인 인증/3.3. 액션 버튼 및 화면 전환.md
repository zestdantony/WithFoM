**3.3.** 액션 버튼 및 화면 전환


**3.3.1.** 본인 인증 화면 구성

1: 화면 진입
　A. 트리거: 이용 약관 동의 완료 후 [다음] 버튼 클릭
　B. 이전 단계: 7-1-1-2. 이용 약관 및 개인정보 처리방침 동의
　C. 진행 상태: 3/9 단계

2: 인증 방법 선택 화면
　A. 휴대폰 인증 버튼
　B. 공동인증서 인증 버튼
　C. 간편인증 버튼 (카카오, 네이버, PASS)


**3.3.2.** 인증 방법 선택 버튼 동작

1: [휴대폰 인증] 버튼 클릭
　A. 트리거: 휴대폰 인증 버튼 탭
　B. 동작: 휴대폰 인증 입력 화면으로 전환
　C. 전환 방식: 화면 내 섹션 전환 또는 새 화면

2: [공동인증서 인증] 버튼 클릭
　A. 트리거: 공동인증서 인증 버튼 탭
　B. 동작: 공동인증서 선택 화면으로 전환
　C. 전환 방식: 외부 인증서 앱 연동 또는 내장 인증서 선택

3: [간편인증] 버튼 클릭
　A. 트리거: 간편인증 (카카오/네이버/PASS) 버튼 탭
　B. 동작: 해당 간편인증 앱으로 이동
　C. 전환 방식: 외부 앱 호출 → 인증 완료 → 앱 복귀


**3.3.3.** 휴대폰 인증 화면 동작

1: 정보 입력 단계
　A. 통신사 선택: 드롭다운 또는 버튼 그룹
　B. 이름 입력: 텍스트 입력 필드
　C. 생년월일 입력: 날짜 선택 또는 숫자 입력
　D. 성별 선택: 남/여 버튼 그룹
　E. 휴대폰 번호 입력: 숫자 입력 필드

2: [인증번호 요청] 버튼 클릭
　A. 버튼 활성화 조건: 모든 필수 정보 입력 완료
　B. 동작: 인증번호 SMS 발송 요청
　C. 화면 변경: 인증번호 입력 필드 표시
　D. 타이머 시작: 3분 카운트다운

3: 인증번호 입력 및 확인
　A. 인증번호 6자리 입력
　B. [확인] 버튼 클릭: 인증번호 검증
　C. 인증 성공: 인증 완료 화면으로 전환
　D. 인증 실패: 오류 메시지 표시, 재입력 요청

4: [인증번호 재요청] 버튼
　A. 트리거: 인증번호 미수신 또는 시간 초과 시
　B. 동작: 새 인증번호 SMS 발송
　C. 타이머 리셋: 3분 카운트다운 재시작
　D. 재요청 제한: 5회 초과 시 잠시 후 시도 안내


**3.3.4.** 공동인증서 인증 화면 동작

1: 인증서 선택
　A. 저장된 인증서 목록 표시
　B. 인증서 선택 후 비밀번호 입력

2: [인증] 버튼 클릭
　A. 동작: 인증서 비밀번호 검증
　B. 인증 성공: 인증 완료 화면으로 전환
　C. 인증 실패: 오류 메시지 표시

3: [인증서 가져오기] 버튼
　A. 동작: 인증서 가져오기 안내 팝업 표시
　B. PC → 모바일 인증서 복사 방법 안내


**3.3.5.** 간편인증 화면 동작

1: 간편인증 앱 호출
　A. 선택한 앱 (카카오/네이버/PASS) 실행
　B. 해당 앱에서 인증 요청 승인

2: 인증 완료 후 앱 복귀
　A. 인증 성공: 위드폼 파트너 앱으로 자동 복귀, 인증 완료 처리
　B. 인증 실패 또는 취소: 위드폼 파트너 앱으로 복귀, 재시도 안내


**3.3.6.** 인증 완료 화면 동작

1: 인증 완료 표시
　A. "본인 인증이 완료되었습니다" 메시지 표시
　B. 인증된 정보 표시 (이름, 생년월일, 휴대폰 번호 - 일부 마스킹)

2: [다음] 버튼 클릭
　A. 트리거: 인증 완료 후 [다음] 버튼 탭
　B. 동작: 다음 단계로 화면 전환
　C. 이동 경로: 7-1-1-4. 아이디(닉네임) 패스워드 설정
　D. 전환 애니메이션: 슬라이드 (좌 → 우)


**3.3.7.** 뒤로가기 동작

1: 인증 방법 선택 화면에서 뒤로가기
　A. 이동 경로: 7-1-1-2. 이용 약관 동의 화면
　B. 동의 정보: 유지됨

2: 휴대폰 인증 입력 화면에서 뒤로가기
　A. 이동 경로: 인증 방법 선택 화면
　B. 입력 정보: 초기화됨

3: 인증 완료 화면에서 뒤로가기
　A. 인증 완료 상태 유지
　B. 재인증 필요 없음 (동일 세션 내)


**3.3.8.** 다음 단계로의 화면 전환

1: 본인 인증 → 아이디/패스워드 설정
　A. 트리거: 인증 완료 후 [다음] 버튼 클릭
　B. 이동 경로: 7-1-1-4. 아이디(닉네임) 패스워드 설정 화면
　C. 전달 데이터: 인증된 이름, 휴대폰 번호

2: 진행 상태 표시 업데이트
　A. 현재 단계: 3/9 → 4/9
　B. 다음 단계 예고: "다음: 아이디/패스워드 설정"


**3.3.9.** 에러 상황 화면 처리

1: 인증번호 불일치
　A. 메시지: "인증번호가 일치하지 않습니다"
　B. 재입력 요청

2: 인증 시간 초과
　A. 메시지: "인증 시간이 초과되었습니다"
　B. [인증번호 재요청] 버튼 강조 표시

3: 인증 실패 (5회 초과)
　A. 메시지: "인증 시도 횟수를 초과했습니다. 잠시 후 다시 시도해주세요."
　B. 일정 시간 후 재시도 가능


• 참조: 7-1-1-4. 아이디(닉네임) 패스워드 설정
