**3.2.** 인증 완료 확인  

↳ 본인 인증 성공 여부 확인 및 다음 단계 진행  


**3.2.1.** 인증 성공  

**3.2.1.1.** "본인 인증이 완료되었습니다" 메시지 표시  
**3.2.1.2.** 인증된 정보 확인  

| 확인 항목 | 표시 내용 |
|---|---|
| 이름 | 본인 인증된 실명 (예: 홍*동) |
| 생년월일 | 생년월일 (예: 1990-01-01) |
| 휴대폰 번호 | 인증된 번호 (예: 010-****-1234) |

**3.2.1.3.** [다음] 버튼 활성화 → 아이디/패스워드 설정 단계로 진행  


**3.2.2.** 자동 반영 안내  

**3.2.2.1.** 인증 완료 시 다음 정보가 이후 단계에 자동 기입됩니다:  

| 인증 정보 | 자동 반영 위치 | 수정 가능 여부 |
|---|---|---|
| 이름 (성명) | 7-1-1-5. 대표자명 입력란 | 수정 불가 (본인 인증 정보 고정) |
| 휴대폰 번호 | 7-1-1-5. 연락처 입력란 | 수정 가능 (매장 연락처로 변경 가능) |

**3.2.2.2.** 자동 기입된 정보 확인 후 다음 단계 진행  
**3.2.2.3.** 대표자명은 본인 인증 정보가 그대로 사용됩니다  


**3.2.3.** 인증 실패  

**3.2.3.1.** 인증번호 불일치: 인증번호 재확인 후 재입력  
**3.2.3.2.** 인증 시간 초과: 인증번호 재요청  
**3.2.3.3.** 본인 확인 불가: 입력 정보 확인 후 재시도  
**3.2.3.4.** 인증서 오류: 인증서 유효성 확인 후 재시도
