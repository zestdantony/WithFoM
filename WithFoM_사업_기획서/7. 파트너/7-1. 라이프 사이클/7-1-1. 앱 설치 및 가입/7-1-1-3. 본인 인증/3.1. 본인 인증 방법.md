**3.1.** 본인 인증 방법  

↳ 휴대폰 또는 공동인증서를 통한 본인 인증  


**3.1.1.** 본인 인증 개요  

**3.1.1.1.** 서비스 가입을 위한 본인 확인 절차  
**3.1.1.2.** 휴대폰 인증, 공동인증서 인증, 간편인증 중 선택  
**3.1.1.3.** 인증 완료 후 다음 단계 진행  


**3.1.2.** 휴대폰 인증  

**3.1.2.1.** 통신사 선택 (SKT, KT, LG U+, 알뜰폰)  
**3.1.2.2.** 이름, 생년월일, 성별, 휴대폰 번호 입력  
**3.1.2.3.** 인증번호 요청 → SMS 수신 → 인증번호 입력  
**3.1.2.4.** 인증 유효 시간: 3분 (초과 시 재요청)  


**3.1.3.** 공동인증서 인증  

**3.1.3.1.** 공동인증서 선택 (개인/범용 공동인증서)  
**3.1.3.2.** 인증서 비밀번호 입력  
**3.1.3.3.** 인증서 유효성 검증 후 본인 확인 완료  


**3.1.4.** 간편인증 (선택)  

**3.1.4.1.** 카카오, 네이버, PASS 등 간편인증 지원  
**3.1.4.2.** 해당 앱에서 인증 요청 승인  
**3.1.4.3.** 인증 완료 후 자동 반영  


**3.1.5.** 인증 완료 시 확보 정보  

| 항목 | 설명 | 자동 반영 위치 |
|---|---|---|
| 이름 (성명) | 본인 인증된 실명 | 7-1-1-5. 매장 기본 정보 입력 → 대표자명 |
| 생년월일 | 본인 확인용 | 계정 정보 저장 |
| 휴대폰 번호 | 인증된 연락처 | 7-1-1-5. 매장 기본 정보 입력 → 연락처 |

**※ 인증 완료 시 위 정보는 이후 단계의 해당 입력란에 자동으로 기입됩니다.**
