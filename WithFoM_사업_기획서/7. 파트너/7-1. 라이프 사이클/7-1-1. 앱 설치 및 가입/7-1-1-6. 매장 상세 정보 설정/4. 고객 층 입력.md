# 4: 고객 층 입력

　A. 주 고객층 드롭다운 선택  
　　a. 선택 개수: 최소 2개 ~ 최대 5개 (복수 선택)  
　　b. 상위 고객층 및 세부 항목  
　　aa. 직장인  
　	• 오피스  
　	• 관공서·공공기관  
　	• 병원·의료기관  
　	• 대형마트·백화점·상가  
　	• 교대·야간 근무  
　	• 프리랜서  
　　bb. 학생  
　	• 초등학생  
　	• 중학생  
　	• 고등학생  
　	• 대학생  
　	• 취업 준비생  
　	• 고시·자격증 준비생  
　　cc. 학부모  
　	• 유아·초등 학부모  
　	• 중·고등 학부모  
　	• 입시·학원 동행 학부모  
　	• 어린이집·유치원 학부모  
　　dd. 기타  
　	• 주부/가사 중심  
　	• 자영업자  
　	• 지역 주민  
　	• 여행객/관광객  
　	• 외국인  
　	• 시니어(60대 이상)  
　	• 커플/데이트  
　	• 가족 단위 방문객  
　	• 참조: WithFoM_Master_Planning_v3.10.md 3장 3-2절 3-2-2. 온보딩 단계 (파트너 가입)의 "주 고객층 드롭다운 (복수 선택, 최소 2개 ~ 최대 5개)" 표  
　B. 화면 구성 및 동작 흐름  
　　a. 상단 영역  
　	aa. 제목: "주 고객층 선택"  
　	bb. 설명 텍스트: "매장의 주 고객층을 선택해 주세요 (최소 2개, 최대 5개)"  
　	cc. 진행 단계 표시: "3단계 중 4단계"  
　　b. 메인 선택 영역  
　	aa. 계층적 드롭다운 메뉴 (상위 고객층 → 세부 항목)  
　	bb. 선택된 항목 목록 표시 영역  
　　c. 하단 영역  
　	aa. 선택 개수 표시: "2개 선택됨 / 최소 2개 필요"  
　	bb. "다음 단계" 버튼  
　	• 최소 2개 선택 시 활성화  
　	• 최대 5개 선택 시 추가 선택 불가 안내  
　	• 버튼 비활성화 시: 회색, 클릭 불가  
　	• 버튼 활성화 시: 파란색, 클릭 가능  
　C. 동작 흐름  
　　a. 고객층 선택  
　	aa. 화면 진입 시 상위 고객층 목록 표시  
　	bb. 상위 고객층 클릭 시 세부 항목 드롭다운 표시  
　	cc. 세부 항목 선택 시 선택 목록에 추가  
　	dd. 최소 2개 선택 시 "다음 단계" 버튼 활성화  
　	ee. 최대 5개 선택 시 추가 선택 불가 안내  
　　b. 다음 단계 진행  
　	aa. 최소 2개 이상 선택 완료 시 "다음 단계" 버튼 클릭 가능  
　	bb. 버튼 클릭 시 다음 화면(고객 구성 입력)으로 이동  
　	cc. 선택한 정보는 자동 저장됨  
　D. 입력 사항 활용 (파트너용 설명)  
　　a. 입력한 고객층 정보는 매장에 맞는 캠페인 매칭에 활용됩니다  
　　b. 광고주는 타깃 고객층을 설정하는데, 이를 바탕으로 적합한 캠페인이 매칭됩니다  
　　c. 고객층 정보는 매장 리포트에도 활용되어 고객 분석에 도움이 됩니다  
　E. 입력 사항 활용 (개발자용)  
　　a. 캠페인 매칭  
　	aa. 광고주 타깃 고객층과 매칭하여 적합한 캠페인 추천  
　	bb. 고객층 정보를 활용하여 매칭 알고리즘에 가중치 부여  
　　b. AI 개인화  
　	aa. 소비자 프로파일링 시 고객층 정보 활용  
　	bb. 고객층별 선호도 분석 및 추천  
　　c. 마케팅 전략  
　	aa. 파트너 맞춤형 마케팅 전략 수립  
　	bb. 광고주에게 제공하는 매장 정보에 고객층 정보 포함  
　　d. 데이터 분석  
　	aa. 고객층별 이용 패턴 분석  
　	bb. 매장 리포트 생성 시 고객층 정보 활용  
　	cc. 고객층별 벤치마크 비교 데이터 생성  
　　e. 매칭 알고리즘  
　	aa. 고객층 정보를 매칭 알고리즘의 주요 변수로 활용  
　	bb. 광고주 타깃 고객층과 파트너 매장 고객층의 일치도를 계산하여 매칭 점수 산출
