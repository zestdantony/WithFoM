　　**6.8.** 컵홀더 수용량 입력  

↳ 매장 상세 정보 설정 단계 중 여덟 번째 화면  

**진행 단계 표시:** 8단계 중 8단계  

　A. 상단 영역  
　　a. 제목: "컵홀더 수용량 입력"  
　　b. 설명 텍스트: "매장에서 보관 가능한 최대 컵홀더 수량을 입력해 주세요"  
　　c. 진행 단계 표시: "8단계 중 8단계" (우측 상단)  
　B. 메인 입력 영역  
　　a. 컵홀더만 선택했을 경우  
　　aa. 단일 수용량 입력창 표시  
　　bb. 입력창 제목: "컵홀더 수용량"  
　　cc. 입력창 구성  
　	• 숫자 입력창 (숫자만 입력 가능)  
　	• 단위 표시: "개" (입력창 우측에 표시)  
　	• 최소/최대 가이드 텍스트: "최소 1,000개 이상 입력해 주세요"  
　dd. 입력 단위: 개 (최소 1,000개 ~ 최대 10,000개)  
　ee. 입력 검증  
　	• 1,000개 미만 입력 시 경고 메시지: "최소 1,000개 이상 입력해 주세요"  
　	• 10,000개 초과 입력 시 경고 메시지: "최대 10,000개까지 입력 가능합니다"  
　	• 숫자가 아닌 값 입력 시 경고 메시지: "숫자만 입력해 주세요"  
　b. 에어 홀더 + 컵홀더 둘 다 선택했을 경우  
　　aa. 두 개의 수용량 입력창 동시 표시  
　　bb. 입력창 1: "에어 홀더 수용량"  
　　cc. 입력창 2: "컵홀더 수용량"  
　　dd. 각각의 입력 단위: 개 (최소 1,000개 ~ 최대 10,000개)  
　　ee. 각각 독립적으로 입력  
　	• 에어 홀더 수용량과 컵홀더 수용량을 각각 별도로 입력  
　	• 하나만 입력해도 다음 단계 진행 가능 (다른 하나는 나중에 입력 가능)  
　	• 두 입력창 모두 입력 완료 시 "다음 단계" 버튼 활성화  
　ff. 입력 검증은 각각 독립적으로 적용  
　C. 하단 영역  
　　a. 입력 가이드: "입력한 수용량은 기본 발주량으로 활용됩니다"  
　　b. "다음 단계" 버튼  
　　aa. 버튼 상태  
　	• 비활성화 시: 회색 (#CCCCCC), 클릭 불가  
　	• 활성화 시: 파란색 (#0066FF), 클릭 가능  
　	• 조건: 수용량 입력 완료 시 활성화  
　bb. 활성화 조건  
　	• 컵홀더만 선택 시: 컵홀더 수용량 입력 완료 시 활성화  
　	• 둘 다 선택 시: 두 입력창 모두 입력 완료 시 활성화 (하나만 입력해도 가능)  
　cc. 버튼 클릭 시  
　	• 다음 화면으로 이동  
　	• 입력한 정보는 자동 저장되며, 발주 시스템에 반영됨  
　D. 동작 흐름  
　　a. 화면 진입  
　　aa. 이전 단계(컵홀더 유형 선택)에서 선택한 컵홀더 종류에 따라 입력창 표시  
　	• 컵홀더만 선택했을 경우: 컵홀더 수용량 입력창 1개만 표시  
　	• 에어 홀더 + 컵홀더 둘 다 선택했을 경우: 에어 홀더 수용량 입력창과 컵홀더 수용량 입력창 2개 표시  
　bb. 입력창은 비어있음  
　　cc. "다음 단계" 버튼 비활성화 상태  
　　b. 수용량 입력  
　　aa. 컵홀더만 선택했을 경우  
　	• 숫자 키패드로 수량 입력  
　	• 최소 1,000개, 최대 10,000개 제한  
　	• 입력 완료 시 "다음 단계" 버튼 활성화  
　bb. 둘 다 선택했을 경우  
　	• 각각 독립적으로 입력 가능  
　	• 에어 홀더 수용량 입력창과 컵홀더 수용량 입력창 각각 입력  
　	• 각각 최소 1,000개, 최대 10,000개 제한  
　	• 하나만 입력해도 다음 단계 진행 가능  
　	• 두 입력창 모두 입력 완료 시 "다음 단계" 버튼 활성화  
　c. 입력 검증  
　　aa. 입력값 검증: 1,000개 미만, 10,000개 초과, 숫자가 아닌 값 입력 시 경고 메시지 표시  
　　bb. 검증 성공 시 입력값 표시 및 버튼 활성화  
　d. 입력 완료 및 저장  
　　aa. 입력 완료 후 "다음 단계" 버튼 클릭  
　　bb. 입력한 정보는 자동 저장되며, 발주 시스템에 반영됨  
　	• 컵홀더 종류 정보 (이전 단계에서 선택한 정보 활용)  
　	• 수용량 정보 (컵홀더 수용량 또는 에어 홀더 수용량 + 컵홀더 수용량)  
　cc. 저장된 수용량은 기본 발주량으로 활용됨  
　dd. 버튼 클릭 시 다음 화면으로 이동  
　E. 수용량 변경  
　　a. 매장 정보 수정에서 언제든 변경 가능  
　　b. 변경 즉시 발주 시스템에 반영  
　F. 입력 사항 활용 (파트너용 설명)  
　　a. 입력한 수용량은 기본 발주량으로 활용됩니다  
　　b. 빠른 발주 시 설정한 수용량을 기준으로 자동 발주됩니다  
　　c. 수용량을 초과하는 발주를 원하시면 추가 발주 옵션을 이용하실 수 있습니다  
　G. 입력 사항 활용 (개발자용)  
　　a. 기본 발주량 산정  
　　aa. 입력한 수용량을 기준으로 기본 최대 발주량 설정  
　　bb. 컵홀더만 선택 시: 컵홀더 수용량을 기본 발주량으로 설정  
　　cc. 둘 다 선택 시: 에어 홀더와 컵홀더 수용량을 각각 독립적으로 기본 발주량으로 설정  
　b. 발주 시스템  
　　aa. 빠른 발주 시 기본 발주량으로 자동 발주  
　　bb. 컵홀더 종류별로 발주량 관리  
　　cc. 에어 홀더와 컵홀더가 둘 다 선택된 경우 각각 독립적으로 발주량 관리  
　c. 재고 관리  
　　aa. 매장 수용량 초과 발주 시 경고 메시지 표시  
　　bb. 수용량 정보를 활용하여 재고 부족 시점 예측  
　d. 물류 계획  
　　aa. 물류팀이 매장별 수용량을 고려한 배송 계획 수립  
　　bb. 컵홀더 종류별 배송 수량 결정  
　　cc. 에어 홀더와 컵홀더가 둘 다 선택된 경우 각각 독립적으로 배송 계획 수립
