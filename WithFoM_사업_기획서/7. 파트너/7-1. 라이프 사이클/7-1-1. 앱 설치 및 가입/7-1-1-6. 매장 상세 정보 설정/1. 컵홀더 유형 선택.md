# 1: 컵홀더 유형 선택

　A. 컵홀더 유형 종류  
　　a. 전체형 (단독)  
　　　aa. 전면 구성  
　　　　• 광고주 비주얼 100% + 광고주 QR (크게)  
　　　bb. 후면 구성  
　　　　• 광고주 비주얼 + 위드폼 로고/QR (작게)  
　　　cc. 파트너 데이터 수집  
　　　　• 수집 어려움  
　　　dd. 컵홀더 비용 혜택  
　　　　• 컵홀더 무료 제공 혜택  
　　　　• 카드 하단에 "컵홀더 무료 제공" 뱃지 표시  
　　　　• 참조: WithFoM_Master_Planning_v3.10.md 3장 3-2절 3-2-4. 배포 단계의 "컵홀더 유형별 구성" 표에서 "전체형 (단독)" 항목  
　　b. 혼합형 (콜라보)  
　　　aa. 전면 구성  
　　　　• 광고주 비주얼 100% + 광고주 QR (크게)  
　　　bb. 후면 구성  
　　　　• 파트너 로고 60% + 위드폼 로고/QR 40% (작게)  
　　　cc. 파트너 데이터 수집  
　　　　• 수집 가능  
　　　dd. 컵홀더 비용 혜택  
　　　　• 컵홀더 반값 지원 혜택  
　　　　• 카드 하단에 "컵홀더 반값 지원" 뱃지 표시  
　　　ee. 로고 업로드 안내  
　　　　• 혼합형 선택 시에도 바로 다음 단계로 진행 가능  
　　　　• 파트너 로고 업로드는 추후 캠페인 매칭 시 진행  
　　　　• 참조: WithFoM_Master_Planning_v3.10.md 3장 3-2절 3-2-4. 배포 단계의 "컵홀더 유형별 구성" 표에서 "혼합형 (콜라보)" 항목  
　B. 컵홀더 QR 매핑 방식  
　　a. 기본 원칙  
　　　aa. 1 컵홀더 = 1 캠페인  
　　　　• 컵홀더 1개에는 광고주 캠페인 1개만 인쇄  
　　　　• 참조: WithFoM_Master_Planning_v3.10.md 3장 3-2절 3-2-4. 배포 단계의 "컵홀더 QR 매핑 방식" 표에서 "1 컵홀더 = 1 캠페인" 원칙  
　　b. QR 종류 및 위치  
　　　aa. 광고주 QR  
　　　　• 위치: 전면  
　　　　• 크기: 크게  
　　　　• 용도: 소비자 캠페인 참여 (스캔 → 설문 → 게임 → 리워드)  
　　　　• 참조: WithFoM_Master_Planning_v3.10.md 3장 3-2절 3-2-4. 배포 단계의 "QR 역할 구분" 표에서 "광고주 QR" 항목  
　　　bb. 위드폼 QR  
　　　　• 위치: 후면  
　　　　• 크기: 작게  
　　　　• 용도: 광고주/파트너 문의, 위드폼 서비스 안내  
　　　　• 참조: WithFoM_Master_Planning_v3.10.md 3장 3-2절 3-2-4. 배포 단계의 "QR 역할 구분" 표에서 "위드폼 QR" 항목  
　C. 컵홀더 유형 선택 시 고려사항  
　　a. 데이터 수집 목적  
　　aa. 전체형 (단독) 선택 시  
　　• 파트너 데이터 수집이 어려움  
　　bb. 혼합형 (콜라보) 선택 시  
　　• 파트너 데이터 수집 가능  
　　• 매장 브랜딩 효과 (로고 노출)  
　　b. 브랜딩 효과  
　　aa. 혼합형 선택 시 매장 로고 노출로 브랜딩 효과  
　　bb. 후면에 매장명과 로고가 함께 표시되어 고객 인지도 향상  
　　c. 물류 및 배포  
　　aa. 컵홀더는 분할 발주 방식으로 배포  
　　bb. 각 파트너 매장의 소진 속도에 맞춰 필요한 만큼만 배송  
　　cc. 재고 리스크를 줄이고 캠페인 진행 중 유연하게 대응  
　　• 참조: WithFoM_Master_Planning_v3.10.md 3장 3-2절 3-2-4. 배포 단계의 "물류 분할 발주 정책 상세" 항목  
　D. 화면 구성 및 동작 흐름  
　　a. 상단 영역  
　　aa. 제목: "컵홀더 유형 선택"  
　　bb. 설명 텍스트: "매장에 맞는 컵홀더 유형을 선택해 주세요"  
　　cc. 진행 단계 표시: "3단계 중 1단계"  
　　b. 메인 선택 영역  
　　aa. 두 개의 큰 카드 형태 버튼 (전체형/혼합형)  
　　bb. 각 카드 구성  
　　• 유형명 (큰 글씨)  
　　• 유형별 설명 텍스트  
　　• 주요 특징 리스트  
　　• 혜택 안내 문구 (카드 하단, 뱃지 형태)  
　　cc. 카드 선택 시  
　　• 테두리 색상 변경 (파란색)  
　　• 체크 아이콘 표시 (우측 상단)  
　　• 배경색 약간 변경  
　　c. 컵홀더 종류 선택 영역  
　　aa. 제목: "컵홀더 종류 선택"  
　　bb. 선택 옵션 (양자 택일)  
　　• 라디오 버튼 1: "컵홀더만 선택"  
　	• 라디오 버튼 2: "에어 홀더 + 컵홀더 둘 다 선택"  
　　cc. 선택 방식  
　　• 라디오 버튼 형태로 양자 택일 선택  
　　• 둘 중 하나만 선택 가능  
　　• 컵홀더만 원하면 라디오 버튼 1 선택  
　	• 둘 다 원하면 라디오 버튼 2 선택  
　　d. 하단 영역  
　　aa. "다음 단계" 버튼  
　　• 유형 선택 및 컵홀더 종류 선택 완료 시 활성화  
　　• 버튼 비활성화 시: 회색, 클릭 불가  
　	• 버튼 활성화 시: 파란색, 클릭 가능  
　E. 동작 흐름  
　　a. 유형 선택  
　　aa. 화면 진입 시 두 가지 유형 카드 표시  
　　bb. 카드 클릭 시 해당 카드 하이라이트 및 선택 표시  
　　cc. 전체형 선택 시: "컵홀더 무료 제공" 뱃지 표시, 바로 다음 단계 가능  
　	dd. 혼합형 선택 시: "컵홀더 반값 지원" 뱃지 표시, 바로 다음 단계 가능 (로고 업로드는 캠페인 매칭 시 진행)  
　　b. 컵홀더 종류 선택  
　　aa. 유형 선택 후 컵홀더 종류 선택 영역 표시  
　	bb. 라디오 버튼으로 양자 택일 선택  
　	cc. 컵홀더만 선택 또는 에어 홀더 + 컵홀더 둘 다 선택  
　　c. 다음 단계 진행  
　　aa. 유형 선택 및 컵홀더 종류 선택 완료 시 "다음 단계" 버튼 활성화  
　	bb. 버튼 클릭 시 다음 화면(리워드/오더 선택)으로 이동  
　	cc. 선택한 정보는 자동 저장됨  
　F. 광고군 제외 설정  
　　a. 매장 이미지와 맞지 않는 광고군 입력  
　　aa. 최대 3개까지 입력 가능  
　　b. 참고 사항  
　　aa. "우선 순위에 따라 매칭 시 제외 대상으로 선정, 후순위로 밀리지만 정책상 100% 제외 되는 것은 아닙니다"  
　G. 입력 사항 활용 (파트너용 설명)  
　　a. 선택한 유형에 따라 캠페인 매칭 우선순위가 달라집니다  
　　b. 전체형 선택 시 컵홀더 무료 제공, 혼합형 선택 시 반값 지원 혜택을 받을 수 있습니다  
　　c. 혼합형 선택 시 매장 브랜딩 효과를 얻을 수 있으며, 파트너 데이터 수집이 가능합니다  
　H. 입력 사항 활용 (개발자용)  
　　a. 캠페인 매칭  
　　aa. 선택한 유형에 따라 매칭되는 캠페인 필터링  
　	bb. 전체형 선택 시: 전체형 캠페인 우선 매칭  
　	cc. 혼합형 선택 시: 혼합형 캠페인 우선 매칭  
　　b. 컵홀더 디자인  
　	aa. 혼합형 선택 시 파트너 로고 포함 디자인 생성 (캠페인 매칭 시 로고 업로드)  
　	bb. 전체형 선택 시 광고주 비주얼 중심 디자인 생성  
　　c. 정산  
　	aa. 유형별 혜택 차등 적용 (전체형 무료, 혼합형 반값)  
　	bb. 정산 시스템에서 유형 정보를 참조하여 비용 계산  
　　d. 데이터 수집  
　	aa. 혼합형 선택 시 파트너 데이터 수집 가능  
　	bb. 수집된 데이터는 파트너 리포트 및 벤치마크 분석에 활용  
　　e. 컵홀더 종류  
　	aa. 선택한 종류에 따라 발주 시스템에서 종류별 발주량 관리  
　	bb. 에어 홀더 + 컵홀더 선택 시 각각 독립적으로 수용량 관리
