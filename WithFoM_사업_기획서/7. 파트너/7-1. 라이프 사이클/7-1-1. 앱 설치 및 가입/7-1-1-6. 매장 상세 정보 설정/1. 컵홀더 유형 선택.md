　　**6.1.** 컵홀더 유형 선택  

↳ 매장 상세 정보 설정 단계 중 첫 번째 화면  

**진행 단계 표시:** 8단계 중 1단계  

　A. 상단 영역  
　　a. 제목: "컵홀더 유형 선택"  
　　b. 설명 텍스트: "매장에 맞는 컵홀더 유형을 선택해 주세요 (1개 이상 선택)"  
　　c. 진행 단계 표시: "8단계 중 1단계" (우측 상단)  
　B. 메인 선택 영역  
　　a. 컵홀더 유형 카드 (2개)  
　　aa. 카드 1: 단독  
　　• 카드 구성  
　	- 유형명: "단독" (큰 글씨, 상단 중앙)  
　	- 전면 구성 설명: "광고주 비주얼 100% + 광고주 QR (크게)"  
　	- 후면 구성 설명: "광고주 비주얼 + 위드폼 로고/QR (작게)"  
　	- 혜택 뱃지: "컵홀더 무료 제공" (카드 하단, 초록색 뱃지)  
　	- 체크박스: 좌측 상단에 체크박스 표시  
　• 카드 선택 시  
　	- 체크박스 선택 표시 (체크마크)  
　	- 테두리 색상 변경: 회색 → 파란색 (#0066FF)  
　	- 배경색 약간 변경: 밝은 파란색 배경 (#F0F7FF)  
　bb. 카드 2: 콜라보  
　　• 카드 구성  
　	- 유형명: "콜라보" (큰 글씨, 상단 중앙)  
　	- 전면 구성 설명: "광고주 비주얼 100% + 광고주 QR (크게)"  
　	- 후면 구성 설명: "파트너 로고 60% + 위드폼 로고/QR 40% (작게)"  
　	- 브랜딩 효과 안내: "매장 브랜딩 효과 (로고 노출)"  
　	- 혜택 뱃지: "컵홀더 반값 지원" (카드 하단, 주황색 뱃지)  
　	- 로고 업로드 안내: "로고 업로드는 캠페인 매칭 시 진행됩니다" (작은 글씨, 회색)  
　	- 체크박스: 좌측 상단에 체크박스 표시  
　• 카드 선택 시  
　	- 체크박스 선택 표시 (체크마크)  
　	- 테두리 색상 변경: 회색 → 파란색 (#0066FF)  
　	- 배경색 약간 변경: 밝은 파란색 배경 (#F0F7FF)  
　b. 컵홀더 종류 선택 영역 (유형 1개 이상 선택 후 표시)  
　　aa. 제목: "컵홀더 종류 선택"  
　　bb. 선택 옵션 (양자 택일)  
　	• 라디오 버튼 1: "컵홀더만 선택"  
　	• 라디오 버튼 2: "에어 홀더 + 컵홀더 둘 다 선택"  
　cc. 선택 방식  
　	• 라디오 버튼 형태로 양자 택일 선택  
　	• 둘 중 하나만 선택 가능  
　c. 광고군 제외 설정 영역 (컵홀더 종류 선택 후 표시)  
　　aa. 제목: "광고군 제외 설정 (선택 사항)"  
　　bb. 설명 텍스트: "매장 이미지와 맞지 않는 광고군을 선택해 주세요 (최대 3개, 선택 사항)"  
　　cc. 안내 문구: "우선 순위에 따라 매칭 시 제외 대상으로 선정, 후순위로 밀리지만 정책상 100% 제외 되는 것은 아닙니다" (작은 글씨, 회색)  
　　dd. 광고군 선택창  
　	• 체크박스 형태로 표시  
　	• 대분류별로 그룹화하여 표시  
　	• 선택 가능한 광고군 (대분류별)  
　		- 공공/기관  
　			• 정부 정책·행사  
　			• 지자체 축제  
　			• 경찰·소방 홍보  
　			• 공공기관 홍보  
　		- 엔터/문화  
　			• 아이돌 신곡  
　			• 신인 홍보  
　			• 공연·뮤지컬·오페라  
　			• 영화·드라마  
　			• 예술·전시  
　		- 의료/건강  
　			• 병원  
　			• 성형외과  
　			• 치과  
　			• 한의원  
　			• 약국  
　			• 헬스·피트니스  
　		- 교육  
　			• 학원  
　			• 온라인 강사  
　			• 유치원·어린이집  
　			• 대학교  
　			• 평생교육원  
　		- 금융  
　			• 은행  
　			• 보험  
　			• 증권  
　			• 카드사  
　			• 대출·금융상품  
　		- 기업/브랜딩  
　			• 대기업 브랜딩  
　			• 스타트업 홍보  
　			• 제조업  
　			• IT·테크  
　		- 패션·뷰티  
　			• 의류  
　			• 화장품  
　			• 액세서리  
　			• 뷰티 서비스  
　		- 커머스/쇼핑  
　			• 온라인 쇼핑몰  
　			• 오프라인 매장  
　			• 배달 앱  
　			• 중고거래  
　		- 음식/외식  
　			• 프랜차이즈  
　			• 배달음식  
　			• 식품  
　			• 주류  
　		- 여행/레저  
　			• 여행사  
　			• 호텔·숙박  
　			• 레저·스포츠  
　			• 관광지  
　	• 최대 3개까지 선택 가능  
　	• 선택 안 해도 다음 단계 진행 가능  
　D. 하단 영역  
　　a. "다음 단계" 버튼  
　　aa. 버튼 위치: 화면 하단 고정, 전체 너비  
　　bb. 버튼 상태  
　	• 비활성화 시  
　	- 배경색: 회색 (#CCCCCC)  
　	- 텍스트 색상: 흰색 (#FFFFFF)  
　	- 클릭 불가 (cursor: not-allowed)  
　	- 조건: 유형 1개 이상 선택 및 컵홀더 종류 선택 미완료 시  
　	• 활성화 시  
　	- 배경색: 파란색 (#0066FF)  
　	- 텍스트 색상: 흰색 (#FFFFFF)  
　	- 클릭 가능 (cursor: pointer)  
　	- 조건: 유형 1개 이상 선택 및 컵홀더 종류 선택 완료 시  
　cc. 버튼 클릭 시  
　	• 다음 화면(리워드/오더 선택)으로 이동  
　	• 선택한 정보는 자동 저장됨  
　E. 동작 흐름  
　　a. 화면 진입  
　　aa. 두 가지 유형 카드 표시 (단독, 콜라보)  
　　bb. 각 카드에 체크박스 표시 (미선택 상태)  
　　cc. 컵홀더 종류 선택 영역 및 광고군 제외 설정 영역은 아직 표시되지 않음  
　　dd. "다음 단계" 버튼 비활성화 상태  
　　b. 유형 선택  
　　aa. 카드 내 체크박스 클릭으로 선택/해제 가능  
　	• 체크박스 클릭 시 선택/해제 토글  
　	• 각 카드는 독립적으로 선택 가능 (단독만, 콜라보만, 둘 다 선택 가능)  
　bb. 단독 선택 시  
　	• "컵홀더 무료 제공" 뱃지 강조 표시  
　	• 컵홀더 종류 선택 영역 표시  
　cc. 콜라보 선택 시  
　	• "컵홀더 반값 지원" 뱃지 강조 표시  
　	• "로고 업로드는 캠페인 매칭 시 진행됩니다" 안내 표시  
　	• 컵홀더 종류 선택 영역 표시  
　dd. 둘 다 선택 시  
　	• 두 뱃지 모두 표시  
　	• 컵홀더 종류 선택 영역 표시  
　c. 컵홀더 종류 선택  
　　aa. 유형 1개 이상 선택 시 컵홀더 종류 선택 영역 표시  
　　bb. 라디오 버튼으로 양자 택일 선택  
　	• "컵홀더만 선택" 또는 "에어 홀더 + 컵홀더 둘 다 선택" 중 하나만 선택  
　	• 한쪽 선택 시 다른 쪽 자동 해제  
　cc. 선택 완료 시 광고군 제외 설정 영역 표시  
　d. 광고군 제외 설정  
　　aa. 컵홀더 종류 선택 완료 시 광고군 제외 설정 영역 표시  
　　bb. 광고군 선택창에서 최대 3개까지 선택 가능  
　	• 대분류별로 그룹화된 체크박스 목록 표시  
　	• 각 광고군에 체크박스 클릭으로 선택/해제  
　	• 선택 안 해도 다음 단계 진행 가능  
　cc. 선택 완료 시 "다음 단계" 버튼 활성화  
　e. 다음 단계 진행  
　　aa. "다음 단계" 버튼 클릭  
　　bb. 선택한 정보 저장  
　	• 유형 정보 (단독/콜라보, 각각 또는 둘 다)  
　	• 컵홀더 종류 정보 (컵홀더만/에어 홀더+컵홀더)  
　	• 광고군 제외 정보 (선택한 광고군 목록)  
　cc. 다음 화면(리워드/오더 선택)으로 이동  
　F. 입력 사항 활용 (파트너용 설명)  
　　a. 선택한 유형에 따라 캠페인 매칭 우선순위가 달라집니다  
　　b. 단독 선택 시 컵홀더 무료 제공, 콜라보 선택 시 반값 지원 혜택을 받을 수 있습니다  
　　c. 콜라보 선택 시 매장 브랜딩 효과를 얻을 수 있습니다  
　　d. 광고군 제외 설정을 통해 매장 이미지와 맞지 않는 광고군을 제외할 수 있습니다 (우선 순위에 따라 매칭 시 제외 대상으로 선정, 후순위로 밀리지만 정책상 100% 제외 되는 것은 아닙니다)  
　G. 입력 사항 활용 (개발자용)  
　　a. 캠페인 매칭  
　　aa. 선택한 유형에 따라 매칭되는 캠페인 필터링  
　	• 단독 선택 시: 단독 캠페인 우선 매칭  
　	• 콜라보 선택 시: 콜라보 캠페인 우선 매칭  
　	• 둘 다 선택 시: 단독 및 콜라보 캠페인 모두 매칭 가능  
　bb. 광고군 제외 설정  
　	• 선택한 광고군은 매칭 시 제외 대상으로 선정  
　	• 우선 순위에 따라 후순위로 밀리지만 정책상 100% 제외 되는 것은 아님  
　b. 컵홀더 디자인  
　　aa. 단독 선택 시 디자인 생성 프로세스  
　	• 광고주 디자인이 생성됨  
　	• 전면과 후면 모두 광고주 비주얼 중심으로 디자인 생성  
　bb. 콜라보 선택 시 디자인 생성 프로세스  
　	• 광고주 디자인이 먼저 생성됨  
　	• 파트너 매칭이 완료된 후 파트너 로고 디자인이 생성됨  
　	• 파트너 로고 업로드 시  
　		- 로고에 배경이 있는 경우: 광고주 배경 디자인과 파트너 로고가 만나는 경계선만 자연스럽게 연결  
　		- 로고에 배경이 없는 경우: 광고주 배경을 사용하고 파트너 로고만 삽입  
　c. 정산  
　　aa. 유형별 혜택 차등 적용 (단독 무료, 콜라보 반값)  
　　bb. 정산 시스템에서 유형 정보를 참조하여 비용 계산  
　d. 데이터 수집  
　　aa. 콜라보 선택 시 파트너 데이터 수집 가능  
　　bb. 수집된 데이터는 파트너 리포트 및 벤치마크 분석에 활용  
　e. 컵홀더 종류  
　　aa. 선택한 종류에 따라 발주 시스템에서 종류별 발주량 관리  
　　bb. 에어 홀더 + 컵홀더 선택 시 각각 독립적으로 수용량 관리  
　H. 컵홀더 QR 매핑 방식  
　　a. 기본 원칙  
　　aa. 1 컵홀더 = 1 캠페인  
　	• 컵홀더 1개에는 광고주 캠페인 1개만 인쇄  
　b. QR 종류 및 위치  
　　aa. 광고주 QR  
　	• 위치: 전면  
　	• 크기: 크게  
　	• 용도: 소비자 캠페인 참여 (스캔 → 설문 → 게임 → 리워드)  
　　bb. 위드폼 QR  
　	• 위치: 후면  
　	• 크기: 작게  
　	• 용도: 광고주/파트너 문의, 위드폼 서비스 안내  
　I. 컵홀더 유형 선택 시 고려사항  
　　a. 브랜딩 효과  
　　aa. 콜라보 선택 시 매장 로고 노출로 브랜딩 효과  
　　bb. 후면에 매장명과 로고가 함께 표시되어 고객 인지도 향상  
　b. 물류 및 배포  
　　aa. 컵홀더는 분할 발주 방식으로 배포  
　　bb. 각 파트너 매장의 소진 속도에 맞춰 필요한 만큼만 배송  
　　cc. 재고 리스크를 줄이고 캠페인 진행 중 유연하게 대응
