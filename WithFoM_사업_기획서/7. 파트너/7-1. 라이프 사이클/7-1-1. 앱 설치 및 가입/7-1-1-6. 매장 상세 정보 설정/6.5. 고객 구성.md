　　**6.5.** 고객 구성  


↳ 매장 상세 정보 설정 단계 중 다섯 번째 화면  

**진행 단계 표시:** 8단계 중 5단계  


　　　　　1: 상단 영역  
　　　　　　A. 제목: "고객 구성 입력"  
　　　　　　B. 설명 텍스트: "매장 고객의 성별 및 연령대 비율을 입력해 주세요"  
　　　　　　C. 진행 단계 표시: "8단계 중 5단계" (우측 상단)  

　　　　　2: 메인 입력 영역  

　　　　　　A. 연령대별 입력 섹션  
　　　　　　　　a. 연령대: 10대, 20대, 30대, 40대, 50대+  
　　　　　　　　b. 각 연령대별 입력 섹션 구성  
　　　　　　　　　　aa. 연령대명 표시 (예: "10대")  
　　　　　　　　　　bb. 성별 비율 슬라이더 (0%~100%)  
　　　　　　　　　　　　a. 슬라이더 왼쪽: 남성  
　　　　　　　　　　　　b. 슬라이더 오른쪽: 여성  
　　　　　　　　　　　　c. 슬라이더 조정으로 남/여 비율 입력  
　　　　　　　　　　　　d. 슬라이더를 왼쪽으로 이동하면 남성 비율 증가, 오른쪽으로 이동하면 여성 비율 증가  

　　　　　　　　　　cc. 현재 비율 표시: "남성: N%, 여성: M%" (슬라이더 하단)  

　　　　　　　　c. 슬라이더 동작  
　　　　　　　　　　aa. 각 연령대별 슬라이더는 독립적으로 조작 가능  
　　　　　　　　　　bb. 슬라이더 조작 시 실시간으로 비율 변경 및 표시  
　　　　　　　　　　cc. 슬라이더 위치에 따라 남/여 비율이 자동 계산되어 표시됨  

　　　　　　B. 총 남/여 비율 표시 영역  
　　　　　　　　a. 위치: 모든 연령대 입력 섹션 하단  
　　　　　　　　b. 표시 형식: "총 남성 비율: N%, 총 여성 비율: M%"  
　　　　　　　　c. 표시 조건: 모든 연령대의 남여 비율 입력 완료 시 자동 계산 및 표시  

　　　　　3: 하단 영역  

　　　　　　A. "다음 단계" 버튼  
　　　　　　　　a. 버튼 상태  
　　　　　　　　　　aa. 비활성화 시: 회색 (#CCCCCC), 클릭 불가  
　　　　　　　　　　bb. 활성화 시: 파란색 (#0066FF), 클릭 가능  
　　　　　　　　　　cc. 조건: 모든 연령대의 남여 비율 입력 완료 및 총 남/여 비율 표시 시 활성화  

　　　　　　　　b. 버튼 클릭 시  
　　　　　　　　　　aa. 다음 화면(음료 3종 가격 입력)으로 이동  
　　　　　　　　　　bb. 입력한 정보는 자동 저장됨  


　　　　　4: 동작 흐름  

　　　　　　A. 화면 진입  
　　　　　　　　a. 각 연령대별 입력 섹션 표시 (10대, 20대, 30대, 40대, 50대+)  
　　　　　　　　b. 모든 슬라이더 중앙 위치 (50%:50%)  
　　　　　　　　c. 총 남/여 비율 표시 영역은 아직 표시되지 않음  
　　　　　　　　d. "다음 단계" 버튼 비활성화 상태  

　　　　　　B. 연령대별 남여 비율 입력  
　　　　　　　　a. 슬라이더 조작으로 남/여 비율 입력  
　　　　　　　　　　aa. 슬라이더를 왼쪽으로 이동: 남성 비율 증가, 여성 비율 감소  
　　　　　　　　　　bb. 슬라이더를 오른쪽으로 이동: 여성 비율 증가, 남성 비율 감소  
　　　　　　　　　　cc. 슬라이더 위치에 따라 남/여 비율이 자동 계산되어 표시됨  

　　　　　　　　b. 슬라이더 조작 시 실시간으로 비율 변경 및 표시  
　　　　　　　　　　aa. 현재 비율 표시 업데이트: "남성: N%, 여성: M%"  

　　　　　　　　c. 모든 연령대의 남여 비율 입력 완료 시  
　　　　　　　　　　aa. 총 남/여 비율 자동 계산  
　　　　　　　　　　bb. 총 남성 비율 = (10대 남성 비율 × 10대 전체 비중) + (20대 남성 비율 × 20대 전체 비중) + ...  
　　　　　　　　　　cc. 총 여성 비율 = (10대 여성 비율 × 10대 전체 비중) + (20대 여성 비율 × 20대 전체 비중) + ...  
　　　　　　　　　　dd. 계산된 총 남/여 비율이 하단에 표시됨  
　　　　　　　　　　ee. "다음 단계" 버튼 활성화  

　　　　　　C. 다음 단계 진행  
　　　　　　　　a. 모든 연령대 입력 완료 및 총 남/여 비율 표시 시 "다음 단계" 버튼 클릭 가능  
　　　　　　　　b. 버튼 클릭 시 다음 화면(음료 3종 가격 입력)으로 이동  
　　　　　　　　c. 입력한 정보는 자동 저장됨  


　　　　　5: 입력 사항 활용 (파트너용 설명)  

　　　　　　A. 입력한 고객 구성 정보는 매장에 맞는 캠페인 매칭에 활용되며, 광고주가 타깃 고객층을 설정할 때 이를 바탕으로 적합한 캠페인이 매칭됩니다  
　　　　　　B. 고객 구성 정보는 매장 리포트에도 활용되어 고객 분석에 도움이 됩니다  


　　　　　6: 입력 사항 활용 (개발자용)  

　　　　　　A. 캠페인 매칭  
　　　　　　　　a. 광고주 타깃 연령대/성별과 매칭하여 적합한 캠페인 추천  
　　　　　　　　b. 각 연령대별 남/여 비율 정보를 활용하여 정밀 매칭  
　　　　　　　　c. 총 남/여 비율 정보를 활용하여 전체적인 매칭 우선순위 결정  

　　　　　　B. AI 개인화  
　　　　　　　　a. 소비자 프로파일링 시 고객 구성 정보 활용  
　　　　　　　　b. 연령대별 및 성별별 선호도 분석  

　　　　　　C. 데이터 분석  
　　　　　　　　a. 고객 구성별 이용 패턴 및 선호도 분석  
　　　　　　　　b. 연령대별 및 성별별 벤치마크 비교  
　　　　　　　　c. 매장 리포트 생성 시 고객 구성 정보 활용  

　　　　　　D. 마케팅 전략  
　　　　　　　　a. 타겟 고객층에 맞는 마케팅 전략 수립  
　　　　　　　　b. 광고주에게 제공하는 매장 정보에 고객 구성 정보 포함
