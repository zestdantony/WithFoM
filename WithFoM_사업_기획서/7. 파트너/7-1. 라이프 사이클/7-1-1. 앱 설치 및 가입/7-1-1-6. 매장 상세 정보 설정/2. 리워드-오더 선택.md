# 2: 리워드/오더 선택

　A. 서비스 타입 선택  
　　a. 리워드형  
　　aa. 리워드 기능 (☑️): 사용 가능  
　　bb. 설명: QR 스캔 후 게임/리워드만 제공, 앱 주문 불가, 메뉴 오더/관리 미노출  
　　b. 오더형  
　　aa. 오더 기능 (☑️): 사용 가능  
　　bb. 설명: 앱 주문·결제만 제공, 게임/리워드 없음  
　　c. 복합형 (리워드형 + 오더형)  
　　aa. 리워드 기능 (☑️): 사용 가능  
　　bb. 오더 기능 (☑️): 사용 가능  
　　cc. 설명: 리워드 + 오더 모두 사용, 쿠폰/포인트 결제 시 자동 적용  
　B. 화면 구성 및 동작 흐름  
　　a. 상단 영역  
　　aa. 제목: "서비스 타입 선택"  
　　bb. 설명 텍스트: "매장에서 제공할 서비스 타입을 선택해 주세요"  
　　cc. 진행 단계 표시: "3단계 중 2단계"  
　　b. 메인 선택 영역  
　　aa. 두 개의 큰 카드 형태 박스 (리워드형/오더형)  
　	bb. 각 카드 구성  
　	• 서비스 타입명 (큰 글씨)  
　	• 기능 상태 표시 (체크박스 형태)  
　	• 설명 텍스트  
　	cc. 카드 선택 방식  
　	• 각 카드에 체크박스 표시  
　	• 체크박스 클릭으로 선택/해제 가능  
　	• 하나만 선택 시: 각각의 기능만 적용  
　	• 둘 다 선택 시: 복합형으로 적용  
　	dd. 카드 선택 시  
　	• 체크박스 선택 표시  
　	• 테두리 색상 변경 (파란색)  
　	• 배경색 약간 변경  
　　c. 하단 영역  
　	aa. "다음 단계" 버튼  
　	• 최소 1개 이상 선택 완료 시 활성화  
　	• 버튼 비활성화 시: 회색, 클릭 불가  
　	• 버튼 활성화 시: 파란색, 클릭 가능  
　C. 동작 흐름  
　　a. 서비스 타입 선택  
　	aa. 화면 진입 시 두 가지 서비스 타입 카드 표시  
　	bb. 카드 내 체크박스 클릭으로 선택/해제  
　	cc. 리워드형만 선택 시: 리워드 기능만 활성화, 오더 기능 비활성화  
　	dd. 오더형만 선택 시: 오더 기능만 활성화, 리워드 기능 비활성화  
　	ee. 둘 다 선택 시: 복합형으로 적용, 리워드와 오더 기능 모두 활성화  
　　b. 선택 사항에 따른 표시  
　	aa. 리워드형 선택 시: 리워드 관련 메뉴 및 화면 표시  
　	bb. 오더형 선택 시: 오더 관련 메뉴 및 화면 표시  
　	cc. 복합형 선택 시: 리워드와 오더 관련 메뉴 및 화면 모두 표시  
　　c. 다음 단계 진행  
　	aa. 최소 1개 이상 선택 완료 시 "다음 단계" 버튼 활성화  
　	bb. 버튼 클릭 시 다음 화면(상권 입력)으로 이동  
　	cc. 선택한 정보는 자동 저장되며, 해당 기능들이 활성화됨  
　D. 입력 사항 활용 (파트너용 설명)  
　　a. 선택한 서비스 타입에 따라 매장에서 제공하는 서비스가 달라집니다  
　　b. 리워드형을 선택하면 고객에게 리워드를 제공할 수 있고, 오더형을 선택하면 앱 주문 서비스를 제공할 수 있습니다  
　　c. 복합형을 선택하면 리워드와 오더 기능을 모두 사용할 수 있어 더 다양한 서비스를 제공할 수 있습니다  
　E. 입력 사항 활용 (개발자용)  
　　a. 서비스 기능 활성화  
　	aa. 선택한 타입에 따라 리워드/오더 기능 활성화  
　	bb. 리워드형 선택 시: 리워드 관련 API 및 화면 활성화  
　	cc. 오더형 선택 시: 오더 관련 API 및 화면 활성화  
　	dd. 복합형 선택 시: 리워드와 오더 관련 API 및 화면 모두 활성화  
　　b. 캠페인 매칭  
　	aa. 리워드형 선택 시: 리워드 캠페인 매칭 우선  
　	bb. 오더형 선택 시: 오더 관련 캠페인 매칭 우선  
　	cc. 복합형 선택 시: 리워드와 오더 캠페인 모두 매칭 가능  
　　c. 메뉴 관리  
　	aa. 오더형/복합형 선택 시: 메뉴 등록 및 관리 기능 활성화  
　	bb. 리워드형만 선택 시: 메뉴 관리 기능 비활성화  
　　d. 정산  
　	aa. 서비스 타입에 따라 정산 항목 차등 적용  
　	bb. 리워드 사용 내역, 오더 결제 내역 등 타입별 정산 데이터 수집  
　　e. 화면 구성  
　	aa. 선택한 타입에 따라 표시되는 메뉴 및 화면 구성 변경  
　	bb. 리워드형 선택 시: 리워드 사용 내역 보드, 리워드 승인 기능 등 표시  
　	cc. 오더형 선택 시: 결제 내역 보드, 주문 승인 기능 등 표시  
　	dd. 복합형 선택 시: 리워드와 오더 관련 보드 및 기능 모두 표시
