# 6: 음료 3종 가격

　A. 음료 가격 입력  
　　a. 아메리카노 (필수 입력)  
　　aa. 반드시 입력해야 하는 필수 항목  
　	bb. 입력창 구성  
　	• 음료명: "아메리카노" (고정, 수정 불가, 회색 배경)  
　	• 가격 입력창: 숫자만 입력 가능  
　	• 단위 표시: "원" (입력창 우측에 표시)  
　	cc. 입력 방법  
　	• 숫자 키패드로 가격 입력  
　	• 500원 단위로 입력 (예: 3000, 3500, 4000 등)  
　	• 입력 완료 시 자동으로 "원" 단위 표시  
　	dd. 입력 검증  
　	• 최소 가격: 1000원  
　	• 최대 가격: 50000원  
　	• 500원 단위가 아닌 경우 경고 메시지 표시: "500원 단위로 입력해 주세요"  
　　b. 다른 음료 2종 (선택 입력)  
　	aa. 아메리카노 입력 완료 후 추가 가능  
　	bb. 카테고리 선택  
　	• 입력창 클릭 시 드롭다운 표시  
　	• 선택 가능 카테고리: 커피, 논커피, 에이드, 스무디, 티  
　	• 카테고리 선택 후 해당 카테고리 표시  
　	cc. 음료명 입력  
　	• 두 번째 입력창에 음료명 직접 입력 (텍스트 입력, 최대 20자)  
　	• 예시: "카페라떼", "카푸치노", "녹차라떼", "망고 에이드" 등  
　	• 특수문자 사용 가능 (이모지 제외)  
　	dd. 가격 입력  
　	• 세 번째 입력창에 가격 입력 (숫자 입력)  
　	• 500원 단위로 입력 (예: 3000, 3500, 4000 등)  
　	• 500원 단위로 입력하게 안내 문구 표시: "500원 단위로 입력해 주세요"  
　	• 입력 완료 시 자동으로 "원" 단위 표시  
　	ee. 입력창 생성 방법  
　	• "다른 음료 추가" 버튼 클릭  
　	• 클릭 시 카테고리 선택 드롭다운, 음료명 입력창, 가격 입력창이 함께 생성  
　	• 최대 2개까지 추가 가능 (총 3개 음료)  
　	ff. 입력창 삭제  
　	• 각 음료 입력창 우측에 "삭제" 버튼 표시 (X 버튼)  
　	• 삭제 버튼 클릭 시 해당 음료 입력창 제거  
　	• 삭제 후 다시 추가 가능  
　B. 입력 완료 및 저장  
　　a. 3가지 음료 모두 입력해야 "다음 단계" 버튼 활성화  
　　b. 아메리카노는 필수 입력 항목  
　　c. 나머지 2종의 음료는 카페마다 원하는 음료 선택하여 입력  
　　d. 입력한 정보는 자동 저장되며, 다음 단계로 진행  
　C. 화면 구성 및 동작 흐름  
　　a. 상단 영역  
　	aa. 제목: "음료 가격 입력"  
　	bb. 설명 텍스트: "매장 대표 음료 3종의 가격을 입력해 주세요"  
　	cc. 진행 단계 표시: "3단계 중 6단계"  
　　b. 메인 입력 영역  
　	aa. 음료 1: 아메리카노 (필수 입력)  
　	• 음료명: "아메리카노" (고정, 수정 불가, 회색 배경)  
　	• 가격 입력창: 숫자 키패드, "원" 단위 표시  
　	• 필수 표시: 빨간색 별표 (*)  
　	bb. 음료 2: 다른 음료 1 (선택 입력)  
　	• "다른 음료 추가" 버튼 (아메리카노 입력 완료 후 표시)  
　	• 버튼 클릭 시 카테고리 선택 드롭다운, 음료명 입력창, 가격 입력창 생성  
　	cc. 음료 3: 다른 음료 2 (선택 입력)  
　	• 첫 번째 다른 음료 입력 완료 후 두 번째 추가 버튼 표시  
　	• 동일한 방식으로 카테고리 선택, 음료명 입력, 가격 입력  
　　c. 하단 영역  
　	aa. 입력 가이드: "아메리카노는 필수 입력 항목입니다. 나머지 2종은 원하는 음료를 선택하여 입력해 주세요"  
　	bb. "다음 단계" 버튼  
　	• 3가지 음료 모두 입력 완료 시 활성화  
　	• 버튼 비활성화 시: 회색, 클릭 불가  
　	• 버튼 활성화 시: 파란색, 클릭 가능  
　D. 동작 흐름  
　　a. 아메리카노 입력 (필수)  
　	aa. 화면 진입 시 아메리카노 입력창 표시  
　	bb. 가격 입력창 클릭 시 숫자 키패드 팝업  
　	cc. 가격 입력 및 검증 (500원 단위, 1000원~50000원)  
　	dd. 아메리카노 입력 완료 시 "다른 음료 추가" 버튼 표시  
　　b. 다른 음료 추가 (2종 필수)  
　	aa. "다른 음료 추가" 버튼 클릭  
　	bb. 입력창 생성: 카테고리 선택 드롭다운 + 음료명 입력창 + 가격 입력창  
　	cc. 카테고리 선택: 드롭다운에서 커피, 논커피, 에이드, 스무디, 티 중 선택  
　	dd. 음료명 입력: 텍스트 입력창에 음료명 입력 (최대 20자)  
　	ee. 가격 입력: 숫자 입력, 500원 단위 (안내 문구 표시)  
　	ff. 입력 완료 후 "다른 음료 추가" 버튼 다시 표시 (최대 2개까지)  
　	gg. 삭제 버튼 클릭 시 해당 음료 입력창 제거  
　　c. 입력 완료 및 저장  
　	aa. 3가지 음료 모두 입력 완료 시 "다음 단계" 버튼 활성화  
　	bb. 버튼 클릭 시 다음 화면(1일 평균 컵홀더 소진량 입력)으로 이동  
　	cc. 입력한 정보는 자동 저장됨  
　E. 입력 사항 활용 (파트너용 설명)  
　　a. 입력한 음료 가격 정보는 매장에 맞는 캠페인 매칭에 활용됩니다  
　　b. 광고주는 제품 가격대를 고려하여 캠페인을 매칭하는데, 음료 가격 정보가 이를 도와줍니다  
　F. 입력 사항 활용 (개발자용)  
　　a. 캠페인 매칭  
　	aa. 음료 가격 정보를 바탕으로 적합한 캠페인 매칭  
　	bb. 광고주 제품 가격대와 매장 음료 가격대를 비교하여 매칭 우선순위 결정  
　	cc. 카테고리 정보를 활용하여 음료 유형별 캠페인 매칭
