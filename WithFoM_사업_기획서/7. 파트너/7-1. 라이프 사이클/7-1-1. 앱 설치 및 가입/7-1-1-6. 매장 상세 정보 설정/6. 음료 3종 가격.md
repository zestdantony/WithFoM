　　**6.6.** 음료 3종 가격  

↳ 매장 상세 정보 설정 단계 중 여섯 번째 화면  

**진행 단계 표시:** 8단계 중 6단계  

　A. 상단 영역  
　　a. 제목: "음료 가격 입력"  
　　b. 설명 텍스트: "매장 대표 음료 3종의 가격을 입력해 주세요"  
　　c. 진행 단계 표시: "8단계 중 6단계" (우측 상단)  
　B. 메인 입력 영역  
　　a. 음료 1: 아메리카노 (필수 입력)  
　　aa. 음료명: "아메리카노" (고정, 수정 불가, 회색 배경)  
　　bb. 가격 입력창  
　	• 숫자 키패드로 가격 입력  
　	• 단위 표시: "원" (입력창 우측에 표시)  
　	• 필수 표시: 빨간색 별표 (*)  
　	• 판매 가격 그대로 입력 (500원 단위 제한 없음)  
　cc. 입력 검증  
　	• 최소 가격: 1000원  
　	• 최대 가격: 50000원  
　b. 음료 2: 다른 음료 1 (필수 입력)  
　　aa. "다른 음료 추가" 버튼 (아메리카노 입력 완료 후 표시)  
　　bb. 버튼 클릭 시 입력창 생성  
　	• 카테고리 선택 드롭다운  
　		- 선택 가능 카테고리: 커피, 논커피, 에이드, 스무디, 티, 기타, 직접 입력  
　		- 입력창 클릭 시 드롭다운 표시  
　		- 카테고리 선택 후 해당 카테고리 표시  
　		- "기타" 선택 시 추가 입력창 없이 바로 음료명 입력 가능  
　		- "직접 입력" 선택 시 카테고리 없이 음료명만 입력 가능  
　	• 음료명 입력창 (텍스트 입력, 최대 20자)  
　		- 예시: "카페라떼", "카푸치노", "녹차라떼", "망고 에이드" 등  
　		- 특수문자 사용 가능 (이모지 제외)  
　	• 가격 입력창 (숫자 입력)  
　		- 판매 가격 그대로 입력 (500원 단위 제한 없음)  
　		- 입력 완료 시 자동으로 "원" 단위 표시  
　	• 삭제 버튼 (우측 상단 X 버튼)  
　cc. 입력 검증은 아메리카노와 동일  
　c. 음료 3: 다른 음료 2 (필수 입력)  
　　aa. 첫 번째 다른 음료 입력 완료 후 두 번째 "다른 음료 추가" 버튼 표시  
　　bb. 동일한 방식으로 카테고리 선택, 음료명 입력, 가격 입력  
　C. 하단 영역  
　　a. 입력 가이드: "아메리카노는 필수 입력 항목입니다. 나머지 2종은 원하는 음료를 선택하여 입력해 주세요"  
　　b. "다음 단계" 버튼  
　　aa. 버튼 상태  
　	• 비활성화 시: 회색 (#CCCCCC), 클릭 불가  
　	• 활성화 시: 파란색 (#0066FF), 클릭 가능  
　	• 조건: 3가지 음료 모두 입력 완료 시 활성화  
　bb. 버튼 클릭 시  
　	• 다음 화면(1일 평균 컵홀더 소진량 입력)으로 이동  
　	• 입력한 정보는 자동 저장됨  
　D. 동작 흐름  
　　a. 화면 진입  
　　aa. 아메리카노 입력창 표시  
　　bb. "다른 음료 추가" 버튼은 아직 표시되지 않음  
　　cc. "다음 단계" 버튼 비활성화 상태  
　　b. 아메리카노 입력 (필수)  
　　aa. 가격 입력창 클릭 시 숫자 키패드 팝업  
　　bb. 가격 입력 및 검증 (1000원~50000원, 판매 가격 그대로 입력)  
　　cc. 아메리카노 입력 완료 시 "다른 음료 추가" 버튼 표시  
　　c. 다른 음료 추가 (2종 필수)  
　　aa. "다른 음료 추가" 버튼 클릭  
　　bb. 입력창 생성: 카테고리 선택 드롭다운 + 음료명 입력창 + 가격 입력창  
　　cc. 카테고리 선택: 드롭다운에서 커피, 논커피, 에이드, 스무디, 티, 기타, 직접 입력 중 선택  
　	• "기타" 선택 시: 카테고리 없이 바로 음료명 입력 가능  
　	• "직접 입력" 선택 시: 카테고리 없이 음료명만 입력 가능  
　dd. 음료명 입력: 텍스트 입력창에 음료명 입력 (최대 20자)  
　ee. 가격 입력: 숫자 입력, 판매 가격 그대로 입력 (500원 단위 제한 없음)  
　ff. 첫 번째 다른 음료 입력 완료 후 두 번째 "다른 음료 추가" 버튼 표시  
　gg. 두 번째 다른 음료도 동일한 방식으로 입력  
　hh. 삭제 버튼 클릭 시 해당 음료 입력창 제거 (다시 추가 가능)  
　d. 입력 완료 및 저장  
　　aa. 3가지 음료 모두 입력 완료 시 "다음 단계" 버튼 활성화  
　　bb. 버튼 클릭 시 다음 화면(1일 평균 컵홀더 소진량 입력)으로 이동  
　　cc. 입력한 정보는 자동 저장됨  
　E. 입력 사항 활용 (파트너용 설명)  
　　a. 입력한 음료 가격 정보는 매장에 맞는 캠페인 매칭에 활용됩니다  
　　b. 광고주는 제품 가격대를 고려하여 캠페인을 매칭하는데, 음료 가격 정보가 이를 도와줍니다  
　F. 입력 사항 활용 (개발자용)  
　　a. 캠페인 매칭  
　　aa. 음료 가격 정보를 바탕으로 적합한 캠페인 매칭  
　　bb. 광고주 제품 가격대와 매장 음료 가격대를 비교하여 매칭 우선순위 결정  
　　cc. 카테고리 정보를 활용하여 음료 유형별 캠페인 매칭
