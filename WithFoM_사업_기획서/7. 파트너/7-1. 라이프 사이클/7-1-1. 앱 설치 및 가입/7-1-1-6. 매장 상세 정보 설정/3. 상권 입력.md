# 3: 상권 입력

　A. 주변 상권 유형 선택  
　　a. 선택 개수  
　　aa. 최소 2개 선택  
　　bb. 최대 4개 선택  
　　b. 상권 유형 종류  
　　aa. 주거지역  
　　bb. 오피스  
　　cc. 상가 밀집  
　　dd. 역세권  
　　ee. 관광지  
　　ff. 학교  
　	• 학교 선택 시 세부 유형 의무 선택  
　	• 초등학교  
　	• 중학교  
　	• 고등학교  
　	• 대학교  
　B. 화면 구성 및 동작 흐름  
　　a. 상단 영역  
　	aa. 제목: "상권 정보 입력"  
　	bb. 설명 텍스트: "매장 주변 상권 유형을 선택해 주세요 (최소 2개, 최대 4개)"  
　	cc. 진행 단계 표시: "3단계 중 3단계"  
　　b. 메인 선택 영역  
　	aa. 체크박스 형태의 상권 유형 목록  
　	bb. 학교 선택 시 세부 유형 선택 드롭다운 표시  
　　c. 하단 영역  
　	aa. 선택 개수 표시: "2개 선택됨 / 최소 2개 필요"  
　	bb. "다음 단계" 버튼  
　	• 최소 2개 선택 시 활성화  
　	• 최대 4개 선택 시 나머지 체크박스 비활성화  
　	• 버튼 비활성화 시: 회색, 클릭 불가  
　	• 버튼 활성화 시: 파란색, 클릭 가능  
　C. 동작 흐름  
　　a. 상권 유형 선택  
　	aa. 화면 진입 시 상권 유형 목록 표시  
　	bb. 체크박스 클릭으로 선택/해제  
　	cc. 학교 선택 시 세부 유형 드롭다운 표시 및 의무 선택  
　	dd. 최소 2개 선택 시 "다음 단계" 버튼 활성화  
　	ee. 최대 4개 선택 시 나머지 체크박스 비활성화  
　　b. 다음 단계 진행  
　	aa. 최소 2개 이상 선택 완료 시 "다음 단계" 버튼 클릭 가능  
　	bb. 버튼 클릭 시 다음 화면(고객 층 입력)으로 이동  
　	cc. 선택한 정보는 자동 저장됨  
　D. 상권 정보 활용  
　　a. 캠페인 매칭  
　	aa. 광고주 타깃 설정과 매칭  
　	bb. 동일 상권 우선 배정  
　　b. 데이터 분석  
　	aa. 동일 상권(반경 500m) 평균 대비 지표 제공  
　	bb. 상권별 벤치마크 비교  
　E. 입력 사항 활용 (파트너용 설명)  
　　a. 입력한 상권 정보는 매장에 맞는 캠페인 매칭에 활용됩니다  
　　b. 광고주는 타깃 상권을 설정하는데, 이를 바탕으로 적합한 캠페인이 매칭됩니다  
　　c. 동일 상권의 다른 매장과 벤치마크 비교를 통해 매장 운영에 도움이 되는 데이터를 제공받을 수 있습니다  
　F. 입력 사항 활용 (개발자용)  
　　a. 캠페인 매칭  
　	aa. 광고주 타깃 설정과 매칭하여 적합한 캠페인 추천  
　	bb. 동일 상권 우선 배정으로 매칭 확률 향상  
　	cc. 상권 유형 정보를 활용하여 매칭 알고리즘에 가중치 부여  
　　b. 우선 배정  
　	aa. 동일 상권 우선 배정으로 매칭 확률 향상  
　	bb. 상권 유형 일치도에 따라 매칭 우선순위 결정  
　　c. 데이터 분석  
　	aa. 동일 상권(반경 500m) 평균 대비 지표 제공  
　	bb. 상권별 벤치마크 비교 데이터 생성  
　	cc. 매장 리포트 생성 시 상권 정보 활용  
　　d. 매칭 알고리즘  
　	aa. 상권 유형 정보를 매칭 알고리즘의 주요 변수로 활용  
　	bb. 광고주 타깃 상권과 파트너 매장 상권의 일치도를 계산하여 매칭 점수 산출
