# 5: 고객 구성

　A. 성별·연령대 비율 입력  
　　a. 각 연령대별 남여 비율 입력  
　　aa. 10대 남/여 비율 슬라이더(%)  
　　bb. 20대 남/여 비율 슬라이더(%)  
　　cc. 30대 남/여 비율 슬라이더(%)  
　　dd. 40대 남/여 비율 슬라이더(%)  
　　ee. 50대+ 남/여 비율 슬라이더(%)  
　　ff. 각 연령대별 남/여 슬라이더는 연동되어 합계 100% 유지  
　　gg. 한쪽 슬라이더 조작 시 다른 쪽 자동 조정  
　　hh. 슬라이더 조작 시 실시간 합계 표시  
　　b. 총 남/여 비율 표시  
　　aa. 모든 연령대의 남여 비율 입력 완료 시 하단에 총 남/여 비율 자동 계산 및 표시  
　　bb. 총 남성 비율: 모든 연령대 남성 비율의 합  
　　cc. 총 여성 비율: 모든 연령대 여성 비율의 합  
　　dd. 총 남/여 비율의 합계는 100%가 되어야 함  
　B. 화면 구성 및 동작 흐름  
　　a. 상단 영역  
　　aa. 제목: "고객 구성 입력"  
　　bb. 설명 텍스트: "매장 고객의 성별 및 연령대 비율을 입력해 주세요"  
　　cc. 진행 단계 표시: "3단계 중 5단계"  
　　b. 메인 입력 영역  
　　aa. 연령대별 입력 섹션 (10대, 20대, 30대, 40대, 50대+)  
　	bb. 각 연령대별 입력 섹션 구성  
　	• 연령대명 표시  
　	• 남성 비율 슬라이더 (0%~100%)  
　	• 여성 비율 슬라이더 (0%~100%)  
　	• 현재 비율 표시: "남성: N%, 여성: M%"  
　	cc. 각 연령대별 남/여 슬라이더는 연동됨  
　	• 남성 슬라이더 조작 시 여성 슬라이더 자동 조정  
　	• 여성 슬라이더 조작 시 남성 슬라이더 자동 조정  
　	• 합계 항상 100% 유지  
　　c. 하단 영역  
　	aa. 총 남/여 비율 표시 영역  
　	• 모든 연령대 입력 완료 시 표시  
　	• "총 남성 비율: N%, 총 여성 비율: M%" 형식으로 표시  
　	bb. "다음 단계" 버튼  
　	• 모든 연령대의 남여 비율 입력 완료 시 활성화  
　	• 총 남/여 비율 표시와 동시에 활성화  
　	• 버튼 비활성화 시: 회색, 클릭 불가  
　	• 버튼 활성화 시: 파란색, 클릭 가능  
　C. 동작 흐름  
　　a. 연령대별 남여 비율 입력  
　	aa. 화면 진입 시 각 연령대별 입력 섹션 표시  
　	bb. 슬라이더 조작으로 남/여 비율 입력  
　	cc. 슬라이더 조작 시 실시간으로 비율 변경 및 표시  
　	dd. 남성 슬라이더 조작 시 여성 슬라이더 자동 조정 (합계 100% 유지)  
　	ee. 여성 슬라이더 조작 시 남성 슬라이더 자동 조정 (합계 100% 유지)  
　　b. 총 남/여 비율 계산 및 표시  
　	aa. 모든 연령대의 남여 비율 입력 완료 시 자동 계산  
　	bb. 총 남성 비율 = (10대 남성 비율 × 10대 전체 비중) + (20대 남성 비율 × 20대 전체 비중) + ...  
　	cc. 총 여성 비율 = (10대 여성 비율 × 10대 전체 비중) + (20대 여성 비율 × 20대 전체 비중) + ...  
　	dd. 계산된 총 남/여 비율이 하단에 표시됨  
　　c. 다음 단계 진행  
　	aa. 모든 연령대 입력 완료 및 총 남/여 비율 표시 시 "다음 단계" 버튼 활성화  
　	bb. 버튼 클릭 시 다음 화면(음료 3종 가격 입력)으로 이동  
　	cc. 입력한 정보는 자동 저장됨  
　D. 입력 사항 활용 (파트너용 설명)  
　　a. 입력한 고객 구성 정보는 매장에 맞는 캠페인 매칭에 활용됩니다  
　　b. 광고주는 타깃 고객층을 설정하는데, 이를 바탕으로 적합한 캠페인이 매칭됩니다  
　　c. 고객 구성 정보는 매장 리포트에도 활용되어 고객 분석에 도움이 됩니다  
　E. 입력 사항 활용 (개발자용)  
　　a. 캠페인 매칭  
　	aa. 광고주 타깃 연령대/성별과 매칭하여 적합한 캠페인 추천  
　	bb. 각 연령대별 남/여 비율 정보를 활용하여 정밀 매칭  
　	cc. 총 남/여 비율 정보를 활용하여 전체적인 매칭 우선순위 결정  
　　b. AI 개인화  
　	aa. 소비자 프로파일링 시 고객 구성 정보 활용  
　	bb. 연령대별 및 성별별 선호도 분석  
　　c. 데이터 분석  
　	aa. 고객 구성별 이용 패턴 및 선호도 분석  
　	bb. 연령대별 및 성별별 벤치마크 비교  
　	cc. 매장 리포트 생성 시 고객 구성 정보 활용  
　　d. 마케팅 전략  
　	aa. 타겟 고객층에 맞는 마케팅 전략 수립  
　	bb. 광고주에게 제공하는 매장 정보에 고객 구성 정보 포함
