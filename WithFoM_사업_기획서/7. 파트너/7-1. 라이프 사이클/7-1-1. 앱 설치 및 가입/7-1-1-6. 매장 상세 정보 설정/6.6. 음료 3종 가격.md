　　**6.6.** 음료 3종 가격  


↳ 매장 상세 정보 설정 단계 중 여섯 번째 화면  

**진행 단계 표시:** 8단계 중 6단계  


　　　　　1: 상단 영역  
　　　　　　A. 제목: "음료 가격 입력"  
　　　　　　B. 설명 텍스트: "매장 대표 음료 3종의 가격을 입력해 주세요"  
　　　　　　C. 진행 단계 표시: "8단계 중 6단계" (우측 상단)  

　　　　　2: 메인 입력 영역  

　　　　　　A. 음료 1: 아메리카노 (필수 입력)  
　　　　　　　　a. 음료명: "아메리카노" (고정, 수정 불가, 회색 배경)  
　　　　　　　　b. 가격 입력창  
　　　　　　　　　　aa. 숫자 키패드로 가격 입력  
　　　　　　　　　　bb. 단위 표시: "원" (입력창 우측에 표시)  
　　　　　　　　　　cc. 필수 표시: 빨간색 별표 (*)  
　　　　　　　　　　dd. 판매 가격 그대로 입력 (500원 단위 제한 없음)  

　　　　　　　　c. 입력 검증  
　　　　　　　　　　aa. 최소 가격: 1000원  
　　　　　　　　　　bb. 최대 가격: 50000원  

　　　　　　B. 음료 2: 다른 음료 1 (필수 입력)  
　　　　　　　　a. "다른 음료 추가" 버튼 (아메리카노 입력 완료 후 표시)  
　　　　　　　　b. 버튼 클릭 시 입력창 생성  
　　　　　　　　　　aa. 카테고리 선택 드롭다운  
　　　　　　　　　　　　a. 선택 가능 카테고리: 커피, 논커피, 에이드, 스무디, 티, 기타, 직접 입력  
　　　　　　　　　　　　b. 입력창 클릭 시 드롭다운 표시  
　　　　　　　　　　　　c. 카테고리 선택 후 해당 카테고리 표시  
　　　　　　　　　　　　d. "기타" 선택 시 추가 입력창 없이 바로 음료명 입력 가능  
　　　　　　　　　　　　e. "직접 입력" 선택 시 카테고리 없이 음료명만 입력 가능  

　　　　　　　　　　bb. 음료명 입력창 (텍스트 입력, 최대 20자)  
　　　　　　　　　　　　a. 예시: "카페라떼", "카푸치노", "녹차라떼", "망고 에이드" 등  
　　　　　　　　　　　　b. 특수문자 사용 가능 (이모지 제외)  

　　　　　　　　　　cc. 가격 입력창 (숫자 입력)  
　　　　　　　　　　　　a. 판매 가격 그대로 입력 (500원 단위 제한 없음)  
　　　　　　　　　　　　b. 입력 완료 시 자동으로 "원" 단위 표시  

　　　　　　　　　　dd. 삭제 버튼 (우측 상단 X 버튼)  

　　　　　　　　c. 입력 검증은 아메리카노와 동일  

　　　　　　C. 음료 3: 다른 음료 2 (필수 입력)  
　　　　　　　　a. 첫 번째 다른 음료 입력 완료 후 두 번째 "다른 음료 추가" 버튼 표시  
　　　　　　　　b. 동일한 방식으로 카테고리 선택, 음료명 입력, 가격 입력  

　　　　　3: 하단 영역  

　　　　　　A. 입력 가이드: "아메리카노는 필수 입력 항목입니다. 나머지 2종은 원하는 음료를 선택하여 입력해 주세요"  

　　　　　　B. "다음 단계" 버튼  
　　　　　　　　a. 버튼 상태  
　　　　　　　　　　aa. 비활성화 시: 회색 (#CCCCCC), 클릭 불가  
　　　　　　　　　　bb. 활성화 시: 파란색 (#0066FF), 클릭 가능  
　　　　　　　　　　cc. 조건: 3가지 음료 모두 입력 완료 시 활성화  

　　　　　　　　b. 버튼 클릭 시  
　　　　　　　　　　aa. 다음 화면(1일 평균 컵홀더 소진량 입력)으로 이동  
　　　　　　　　　　bb. 입력한 정보는 자동 저장됨  


　　　　　4: 동작 흐름  

　　　　　　A. 화면 진입  
　　　　　　　　a. 아메리카노 입력창 표시  
　　　　　　　　b. "다른 음료 추가" 버튼은 아직 표시되지 않음  
　　　　　　　　c. "다음 단계" 버튼 비활성화 상태  

　　　　　　B. 아메리카노 입력 (필수)  
　　　　　　　　a. 가격 입력창 클릭 시 숫자 키패드 팝업  
　　　　　　　　b. 가격 입력 및 검증 (1000원~50000원, 판매 가격 그대로 입력)  
　　　　　　　　c. 아메리카노 입력 완료 시 "다른 음료 추가" 버튼 표시  

　　　　　　C. 다른 음료 추가 (2종 필수)  
　　　　　　　　a. "다른 음료 추가" 버튼 클릭  
　　　　　　　　b. 입력창 생성: 카테고리 선택 드롭다운 + 음료명 입력창 + 가격 입력창  
　　　　　　　　c. 카테고리 선택: 드롭다운에서 커피, 논커피, 에이드, 스무디, 티, 기타, 직접 입력 중 선택  
　　　　　　　　　　aa. "기타" 선택 시: 카테고리 없이 바로 음료명 입력 가능  
　　　　　　　　　　bb. "직접 입력" 선택 시: 카테고리 없이 음료명만 입력 가능  

　　　　　　　　d. 음료명 입력: 텍스트 입력창에 음료명 입력 (최대 20자)  
　　　　　　　　e. 가격 입력: 숫자 입력, 판매 가격 그대로 입력 (500원 단위 제한 없음)  
　　　　　　　　f. 첫 번째 다른 음료 입력 완료 후 두 번째 "다른 음료 추가" 버튼 표시  
　　　　　　　　g. 두 번째 다른 음료도 동일한 방식으로 입력  
　　　　　　　　h. 삭제 버튼 클릭 시 해당 음료 입력창 제거 (다시 추가 가능)  

　　　　　　D. 입력 완료 및 저장  
　　　　　　　　a. 3가지 음료 모두 입력 완료 시 "다음 단계" 버튼 활성화  
　　　　　　　　b. 버튼 클릭 시 다음 화면(1일 평균 컵홀더 소진량 입력)으로 이동  
　　　　　　　　c. 입력한 정보는 자동 저장됨  


　　　　　5: 입력 사항 활용 (파트너용 설명)  

　　　　　　A. 입력한 음료 가격 정보는 매장에 맞는 캠페인 매칭에 활용됩니다  
　　　　　　B. 광고주는 제품 가격대를 고려하여 캠페인을 매칭하는데, 음료 가격 정보가 이를 도와줍니다  


　　　　　6: 입력 사항 활용 (개발자용)  

　　　　　　A. 캠페인 매칭  
　　　　　　　　a. 음료 가격 정보를 바탕으로 적합한 캠페인 매칭  
　　　　　　　　b. 광고주 제품 가격대와 매장 음료 가격대를 비교하여 매칭 우선순위 결정  
　　　　　　　　c. 카테고리 정보를 활용하여 음료 유형별 캠페인 매칭
