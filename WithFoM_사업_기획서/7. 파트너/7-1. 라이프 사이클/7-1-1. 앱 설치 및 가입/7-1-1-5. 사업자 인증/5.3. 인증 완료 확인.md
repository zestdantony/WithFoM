　　**5.3.** 인증 완료 확인  


　　　　**5.3.1.** 개요  

사업자 인증이 완료되면 인증 결과를 확인하고 다음 단계로 진행할 수 있습니다.  


　　　　**5.3.2.** 인증 완료 확인  

　　　　　1: 인증 성공 화면  

인증이 성공하면 아래와 같은 화면이 표시됩니다:  

```
┌─────────────────────────┐
│   인증 완료 ✅           │
├─────────────────────────┤
│                         │
│  사업자 인증이          │
│  완료되었습니다.        │
│                         │
│  확인된 정보:           │
│  • 사업자명: (자동 입력)│
│  • 대표자명: (자동 입력)│
│  • 사업장 주소: (자동 입력)│
│                         │
│  [다음 단계]            │
│                         │
└─────────────────────────┘
```


　　　　**5.3.3.** 인증 완료 후 자동 입력 정보  

국세청 API 인증을 통해 자동으로 입력되는 정보:  

|| 항목 | 설명 |  
|---|---|  
|| 사업자명 | 사업자등록증상 상호명 (국세청 API에서 자동 입력) |  
|| 대표자명 | 대표자명 (국세청 API에서 자동 입력) |  
|| 사업장 주소 | 사업장 소재지 (국세청 API에서 자동 입력) |  

※ 국세청 API는 정부 공식 데이터이므로 정확하며, 자동 입력된 정보는 수정할 필요가 없습니다.  


　　　　**5.3.4.** 정보 확인  

　　　　　1: 자동 입력 정보 확인  
　　　　　　A. 국세청 API에서 자동으로 입력된 정보 확인  
　　　　　　B. 사업자등록증과 일치하는지 확인  
　　　　　　C. 매장 기본 정보 입력 단계에서 입력한 정보와 일치하는지 확인  

> ✅ **확인 완료:** 국세청 API 정보는 정부 공식 데이터이므로 정확합니다. 별도 수정이 필요하지 않습니다.  


　　　　**5.3.5.** 다음 단계 진행  

인증 완료 후:  

　　　　　1: 정보 확인 완료  
　　　　　　A. 자동 입력된 정보 확인  
　　　　　　B. 국세청 API 정보는 정확하므로 수정 불필요  

　　　　　2: 다음 단계로 진행  
　　　　　　A. "다음 단계" 버튼 클릭  
　　　　　　B. 매장 상세 정보 설정 단계로 이동  


　　　　**5.3.6.** 인증 완료 상태 확인  

앱 내에서 인증 완료 상태를 확인할 수 있습니다:  

|| 확인 위치 | 내용 |  
|---|---|  
|| 가입 진행 상황 | "3. 사업자 인증" 단계에 체크 표시 |  
|| 인증 정보 | 설정 > 매장 정보에서 인증 정보 확인 |  


　　　　**5.3.7.** 참고  

• 국세청 API는 정부 공식 데이터이므로 자동 입력된 정보는 정확합니다  
• 인증 완료 후 다음 단계로 진행할 수 있습니다  
• 인증 정보는 사업자등록증과 일치하며, 별도 수정이 필요하지 않습니다
