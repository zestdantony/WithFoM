　　**5.1.** 사업자 인증 방법  


　　　1: 개요  

매장 기본 정보 입력 단계에서 입력한 사업자등록번호를 기반으로, 국세청 공공데이터포털의 사업자등록번호 진위확인 API를 통해 자동으로 사업자 정보를 인증합니다.  


　　　2: 사업자등록번호 자동 반영  

　A. 이전 단계 연동  
　　a. 매장 기본 정보 입력 단계에서 입력한 사업자등록번호가 본 화면의 입력창에 자동으로 반영됩니다  
　　b. 중복 입력 없이 바로 인증 절차를 진행할 수 있습니다  
　　c. 필요 시 사업자등록번호를 수정할 수 있습니다  
　B. 입력 형식  
　　a. 하이픈 포함: 123-45-67890  
　　b. 하이픈 미포함: 1234567890  
　　c. 두 형식 모두 입력 가능하며, 시스템이 자동으로 형식 통일  


　　　3: 인증 절차  

　A. 1단계: 사업자등록번호 확인  

　　a. 자동 반영된 사업자등록번호 확인  
　　b. 필요 시 번호 수정 가능  
　　c. "인증 시작" 버튼 클릭  

　B. 2단계: 자동 인증 진행  

| 단계 | 내용 |
|---|---|
| API 호출 | 국세청 공공 API로 사업자등록번호 진위확인 요청 |
| 정보 확인 | 사업자명, 대표자명, 사업장 주소 등 정보 확인 |
| 일치 검증 | 매장 기본 정보 입력 단계에서 입력한 정보와 API에서 받은 정보 일치 여부 확인 |

### 3단계: 인증 결과

| 결과 | 설명 | 다음 단계 |
|---|---|---|
| 인증 성공 | 사업자등록번호가 유효하고 정보가 일치함 | 다음 단계 진행 |
| 인증 실패 | 사업자등록번호가 유효하지 않거나 정보 불일치 | 오류 메시지 표시 및 재입력 요청 |
| API 오류 | 국세청 API 서버 오류 | 잠시 후 재시도 안내 |

## 인증 확인 항목

국세청 API를 통해 확인하는 정보:

| 확인 항목 | 설명 |
|---|---|
| 사업자등록번호 유효성 | 등록된 사업자등록번호인지 확인 |
| 사업자명 | 사업자등록증상 상호명 확인 |
| 대표자명 | 대표자명 확인 |
| 사업장 주소 | 사업장 소재지 확인 |
| 사업 상태 | 휴업/폐업 여부 확인 |

## 인증 완료 후 자동 입력

인증이 성공하면 국세청 API에서 받은 정보가 자동으로 입력됩니다:

| 자동 입력 항목 | 설명 |
|---|---|
| 사업자명 | 사업자등록증상 상호명 (매장 기본 정보 입력 단계의 사업자명과 일치 여부 확인) |
| 대표자명 | 대표자명 (매장 기본 정보 입력 단계의 대표자명과 일치 여부 확인) |
| 사업장 주소 | 사업장 소재지 (매장 기본 정보 입력 단계의 매장 주소와 일치 여부 확인) |

※ 국세청 API는 정부 공식 데이터이므로 정확하며, 자동 입력된 정보는 수정할 필요가 없습니다.

## 인증 실패 시 대응

### 사업자등록번호 오류

　　a. 증상: "유효하지 않은 사업자등록번호입니다" 메시지  
　　b. 해결 방법  
　　　aa. 사업자등록증에서 정확한 번호 확인  
　　　bb. 오타 확인 (0과 O, 1과 I 등)  
　　　cc. 정확한 번호로 재입력  

　B. 정보 불일치  
　　a. 증상: "입력하신 정보와 등록된 정보가 일치하지 않습니다" 메시지  
　　b. 해결 방법  
　　　aa. 사업자등록증에서 정확한 정보 확인  
　　　bb. 매장 기본 정보 입력 단계에서 입력한 사업자명, 대표자명 등 재확인  
　　　cc. 정확한 정보로 수정  

　C. API 서버 오류  
　　a. 증상: "인증 서비스에 일시적인 오류가 발생했습니다" 메시지  
　　b. 해결 방법  
　　　aa. 잠시 후 재시도  
　　　bb. 네트워크 연결 확인  
　　　cc. 지속적인 오류 시 고객센터 문의  


　　　7: 다음 단계 진행  

인증이 완료되면:  

　A. 정보 자동 입력 확인  
　　a. 국세청 API에서 받은 정보가 자동으로 입력됨  
　　b. 입력된 정보 확인 (수정 불필요)  

　B. 다음 단계 진행  
　　a. 인증 완료 후 다음 단계(매장 상세 정보 설정)로 진행 가능  


　　　8: 개인정보 보호  

| 항목 | 내용 |
|---|---|
| API 사용 목적 | 사업자등록번호 진위확인 및 정보 확인 |
| 정보 보관 | 인증 완료 후 필요한 정보만 저장 |
| 정보 공유 | 국세청 API 외 제3자와 공유하지 않음 |

## 참고

- 국세청 공공 API는 정부에서 제공하는 공식 서비스입니다
- 인증은 실시간으로 진행되며 보통 몇 초 내 완료됩니다
- 인증 실패 시 정확한 정보로 재시도하면 대부분 해결됩니다
- 매장 기본 정보 입력 단계에서 입력한 정보와 국세청 API 정보가 일치해야 인증이 완료됩니다
