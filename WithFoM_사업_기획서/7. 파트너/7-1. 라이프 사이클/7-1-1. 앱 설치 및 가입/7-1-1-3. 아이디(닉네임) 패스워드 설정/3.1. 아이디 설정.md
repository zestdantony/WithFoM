　　**3.1.** 아이디 설정  


　　　　　1: 아이디 형식  
　　　　　　A. 영문, 숫자 조합  
　　　　　　B. 특수문자 사용 불가  

　　　　　2: 사용 가능 아이디 확인  
　　　　　　A. 중복 확인 버튼 클릭  
　　　　　　B. 사용 가능 시 "사용 가능한 아이디입니다" 메시지 표시  
　　　　　　C. 중복 시 "이미 사용 중인 아이디입니다" 메시지 표시  

　　　　　3: 아이디 활용  
　　　　　　A. 아이디는 로그인에 사용됩니다  
　　　　　　B. 관리자가 파트너를 검색할 때 사용됩니다  
　　　　　　C. 소비자에게 노출되는 것은 매장명이며, 매장명은 별도로 입력합니다  

　　　　　4: 아이디 수정  
　　　　　　A. 아이디는 가입 후 수정 불가합니다
