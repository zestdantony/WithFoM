**8.4.** 액션 버튼 및 화면 전환


**8.4.1.** 파트너 계약서 작성 화면 구성

1: 화면 진입
　A. 트리거: 매장 상세 정보 설정 완료 후 [완료] 버튼 클릭
　B. 이전 단계: 7-1-1-7. 매장 상세 정보 설정
　C. 진행 상태: 8/9 단계

2: 화면 요소
　A. 자동 생성된 계약서 내용 표시
　B. 스크롤 가능한 계약서 본문
　C. 서명 영역
　D. [서명하기] 버튼
　E. [다운로드] 버튼 (서명 완료 후)


**8.4.2.** 계약서 확인 화면

1: 계약서 자동 생성
　A. 이전 단계에서 입력한 모든 정보 반영
　B. 파트너 정보, 매장 정보, 서비스 조건 등 표시

2: 계약서 스크롤
　A. 세로 스크롤로 전체 계약서 확인
　B. 페이지 인디케이터 또는 진행률 표시

3: 정보 수정 링크
　A. 정보 오류 발견 시 [정보 수정] 링크 탭
　B. 해당 정보 입력 화면으로 이동
　C. 수정 후 계약서 자동 재생성


**8.4.3.** [서명하기] 버튼 동작

1: 버튼 활성화 조건
　A. 계약서 전체 스크롤 확인 (선택적)
　B. 정보 확인 체크박스 선택 (선택적)

2: 버튼 클릭 시 동작
　A. 서명 화면/모달 표시
　B. 서명 입력 영역 활성화


**8.4.4.** 서명 입력 화면

1: 서명 방법 선택
　A. [직접 서명] 버튼: 손가락/스타일러스로 서명
　B. [전자서명] 버튼: 공동인증서 또는 간편인증 연동

2: 직접 서명
　A. 서명 캔버스 표시
　B. 손가락 또는 스타일러스로 서명 그리기
　C. [다시 쓰기] 버튼: 서명 지우고 재입력
　D. [확인] 버튼: 서명 저장

3: 전자서명
　A. 인증 방법 선택 (공동인증서/간편인증)
　B. 인증 완료 후 자동 서명 반영


**8.4.5.** 서명 완료 후 화면

1: 서명 확인
　A. 계약서에 서명이 반영된 미리보기 표시
　B. "계약서 서명이 완료되었습니다" 메시지

2: 버튼 변경
　A. [서명하기] → [서명 완료] (비활성화)
　B. [계약서 다운로드] 버튼 활성화
　C. [이메일로 받기] 버튼 활성화
　D. [다음] 버튼 활성화


**8.4.6.** [계약서 다운로드] 버튼 동작

1: 버튼 클릭 시 동작
　A. PDF 파일 생성 및 다운로드
　B. 기기 저장소에 파일 저장
　C. 다운로드 완료 토스트 메시지

2: 파일 정보
　A. 파일명: WithFoM_파트너_계약서_[매장명]_[날짜].pdf
　B. 파일 형식: PDF
　C. 파일 크기: 약 500KB~1MB


**8.4.7.** [이메일로 받기] 버튼 동작

1: 자동 발송
　A. 서명 완료 즉시 등록된 이메일로 자동 발송

2: 버튼 클릭 시 동작 (재발송)
　A. 등록된 이메일 주소 확인 팝업
　B. [발송] 버튼: 이메일 재발송
　C. 발송 완료 토스트 메시지


**8.4.8.** [다음] 버튼 동작

1: 버튼 활성화 조건
　A. 서명 완료

2: 버튼 클릭 시 동작
　A. 계약 정보 최종 저장
　B. 다음 단계로 화면 전환: 7-1-1-9. 가입 완료
　C. 전환 애니메이션: 슬라이드 (좌 → 우)


**8.4.9.** 뒤로가기 동작

1: 계약서 확인 화면에서 뒤로가기
　A. 확인 팝업: "계약서 작성을 취소하시겠습니까?"
　B. [취소] 버튼: 팝업 닫힘, 현재 화면 유지
　C. [확인] 버튼: 7-1-1-7. 매장 상세 정보 설정 화면으로 이동

2: 서명 화면에서 뒤로가기
　A. 서명 화면/모달 닫힘
　B. 계약서 확인 화면으로 복귀

3: 서명 완료 후 뒤로가기
　A. 서명 취소 확인 팝업
　B. 서명 취소 시 서명 정보 초기화


**8.4.10.** 다음 단계로의 화면 전환

1: 파트너 계약서 작성 → 가입 완료
　A. 트리거: 서명 완료 후 [다음] 버튼 클릭
　B. 이동 경로: 7-1-1-9. 가입 완료 화면

2: 진행 상태 표시 업데이트
　A. 현재 단계: 8/9 → 9/9
　B. 다음 단계 예고: "다음: 가입 완료"


**8.4.11.** 모바일 환경 고려사항

1: 계약서 확인
　A. 핀치 줌 지원 (확대/축소)
　B. 더블 탭 확대 지원

2: 서명 입력
　A. 서명 캔버스 충분한 크기 확보
　B. 손가락 터치 최적화
　C. 스타일러스 지원 (iPad Pro 등)

3: 버튼 배치
　A. 하단 고정 버튼 영역
　B. 스크롤 시에도 항상 접근 가능


• 참조: 7-1-1-9. 가입 완료
