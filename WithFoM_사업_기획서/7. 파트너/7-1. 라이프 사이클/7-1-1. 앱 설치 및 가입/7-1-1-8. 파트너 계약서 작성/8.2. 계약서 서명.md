**8.2.** 계약서 서명

↳ 계약서 내용 확인 후 전자서명 진행

**📌 중요:** 본 문서는 가입 절차 안내용입니다. 실제 파트너 계약서 양식 전문(서명 절차 포함)은 아래 경로에서 확인하실 수 있습니다.

**파트너 계약서 양식 전문:** `11. 법무 및 정책\11-2. 계약서 양식\1. 파트너 계약서 양식.md`


**8.2.1.** 서명 절차

**8.2.1.1.** 1단계: 계약서 내용 최종 확인
　1: 계약서 전체 내용 재확인
　2: 모든 정보 정확성 확인
　3: 계약 조건 이해 확인

**8.2.1.2.** 2단계: 서명 진행

| 단계 | 내용 |
|---|---|
| 서명 버튼 클릭 | [서명하기] 버튼 클릭 |
| 서명 방법 선택 | 직접 서명 또는 전자서명 선택 |
| 서명 완료 | 서명 완료 후 확인 |

**8.2.1.3.** 3단계: 서명 확인
　1: 서명 완료 확인 팝업 표시
　2: 서명된 계약서 미리보기
　3: [확인] 버튼 클릭하여 확정


**8.2.2.** 서명 방법

**8.2.2.1.** 직접 서명

| 방법 | 설명 |
|---|---|
| 터치 서명 | 화면에 손가락으로 직접 서명 |
| 스타일러스 서명 | 스타일러스 펜으로 서명 (지원 기기) |
| 서명 확인 | 서명 후 미리보기로 확인 |

**8.2.2.2.** 서명 시 주의사항
　1: 명확하고 읽기 쉬운 서명
　2: 서명 영역 내에 서명
　3: 서명 후 재확인

**8.2.2.3.** 전자서명

| 방법 | 설명 |
|---|---|
| 공동인증서 | 공동인증서(구 공인인증서)를 통한 전자서명 |
| 간편인증 | 카카오, 네이버, PASS 등 간편인증 연동 |

**8.2.2.4.** 전자서명 완료 시 계약서에 자동 반영


**8.2.3.** 서명 완료 후

**8.2.3.1.** 계약서 확정

| 단계 | 내용 |
|---|---|
| 서명 완료 | 서명이 계약서에 반영됨 |
| 계약서 저장 | 서명된 계약서가 시스템에 저장됨 |
| 다음 단계 진행 | 계약서 다운로드 및 이메일 수신 단계로 진행 |

**8.2.3.2.** 서명 수정 (확정 전)
　1: [다시 서명] 버튼으로 서명 취소
　2: 새로운 서명 진행
　3: 재서명 후 최종 확인


**8.2.4.** 개인사업자 vs 법인사업자

**8.2.4.1.** 개인사업자

| 항목 | 내용 |
|---|---|
| 서명자 | 대표자 본인 서명 |
| 서명 방법 | 직접 서명 또는 전자서명 |

**8.2.4.2.** 법인사업자

| 항목 | 내용 |
|---|---|
| 서명자 | 대표자 또는 법인 인감 |
| 서명 방법 | 대표자 서명 또는 법인 인감 날인 (전자서명) |


**8.2.5.** 서명 관련 주의사항

**8.2.5.1.** 법적 효력 안내

| 주의사항 | 내용 |
|---|---|
| 법적 효력 | 서명 완료 시 계약서가 법적 구속력을 가짐 |
| 서명 전 확인 | 서명 전 반드시 계약서 전체 내용 확인 |
| 서명 후 수정 | 서명 완료 후 계약서 내용 수정 불가 |
| 서명 취소 | 서명 완료 전에만 서명 취소 가능 |


**8.2.6.** 서명 완료 확인

**8.2.6.1.** 확인 사항
　1: 서명이 명확하게 표시됨
　2: 서명이 계약서에 정상적으로 반영됨
　3: 서명 완료 확인 팝업 확인
　4: 다음 단계(계약서 다운로드) 진행 가능
