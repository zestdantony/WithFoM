**8.1.** 계약서 내용 확인

↳ 자동 생성된 파트너 계약서 확인 절차  
**진행 단계 표시:** 8/9 단계


**8.1.1.** 개요

매장 상세 정보 입력이 완료되면 자동으로 생성된 계약서를 확인합니다. 계약서에는 입력한 모든 정보가 자동으로 반영됩니다.

**📌 중요:** 본 문서는 가입 절차 안내용입니다. 실제 파트너 계약서 양식 전문은 아래 경로에서 확인하실 수 있습니다.

**파트너 계약서 양식 전문:** `11. 법무 및 정책\11-2. 계약서 양식\1. 파트너 계약서 양식.md`

**8.1.4.** 계약서 내용 구성

**8.1.4.1.** 계약서 주요 조항

| 구분 | 내용 |
|---|---|
| 제1조 | 목적 및 정의 |
| 제2조 | 계약의 체결 및 효력 |
| 제3조 | 서비스 이용 조건 |
| 제4조 | 파트너의 권리 및 의무 |
| 제5조 | 위드폼의 권리 및 의무 |
| 제6조 | 정산 및 지급 |
| 제7조 | 등급 및 혜택 |
| 제8조 | 계약 기간 및 해지 |
| 제9조 | 손해배상 및 면책 |
| 제10조 | 분쟁 해결 및 준거법 |
| 부칙 | 계약서 일자 및 서명란 |

**8.1.4.2.** 주요 조항 요약
　1: 제3조 — 서비스 이용 조건

| 항목 | 내용 |
|---|---|
| 컵홀더 배포 | 위드폼이 제공하는 컵홀더의 배포 및 관리 (필수) |
| 일일 소진 보고 | 영업 종료 후 일일 소진량 보고 의무 |

　2: 제4조 — 파트너의 권리 및 의무

| 권리 | 의무 |
|---|---|
| 캠페인 매칭 및 수익 기회 | 정확한 매장 정보 제공 |
| 등급별 혜택 수령 | 일일 소진 보고 성실 이행 |
| 정산 금액 수령 | 서비스 이용약관 준수 |
| - | 매칭된 캠페인 참여 (필수) |
| - | 부정 사용 방지 (FDS 위반 금지) |
| - | 소비자의 QR 스캔 유도 |

　3: 제6조 — 정산 및 지급

| 항목 | 내용 |
|---|---|
| 정산 주기 | 매월 말일 기준 월말 정산 |
| 정산 금액 | 인센티브(플래티넘 등급) 지급 |
| 정산 계좌 | 파트너가 등록한 정산 계좌로 입금 |
| 정산 확정 | 정산 내역 확인 후 확정 (이의 신청 7일 이내) |

　4: 제7조 — 등급 및 혜택

| 등급 | 혜택 |
|---|---|
| 브론즈 | 위드폼 서비스 이용 |
| 실버 | 컵홀더 무료 제공 |
| 골드 | 컵홀더 무료 제공 + 우선 매칭 + 멤버십 10% 할인 |
| 플래티넘 | 컵홀더 무료 제공 + 최우선 매칭 + 멤버십 30% 할인 + 인센티브 지급 |

　5: 제8조 — 계약 기간 및 해지

| 항목 | 내용 |
|---|---|
| 계약 기간 | 2년 |
| 자동 연장 | 2년 후 자동 연장 (별도 해지 의사 표시 없는 경우) |
| 파트너 해지 | 30일 전 서면 통보, 미처리 주문/미정산 없어야 함 |
| 조기 해지 | 최소 2년 전 계약 해지 시 인센티브 반환 |
| 캠페인 포기 누적 | 총 3회 포기 시 계약 일시 정지 (관리자 합당 사유 인정 시 포기 횟수 미포함, 정지 해제 시 초기화) |
| 캠페인 참여 중 중도 포기 | 남은 컵홀더에 대한 비용을 파트너가 회사에 즉시 정산 (남은 수량 × 포기 시점 기준가) |
| 강제 해지 | 파트너의 잘못(약관 위반 등)으로 인한 해지 시 인센티브 반환 |
| 위드폼 해지 | 서비스 이용약관 중대 위반 시 즉시 해지 가능 |
| 데이터 보관 | 해지 후 30일 유예 기간, 이후 영구 삭제 |


**8.1.5.** 계약서 확인 방법

**8.1.5.1.** 1단계: 계약서 화면 확인
　1: 매장 상세 정보 입력 완료 후 자동으로 생성된 계약서 화면 표시
　2: 계약서 전체 내용 세로 스크롤하여 확인
　3: 핀치 줌 지원 (확대/축소)

**8.1.5.2.** 2단계: 정보 정확성 확인

| 확인 항목 | 확인 방법 |
|---|---|
| 파트너 정보 | 대표자명, 사업자등록번호, 사업자명 등 정확성 확인 |
| 매장 정보 | 매장명, 주소, 연락처 등 정확성 확인 |
| 계약 조건 | 서비스 이용 조건, 정산 조건 등 확인 |

**8.1.5.3.** 3단계: 계약서 내용 검토
　1: 계약서 전체 조항 읽기
　2: 이해가 되지 않는 부분 확인
　3: 필요 시 고객센터 문의


**8.1.6.** 정보 수정

**8.1.6.1.** 수정 절차

| 수정 방법 | 내용 |
|---|---|
| 이전 단계로 이동 | [정보 수정] 링크 탭 → 해당 정보가 입력된 단계로 이동 |
| 정보 수정 후 재생성 | 정보 수정 후 계약서 자동 재생성 |
| 최종 확인 | 수정된 계약서 내용 재확인 |


**8.1.7.** 계약서 확인 체크리스트

**8.1.7.1.** 확인 항목
　1: 대표자 성명 정확함
　2: 사업자등록번호 정확함 (10자리)
　3: 사업자명 정확함
　4: 매장명 정확함
　5: 사업장 소재지 정확함
　6: 연락처 정확함
　7: 이메일 정확함
　8: 정산 계좌 정보 정확함
　9: 서비스 타입 정확함
　10: 계약서 전체 내용 검토 완료
