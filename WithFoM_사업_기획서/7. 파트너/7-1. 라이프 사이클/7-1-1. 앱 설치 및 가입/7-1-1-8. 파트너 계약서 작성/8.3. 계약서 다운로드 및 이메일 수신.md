**8.3.** 계약서 다운로드 및 이메일 수신

↳ 서명 완료된 계약서의 다운로드 및 이메일 수신

**📌 중요:** 본 문서는 가입 절차 안내용입니다. 실제 파트너 계약서 양식 전문(다운로드 및 이메일 수신 절차 포함)은 아래 경로에서 확인하실 수 있습니다.

**파트너 계약서 양식 전문:** `11. 법무 및 정책\11-2. 계약서 양식\1. 파트너 계약서 양식.md`


**8.3.1.** 계약서 다운로드

**8.3.1.1.** 다운로드 방법

| 방법 | 설명 |
|---|---|
| 앱 내 다운로드 | 앱 내에서 직접 PDF 파일 다운로드 |
| 다운로드 버튼 | [계약서 다운로드] 버튼 클릭 |
| 파일 저장 | 기기 저장 공간에 PDF 파일 저장 |

**8.3.1.2.** 다운로드 파일 정보

| 항목 | 내용 |
|---|---|
| 파일 형식 | PDF (Portable Document Format) |
| 파일명 | WithFoM_파트너_계약서_[매장명]_[날짜].pdf |
| 파일 크기 | 약 500KB ~ 1MB |
| 포함 내용 | 계약서 전체 내용 및 서명 |

**8.3.1.3.** 다운로드 절차
　1: 서명 완료 후 [계약서 다운로드] 버튼 클릭
　2: 다운로드 진행 상태 확인 (프로그레스 바)
　3: 다운로드 완료 토스트 메시지: "계약서가 다운로드되었습니다"
　4: 파일 저장 위치 확인


**8.3.2.** 이메일 수신

**8.3.2.1.** 이메일 자동 발송

| 항목 | 내용 |
|---|---|
| 발송 시점 | 계약서 서명 완료 즉시 |
| 수신 주소 | 가입 시 입력한 이메일 주소 |
| 발송 내용 | 서명된 계약서 PDF 파일 첨부 |

**8.3.2.2.** 이메일 내용

| 항목 | 내용 |
|---|---|
| 제목 | [위드폼 파트너] 파트너 계약서 |
| 본문 | 계약서 서명 완료 안내 및 계약서 첨부 |
| 첨부 파일 | 서명된 계약서 PDF 파일 |

**8.3.2.3.** 이메일 수신 확인

| 확인 방법 | 내용 |
|---|---|
| 이메일 수신함 확인 | 등록한 이메일 주소의 수신함 확인 |
| 스팸 폴더 확인 | 스팸 폴더에 분류되었을 수 있음 |
| 이메일 재발송 | 미수신 시 [이메일로 받기] 버튼 클릭하여 재발송 |


**8.3.3.** 계약서 보관

**8.3.3.1.** 보관 방법

| 방법 | 설명 |
|---|---|
| 앱 내 보관 | 설정 > 계약서에서 언제든 확인 가능 |
| 다운로드 보관 | 기기에 다운로드하여 보관 |
| 이메일 보관 | 이메일로 받은 계약서 보관 |

**8.3.3.2.** 보관 권장사항
　1: 여러 방법으로 다중 보관 권장
　2: 중요한 문서이므로 백업 권장
　3: 필요 시 쉽게 접근할 수 있는 위치에 보관


**8.3.4.** 계약서 재다운로드

**8.3.4.1.** 재다운로드 방법

| 방법 | 설명 |
|---|---|
| 앱 내 메뉴 | 설정 > 계약서 > 계약서 다운로드 |
| 이메일 재발송 | 설정 > 계약서 > 이메일 재발송 |

**8.3.4.2.** 재다운로드 가능 시점
　1: 계약서 분실 시
　2: 추가 백업이 필요한 경우
　3: 이메일 미수신 시


**8.3.5.** 다운로드 및 이메일 수신 오류 해결

**8.3.5.1.** 다운로드 오류

| 오류 | 해결 방법 |
|---|---|
| 다운로드 실패 | 네트워크 연결 확인 후 재시도 |
| 저장 공간 부족 | 저장 공간 확보 후 재시도 |
| 파일 손상 | 재다운로드 시도 |

**8.3.5.2.** 이메일 미수신

| 원인 | 해결 방법 |
|---|---|
| 이메일 주소 오류 | 이메일 주소 확인 및 수정 |
| 스팸 분류 | 스팸 폴더 확인 |
| 이메일 서버 오류 | 이메일 재발송 요청 |


**8.3.6.** 다운로드한 계약서 확인

**8.3.6.1.** 확인 항목
　1: PDF 뷰어로 파일이 정상적으로 열리는지 확인
　2: 계약서 내용이 정상적으로 표시되는지 확인
　3: 서명이 정상적으로 표시되는지 확인
