　　**6.1.** 사업자 인증 방법  


　　　　　1: 사업자 인증 개요  
　　　　　　A. 국세청 API 연동을 통한 사업자 진위 확인  
　　　　　　B. 사업자등록번호 입력 후 인증 버튼 클릭  
　　　　　　C. 실시간 조회를 통해 사업자 상태 확인  

　　　　　2: 인증 절차  
　　　　　　A. 사업자등록번호 입력 (이전 단계에서 자동 반영)  
　　　　　　B. [인증하기] 버튼 클릭  
　　　　　　C. 국세청 API 조회 진행  
　　　　　　D. 인증 결과 표시  

　　　　　3: 인증 결과  
　　　　　　A. 인증 성공: 사업자명, 대표자명, 사업장 주소 자동 표시  
　　　　　　B. 인증 실패: 오류 메시지 표시 및 재시도 안내
