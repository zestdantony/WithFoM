**6.3.** 인증 완료 확인  

↳ 사업자 인증 성공 후 확인 사항 및 자동 반영 정보  


**6.3.1.** 인증 완료 표시  

**6.3.1.1.** 상태 아이콘: 초록색 체크마크  

**6.3.1.2.** 상태 메시지: "사업자 인증이 완료되었습니다"  

**6.3.1.3.** 인증 완료 아이콘은 사업자등록번호 필드 옆에 표시  


**6.3.2.** 인증된 사업자 정보 표시  

**6.3.2.1.** 표시 항목  

| 항목 | 출처 | 수정 가능 여부 |
|---|---|---|
| 사업자명 | 국세청 API 조회 결과 | 수정 불가 |
| 대표자명 | 국세청 API 조회 결과 | 수정 불가 |
| 사업장 소재지 | 국세청 API 조회 결과 | 수정 불가 |
| 업태 | 국세청 API 조회 결과 | 수정 불가 |
| 종목 | 국세청 API 조회 결과 | 수정 불가 |

**6.3.2.2.** 필드 상태  
　1: 배경색: 연한 회색 (자동 기입 표시)  
　2: 잠금 아이콘 표시 (수정 불가)  


**6.3.3.** 자동 반영 정보  

**6.3.3.1.** 사업자명 → 7-1-1-5. 매장 기본 정보 입력의 사업자명(5.2) 필드에 자동 반영  

**6.3.3.2.** 대표자명과 본인 인증(7-1-1-3) 이름 일치 여부 확인  
　1: 일치: 정상 진행  
　2: 불일치: 안내 메시지 표시 — "본인 인증 이름과 대표자명이 다릅니다. 확인 후 진행해주세요"  


**6.3.4.** 인증 후 수정  

**6.3.4.1.** 사업자등록번호 수정 시 인증 결과 초기화  
　1: 인증 완료 상태 해제  
　2: [인증하기] 버튼 재활성화  
　3: [다음] 버튼 비활성화  

**6.3.4.2.** 인증된 사업자 정보는 직접 수정 불가 (재인증 필요)  


**6.3.5.** 다음 단계  

**6.3.5.1.** 인증 완료 후 [다음] 버튼 활성화  

**6.3.5.2.** 이동 경로: 7-1-1-7. 매장 상세 정보 설정  

**6.3.5.3.** 전달 데이터: 인증된 사업자 정보 (사업자명, 대표자명, 주소)
