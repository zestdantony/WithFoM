**6.2.** 인증 오류 해결 가이드  

↳ 사업자 인증 실패 시 유형별 해결 방법  


**6.2.1.** 일반적인 오류  

**6.2.1.1.** 사업자등록번호 오류  
　1: 원인: 번호 입력 오류 또는 형식 불일치  
　2: 해결: 사업자등록증 원본 확인 후 번호 재입력  
　3: 안내 메시지: "사업자등록번호를 확인해주세요"  

**6.2.1.2.** 휴업 상태 사업자  
　1: 원인: 국세청 등록 상태가 "휴업"  
　2: 해결: 국세청 홈택스에서 사업자 상태 확인  
　3: 안내 메시지: "휴업 상태의 사업자입니다. 고객센터로 문의해주세요"  

**6.2.1.3.** 폐업 상태 사업자  
　1: 원인: 국세청 등록 상태가 "폐업"  
　2: 해결: 위드폼 서비스 가입 불가, 사업자 상태 변경 후 재시도  
　3: 안내 메시지: "폐업 상태의 사업자입니다"  

**6.2.1.4.** 네트워크 오류  
　1: 원인: 인터넷 연결 불안정 또는 서버 통신 실패  
　2: 해결: 인터넷 연결 확인 후 [다시 시도] 버튼 클릭  
　3: 안내 메시지: "네트워크 오류가 발생했습니다"  


**6.2.2.** 조회 불가 오류  

**6.2.2.1.** 신규 사업자  
　1: 원인: 국세청 데이터베이스에 아직 등록되지 않음  
　2: 해결: 사업자등록 후 1~2일 대기 후 재시도  
　3: 안내 메시지: "신규 등록 사업자입니다. 1~2일 후 다시 시도해주세요"  

**6.2.2.2.** 간이과세자 등 일부 사업자  
　1: 원인: API 조회 범위 밖의 사업자 유형  
　2: 해결: 수동 인증 요청 필요  
　3: 안내 메시지: "사업자 정보를 조회할 수 없습니다"  


**6.2.3.** 수동 인증 요청  

**6.2.3.1.** 수동 인증 대상  
　1: 자동 인증 반복 실패 시  
　2: 조회 불가 사업자 (간이과세자 등)  
　3: 신규 사업자로 대기가 어려운 경우  

**6.2.3.2.** 수동 인증 절차  
　1: "수동 인증 요청하기" 링크 클릭  
　2: 사업자등록증 사본 촬영 또는 갤러리에서 선택  
　3: 이미지 업로드 및 검토 요청 제출  
　4: 영업일 기준 1~2일 내 검토 완료  

**6.2.3.3.** 수동 인증 결과 알림  
　1: 승인: 푸시 알림 + 앱 내 알림으로 안내, [다음] 버튼 자동 활성화  
　2: 반려: 반려 사유 안내 + 재제출 요청  


**6.2.4.** 반복 실패 시 고객센터 안내  

**6.2.4.1.** 3회 이상 연속 실패 시 고객센터 연결 안내 표시  

**6.2.4.2.** [고객센터 문의] 버튼: 고객센터 화면 또는 전화 연결
