**6.4.** 액션 버튼 및 화면 전환


**6.4.1.** 사업자 인증 화면 구성

**6.4.1.1.** 화면 진입
　1: 트리거: 매장 기본 정보 입력 완료 후 [다음] 버튼 클릭
　2: 이전 단계: 7-1-1-5. 매장 기본 정보 입력
　3: 진행 상태: 6/9 단계

**6.4.1.2.** 화면 요소
　1: 사업자등록번호 입력/표시 필드 (이전 단계에서 자동 반영)
　2: [인증하기] 버튼
　3: 인증 결과 표시 영역
　4: 수동 인증 요청 링크 (자동 인증 실패 시)


**6.4.2.** 사업자등록번호 자동 반영

**6.4.2.1.** 이전 단계 데이터 표시
　1: 7-1-1-5에서 입력한 사업자등록번호 자동 표시
　2: 필드 상태: 수정 가능 (필요 시 재입력)

**6.4.2.2.** 번호 수정 시 동작
　1: 이전 인증 결과 초기화
　2: [인증하기] 버튼 재활성화


**6.4.3.** [인증하기] 버튼 동작

**6.4.3.1.** 버튼 활성화 조건
　1: 사업자등록번호 10자리 형식 유효

**6.4.3.2.** 버튼 클릭 시 동작
　1: 로딩 스피너 표시
　2: 국세청 API 연동하여 사업자 진위 확인
　3: 인증 결과에 따라 화면 업데이트

**6.4.3.3.** 인증 중 UI
　1: 버튼 비활성화 + 로딩 스피너
　2: "사업자 정보를 확인 중입니다..." 메시지


**6.4.4.** 인증 결과 표시

**6.4.4.1.** 인증 성공 시

| 표시 항목 | 내용 |
|---|---|
| 상태 아이콘 | 초록색 체크마크 |
| 상태 메시지 | "사업자 인증이 완료되었습니다" |
| 사업자명 | 국세청 API에서 조회된 사업자명 |
| 대표자명 | 국세청 API에서 조회된 대표자명 |
| 사업장 소재지 | 국세청 API에서 조회된 주소 |

**6.4.4.2.** 인증 실패 시

| 실패 유형 | 메시지 | 추가 동작 |
|---|---|---|
| 번호 오류 | "사업자등록번호를 확인해주세요" | 재입력 요청 |
| 휴업/폐업 | "휴업 또는 폐업 상태의 사업자입니다" | 고객센터 안내 |
| 조회 불가 | "사업자 정보를 조회할 수 없습니다" | 수동 인증 안내 |
| 네트워크 오류 | "네트워크 오류가 발생했습니다" | 재시도 버튼 |


**6.4.5.** 수동 인증 요청

**6.4.5.1.** 수동 인증 링크 클릭
　1: 트리거: "수동 인증 요청하기" 링크 탭
　2: 동작: 수동 인증 안내 화면/팝업 표시

**6.4.5.2.** 수동 인증 절차 안내
　1: 사업자등록증 사본 업로드
　2: 업로드 완료 시 검토 요청 제출
　3: 영업일 기준 1~2일 내 검토 완료 안내

**6.4.5.3.** 사업자등록증 업로드
　1: [사진 촬영] 버튼: 카메라 앱 실행
　2: [앨범에서 선택] 버튼: 갤러리 앱 실행
　3: 업로드 완료 후 미리보기 표시


**6.4.6.** [다음] 버튼 동작

**6.4.6.1.** 버튼 활성화 조건
　1: 사업자 인증 완료 (자동 또는 수동 검토 완료)

**6.4.6.2.** 버튼 클릭 시 동작
　1: 인증 정보 저장
　2: 다음 단계로 화면 전환: 7-1-1-7. 매장 상세 정보 설정
　3: 전환 애니메이션: 슬라이드 (좌 → 우)

**6.4.6.3.** 수동 인증 대기 중
　1: [다음] 버튼: 비활성화
　2: 안내 메시지: "인증 검토가 완료되면 알림을 보내드립니다"
　3: 앱 종료 후에도 검토 완료 시 푸시 알림 발송


**6.4.7.** 뒤로가기 동작

**6.4.7.1.** 사업자 인증 화면에서 뒤로가기
　1: 이동 경로: 7-1-1-5. 매장 기본 정보 입력 화면
　2: 입력 정보: 유지됨

**6.4.7.2.** 수동 인증 화면에서 뒤로가기
　1: 이동 경로: 사업자 인증 메인 화면
　2: 업로드 정보: 초기화


**6.4.8.** 다음 단계로의 화면 전환

**6.4.8.1.** 사업자 인증 → 매장 상세 정보 설정
　1: 트리거: 인증 완료 후 [다음] 버튼 클릭
　2: 이동 경로: 7-1-1-7. 매장 상세 정보 설정 화면
　3: 전달 데이터: 인증된 사업자 정보 (사업자명, 대표자명, 주소)

**6.4.8.2.** 진행 상태 표시 업데이트
　1: 현재 단계: 6/9 → 7/9
　2: 다음 단계 예고: "다음: 매장 상세 정보 설정"


**6.4.9.** 에러 상황 화면 처리

**6.4.9.1.** 인증 재시도
　1: [다시 시도] 버튼 표시
　2: 버튼 클릭: 인증 프로세스 재시작

**6.4.9.2.** 고객센터 연결
　1: 반복 실패 시 고객센터 연결 안내
　2: [고객센터 문의] 버튼: 고객센터 화면/전화 연결


• 참조: 7-1-1-7. 매장 상세 정보 설정
