　　**1.2.** 리워드 승인/거절 처리  

　A. 승인  
　　　a. 직접 승인  
　　　　aa. 파트너가 직접 승인 버튼을 클릭하여 즉시 승인 처리합니다.  
　　　b. 자동 승인  
　　　　aa. 1분(60초) 후 자동 승인 처리됩니다. (Default)  
　　　　bb. 원형 카운터 UI로 60초 카운트다운이 표시됩니다.  
　　　　cc. 타이머 종료 시 자동 승인 처리되며, 파트너에게 자동 승인 알림이 발송됩니다.  
　　　c. 승인 이력 관리  
　　　　aa. 승인 완료 후 최근 5개 이력이 유지됩니다.  
　　　　bb. 매일 영업시간 시작 시 초기화됩니다. (24시간 매장 제외)  
　　　d. 사용 완료 처리  
　　　　aa. 승인되면 즉시 "사용 완료" 처리됩니다.  
　　　　bb. 소비자 계정에서 쿠폰이 회수됩니다.  
　　　• 참조: WithFoM_Master_Planning_v3.10.md 11장 11-6절 11-6-3. 리워드 운영 정책 (광고주·파트너 공통)의 "리워드 사용 흐름 (최종 확정)" 표 및 3장 3-8절 3-8-3. 1분 자동 승인 타이머 UI 표  
　B. 거절  
　　　a. 거절 사유 선택  
　　　　aa. 재고 소진  
　　　　bb. 영업시간 외  
　　　　cc. 조건 불일치  
　　　　dd. 기타(직접 입력)  
　　　b. 소비자 안내 문구  
　　　　aa. "죄송합니다. [사유]로 인해 사용이 어렵습니다. 다른 매장에서 이용해주세요." 문구가 표시됩니다.  
　　　c. 재참여 정책  
　　　　aa. 거절된 리워드는 유효기간 내 다른 매장에서 재사용 가능합니다.  
　　　　bb. 유효기간 자동 연장은 없습니다.  
　　　d. 거절 이력 관리  
　　　　aa. 파트너별 거절 사유 통계가 관리됩니다.  
　　　　bb. 거절률이 높은 매장은 관리자에게 알림이 발송됩니다.
