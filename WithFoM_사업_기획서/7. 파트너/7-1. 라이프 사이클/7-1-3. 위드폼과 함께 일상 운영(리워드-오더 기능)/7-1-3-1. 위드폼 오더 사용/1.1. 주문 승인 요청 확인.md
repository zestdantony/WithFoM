**1.1.1.** 결제 내역 보드 확인  

**1.1.1.1.** 보드 구성 및 기능  

　1: 대상  
　　A. 위드폼 결제 기능을 사용하는 파트너만 해당 보드가 노출됩니다.  
　　B. 결제 기능 미사용 매장은 보드가 미노출됩니다.  

　2: 구성  
　　A. 최대 5개 카드에 최근 결제 내역이 표시됩니다.  

　3: 카드 표시 항목  
　　A. 주문 내용(메뉴·옵션·금액)  
　　B. 사용된 리워드 정보  
　　C. 사용자명  
　　D. 매장명/로고(배경)  

　4: 하단 버튼  
　　A. 승인 / 거절 버튼  
　　B. 60초 카운트다운 타이머 표시  

　5: 자동 승인  
　　A. 60초 이내 미응답 시 자동 승인 처리됩니다.  

　6: 클릭 시 이동  
　　A. 카드 클릭 시 "결제 페이지"로 이동합니다.
