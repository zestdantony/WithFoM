**3.6.** 일일 소진 보고 보드  


**3.6.1.** 보드 상태 표시  

**3.6.1.1.** 상태 정보  

　1: 보드 이름 옆 상태 표시  
　　A. 보드 이름 옆에 "오늘 보고 완료/미완료" 표시  
　　B. 보고 완료 시: "완료" 표시  
　　C. 보고 미완료 시: "미완료" 표시  

　2: 보드 색상  
　　A. 미완료: 빨간색  
　　B. 완료: 그린 컬러  
　　C. 색상으로 보고 상태를 직관적으로 파악  


**3.6.2.** 입력 구조  

**3.6.2.1.** 입력 방법  

　1: 소진량 입력창  
　　A. 입력창에 소진 수량(갯수) 입력  
　　B. 입력창 안에 '오늘 소진량을 입력 하세요' 플레이스홀더 표시  

　2: 제출 버튼  
　　A. 입력창 우측에 제출 버튼 위치  
　　B. 버튼 클릭 시 소진량 제출  


**3.6.3.** 경고 로직  

**3.6.3.1.** 경고 처리  

　1: 경고 조건  
　　A. 파트너는 가입 시 "하루 평균 소진량"을 입력  
　　B. 입력값이 평균 대비 1.5배 이상 차이 나면 경고 메세지 표시  

　2: 경고 발생 시 처리  
　　A. "제대로 입력했는지 확인" 경고 메세지 표시  
　　B. 맞으면 제출 버튼 클릭  
　　C. 실수 입력이면 수정 후 다시 제출  


**3.6.4.** 클릭 시 이동  

**3.6.4.1.** 이동 경로  

　1: 보드 클릭 시  
　　A. 사이드바의 일일 소진 보고 페이지로 이동  
　　B. 상세 보고 내역 및 캘린더 확인 가능
