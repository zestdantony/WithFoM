**3.8.** 리워드 사용 내역 보드  


**3.8.1.** 노출 조건  

**3.8.1.1.** 활성화 조건  

　1: 활성화 조건  
　　A. 리워드 기능을 On한 파트너에게만 노출  
　　B. "리워드 On/Off"는 서비스 타입(리워드/오더) 및 리워드 설정 > 참여 여부와 연동됨  
　　C. 리워드 Off 시 보드 미노출  


**3.8.2.** 카드 표시 구조  

**3.8.2.1.** 카드 정보  

　1: 노출 수량  
　　A. 최근 사용된 리워드 목록을 5개 카드로 노출  
　　B. 최신순으로 정렬  

　2: 카드 표시 정보  
　　A. 사용된 쿠폰/포인트/스탬프 이용 정보  
　　B. 사용자명 (작게 표시)  
　　C. 배경: 매장명 또는 로고 이미지  


**3.8.3.** 승인/거절 버튼  

**3.8.3.1.** 버튼 기능  

　1: 승인 버튼  
　　A. 60초 카운트다운 표시  
　　B. 60초 후 자동 승인 처리  

　2: 거절 버튼 목적  
　　A. 쿠폰/리워드가 잘못 사용되었을 때  
　　B. 매장 사정으로 리워드를 사용하지 못할 때  


**3.8.4.** 카드 클릭 시 이동  

**3.8.4.1.** 이동 경로  

　1: 연결 페이지  
　　A. 카드 클릭 시 사이드바의 내역 페이지로 이동  
　　B. 이동 후 더 디테일한 리워드 정보를 확인  


**3.8.5.** 결제 연동 시 표시  

**3.8.5.1.** 연동 정보  

　1: 위드폼 결제 기능 이용 파트너의 경우  
　　A. 결제 내역 보드에 결제 내역 카드가 뜨는 동시에  
　　B. 리워드 사용 내역 보드에도 "결제하면서 사용한 쿠폰"이 함께 표시됨  
　　C. 두 보드에서 연동된 정보 확인 가능
