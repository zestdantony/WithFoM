**1.3.1.** 메뉴 기본 정보 입력  


**1.3.1.1.** 메뉴 추가 버튼  

　1: 메뉴 추가 버튼 위치  
　　A. 메뉴 입력 영역 상단 우측  
　　B. 버튼 텍스트: "메뉴 추가" 또는 "+ 메뉴 추가"  
　　C. 클릭 시: 메뉴 입력 폼 표시  

　2: 메뉴 추가 조건  
　　A. 최소 1개 이상의 카테고리가 생성되어 있어야 함  
　　B. 카테고리가 없을 경우: "먼저 카테고리를 생성해주세요" 안내 표시  


**1.3.1.2.** 메뉴 기본 정보 입력 폼  

　1: 메뉴명 입력 (필수)  
　　A. 입력 필드: 메뉴명 (예: 아메리카노)  
　　B. 필수 입력: 메뉴명은 필수 항목  
　　C. 최대 길이: 50자  
　　D. 입력 예시: "아메리카노", "카페라떼", "초콜릿 케이크" 등  

　2: 메뉴 설명 입력 (선택)  
　　A. 입력 필드: 메뉴 설명  
　　B. 선택 입력: 메뉴 설명은 선택 사항  
　　C. 최대 길이: 200자  
　　D. 입력 예시: "진한 에스프레소에 물을 더한 커피", "부드러운 우유와 에스프레소의 조화" 등  

　3: 카테고리 선택 (필수)  
　　A. 드롭다운 선택: 생성된 카테고리 목록에서 선택  
　　B. 필수 선택: 카테고리는 필수 항목  
　　C. 기본값: 첫 번째 카테고리 또는 "카테고리 선택" 플레이스홀더  


**1.3.1.3.** 메뉴명 국문/영문 입력 (선택 사항)  

　1: 국문 메뉴명  
　　A. 기본 입력 필드: 국문 메뉴명 (예: 아메리카노)  
　　B. 필수 입력: 국문 메뉴명은 필수 항목  

　2: 영문 메뉴명 (선택)  
　　A. 추가 입력 필드: 영문 메뉴명 (예: Americano)  
　　B. 선택 입력: 영문 메뉴명은 선택 사항  
　　C. 입력 시: 소비자 앱에서 영문명도 함께 표시 가능  


**1.3.1.4.** 메뉴 입력 폼 레이아웃  

　1: 입력 폼 구성  
　　A. 상단: 메뉴명 입력 필드 (필수)  
　　B. 중간: 메뉴 설명 입력 필드 (선택)  
　　C. 하단: 카테고리 선택 드롭다운 (필수)  

　2: 입력 필드 스타일  
　　A. 필수 항목: 레이블에 "*" 표시 또는 "필수" 텍스트  
　　B. 선택 항목: 레이블에 "(선택)" 표시  
　　C. 플레이스홀더: 입력 예시 또는 안내 문구 표시  


**1.3.1.5.** 메뉴 입력 유효성 검사  

　1: 실시간 유효성 검사  
　　A. 메뉴명 입력 시: 최대 길이 초과 시 경고 표시  
　　B. 카테고리 선택 시: 미선택 시 경고 표시  
　　C. 저장 버튼 클릭 시: 필수 항목 미입력 시 경고 표시  

　2: 에러 메시지  
　　A. 메뉴명 미입력: "메뉴명을 입력해주세요"  
　　B. 카테고리 미선택: "카테고리를 선택해주세요"  
　　C. 메뉴명 길이 초과: "메뉴명은 50자 이하로 입력해주세요"  


**1.3.1.6.** 메뉴 입력 폼 저장  

　1: 저장 버튼  
　　A. 위치: 메뉴 입력 폼 하단  
　　B. 버튼 텍스트: "저장" 또는 "메뉴 추가"  
　　C. 클릭 시: 입력된 정보 검증 후 저장  

　2: 저장 후 처리  
　　A. 저장 성공: 메뉴 목록에 추가된 메뉴 표시  
　　B. 저장 실패: 에러 메시지 표시 및 재입력 유도  
　　C. 저장 후 입력 폼 초기화 또는 다음 메뉴 추가 모드로 전환
