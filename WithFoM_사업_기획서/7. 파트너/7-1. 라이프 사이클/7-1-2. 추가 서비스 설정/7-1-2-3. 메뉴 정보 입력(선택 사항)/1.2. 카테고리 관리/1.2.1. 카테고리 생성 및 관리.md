**1.2.1.** 카테고리 생성 및 관리  


**1.2.1.1.** 카테고리 생성  

　1: 카테고리 추가 버튼  
　　A. 위치: 카테고리 관리 영역 상단  
　　B. 버튼 텍스트: "카테고리 추가" 또는 "+ 카테고리 추가"  
　　C. 클릭 시: 카테고리 입력 모달 또는 인라인 입력 폼 표시  

　2: 카테고리명 입력  
　　A. 입력 필드: 카테고리명 (예: 음료, 디저트, 기타)  
　　B. 필수 입력: 카테고리명은 필수 항목  
　　C. 최대 길이: 20자  
　　D. 중복 체크: 동일한 카테고리명이 이미 존재하는지 확인  

　3: 카테고리 저장  
　　A. 저장 버튼 클릭 시 카테고리 생성 완료  
　　B. 생성된 카테고리는 카테고리 목록에 즉시 반영  
　　C. 기본 카테고리 예시: "음료", "디저트", "기타"  


**1.2.1.2.** 카테고리 목록 표시  

　1: 카테고리 리스트  
　　A. 생성된 카테고리 목록을 리스트 형태로 표시  
　　B. 각 카테고리별 메뉴 개수 표시 (예: "음료 (5개)")  
　　C. 카테고리 선택 시 해당 카테고리의 메뉴만 중앙 리스트에 표시  

　2: 카테고리 정렬  
　　A. 위/아래 이동 버튼으로 노출 순서 조정  
　　B. 드래그 앤 드롭으로 순서 변경 가능 (선택 사항)  
　　C. 정렬 순서는 소비자 앱 메뉴 노출 순서에 반영  


**1.2.1.3.** 카테고리 수정  

　1: 카테고리명 수정  
　　A. 카테고리 목록에서 수정 버튼 클릭  
　　B. 카테고리명 입력 필드 수정 모드로 전환  
　　C. 수정 완료 후 저장 버튼 클릭  

　2: 수정 시 주의 사항  
　　A. 카테고리명 변경 시 해당 카테고리의 모든 메뉴에 자동 반영  
　　B. 중복 체크: 다른 카테고리와 동일한 이름인지 확인  


**1.2.1.4.** 카테고리 삭제  

　1: 카테고리 삭제 버튼  
　　A. 카테고리 목록에서 삭제 버튼 클릭  
　　B. 삭제 확인 모달 표시  

　2: 삭제 확인 모달  
　　A. 경고 문구: "이 카테고리를 삭제하면 해당 카테고리의 메뉴는 '미분류'로 이동합니다. 정말 삭제하시겠습니까?"  
　　B. 취소 버튼: 삭제 취소  
　　C. 확인 버튼: 카테고리 삭제 실행  

　3: 삭제 후 처리  
　　A. 삭제된 카테고리의 메뉴는 "미분류" 카테고리로 자동 이동  
　　B. "미분류" 카테고리는 기본 제공되며 삭제 불가  


**1.2.1.5.** 카테고리 노출 여부 설정  

　1: 노출 ON/OFF 토글  
　　A. 각 카테고리별로 노출 여부 토글 버튼 제공  
　　B. OFF 상태: 소비자 앱에서 해당 카테고리 및 메뉴 미노출  
　　C. ON 상태: 소비자 앱에서 해당 카테고리 및 메뉴 노출  

　2: 노출 상태 표시  
　　A. ON 상태: 초록색 또는 활성화 아이콘  
　　B. OFF 상태: 회색 또는 비활성화 아이콘  
　　C. 상태 변경 시 즉시 반영  


**1.2.1.6.** 카테고리 선택 및 필터링  

　1: 카테고리 선택  
　　A. 카테고리 목록에서 카테고리 클릭  
　　B. 선택된 카테고리는 하이라이트 표시  
　　C. 선택된 카테고리의 메뉴만 중앙 메뉴 리스트에 표시  

　2: 전체 메뉴 보기  
　　A. "전체" 옵션 제공  
　　B. "전체" 선택 시 모든 카테고리의 메뉴 표시
