**1.1.2.** 메뉴 입력 개요  


**1.1.2.1.** 메뉴 입력 프로세스 개요  

　1: 전체 프로세스 흐름  
　　A. 1단계: 카테고리 생성 (예: 음료, 디저트, 기타)  
　　B. 2단계: 메뉴 추가 및 기본 정보 입력  
　　C. 3단계: 메뉴 가격 및 옵션 설정  
　　D. 4단계: 메뉴 이미지 업로드 (선택 사항)  
　　E. 5단계: 리워드 사용 가능 여부 설정 (선택 사항)  
　　F. 6단계: 저장 및 완료  

　2: 필수 입력 항목  
　　A. 메뉴 카테고리 (필수)  
　　B. 메뉴명 (필수)  
　　C. 가격 (필수, 500원 단위 권장)  

　3: 선택 입력 항목  
　　A. 메뉴 설명 (선택)  
　　B. 메뉴 이미지 (선택)  
　　C. 옵션 (사이즈, 온도, 당도 등, 선택)  


**1.1.2.2.** 메뉴 입력 항목 상세  

| 항목 | 설명 | 필수 여부 | 입력 형식 |  
|---|---|---|---|  
| 메뉴 카테고리 | 음료, 디저트, 기타 등 | 필수 | 드롭다운 선택 또는 직접 입력 |  
| 메뉴명 | 메뉴 이름 | 필수 | 텍스트 입력 (최대 50자) |  
| 가격 | 메뉴 가격 | 필수 | 숫자 입력 (500원 단위 권장) |  
| 메뉴 설명 | 메뉴 설명 | 선택 | 텍스트 입력 (최대 200자) |  
| 메뉴 이미지 | 메뉴 사진 | 선택 | 이미지 업로드 (1:1 정사각형, 최대 5MB) |  
| 옵션 | 사이즈, 온도, 당도 등 | 선택 | 옵션 그룹 선택 또는 직접 생성 |  


**1.1.2.3.** 카테고리 관리 개요  

　1: 카테고리 생성  
　　A. 카테고리명 입력 (예: 음료, 디저트, 기타)  
　　B. 카테고리 저장  
　　C. 카테고리별로 메뉴를 분류하여 관리  

　2: 카테고리 수정 및 삭제  
　　A. 카테고리명 수정 가능  
　　B. 카테고리 삭제 시 해당 카테고리의 메뉴는 "미분류"로 이동  
　　C. 카테고리 노출 여부 ON/OFF 설정 가능  

　3: 카테고리 정렬  
　　A. 위/아래 이동으로 노출 순서 조정 가능  


**1.1.2.4.** 옵션 그룹 관리 개요  

　1: 옵션 그룹 종류  
　　A. 사이즈: Small, Medium, Large 등  
　　B. 온도: Hot, Ice 등  
　　C. 당도: 당도 조절 (0~100%)  
　　D. 추가 옵션: 샷 추가, 시럽 추가 등  

　2: 옵션 그룹 생성  
　　A. 옵션 그룹명 입력 (예: 사이즈 선택)  
　　B. 옵션 항목 및 가격 설정  
　　C. 여러 메뉴에 일괄 적용 가능  

　3: 옵션 유형  
　　A. 단일형: 옵션을 1개만 선택할 수 있는 형태  
　　B. 복수형: 옵션을 1개 이상 선택할 수 있는 형태  
　　C. 수량형: 옵션을 1개 이상 선택하면서 개수를 선택하는 형태  


**1.1.2.5.** 메뉴 이미지 업로드 규칙  

　1: 이미지 비율  
　　A. 1:1 정사각형 비율 Crop Tool 강제 실행  
　　B. 가이드 박스 안에서 위치·확대/축소 조정 후 [자르기] 필수  

　2: 이미지 목적  
　　A. 소비자 앱 메뉴 리스트에서 모든 썸네일이 통일된 규격으로 노출  
　　B. 메뉴 시각적 식별 용이성 향상  

　3: 이미지 제한 사항  
　　A. 파일 크기: 최대 5MB  
　　B. 파일 형식: JPG, PNG, WebP  
　　C. 해상도: 최소 500x500px 권장  


**1.1.2.6.** 리워드 연동 설정  

　1: 리워드 사용 가능 여부  
　　A. 쿠폰/포인트/스탬프 사용 가능 여부 체크  
　　B. 리워드 제외 메뉴 설정 가능  

　2: 향후 확장 기능  
　　A. 특정 리워드 캠페인과 연동되는 메뉴 일괄 설정 여지  
　　B. 시간대별 판매 설정 (모닝, 티타임 등)
