**3.6.1.** 화면 초기 로딩  


**3.6.1.1.** API 호출  

　1: 엔드포인트: GET /api/partner/onboarding/menu-info  
　　A. 응답 데이터  
　　　　a. service_type: 서비스 타입 (reward | order | hybrid)  
　　　　b. existing_categories: 기존에 생성된 카테고리 정보 배열 (있을 경우)  
　　　　c. existing_menus: 기존에 입력된 메뉴 정보 배열 (있을 경우)  
　　　　d. existing_option_groups: 기존에 생성된 옵션 그룹 정보 배열 (있을 경우)  

　2: 로딩 상태 처리  
　　A. 메뉴 정보 로딩 중: 스켈레톤 UI 또는 로딩 스피너 표시  
　　B. 로딩 완료 후: 카테고리 및 메뉴 입력 UI 렌더링  


**3.6.1.2.** 기존 메뉴 정보 표시  

　1: 기존 카테고리가 있는 경우  
　　A. 카테고리 목록에 기존 카테고리 표시  
　　B. 카테고리별 메뉴 개수 표시  
　　C. 카테고리 수정/삭제 가능  

　2: 기존 메뉴가 있는 경우  
　　A. 메뉴 목록에 기존 메뉴 표시  
　　B. 메뉴 정보 수정/삭제 가능  
　　C. 메뉴 이미지, 가격, 옵션 등 모든 정보 표시  

　3: 기존 옵션 그룹이 있는 경우  
　　A. 옵션 그룹 목록에 기존 옵션 그룹 표시  
　　B. 옵션 그룹 수정/삭제 가능
