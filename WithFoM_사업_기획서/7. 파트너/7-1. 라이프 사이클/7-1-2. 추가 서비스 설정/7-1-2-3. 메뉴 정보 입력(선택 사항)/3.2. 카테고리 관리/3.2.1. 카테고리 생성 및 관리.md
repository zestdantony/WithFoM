**3.2.1.** 카테고리 생성 및 관리  


**3.2.1.1.** 카테고리 생성  

　1: 카테고리 추가 버튼  
　　A. 위치: 카테고리 관리 영역 상단  
　　B. 버튼 텍스트: "카테고리 추가" 또는 "+ 카테고리 추가"  
　　C. 클릭 시: 카테고리 입력 모달 또는 인라인 입력 폼 표시  

　2: 카테고리명 입력  
　　A. 입력 필드: 카테고리명 (예: 음료, 디저트, 기타)  
　　B. 필수 입력: 카테고리명은 필수 항목  
　　C. 최대 길이: 20자  
　　D. 중복 체크: 동일한 카테고리명이 이미 존재하는지 확인  

　3: 카테고리 저장  
　　A. 저장 버튼 클릭 시 카테고리 생성 완료  
　　B. 생성된 카테고리는 카테고리 목록에 즉시 반영  
　　C. 기본 카테고리 예시: "음료", "디저트", "기타"  


**3.2.1.2.** 카테고리 목록 표시  

　1: 카테고리 리스트  
　　A. 생성된 카테고리 목록을 리스트 형태로 표시  
　　B. 각 카테고리별 메뉴 개수 표시 (예: "음료 (5개)")  
　　C. 카테고리 선택 시 해당 카테고리의 메뉴만 중앙 리스트에 표시  

　2: 카테고리 정렬  
　　A. 위/아래 이동 버튼으로 노출 순서 조정  
　　B. 드래그 앤 드롭으로 순서 변경 가능  
　　C. 정렬 순서는 소비자 앱 메뉴 노출 순서에 반영  


**3.2.1.3.** 카테고리 수정  

　1: 카테고리명 수정  
　　A. 카테고리 목록에서 수정 버튼 클릭  
　　B. 카테고리명 입력 필드 수정 모드로 전환  
　　C. 수정 완료 후 저장 버튼 클릭  

　2: 수정 시 주의 사항  
　　A. 카테고리명 변경 시 해당 카테고리의 모든 메뉴에 자동 반영  
　　B. 중복 체크: 다른 카테고리와 동일한 이름인지 확인  


**3.2.1.4.** 카테고리 삭제  

　1: 카테고리 삭제 버튼  
　　A. 카테고리 목록에서 삭제 버튼 클릭  
　　B. 삭제 옵션 선택 모달 표시  

　2: 삭제 옵션 선택 모달  
　　A. 모달 제목: "카테고리 삭제"  
　　B. 안내 문구: "삭제 방법을 선택해주세요."  
　　C. 삭제 옵션  
　　　　• 옵션 1: "카테고리만 삭제 (메뉴는 미분류로 이동)"  
　　　　• 옵션 2: "카테고리와 모든 메뉴 함께 삭제"  
　　D. 취소 버튼: 삭제 취소  
　　E. 삭제 버튼: 선택한 옵션으로 삭제 실행  

　3: 카테고리만 삭제 (메뉴 유지)  
　　A. 카테고리만 삭제되고 메뉴는 "미분류" 카테고리로 자동 이동  
　　B. 미분류된 메뉴는 추후 다른 카테고리로 이동 가능  
　　C. 확인 문구: "카테고리 '[카테고리명]'이 삭제되었습니다. N개의 메뉴가 미분류로 이동되었습니다."  

　4: 카테고리 + 메뉴 전체 삭제  
　　A. 경고 문구: "⚠️ 이 카테고리와 포함된 N개의 메뉴가 모두 삭제됩니다. 삭제된 메뉴는 복구할 수 없습니다."  
　　B. 최종 확인: 삭제 버튼 클릭 시 재확인 모달 표시  
　　C. 재확인 문구: "정말 삭제하시겠습니까? 이 작업은 되돌릴 수 없습니다."  
　　D. 삭제 완료: 카테고리와 모든 메뉴 삭제 완료  
　　E. 메뉴 재등록: 삭제된 메뉴는 다시 직접 등록해야 함  

　5: 미분류 카테고리  
　　A. "미분류" 카테고리는 시스템 기본 제공  
　　B. "미분류" 카테고리는 삭제 불가  
　　C. 미분류된 메뉴를 다른 카테고리로 이동 가능  


**3.2.1.5.** 카테고리 노출 여부 설정  

　1: 노출 ON/OFF 토글  
　　A. 각 카테고리별로 노출 여부 토글 버튼 제공  
　　B. OFF 상태: 소비자 앱에서 해당 카테고리 및 메뉴 미노출  
　　C. ON 상태: 소비자 앱에서 해당 카테고리 및 메뉴 노출  

　2: 노출 상태 표시  
　　A. ON 상태: 초록색 또는 활성화 아이콘  
　　B. OFF 상태: 회색 또는 비활성화 아이콘  
　　C. 상태 변경 시 즉시 반영  


**3.2.1.6.** 카테고리 선택 및 필터링  

　1: 카테고리 선택  
　　A. 카테고리 목록에서 카테고리 클릭  
　　B. 선택된 카테고리는 하이라이트 표시  
　　C. 선택된 카테고리의 메뉴만 중앙 메뉴 리스트에 표시  

　2: 전체 메뉴 보기  
　　A. "전체" 옵션 제공  
　　B. "전체" 선택 시 모든 카테고리의 메뉴 표시  


**3.2.1.7.** 카테고리 내 메뉴 순서 조정  

　1: 드래그 앤 드롭 정렬  
　　A. 메뉴 목록에서 메뉴 항목을 드래그하여 순서 변경  
　　B. 드래그 핸들: 메뉴 항목 좌측에 드래그 아이콘 (≡) 표시  
　　C. 드롭 시 순서 즉시 반영  
　　D. 동일 카테고리 내에서만 순서 변경 가능  

　2: 위/아래 이동 버튼  
　　A. 메뉴 항목 우측에 위/아래 이동 버튼  
　　B. 클릭 시 한 칸씩 이동  
　　C. 맨 위/맨 아래 도달 시 버튼 비활성화  

　3: 순서 저장  
　　A. 자동 저장: 순서 변경 시 자동 저장  
　　B. 저장 완료 토스트: "메뉴 순서가 저장되었습니다"  
　　C. 저장된 순서는 소비자 앱 메뉴 노출에 반영  

　4: 순서 초기화  
　　A. "순서 초기화" 버튼 제공 (선택 사항)  
　　B. 초기화 시: 메뉴명 가나다순 또는 등록순으로 정렬  
　　C. 초기화 확인 모달: "메뉴 순서를 초기화하시겠습니까?"
