# 1: 메뉴 정보 입력

## 개요

오더형 또는 복합형 서비스를 선택한 파트너는 메뉴 정보를 입력하여 소비자가 앱을 통해 주문할 수 있도록 합니다.

## 메뉴 정보 입력 항목

| 항목 | 설명 | 필수 여부 |
|---|---|---|
| 메뉴 카테고리 | 음료, 디저트, 기타 등 | 필수 |
| 메뉴명 | 메뉴 이름 | 필수 |
| 가격 | 메뉴 가격 | 필수 |
| 메뉴 설명 | 메뉴 설명 (선택) | 선택 |
| 메뉴 이미지 | 메뉴 사진 (선택) | 선택 |
| 옵션 | 사이즈, 온도, 당도 등 (선택) | 선택 |

## 메뉴 입력 방법

### 1단계: 카테고리 생성

1. 메뉴 관리 > 카테고리 추가 메뉴 선택
2. 카테고리명 입력 (예: 음료, 디저트, 기타)
3. 카테고리 저장

### 2단계: 메뉴 추가

1. 해당 카테고리 선택
2. "메뉴 추가" 버튼 클릭
3. 메뉴 정보 입력
4. 메뉴 저장

### 3단계: 메뉴 정보 입력

| 입력 항목 | 내용 |
|---|---|
| 메뉴명 | 메뉴 이름 (예: 아메리카노) |
| 가격 | 메뉴 가격 (500원 단위) |
| 메뉴 설명 | 메뉴 설명 (선택) |
| 메뉴 이미지 | 메뉴 사진 업로드 (선택) |
| 옵션 | 사이즈, 온도, 당도 등 (선택) |

## 메뉴 옵션 설정

### 옵션 종류

| 옵션 종류 | 예시 |
|---|---|
| 사이즈 | Small, Medium, Large |
| 온도 | Hot, Ice |
| 당도 | 당도 조절 (0~100%) |
| 추가 옵션 | 샷 추가, 시럽 추가 등 |

### 옵션 설정 방법

1. 메뉴 추가/수정 시 "옵션 추가" 버튼 클릭
2. 옵션 종류 선택
3. 옵션 항목 및 가격 설정
4. 옵션 저장

## 메뉴 관리

### 메뉴 수정

| 관리 항목 | 내용 |
|---|---|
| 메뉴 정보 수정 | 메뉴명, 가격, 설명 등 수정 |
| 메뉴 이미지 변경 | 메뉴 사진 변경 |
| 옵션 수정 | 옵션 추가/삭제/수정 |
| 메뉴 삭제 | 메뉴 삭제 (주의 필요) |

### 메뉴 활성화/비활성화

| 상태 | 설명 |
|---|---|
| 활성화 | 소비자에게 메뉴 노출 및 주문 가능 |
| 비활성화 | 메뉴 노출 중단 (일시 품절 등) |

## 메뉴 가격 설정

| 가격 설정 | 내용 |
|---|---|
| 기본 가격 | 메뉴 기본 가격 |
| 옵션 가격 | 옵션별 추가 가격 |
| 할인 가격 | 할인 적용 시 가격 (선택) |

## 참고

- 메뉴 정보 입력은 선택 사항입니다
- 오더형 또는 복합형 서비스를 선택한 경우에만 입력 가능합니다
- 메뉴 정보는 나중에 언제든지 수정할 수 있습니다
- 메뉴 가격은 500원 단위로 입력하는 것을 권장합니다
