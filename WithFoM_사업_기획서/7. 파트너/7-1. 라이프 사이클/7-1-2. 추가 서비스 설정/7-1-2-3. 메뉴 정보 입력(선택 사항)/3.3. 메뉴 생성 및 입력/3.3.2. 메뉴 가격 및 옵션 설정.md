**3.3.2.** 메뉴 가격 및 옵션 설정  


**3.3.2.1.** 메뉴 가격 입력  

　1: 기본 가격 입력 (필수)  
　　A. 입력 필드: 메뉴 기본 가격  
　　B. 필수 입력: 가격은 필수 항목  
　　C. 입력 형식: 숫자만 입력 (원 단위)  
　　D. 입력 예시: 1,900원, 4,800원 등  

　2: 가격 입력 제한  
　　A. 최소 가격: 0원 이상  
　　B. 최대 가격: 제한 없음 (실제 운영에 맞게 설정)  
　　C. 소수점 입력 불가: 정수만 입력 가능  

　3: 가격 표시 형식  
　　A. 입력 시: 천 단위 구분 기호(,) 자동 표시  
　　B. 저장 시: 숫자로 저장, 표시 시 "원" 단위 추가  


**3.3.2.2.** 사이즈별 가격 설정 (선택 사항)  

　1: 사이즈 옵션 선택  
　　A. 사이즈 옵션 그룹 선택 또는 직접 생성  
　　B. 사이즈 예시: Small, Medium, Large  
　　C. 각 사이즈별 가격 설정 가능  

　2: 사이즈별 가격 입력  
　　A. Small: 기본 가격 또는 추가 금액 입력  
　　B. Medium: 기본 가격 또는 추가 금액 입력  
　　C. Large: 기본 가격 또는 추가 금액 입력  
　　D. 사이즈별 가격 차등 설정 가능  

　3: 사이즈 옵션 미사용 시  
　　A. 사이즈 옵션을 선택하지 않으면 기본 가격만 적용  
　　B. 소비자 앱에서 사이즈 선택 옵션 미노출  


**3.3.2.3.** 옵션 그룹 적용  

　1: 옵션 그룹 선택  
　　A. 드롭다운: 생성된 옵션 그룹 목록에서 선택  
　　B. 여러 옵션 그룹 선택 가능 (예: 사이즈 + 온도 + 당도)  
　　C. 옵션 그룹 미선택 시: 옵션 없음으로 설정  

　2: 옵션 그룹별 추가 금액  
　　A. 각 옵션 그룹의 옵션 항목별 추가 금액 표시  
　　B. 옵션 그룹 수정: 옵션 그룹 관리 화면에서 수정 가능  
　　C. 옵션 그룹 변경 시: 해당 메뉴의 옵션 설정 자동 반영  


**3.3.2.4.** 옵션 유형별 설정  

　1: 단일형 옵션  
　　A. 정의: 옵션을 1개만 선택할 수 있는 형태  
　　B. UI 형태: 라디오 버튼 또는 세그먼트 컨트롤  
　　C. 예시  
　　　　• 온도 선택: ○ Hot ● Ice (1개만 선택)  
　　　　• 사이즈 선택: ○ Small ● Medium ○ Large (1개만 선택)  
　　D. 옵션 항목별 추가 금액 설정 가능  
　　E. 기본 선택값 지정 가능  

　2: 복수형 옵션  
　　A. 정의: 옵션을 1개 이상 여러 개 선택할 수 있는 형태  
　　B. UI 형태: 체크박스  
　　C. 예시  
　　　　• 토핑 추가: ☑ 휘핑크림 (+500원) ☑ 초코칩 (+300원) ☐ 카라멜 소스  
　　　　• 시럽 추가: ☑ 바닐라 ☑ 헤이즐넛 ☐ 카라멜 (여러 개 선택 가능)  
　　D. 각 옵션 항목은 선택/미선택만 가능 (개수 지정 불가)  
　　E. 선택된 모든 옵션의 추가 금액 합산  

　3: 수량형 옵션  
　　A. 정의: 옵션을 선택하면서 개수(수량)까지 지정하는 형태  
　　B. UI 형태: 수량 조절 버튼 (-, 숫자, +)  
　　C. 예시  
　　　　• 샷 추가: 에스프레소 샷 [ - ] 2 [ + ] (+500원/개) → +1,000원  
　　　　• 시럽 추가: 바닐라 시럽 [ - ] 3 [ + ] (+200원/펌프) → +600원  
　　　　• 토핑 추가: 휘핑크림 [ - ] 2 [ + ] (+500원/개) → +1,000원  
　　D. 설정 항목  
　　　　• 옵션명: 에스프레소 샷  
　　　　• 단가: 500원/개  
　　　　• 최소 수량: 0 (기본값)  
　　　　• 최대 수량: 10 (설정 가능)  
　　E. 복수형과의 차이점  
　　　　• 복수형: 휘핑크림 ☑ (+500원) → 1개만 추가됨  
　　　　• 수량형: 휘핑크림 [2] (+500원/개) → 2개 추가, +1,000원  


**3.3.2.5.** 할인 가격 설정 (선택 사항)  

　1: 할인 가격 입력  
　　A. 할인 가격 입력 필드 (선택 사항)  
　　B. 할인 가격은 기본 가격보다 낮아야 함  
　　C. 할인 가격 설정 시: 소비자 앱에서 할인 가격 표시  

　2: 할인 가격 표시  
　　A. 기본 가격: 취소선 표시 또는 작은 글씨로 표시  
　　B. 할인 가격: 강조 표시 (큰 글씨, 색상 강조)  
　　C. 할인율 자동 계산 표시 (예: "20% 할인")  

　3: 할인 적용 메뉴의 리워드/스탬프 적용 안내  
　　A. 안내 문구: "할인 적용된 메뉴는 리워드/스탬프 적립에서 제외될 수 있습니다"  
　　B. 상세 설정: 리워드/스탬프 설정 화면에서 제외 여부 설정  


**3.3.2.6.** 가격 및 옵션 설정 저장  

　1: 저장 버튼  
　　A. 위치: 가격 및 옵션 설정 영역 하단  
　　B. 버튼 텍스트: "저장" 또는 "가격 설정 완료"  
　　C. 클릭 시: 입력된 가격 및 옵션 정보 검증 후 저장  

　2: 저장 후 처리  
　　A. 저장 성공: 메뉴 정보에 가격 및 옵션 정보 반영  
　　B. 저장 실패: 에러 메시지 표시 및 재입력 유도  
　　C. 가격 미입력 시: "가격을 입력해주세요" 경고 표시
