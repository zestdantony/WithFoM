**3.3.3.** 메뉴 이미지 업로드  


**3.3.3.1.** 이미지 업로드 버튼  

　1: 이미지 업로드 영역  
　　A. 위치: 메뉴 입력 폼 내 이미지 업로드 섹션  
　　B. 업로드 버튼: "이미지 업로드" 또는 "+ 이미지 추가"  
　　C. 드래그 앤 드롭 영역: 이미지 파일을 드래그하여 업로드 가능  

　2: 이미지 미리보기  
　　A. 업로드된 이미지 썸네일 표시  
　　B. 이미지 삭제 버튼: 업로드된 이미지 삭제 가능  
　　C. 이미지 교체 버튼: 다른 이미지로 교체 가능  


**3.3.3.2.** 이미지 파일 제한 사항  

　1: 파일 형식  
　　A. 지원 형식: JPG, JPEG, PNG, WebP  
　　B. 파일 크기: 최대 5MB  
　　C. 파일 크기 초과 시: "파일 크기는 5MB 이하여야 합니다" 경고 표시  

　2: 이미지 해상도  
　　A. 권장 해상도: 최소 500x500px  
　　B. 최적 해상도: 1000x1000px  
　　C. 권장 비율: 1:1 정사각형  
　　D. 해상도 부족 시: "이미지 품질이 낮을 수 있습니다" 경고 표시  


**3.3.3.3.** 이미지 자동 최적화  

　1: 자동 리사이징  
　　A. 업로드된 이미지 자동으로 적정 크기로 리사이징  
　　B. 최대 저장 크기: 1000x1000px  
　　C. 원본 비율 유지하며 리사이징  

　2: 자동 압축  
　　A. 이미지 품질 유지하며 파일 크기 압축  
　　B. 압축률: 품질 80~90% 유지  
　　C. 로딩 속도 개선  

　3: CDN 연동  
　　A. 업로드된 이미지 CDN에 자동 배포  
　　B. 빠른 로딩 및 캐싱 적용  
　　C. 전국 어디서나 빠른 이미지 로딩  

　4: 썸네일 자동 생성  
　　A. 목록용 썸네일: 200x200px  
　　B. 상세용 이미지: 500x500px  
　　C. 고화질 이미지: 1000x1000px  


**3.3.3.4.** 이미지 Crop Tool (강제 실행)  

　1: Crop Tool 실행  
　　A. 이미지 업로드 후 자동으로 Crop Tool 실행  
　　B. 1:1 정사각형 비율 Crop Tool 강제 실행  
　　C. 가이드 박스 안에서 위치·확대/축소 조정 가능  

　2: Crop Tool 조작  
　　A. 위치 조정: 이미지 드래그로 위치 이동  
　　B. 확대/축소: 핀치 제스처 또는 슬라이더로 크기 조정  
　　C. 회전: 이미지 회전 기능 (선택 사항)  

　3: Crop Tool 완료  
　　A. [자르기] 버튼 클릭 시 Crop 완료  
　　B. Crop 완료 후: 1:1 정사각형 비율로 이미지 저장  
　　C. 목적: 소비자 앱 메뉴 리스트에서 모든 썸네일이 통일된 규격으로 노출  


**3.3.3.5.** 이미지 업로드 프로세스  

　1: 업로드 단계  
　　A. 1단계: 이미지 파일 선택 또는 드래그 앤 드롭  
　　B. 2단계: 파일 형식 및 크기 검증  
　　C. 3단계: Crop Tool 실행 (1:1 비율 강제)  
　　D. 4단계: [자르기] 버튼 클릭으로 Crop 완료  
　　E. 5단계: 이미지 자동 최적화 (리사이징, 압축)  
　　F. 6단계: 이미지 업로드 및 CDN 배포  

　2: 업로드 진행 상태  
　　A. 업로드 중: 프로그레스 바 또는 로딩 스피너 표시  
　　B. 최적화 중: "이미지 최적화 중..." 메시지  
　　C. 업로드 완료: 이미지 썸네일 표시  
　　D. 업로드 실패: 에러 메시지 표시 및 재업로드 유도  


**3.3.3.6.** 이미지 삭제 및 교체  

　1: 이미지 삭제  
　　A. 이미지 썸네일 우측 상단 삭제 버튼 클릭  
　　B. 삭제 확인: "이미지를 삭제하시겠습니까?" 확인 모달  
　　C. 삭제 완료: 이미지 영역이 빈 상태로 복귀  

　2: 이미지 교체  
　　A. 기존 이미지 썸네일 클릭 또는 "이미지 변경" 버튼 클릭  
　　B. 새로운 이미지 파일 선택  
　　C. Crop Tool 다시 실행하여 새 이미지 Crop  
　　D. 교체 완료: 새 이미지로 업로드 및 저장  


**3.3.3.7.** 이미지 업로드 안내 문구  

　1: 업로드 안내  
　　A. 메인 안내: "메뉴 이미지를 업로드해주세요 (권장)"  
　　B. 보조 안내: "1:1 정사각형 비율로 자동 자르기가 적용됩니다"  
　　C. 권장 사항: "명확하고 선명한 이미지를 권장합니다"  
　　D. 추가 안내: "이미지 자동 최적화로 빠른 로딩이 지원됩니다"  

　2: 이미지 미업로드 시  
　　A. 안내 문구: "이미지가 없을 경우 기본 이미지가 표시됩니다"  
　　B. 기본 이미지: 카테고리별 기본 플레이스홀더 이미지 표시  
　　C. 소비자 앱: 기본 이미지 표시 (향후 카테고리별 기본 이미지 제공 예정)  
　　D. 나중에 추가 가능: 메뉴 수정 화면에서 이미지 추가 가능  


**3.3.3.8.** 이미지 권장 가이드  

　1: 좋은 이미지 예시  
　　A. 메뉴가 중앙에 위치한 이미지  
　　B. 밝고 선명한 조명 아래 촬영된 이미지  
　　C. 배경이 깔끔한 이미지  
　　D. 실제 제공되는 메뉴와 동일한 이미지  

　2: 피해야 할 이미지 예시  
　　A. 흐릿하거나 저해상도 이미지  
　　B. 메뉴가 잘려 보이는 이미지  
　　C. 과도한 필터나 보정이 적용된 이미지  
　　D. 저작권 문제가 있는 이미지  

　3: 이미지 촬영 팁  
　　A. 자연광 또는 밝은 조명 활용  
　　B. 스마트폰 세로 모드 활용 (배경 흐림 효과)  
　　C. 깔끔한 배경지 사용  
　　D. 메뉴를 45도 각도에서 촬영
