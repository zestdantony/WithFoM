**1.3.2.** 포인트 생성 및 입력 폼  


**1.3.2.1.** 포인트 생성 인터페이스  

　1: 기본 입력박스  
　　A. 기본적으로 1개 포인트 입력박스 노출  
　　B. 총 6개까지 생성 가능 (쿠폰과 합산)  

　2: 포인트 추가 버튼  
　　A. 위치: 입력박스 영역 우측 상단  
　　B. 버튼 텍스트: "포인트 추가"  
　　C. 클릭 시: 새로운 포인트 입력박스 1개 추가  
　　D. 최대 개수 도달 시: 버튼 비활성화 또는 "최대 6개까지 생성 가능합니다" 메시지 표시  


**1.3.2.2.** 포인트 입력박스 공통 구성  

　1: 박스 헤더  
　　A. 좌측 상단: N번째 포인트 숫자 표시 (예: "포인트 1", "포인트 2")  
　　B. 우측 상단: 삭제 버튼 (X 아이콘)  

　2: 입력 항목  
　　A. 포인트명 (필수)  
　　　　a. 입력 형식: 텍스트 입력창  
　　　　b. 예시: "포인트 500점", "포인트 3000점", "적립 포인트"  
　　　　c. 최대 글자 수: 50자  
　　B. 적립 조건 (필수)  
　　　　a. 입력 형식: 결제 금액 대비 비율 또는 고정 금액  
　　　　b. 비율 적립: 숫자 입력 + "%" (예: 5%)  
　　　　c. 고정 적립: 숫자 입력 + "원당 X점" (예: 100원당 1점)  
　　　　d. 예시: "결제 금액의 5% 적립", "100원당 1점 적립"  
　　C. 적립 한도 (선택)  
　　　　a. 1일 적립 한도: 숫자 입력창, 단위 "점"  
　　　　b. 설명: "고객 1인당 1일 최대 적립 가능한 포인트"  
　　D. 사용 조건 (선택)  
　　　　a. 최소 사용 포인트: 숫자 입력창, 단위 "점"  
　　　　b. 설명: "이 포인트 이상 보유 시 사용 가능"  
　　E. 유효 기간 (필수)  
　　　　a. 입력 형식: 날짜 선택기 (시작일, 종료일)  
　　　　b. 또는 기간 선택: 30일, 60일, 90일, 사용자 지정  
　　　　c. 유효 기간 미설정 체크박스: 체크 시 무기한 유효  


**1.3.2.3.** 포인트 사용 조건 설정  

　1: 최소/최대 결제금액 조건  
　　A. 최소 결제 금액 (선택)  
　　　　a. 입력 형식: 숫자 입력창, 단위 "원"  
　　　　b. 설명: "이 금액 이상 결제 시 포인트 사용 가능"  
　　B. 최대 결제 금액 (선택)  
　　　　a. 입력 형식: 숫자 입력창, 단위 "원"  
　　　　b. 설명: "이 금액 이하 결제 시 포인트 사용 가능"  

　2: 메뉴 조건 (선택)  
　　A. 카테고리 조건  
　　　　a. 형식: 체크박스로 카테고리 선택  
　　　　b. 설명: "선택한 카테고리 메뉴에만 포인트 사용 가능"  
　　B. 단일 메뉴 조건  
　　　　a. 형식: 메뉴 선택 드롭다운  
　　　　b. 설명: "특정 메뉴에만 포인트 사용 가능"  

　3: 채널 조건 (선택)  
　　A. 옵션: 매장 / 테이크아웃 / 오더  
　　B. 형식: 체크박스로 복수 선택 가능  
　　C. 설명: "선택한 채널에서만 포인트 사용 가능"  

　4: 시간/요일 조건 (선택)  
　　A. 시간대 조건  
　　　　a. 형식: 시간 선택 (시작 시간, 종료 시간)  
　　　　b. 예시: "09:00 ~ 18:00"  
　　B. 요일 조건  
　　　　a. 형식: 체크박스로 요일 선택  
　　　　b. 예시: "월, 화, 수, 목, 금"  


**1.3.2.4.** 포인트 사용 제한 설정  

　1: 1일 사용 제한  
　　A. 1일 포인트 사용 가능 총액 (선택)  
　　　　a. 입력 형식: 숫자 입력창, 단위 "원"  
　　　　b. 설명: "고객 1인당 1일 사용 가능한 총 포인트 금액"  

　2: 1회 결제당 사용 제한  
　　A. 1회 결제 기준 최대 사용 비율 (선택)  
　　　　a. 입력 형식: 숫자 입력창, 단위 "%"  
　　　　b. 설명: "1회 결제 금액 대비 사용 가능한 포인트 비율"  
　　　　c. 예시: "결제 금액의 50%까지 포인트 사용 가능"  
　　B. 1회 결제당 최대 사용 금액 (선택)  
　　　　a. 입력 형식: 숫자 입력창, 단위 "원"  
　　　　b. 설명: "1회 결제 시 사용 가능한 최대 포인트 금액"
