**1.6.1.1.** API 호출  

　1: 엔드포인트: GET /api/partner/onboarding/reward-info  
　　A. 응답 데이터  
　　　　a. service_type: 서비스 타입 (reward | order | hybrid)  
　　　　b. reward_enabled: 현재 리워드 기능 활성화 여부 (boolean)  
　　　　c. existing_rewards: 기존에 입력된 리워드 정보 배열 (있을 경우)  

　2: 로딩 상태 처리  
　　A. 리워드 정보 로딩 중: 스켈레톤 UI 또는 로딩 스피너 표시  
　　B. 로딩 완료 후: 리워드 기능 선택 UI 렌더링  


**1.6.1.2.** 기존 리워드 정보 표시  

　1: 기존 리워드가 있는 경우  
　　A. 리워드 기능 On 상태로 자동 설정  
　　B. 기존 리워드 목록 표시
