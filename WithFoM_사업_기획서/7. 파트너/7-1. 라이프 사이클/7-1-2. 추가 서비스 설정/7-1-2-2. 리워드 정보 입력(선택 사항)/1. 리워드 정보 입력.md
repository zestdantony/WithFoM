# 1: 리워드 정보 입력

## 개요

리워드형 또는 복합형 서비스를 선택한 파트너는 리워드 정보를 입력하여 소비자에게 제공할 수 있습니다.

## 리워드 정보 입력 항목

| 항목 | 설명 | 필수 여부 |
|---|---|---|
| 리워드 종류 | 쿠폰, 포인트, 스탬프 등 | 필수 |
| 리워드 조건 | 사용 조건 및 제한 사항 | 필수 |
| 리워드 수량 | 발행 수량 및 사용 제한 | 선택 |
| 유효 기간 | 리워드 사용 가능 기간 | 필수 |

## 리워드 종류별 입력 방법

### 쿠폰 리워드

| 입력 항목 | 내용 |
|---|---|
| 쿠폰명 | 쿠폰 이름 (예: 아메리카노 500원 할인) |
| 할인 금액/율 | 할인 금액 또는 할인율 |
| 사용 조건 | 최소 주문 금액 등 |
| 유효 기간 | 쿠폰 사용 가능 기간 |
| 발행 수량 | 발행할 쿠폰 수량 (선택) |

### 포인트 리워드

| 입력 항목 | 내용 |
|---|---|
| 적립 조건 | 포인트 적립 조건 (예: 결제 금액의 5%) |
| 적립 한도 | 일일 적립 한도 (선택) |
| 사용 조건 | 포인트 사용 조건 |
| 유효 기간 | 포인트 사용 가능 기간 |

### 스탬프 리워드

| 입력 항목 | 내용 |
|---|---|
| 스탬프 기준 | 스탬프 적립 기준 (예: 1회 방문 시 1개) |
| 완성 개수 | 완성 시 필요한 스탬프 개수 |
| 완성 보상 | 완성 시 제공할 보상 |
| 유효 기간 | 스탬프 사용 가능 기간 |

## 리워드 설정 가이드

### 리워드 설정 위치

1. 설정 > 리워드 > 리워드 설정 메뉴 선택
2. 리워드 종류 선택
3. 리워드 정보 입력
4. 저장

### 리워드 활성화

| 항목 | 내용 |
|---|---|
| 활성화 | 리워드를 활성화하면 소비자에게 제공 |
| 비활성화 | 리워드를 비활성화하면 제공 중단 |
| 게임 연동 | 리워드 활성화 시 게임도 자동 활성화 |

## 리워드 중복 사용 규칙

| 규칙 | 설명 |
|---|---|
| 동일 리워드 중복 | 동일 리워드 내 중복 사용 허용 여부 |
| 서로 다른 리워드 | 서로 다른 리워드 간 동시 사용 허용 여부 |
| 단독 사용 | 단독 사용만 가능한 리워드 지정 |

## 참고

- 리워드 정보 입력은 선택 사항입니다
- 리워드형 또는 복합형 서비스를 선택한 경우에만 입력 가능합니다
- 리워드 설정은 나중에 언제든지 수정할 수 있습니다
