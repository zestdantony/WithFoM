**2.1.2.** 리워드 기능 활성화 선택  


**2.1.2.1.** 영역 목적  

　1: 파트너에게 리워드 기능을 소개하고 활성화 여부를 선택하게 함  

　2: 리워드 기능 활성화 시 게임도 자동 활성화됨을 안내  

　3: 리워드형 또는 복합형 서비스를 선택한 파트너만 이 화면에 진입  


**2.1.2.2.** UI 구성 요소  

　1: 리워드 기능 안내 문구  
　　A. 제목: "리워드 기능으로 고객 재방문을 유도하세요"  
　　B. 설명: "쿠폰, 포인트, 스탬프를 제공하여 고객에게 혜택을 주고 재방문을 유도할 수 있습니다"  
　　C. 스타일: 안내 텍스트, 중앙 정렬  

　2: 리워드 기능 On/Off 토글 스위치  
　　A. 위치: 안내 문구 아래, 중앙 배치  
　　B. 기본 상태: Off (비활성화)  
　　C. On 선택 시: 리워드 입력 영역 표시  
　　D. Off 선택 시: 리워드 입력 영역 숨김  

　3: 리워드 기능 주요 혜택 요약  
　　A. 쿠폰: 할인 쿠폰 제공으로 재방문 유도  
　　B. 포인트: 포인트 적립/사용으로 고객 충성도 향상  
　　C. 스탬프: 스탬프 적립 시 보상으로 방문 빈도 증가  
　　D. 게임 연동: 리워드 활성화 시 게임도 자동 활성화  

　4: 건너뛰기 안내 문구  
　　A. 텍스트: "지금 설정하지 않아도 언제든 설정에서 변경할 수 있습니다"  
　　B. 스타일: 안내 텍스트, 회색 컬러  
　　C. 위치: 리워드 기능 선택 영역 하단  


**2.1.2.3.** 리워드 기능 활성화 시 동작  

　1: 리워드 입력 영역 표시  
　　A. 쿠폰/포인트/스탬프 선택 카드 영역 표시  
　　B. 선택한 리워드 종류별 입력 폼 영역 표시  

　2: 게임 자동 활성화  
　　A. 리워드 기능 활성화 시 게임 기능도 자동으로 활성화됨  
　　B. 게임 유형 선택 화면은 이후 별도 단계에서 진행
