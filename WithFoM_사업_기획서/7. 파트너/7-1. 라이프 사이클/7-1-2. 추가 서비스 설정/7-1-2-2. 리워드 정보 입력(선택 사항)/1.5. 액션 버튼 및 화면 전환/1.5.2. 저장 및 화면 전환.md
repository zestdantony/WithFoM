**1.5.2.** 저장 및 화면 전환  


**1.5.2.1.** 저장 성공 시 화면 전환  

　1: 저장 완료 API 응답 수신  
　　A. 리워드 정보 저장 완료 확인  
　　B. 리워드 기능 활성화 상태 업데이트  
　　C. 게임 기능 자동 활성화 (리워드 활성화 시)  

　2: 성공 화면 표시 (선택 사항)  
　　A. 성공 아이콘 및 메시지  
　　B. "리워드 정보가 저장되었습니다"  

　3: 다음 가입 단계로 자동 이동  
　　A. 이동 경로: /partner/onboarding/next-step (메뉴 정보 입력 등)  
　　B. 이동 시간: 2~3초 후 자동 이동 또는 "다음 단계로" 버튼 클릭  


**1.5.2.2.** 저장 실패 시 처리  

　1: 에러 메시지 표시  
　　A. 위치: 저장 버튼 상단 또는 토스트 메시지  
　　B. 메시지: "리워드 정보 저장에 실패했습니다. 다시 시도해주세요."  

　2: 실패 원인 안내 (상세 에러인 경우)  
　　A. 필수 정보 미입력: "필수 항목을 입력해주세요"  
　　B. 입력값 오류: "입력한 값을 확인해주세요"  
　　C. 네트워크 오류: "네트워크 연결을 확인해주세요"  

　3: 재시도 가능  
　　A. 입력 정보 수정 후 재시도  


**1.5.2.3.** 건너뛰기 시 화면 전환  

　1: "건너뛰기" 버튼 클릭  
　　A. 리워드 기능 Off 상태 유지  
　　B. 다음 가입 단계로 이동  
　　C. 이동 경로: /partner/onboarding/next-step  

　2: 건너뛴 경우 안내  
　　A. 이후 설정 메뉴에서 언제든 리워드 설정 가능  
　　B. 사이드바 > 리워드·스탬프 > 설정 메뉴에서 접근  


**1.5.2.4.** 뒤로가기 시 동작  

　1: 헤더 뒤로가기 버튼 클릭  
　　A. 이전 가입 단계 (멤버십 구독 또는 가입 완료 화면)로 이동  
　　B. 확인 다이얼로그: 입력 중인 정보가 있을 경우 확인 후 이동  

　2: 브라우저/기기 뒤로가기  
　　A. 동일하게 이전 가입 단계로 이동  
　　B. 작성 중인 데이터 손실 경고 (입력 중인 정보가 있을 경우)
