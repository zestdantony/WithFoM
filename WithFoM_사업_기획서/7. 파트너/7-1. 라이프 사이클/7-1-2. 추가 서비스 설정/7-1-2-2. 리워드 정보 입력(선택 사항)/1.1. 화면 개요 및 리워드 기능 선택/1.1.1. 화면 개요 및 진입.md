**1.1.1.** 화면 개요 및 진입  


**1.1.1.1.** 화면 목적  

　1: 가입 완료 직후 파트너에게 리워드 기능을 소개하고 리워드 정보 입력을 독려하는 화면  

　2: 파트너는 리워드형 또는 복합형 서비스를 선택한 경우에만 리워드 정보 입력 가능  

　3: 리워드 정보 입력은 선택 사항이므로 "건너뛰기" 버튼을 통해 다음 단계로 이동 가능  


**1.1.1.2.** 화면 진입 경로  

　1: 가입 플로우 내 진입  
　　A. 가입 완료 직후 또는 멤버십 구독 건너뛴 직후 → 리워드 정보 입력 화면 자동 진입  
　　B. URL 경로: /partner/onboarding/reward-setup  

　2: 화면 노출 조건  
　　A. 파트너 가입 완료 직후  
　　B. 서비스 타입이 리워드형 또는 복합형인 경우  


**1.1.1.3.** 전체 화면 레이아웃  

　1: 화면 구성 영역  

| 영역 | 위치 | 설명 |  
|---|---|---|  
| 헤더 영역 | 상단 | 화면 제목, 뒤로가기 버튼 |  
| 리워드 기능 안내 영역 | 헤더 아래 | 리워드 기능 소개 및 활성화 선택 |  
| 리워드 종류 선택 영역 | 중앙 본문 | 쿠폰/포인트/스탬프 선택 카드 |  
| 리워드 입력 영역 | 선택 영역 아래 | 선택한 리워드 종류별 입력 폼 |  
| 하단 액션 영역 | 하단 고정 | 저장 및 건너뛰기 버튼 |  

　2: 반응형 레이아웃  
　　A. 모바일: 세로 스크롤, 하단 버튼 고정  
　　B. 태블릿/데스크톱: 중앙 정렬, 최대 너비 제한  


**1.1.1.4.** 헤더 영역 구성  

　1: 뒤로가기 버튼  
　　A. 위치: 좌측 상단  
　　B. 클릭 시: 이전 가입 단계로 이동  

　2: 화면 제목  
　　A. 텍스트: "리워드 정보 입력" 또는 "리워드 설정"  
　　B. 스타일: 중앙 정렬, 볼드  

　3: 단계 인디케이터 (선택 사항)  
　　A. 현재 가입 진행 단계 표시  
　　B. 예시: "4/5 단계" 또는 프로그레스 바
