**2.5.1.** 하단 액션 영역 구성  


**1.5.1.1.** 저장 버튼  

　1: 버튼 스타일  
　　A. Primary 버튼, 전체 너비 또는 우측 배치  
　　B. 텍스트: "저장하고 다음 단계로" 또는 "저장"  
　　C. 위치: 화면 하단 고정 (Sticky Footer)  

　2: 버튼 상태  
　　A. 활성화: 리워드 기능 Off 선택 시 또는 리워드 정보 입력 완료 시  
　　B. 비활성화: 리워드 기능 On 선택 후 필수 정보 미입력 시  
　　C. 로딩: 저장 처리 중 스피너 표시 및 버튼 비활성화  

　3: 저장 조건  
　　A. 리워드 기능 Off 선택 시: 즉시 저장 가능  
　　B. 리워드 기능 On 선택 시  
　　　　a. 최소 1개 이상의 리워드 정보 입력 필요 (쿠폰, 포인트, 스탬프 중 1개)  
　　　　b. 각 리워드의 필수 항목 입력 완료 필요  


**1.5.1.2.** 건너뛰기 버튼  

　1: 버튼 스타일  
　　A. Secondary 버튼 또는 텍스트 링크  
　　B. 텍스트: "나중에 설정하기" 또는 "건너뛰기"  
　　C. 위치: 저장 버튼 왼쪽 또는 하단 중앙  

　2: 버튼 동작  
　　A. 클릭 시: 다음 가입 단계로 이동  
　　B. 입력 중인 정보는 저장하지 않음  
　　C. 이후 설정 메뉴에서 언제든 리워드 설정 가능
