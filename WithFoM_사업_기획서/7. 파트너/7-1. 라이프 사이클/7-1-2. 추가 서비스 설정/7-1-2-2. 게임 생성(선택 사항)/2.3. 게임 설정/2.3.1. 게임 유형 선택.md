**2.3.1.** 게임 유형 선택  


**2.3.1.1.** 제공 게임 유형  

　1: 룰렛  
　　A. 회전하는 룰렛을 돌려 당첨 칸에 멈추면 상품 획득  
　　B. 칸 개수: 6칸 / 8칸 / 12칸 중 선택  
　　C. 각 칸에 상품 또는 꽝 설정  

　2: 뽑기 (가챠)  
　　A. 여러 개의 캡슐/카드 중 하나를 선택하여 상품 획득  
　　B. 캡슐 개수: 9개 / 16개 / 25개 중 선택  
　　C. 각 캡슐에 상품 또는 꽝 설정  

　3: 스크래치  
　　A. 복권처럼 긁어서 상품 확인  
　　B. 3×3 그리드에서 같은 그림 3개가 나오면 당첨  
　　C. 당첨 확률 및 상품 설정  


**2.3.1.2.** 게임 선택 UI  

　1: 카드형 선택  
　　A. 각 게임 유형을 카드로 표시  
　　B. 미리보기 이미지/애니메이션 포함  
　　C. 클릭 시 해당 게임 설정 화면으로 이동  

　2: 복수 게임 생성  
　　A. 최대 3개까지 게임 생성 가능  
　　B. 게임별로 다른 상품/확률 설정 가능  
　　C. 활성화/비활성화 토글로 운영 관리
