**2.3.1.** 게임 유형 선택  


**2.3.1.1.** 제공 게임 유형  

　1: 룰렛  
　　A. 원형 룰렛 UI, 6개 구간으로 분할  
　　B. 소비자가 룰렛 회전 버튼 클릭 → 랜덤 멈춤 → 결과 표시  
　　C. 칸 수: 6칸 (고정)  

　2: 뽑기 (가챠)  
　　A. 회전 원통 내부 12개 캡슐 배치  
　　B. 소비자가 스타트 버튼 클릭 → 캡슐 1개 사출 → 번호 매칭 리워드 당첨  
　　C. 캡슐 수: 12캡슐 (고정)  

　3: 스크래치  
　　A. 4×4 격자(16칸) 레이아웃  
　　B. 소비자가 터치/드래그로 코팅 긁기 → 16칸 중 같은 번호 3개 일치 시 당첨  
　　C. 칸 수: 16칸 (고정)  

　4: 사다리  
　　A. 6개 시작점-도착점 연결 사다리 레이아웃  
　　B. 소비자가 시작점 하나 선택 → 자동 이동 → 도착점 리워드 당첨  
　　C. 줄 수: 6줄 (고정)  


**2.3.1.2.** 게임 선택 UI  

　1: 카드형 선택  
　　A. 각 게임 유형을 카드로 표시  
　　B. 미리보기 이미지/애니메이션 포함  
　　C. 클릭 시 해당 게임 선택 강조(라디오 방식, 하나만 선택 가능)  

　2: 매장당 1개 게임  
　　A. 매장당 1개 게임만 생성 가능  
　　B. 유형 변경은 플랜별 월 횟수 제한(베이직 월 1회, 플러스 월 2회, 프리미엄 무제한)  
　　C. 활성화/비활성화 토글로 운영 관리
