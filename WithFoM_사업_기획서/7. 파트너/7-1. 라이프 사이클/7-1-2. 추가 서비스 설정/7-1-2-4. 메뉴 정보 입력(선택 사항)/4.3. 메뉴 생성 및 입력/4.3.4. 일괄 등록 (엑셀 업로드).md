**4.3.4.** 일괄 등록 (엑셀 업로드)  


**4.3.4.1.** 일괄 등록 개요  

　1: 기능 소개  
　　A. 엑셀 파일을 통해 여러 메뉴를 한 번에 등록  
　　B. 대량 메뉴 등록 시 시간 절약  
　　C. 기존 메뉴 정보 일괄 수정에도 활용 가능  

　2: 지원 파일 형식  
　　A. .xlsx (Excel 2007 이상)  
　　B. .xls (Excel 97-2003)  
　　C. .csv (쉼표로 구분된 값)  

　3: 파일 크기 제한  
　　A. 최대 파일 크기: 10MB  
　　B. 최대 행 수: 500개 메뉴  


**4.3.4.2.** 엑셀 템플릿 다운로드  

　1: 템플릿 다운로드 버튼  
　　A. 위치: 일괄 등록 영역 상단  
　　B. 버튼 텍스트: "엑셀 템플릿 다운로드"  
　　C. 클릭 시: 표준 템플릿 파일 다운로드  

　2: 템플릿 구조  

| 열 | 필드명 | 필수 | 설명 | 예시 |  
|---|---|---|---|---|  
| A | 카테고리 | 필수 | 메뉴 카테고리명 | 음료 |  
| B | 메뉴명 | 필수 | 메뉴 이름 (국문, 최대 20자) | 아메리카노 |  
| C | 영문명 | 선택 | 메뉴 이름 (영문, 최대 20자) | Americano |  
| D | 설명 | 선택 | 메뉴 설명 (최대 100자) | 진한 에스프레소에 물을 더한 커피 |  
| E | 기본가격 | 필수 | 메뉴 가격 (원) | 4500 |  
| F | 할인가격 | 선택 | 할인된 가격 (원) | 4000 |  
| G | 옵션그룹 | 선택 | 적용할 옵션 그룹명 | 사이즈선택, 온도선택 |  
| H | 판매상태 | 선택 | 판매중/일시중지/솔드아웃 | 판매중 |  
| I | 노출시간 | 선택 | 노출 시간대 (HH:mm-HH:mm) | 11:00-14:00 |  
| J | 노출요일 | 선택 | 노출 요일 (월,화,수 등) | 월,화,수,목,금 |  
| K | 인기메뉴 | 선택 | 인기 표시 여부 | Y 또는 N |  
| L | 알레르기 | 선택 | 알레르기 정보 | 우유, 대두 |  
| M | 노출여부 | 선택 | 노출 여부 | Y 또는 N |  

　3: 템플릿 예시 데이터  
　　A. 첫 번째 행: 필드명 헤더  
　　B. 두 번째 행부터: 예시 데이터 포함  
　　C. 예시 데이터 삭제 후 실제 데이터 입력 안내  


**4.3.4.3.** 엑셀 파일 업로드  

　1: 업로드 버튼  
　　A. 위치: 일괄 등록 영역  
　　B. 버튼 텍스트: "엑셀 파일 업로드" 또는 "파일 선택"  
　　C. 드래그 앤 드롭 영역 제공  

　2: 업로드 프로세스  
　　A. 1단계: 파일 선택 또는 드래그 앤 드롭  
　　B. 2단계: 파일 형식 검증  
　　C. 3단계: 데이터 파싱 및 검증  
　　D. 4단계: 미리보기 표시  

　3: 파일 검증  
　　A. 형식 검증: 지원 파일 형식인지 확인  
　　B. 크기 검증: 파일 크기 제한 확인  
　　C. 구조 검증: 필수 열 존재 여부 확인  


**4.3.4.4.** 데이터 검증 및 미리보기  

　1: 데이터 검증  
　　A. 필수 항목 검증: 카테고리, 메뉴명, 가격 입력 여부  
　　B. 형식 검증: 가격 숫자 여부, 글자 수 제한 등  
　　C. 중복 검증: 동일 메뉴명 존재 여부  
　　D. 참조 검증: 카테고리/옵션그룹 존재 여부  

　2: 검증 결과 표시  
　　A. 전체 행 수: 업로드된 총 메뉴 개수  
　　B. 성공 행 수: 검증 통과한 메뉴 개수  
　　C. 오류 행 수: 검증 실패한 메뉴 개수  
　　D. 경고 행 수: 주의가 필요한 메뉴 개수  

　3: 미리보기 테이블  
　　A. 업로드된 메뉴 목록을 테이블로 표시  
　　B. 정상 행: 기본 스타일  
　　C. 오류 행: 빨간색 하이라이트 + 오류 메시지  
　　D. 경고 행: 노란색 하이라이트 + 경고 메시지  

　4: 오류 상세 정보  
　　A. 오류 행 클릭 시 상세 오류 메시지 표시  
　　B. 오류 예시:  
　　　　a. "메뉴명이 입력되지 않았습니다" (2행)  
　　　　b. "메뉴명은 20자 이내로 입력해주세요" (3행)  
　　　　c. "가격은 숫자만 입력 가능합니다" (5행)  
　　　　d. "카테고리 '간식'이 존재하지 않습니다" (8행)  


**4.3.4.5.** 카테고리 자동 생성 옵션  

　1: 새 카테고리 발견 시  
　　A. 엑셀에 존재하지만 시스템에 없는 카테고리 발견  
　　B. 안내 메시지: "3개의 새 카테고리가 발견되었습니다: 간식, 세트, 시즌"  

　2: 자동 생성 옵션  
　　A. "새 카테고리 자동 생성" 체크박스 제공  
　　B. 체크 시: 새 카테고리 자동 생성 후 메뉴 등록  
　　C. 체크 해제 시: 해당 행 오류 처리  


**4.3.4.6.** 일괄 저장  

　1: 저장 버튼  
　　A. 위치: 미리보기 영역 하단  
　　B. 버튼 텍스트: "일괄 등록" 또는 "메뉴 저장"  
　　C. 오류가 있는 경우: "오류 항목 제외하고 저장" 옵션 제공  

　2: 저장 처리  
　　A. 검증 통과한 메뉴만 저장  
　　B. 진행 상태 프로그레스 바 표시  
　　C. 저장 중 취소 기능 제공  

　3: 저장 결과  
　　A. 성공 메시지: "N개의 메뉴가 등록되었습니다"  
　　B. 부분 성공: "N개 중 M개 메뉴가 등록되었습니다. K개 실패"  
　　C. 실패 목록 다운로드 옵션: 실패한 행만 별도 엑셀로 다운로드  


**4.3.4.7.** 기존 메뉴 일괄 수정  

　1: 수정 모드 선택  
　　A. "새 메뉴 등록" / "기존 메뉴 수정" 모드 선택  
　　B. 수정 모드 선택 시 추가 안내 표시  

　2: 기존 메뉴 내보내기  
　　A. "현재 메뉴 내보내기" 버튼  
　　B. 클릭 시: 현재 등록된 모든 메뉴를 엑셀로 다운로드  
　　C. 다운로드된 파일 수정 후 재업로드  

　3: 수정 처리  
　　A. 메뉴명 기준으로 기존 메뉴 매칭  
　　B. 일치하는 메뉴: 정보 업데이트  
　　C. 일치하지 않는 메뉴: 새 메뉴로 추가 또는 무시 옵션  


**4.3.4.8.** 이미지 일괄 등록  

　1: 이미지 일괄 업로드 (별도 기능)  
　　A. "이미지 일괄 업로드" 버튼 제공  
　　B. ZIP 파일로 여러 이미지 한 번에 업로드  
　　C. 파일명 규칙: [메뉴명].jpg 또는 [메뉴명].png  

　2: 이미지 매칭  
　　A. 파일명과 메뉴명 자동 매칭  
　　B. 매칭되지 않는 이미지 목록 표시  
　　C. 수동 매칭 기능 제공  

　3: 이미지 제한  
　　A. ZIP 파일 최대 크기: 100MB  
　　B. 개별 이미지 최대 크기: 5MB  
　　C. 지원 형식: JPG, PNG, WebP
