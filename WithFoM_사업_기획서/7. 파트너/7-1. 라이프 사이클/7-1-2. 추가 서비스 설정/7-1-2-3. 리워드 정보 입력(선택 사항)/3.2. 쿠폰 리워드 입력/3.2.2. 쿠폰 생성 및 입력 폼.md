**3.2.2.** 쿠폰 생성 및 입력 폼  


**3.2.2.1.** 쿠폰 생성 인터페이스  

　1: 기본 입력박스  
　　A. 기본적으로 1개 쿠폰 입력박스 노출  
　　B. 총 6개까지 생성 가능  

　2: 쿠폰 추가 버튼  
　　A. 위치: 입력박스 영역 우측 상단  
　　B. 버튼 텍스트: "쿠폰 추가"  
　　C. 클릭 시: 새로운 쿠폰 입력박스 1개 추가  
　　D. 최대 개수 도달 시: 버튼 비활성화 또는 "최대 6개까지 생성 가능합니다" 메시지 표시  


**3.2.2.2.** 쿠폰 입력박스 공통 구성  

　1: 박스 헤더  
　　A. 좌측 상단: N번째 쿠폰 숫자 표시 (예: "쿠폰 1", "쿠폰 2")  
　　B. 우측 상단: 삭제 버튼 (X 아이콘)  

　2: 입력 항목  
　　A. 쿠폰명 (필수)  
　　　　a. 입력 형식: 텍스트 입력창  
　　　　b. 예시: "아메리카노 500원 할인", "10% 할인 쿠폰", "1000원 할인 쿠폰"  
　　　　c. 최대 글자 수: 50자  
　　B. 할인 유형 선택 (필수)  
　　　　a. 옵션: 금액 할인 / 비율 할인  
　　　　b. 형식: 라디오 버튼 또는 드롭다운  
　　C. 할인 금액/율 (필수)  
　　　　a. 금액 할인: 숫자 입력, 단위 "원"  
　　　　b. 비율 할인: 숫자 입력, 단위 "%"  
　　　　c. 입력 형식: 숫자만 입력 가능  
　　D. 총 발행 수량 (선택)  
　　　　a. 입력 형식: 숫자 입력창  
　　　　b. 설명: "미입력 시 무제한 발행"  
　　　　c. 최소값: 1  
　　E. 유효 기간 (필수)  
　　　　a. 입력 형식: 날짜 선택기 (시작일, 종료일)  
　　　　b. 또는 기간 선택: 30일, 60일, 90일, 사용자 지정  
　　　　c. 유효 기간 미설정 체크박스: 체크 시 무기한 유효  


**3.2.2.3.** 쿠폰 사용 조건 설정  

　1: 최소 결제금액 조건  
　　A. 최소 결제 금액 (선택)  
　　　　a. 입력 형식: 숫자 입력창, 단위 "원"  
　　　　b. 설명: "이 금액 이상 결제 시 쿠폰 사용 가능"  
　　　　c. 예시: 5,000원 이상 결제 시 사용 가능  

　2: 메뉴 조건 (선택)  
　　A. 카테고리 조건  
　　　　a. 형식: 체크박스로 카테고리 선택  
　　　　b. 설명: "선택한 카테고리 메뉴에만 쿠폰 적용"  
　　B. 단일 메뉴 조건  
　　　　a. 형식: 메뉴 선택 드롭다운  
　　　　b. 설명: "특정 메뉴에만 쿠폰 적용"  

　3: 채널 조건 (선택)  
　　A. 옵션: 매장 / 테이크아웃 / 오더  
　　B. 형식: 체크박스로 복수 선택 가능  
　　C. 설명: "선택한 채널에서만 쿠폰 사용 가능"  

　4: 시간/요일 조건 (선택)  
　　A. 시간대 조건  
　　　　a. 형식: 시간 선택 (시작 시간, 종료 시간)  
　　　　b. 예시: "09:00 ~ 18:00"  
　　B. 요일 조건  
　　　　a. 형식: 체크박스로 요일 선택  
　　　　b. 예시: "월, 화, 수, 목, 금"  


**3.2.2.4.** 쿠폰 사용 제한 설정  

　1: 할인 적용 메뉴 제외  
　　A. 체크박스: "할인 적용 메뉴에는 쿠폰 사용 불가"  
　　B. 기본값: 체크 (권장)  
　　C. 설명: "메뉴 관리에서 할인 가격이 설정된 메뉴에는 쿠폰 적용 불가"  
　　D. 체크 시: 할인 메뉴 선택 시 "할인 메뉴에는 쿠폰을 사용할 수 없습니다" 안내 표시  

　2: 1일 사용 제한  
　　A. 1일 사용 가능 총액 (선택)  
　　　　a. 입력 형식: 숫자 입력창, 단위 "원"  
　　　　b. 설명: "고객 1인당 1일 사용 가능한 총 할인 금액"  
　　B. 1일 사용 횟수 제한 (선택)  
　　　　a. 입력 형식: 숫자 입력창, 단위 "회"  
　　　　b. 설명: "고객 1인당 1일 사용 가능한 쿠폰 개수"  

　3: 1회 결제당 사용 제한  
　　A. 1회 결제 시 최대 사용 금액 (선택)  
　　　　a. 입력 형식: 숫자 입력창, 단위 "원"  
　　　　b. 설명: "1회 결제 시 사용 가능한 최대 할인 금액"  
　　B. 1회 결제당 사용 개수 제한 (선택)  
　　　　a. 입력 형식: 숫자 입력창, 단위 "개"  
　　　　b. 설명: "1회 결제 시 사용 가능한 쿠폰 개수"
