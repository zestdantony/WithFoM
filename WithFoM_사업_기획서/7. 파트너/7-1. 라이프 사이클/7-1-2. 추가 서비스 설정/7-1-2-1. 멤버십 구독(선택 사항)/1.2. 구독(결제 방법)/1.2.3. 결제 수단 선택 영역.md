**1.2.3.** 결제 수단 선택 영역  


**1.2.3.1.** 결제 방법 선택  

　1: 신용/체크카드  
　　A. 라디오 버튼 또는 카드 형태 선택 UI  
　　B. 아이콘: 카드 아이콘 표시  
　　C. 설명: "국내 모든 신용카드 및 체크카드"  

　2: 계좌이체  
　　A. 라디오 버튼 또는 카드 형태 선택 UI  
　　B. 아이콘: 은행 아이콘 표시  
　　C. 설명: "자동이체(CMS) 지원"  

　3: 간편결제  
　　A. 라디오 버튼 또는 카드 형태 선택 UI  
　　B. 아이콘: 간편결제 서비스 로고 (카카오페이, 네이버페이, 토스페이 등)  
　　C. 설명: "카카오페이, 네이버페이, 토스페이 등"  


**1.2.3.2.** 카드 결제 정보 입력  

　1: 입력 필드 구성  
　　A. 카드 번호  
　　　　a. 형식: 16자리 숫자, 4자리씩 구분 표시 (XXXX-XXXX-XXXX-XXXX)  
　　　　b. 유효성 검사: 실시간 카드 번호 검증  
　　B. 유효 기간  
　　　　a. 형식: MM/YY (월/년)  
　　　　b. 입력 방식: 드롭다운 또는 텍스트 입력  
　　C. CVC  
　　　　a. 형식: 카드 뒷면 3자리 또는 앞면 4자리 (아메리칸 익스프레스)  
　　　　b. 보안: 입력 시 마스킹 처리 (●●●)  
　　D. 카드 비밀번호  
　　　　a. 형식: 카드 비밀번호 앞 2자리  
　　　　b. 보안: 입력 시 마스킹 처리 (●●)  
　　E. 카드사  
　　　　a. 자동 인식: 카드 번호 첫 자리로 카드사 자동 인식  
　　　　b. 수동 선택: 드롭다운에서 선택 가능  

　2: 입력 필드 레이아웃  
　　A. 세로 배치, 각 필드 간 적절한 간격  
　　B. 필수 필드 표시 (*)  
　　C. 에러 메시지 표시 영역 (필드 하단)  


**1.2.3.3.** 계좌이체 정보 입력  

　1: 사업자 유형 자동 판별  
　　A. 사업자등록번호 기반 자동 판별  
　　　　a. 개인사업자: 사업자등록번호 첫 자리가 1, 2, 3  
　　　　b. 법인사업자: 사업자등록번호 첫 자리가 4, 5, 6, 7, 8, 9  
　　B. 판별 결과에 따라 입력 화면 자동 분기  


　2: 개인사업자 계좌이체 정보 입력  

　2.1: 입력 필드 구성  
　　A. 은행 선택  
　　　　a. 형식: 드롭다운 메뉴  
　　　　b. 옵션: 국내 모든 은행 목록  
　　B. 계좌번호  
　　　　a. 형식: 숫자만 입력  
　　　　b. 유효성 검사: 은행별 계좌번호 형식 검증  
　　C. 예금주명  
　　　　a. 자동 입력: 사업자 인증 시 확인된 대표자명 자동 입력  
　　　　b. 수정 가능: 필요 시 수정 가능  
　　　　c. 형식: 한글, 영문, 숫자  
　　　　d. 유효성 검사: 대표자명과 일치 여부 확인  

　2.2: 안내 문구  
　　A. 예금주명 필드 하단 표시  
　　　　a. 텍스트: "입력하신 예금주명이 대표자명과 일치해야 합니다"  
　　　　b. 스타일: 안내 텍스트, 작은 폰트, 회색  

　2.3: 계좌 인증 (1원 입금)  
　　A. 인증 버튼  
　　　　a. 위치: 계좌 정보 입력 필드 하단  
　　　　b. 텍스트: "계좌 인증하기"  
　　B. 인증 절차  
　　　　a. 1단계: "인증하기" 버튼 클릭  
　　　　b. 2단계: 1원 입금 안내 모달 표시  
　　　　　　• 입금 은행명  
　　　　　　• 입금 계좌번호  
　　　　　　• 입금자명 (대표자명 또는 지정된 입금자명)  
　　　　c. 3단계: 사용자가 1원 입금  
　　　　d. 4단계: 입금 확인 (PG사 또는 은행 API 연동)  
　　　　e. 5단계: 예금주명 확인  
　　　　　　• 예금주명 = 대표자명 일치 여부 확인  
　　　　　　• 일치 시: 인증 완료  
　　　　　　• 불일치 시: "예금주명이 대표자명과 일치하지 않습니다" 에러 메시지  
　　C. 인증 완료 후  
　　　　a. 인증 완료 표시 (체크 아이콘)  
　　　　b. 계좌 정보 입력 필드 잠금 (수정 불가)  
　　　　c. 다음 단계(약관 동의) 진행 가능  


　3: 법인사업자 계좌이체 정보 입력  

　3.1: 계좌 유형 선택  
　　A. 라디오 버튼 또는 탭 형태 선택 UI  
　　B. 선택 옵션  
　　　　a. 법인 계좌  
　　　　b. 대표자 계좌  

　3.2: 입력 필드 구성  
　　A. 은행 선택  
　　　　a. 형식: 드롭다운 메뉴  
　　　　b. 옵션: 국내 모든 은행 목록  
　　B. 계좌번호  
　　　　a. 형식: 숫자만 입력  
　　　　b. 유효성 검사: 은행별 계좌번호 형식 검증  
　　C. 예금주명  
　　　　a. 법인 계좌 선택 시  
　　　　　• 자동 입력: 사업자 인증 시 확인된 사업자명(법인명) 자동 입력  
　　　　　• 수정 가능: 필요 시 수정 가능  
　　　　　• 유효성 검사: 법인명과 일치 여부 확인  
　　　　b. 대표자 계좌 선택 시  
　　　　　• 자동 입력: 사업자 인증 시 확인된 대표자명 자동 입력  
　　　　　• 수정 가능: 필요 시 수정 가능  
　　　　　• 유효성 검사: 대표자명과 일치 여부 확인  

　3.3: 안내 문구  
　　A. 예금주명 필드 하단 표시  
　　　　a. 법인 계좌 선택 시: "법인 계좌: 법인명과 일치해야 합니다"  
　　　　b. 대표자 계좌 선택 시: "대표자 계좌: 대표자명과 일치해야 합니다"  
　　　　c. 스타일: 안내 텍스트, 작은 폰트, 회색  

　3.4: 계좌 인증 (1원 입금)  
　　A. 인증 버튼  
　　　　a. 위치: 계좌 정보 입력 필드 하단  
　　　　b. 텍스트: "계좌 인증하기"  
　　B. 인증 절차  
　　　　a. 1단계: "인증하기" 버튼 클릭  
　　　　b. 2단계: 1원 입금 안내 모달 표시  
　　　　　• 입금 은행명  
　　　　　• 입금 계좌번호  
　　　　　• 입금자명 (법인명 또는 대표자명)  
　　　　c. 3단계: 사용자가 1원 입금  
　　　　d. 4단계: 입금 확인 (PG사 또는 은행 API 연동)  
　　　　e. 5단계: 예금주명 확인  
　　　　　• 법인 계좌: 예금주명 = 법인명 일치 여부 확인  
　　　　　• 대표자 계좌: 예금주명 = 대표자명 일치 여부 확인  
　　　　　• 일치 시: 인증 완료  
　　　　　• 불일치 시: "예금주명이 법인명(또는 대표자명)과 일치하지 않습니다" 에러 메시지  
　　C. 인증 완료 후  
　　　　a. 인증 완료 표시 (체크 아이콘)  
　　　　b. 계좌 정보 입력 필드 잠금 (수정 불가)  
　　　　c. 다음 단계(약관 동의) 진행 가능  


　4: 본인 확인 로직  

　4.1: 개인사업자  
　　A. 확인 항목  
　　　　a. 예금주명 = 대표자명 (사업자 인증 시 확인된 정보)  
　　　　b. 1원 입금 인증 완료  
　　B. 확인 방법  
　　　　a. 사용자 입력 예금주명  
　　　　b. 사업자 인증 시 확인된 대표자명과 비교  
　　　　c. 일치 여부 확인  
　　　　d. 일치 + 입금 인증 완료 = 본인 확인 완료  

　4.2: 법인사업자  
　　A. 법인 계좌인 경우  
　　　　a. 확인 항목  
　　　　　• 예금주명 = 법인명 (사업자 인증 시 확인된 사업자명)  
　　　　　• 1원 입금 인증 완료  
　　　　b. 확인 방법  
　　　　　• 사용자 입력 예금주명  
　　　　　• 사업자 인증 시 확인된 법인명과 비교  
　　　　　• 일치 여부 확인  
　　　　　• 일치 + 입금 인증 완료 = 본인 확인 완료  
　　B. 대표자 계좌인 경우  
　　　　a. 확인 항목  
　　　　　• 예금주명 = 대표자명 (사업자 인증 시 확인된 대표자명)  
　　　　　• 1원 입금 인증 완료  
　　　　b. 확인 방법  
　　　　　• 사용자 입력 예금주명  
　　　　　• 사업자 인증 시 확인된 대표자명과 비교  
　　　　　• 일치 여부 확인  
　　　　　• 일치 + 입금 인증 완료 = 본인 확인 완료  


　5: 자동이체(CMS) 안내  

　5.1: 안내 문구  
　　A. 텍스트: "매월 결제일에 자동으로 출금됩니다"  
　　B. 스타일: 안내 텍스트, 작은 폰트, 회색  
　　C. 위치: 계좌 인증 완료 후 표시  

　5.2: 추가 안내  
　　A. 계좌 변경 안내  
　　　　a. 텍스트: "계좌 변경이 필요하신가요? 설정 > 결제 관리에서 변경할 수 있습니다"  
　　　　b. 링크: 설정 페이지로 이동 (선택 사항)  


　6: 에러 처리  

　6.1: 예금주명 불일치  
　　A. 에러 메시지  
　　　　a. 개인사업자: "예금주명이 대표자명과 일치하지 않습니다. 정확한 예금주명을 입력해주세요"  
　　　　b. 법인사업자 (법인 계좌): "예금주명이 법인명과 일치하지 않습니다. 정확한 예금주명을 입력해주세요"  
　　　　c. 법인사업자 (대표자 계좌): "예금주명이 대표자명과 일치하지 않습니다. 정확한 예금주명을 입력해주세요"  
　　B. 처리 방법  
　　　　a. 예금주명 필드에 에러 표시 (빨간색 테두리)  
　　　　b. 에러 메시지 표시  
　　　　c. 예금주명 수정 후 재인증 가능  

　6.2: 계좌 인증 실패  
　　A. 에러 메시지  
　　　　a. 입금 미확인: "1원 입금이 확인되지 않았습니다. 입금 후 다시 시도해주세요"  
　　　　b. 입금자명 불일치: "입금자명이 일치하지 않습니다. 정확한 입금자명으로 입금해주세요"  
　　　　c. 시스템 오류: "일시적인 오류가 발생했습니다. 잠시 후 다시 시도해주세요"  
　　B. 처리 방법  
　　　　a. 재시도 버튼 제공  
　　　　b. 고객센터 문의 안내 (지속적인 오류 시)  


**1.2.3.4.** 간편결제 연동  

　1: 간편결제 선택 시  
　　A. 해당 간편결제 서비스 인증 화면으로 이동  
　　B. 간편결제 앱 또는 웹 인증 완료 후 복귀  
　　C. 결제 수단 등록 완료 처리  

　2: 지원 간편결제 서비스  
　　A. 카카오페이  
　　B. 네이버페이  
　　C. 토스페이  
　　D. 기타 (추가 가능)
