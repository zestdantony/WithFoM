　　**1.2.** 구독(결제 방법)  


　　　1: 개요  

위드폼 파트너 멤버십 구독 시 사용할 수 있는 결제 방법을 안내합니다.

## 결제 방법

### 신용/체크카드

| 항목 | 내용 |
|---|---|
| 지원 카드 | 국내 모든 신용카드 및 체크카드 |
| 결제 방식 | 자동 결제 (매월 동일일 결제) |
| 결제 확인 | 결제 완료 시 이메일/앱 알림 발송 |

### 계좌이체

| 항목 | 내용 |
|---|---|
| 지원 은행 | 국내 모든 은행 |
| 결제 방식 | 자동이체 (CMS) |
| 출금일 | 매월 결제일에 자동 출금 |

### 간편결제

| 항목 | 내용 |
|---|---|
| 지원 서비스 | 카카오페이, 네이버페이, 토스페이 등 |
| 결제 방식 | 간편결제 연동 자동 결제 |
| 결제 확인 | 해당 간편결제 앱 및 위드폼 앱 알림 |

## 결제 등록

### 1단계: 결제 수단 선택

1. 설정 > 멤버십 > 결제 수단 관리
2. 결제 방법 선택 (카드/계좌/간편결제)
3. 결제 정보 입력

### 2단계: 결제 정보 입력

#### 카드 결제

| 입력 항목 | 내용 |
|---|---|
| 카드 번호 | 16자리 카드 번호 |
| 유효 기간 | MM/YY 형식 |
| CVC | 카드 뒷면 3자리 |
| 비밀번호 | 카드 비밀번호 앞 2자리 |

#### 계좌이체

| 입력 항목 | 내용 |
|---|---|
| 은행 | 은행 선택 |
| 계좌번호 | 출금 계좌번호 |
| 예금주 | 예금주명 |
| 생년월일 | 예금주 생년월일 |

### 3단계: 결제 확인

1. 입력 정보 확인
2. 약관 동의
3. 결제 등록 완료

## 결제 주기

| 멤버십 | 결제 주기 | 결제일 |
|---|---|---|
| 베이직 | 무료 | - |
| 플러스 | 월 1회 | 구독 시작일 기준 매월 동일일 |
| 프리미엄 | 월 1회 | 구독 시작일 기준 매월 동일일 |

## 결제 실패 시

### 결제 실패 원인

| 원인 | 해결 방법 |
|---|---|
| 잔액 부족 | 계좌 잔액 확인 후 재결제 |
| 카드 한도 초과 | 카드 한도 확인 후 재결제 |
| 카드 유효기간 만료 | 새 카드 정보 등록 |
| 결제 정보 오류 | 결제 정보 확인 및 수정 |

### 결제 재시도

| 항목 | 내용 |
|---|---|
| 재시도 횟수 | 최대 3회 자동 재시도 |
| 재시도 간격 | 1일, 3일, 7일 후 재시도 |
| 실패 시 조치 | 3회 실패 시 멤버십 일시 정지 |

## 결제 취소 및 환불

### 결제 취소

| 항목 | 내용 |
|---|---|
| 취소 가능 기간 | 결제 후 7일 이내 |
| 취소 방법 | 설정 > 멤버십 > 결제 취소 |
| 환불 기간 | 취소 후 3~5영업일 이내 |

### 환불 정책

| 항목 | 내용 |
|---|---|
| 전액 환불 | 결제 후 7일 이내 취소 시 |
| 일할 환불 | 7일 이후 취소 시 (잔여 기간 기준) |
| 환불 불가 | 이용 혜택 사용 시 환불 불가 |

## 영수증 발급

| 항목 | 내용 |
|---|---|
| 영수증 발급 | 설정 > 멤버십 > 결제 내역 > 영수증 발급 |
| 세금계산서 | 사업자 등록 시 세금계산서 발급 가능 |
| 발급 기간 | 결제 완료 후 즉시 발급 가능 |

## 참고

- 결제 정보는 안전하게 암호화되어 저장됩니다
- 결제 실패 시 이메일 및 앱 알림으로 안내됩니다
- 결제 관련 문의는 고객센터로 연락해 주세요
