**1.1.3.** 멤버십 플랜 비교 카드  


**1.1.3.1.** 카드 레이아웃 개요  

　1: 3개의 플랜 카드를 나란히 배치 (베이직, 플러스, 프리미엄)  

　2: 각 카드는 동일한 구조로 구성되며, 플랜별로 내용만 다름  

　3: 플러스 카드에 "추천" 배지를 표시하여 시각적으로 강조  


**1.1.3.2.** 플랜 비교 요약 테이블  

| 항목 | 베이직 | 플러스 | 프리미엄 |  
|---|---|---|---|  
| 월 구독료 | 무료 | ₩29,000 (첫 달 무료 + 2달 50%) | ₩49,000 → ₩34,000 (6개월 할인) |  
| 대시보드 | 기본 | 상세 분석 | AI 분석 + 맞춤 추천 |  
| 캠페인 매칭 | 일반 | 우선 매칭 | 최우선 매칭 |  
| 스탬프 기능 | 기본 템플릿 | 확장 커스텀 (5종) | 고급 커스텀 (5종+) |  
| 휴면 고객 케어 | - | 월 30회 알림톡 | 월 300회 자동화 |  
| 지도 핀 커스텀 | - | 카페 로고 | AI 아이콘 생성 |  
| 폼 코인 증정 | - | 월 200개 (2,000원 상당) | 월 500개 (5,000원 상당) |  


**1.1.3.3.** 베이직 카드 구성  

　1: 카드 스타일  
　　A. 배경색: 회색 또는 연한 색상 (비강조)  
　　B. 테두리: 현재 플랜 강조 (진한 테두리)  
　　C. 상단 라벨: "현재 플랜"  

　2: 플랜 정보 영역  
　　A. 플랜 이름: "베이직"  
　　B. 가격: "무료"  
　　C. 한 줄 설명: "기본 운영 기능 제공"  

　3: 주요 혜택 리스트 (카드 내 표시)  
　　A. 기본 대시보드  
　　B. 캠페인 매칭  
　　C. 리워드/오더 기능  
　　D. 스탬프 기능 (기본 템플릿)  
　　E. 기본 리포트 5종  

　4: 버튼 영역  
　　A. 버튼 없음 (현재 이용 중이므로)  
　　B. 또는 "현재 플랜" 텍스트만 표시 (비활성 상태)  


**1.1.3.4.** 플러스 카드 구성  

　1: 카드 스타일  
　　A. 배경색: 강조 색상 (파란색 계열)  
　　B. 테두리: 일반  
　　C. 상단 라벨: "추천" 배지  

　2: 플랜 정보 영역  
　　A. 플랜 이름: "플러스"  
　　B. 가격: "₩29,000/월"  
　　C. 프로모션: "첫 달 무료 + 2달 50% 할인" (강조 색상)  
　　D. 한 줄 설명: "단골·재방문 관리 중심"  

　3: 주요 혜택 리스트 (카드 내 표시)  
　　A. 베이직 전체 포함  
　　B. 상세 대시보드 분석  
　　C. 캠페인 우선 매칭  
　　D. 스탬프 확장 커스텀 (템플릿 5종)  
　　E. AI 리뷰 답글 생성  
　　F. 휴면 고객 케어 (월 30회)  
　　G. 벤치마크 지표 비교  
　　H. 폼 코인 월 200개 증정  

　4: 버튼 영역  
　　A. 주요 버튼: "플러스 시작하기"  
　　　　a. 스타일: Primary 버튼, 강조 색상  
　　　　b. 클릭 시: 결제 페이지로 이동  
　　B. 보조 버튼: "상세 보기"  
　　　　a. 스타일: 텍스트 링크 또는 아웃라인 버튼  
　　　　b. 클릭 시: 플러스 상세 모달 오픈  


**1.1.3.5.** 프리미엄 카드 구성  

　1: 카드 스타일  
　　A. 배경색: 프리미엄 색상 (금색 또는 다크 계열)  
　　B. 테두리: 골드 또는 그라데이션  
　　C. 상단 라벨: "프리미엄" 배지  

　2: 플랜 정보 영역  
　　A. 플랜 이름: "프리미엄"  
　　B. 가격: "₩49,000/월"  
　　C. 프로모션: "→ ₩34,000 (6개월 할인)" (강조 색상)  
　　D. 한 줄 설명: "정밀 분석 + 고급 기능"  

　3: 주요 혜택 리스트 (카드 내 표시)  
　　A. 플러스 전체 포함  
　　B. AI 대시보드 분석  
　　C. 캠페인 최우선 매칭  
　　D. 휴면/우수 고객 자동 케어 (월 300회)  
　　E. 포스터 AI 툴  
　　F. 세분화 벤치마크 + 인사이트  
　　G. 스탬프 고급 커스텀  
　　H. 폼 코인 월 500개 증정  

　4: 버튼 영역  
　　A. 주요 버튼: "프리미엄 시작하기"  
　　　　a. 스타일: Primary 버튼, 프리미엄 색상  
　　　　b. 클릭 시: 결제 페이지로 이동  
　　B. 보조 버튼: "상세 보기"  
　　　　a. 스타일: 텍스트 링크 또는 아웃라인 버튼  
　　　　b. 클릭 시: 프리미엄 상세 모달 오픈  


**1.1.3.6.** 카드 공통 인터랙션  

　1: 카드 호버/탭 효과  
　　A. 데스크톱: 호버 시 카드 그림자 강조  
　　B. 모바일: 탭 시 시각적 피드백  

　2: 혜택 리스트 더보기  
　　A. 카드 내 혜택은 주요 항목만 표시 (5~8개)  
　　B. 전체 혜택은 "상세 보기" 클릭 시 모달에서 확인
