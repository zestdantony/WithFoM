　　**1.1.** 멤버십 혜택(베이직, 플러스, 프리미엄)  


　　　1: 개요  

위드폼 파트너 멤버십은 베이직, 플러스, 프리미엄 3가지 등급으로 구성되어 있으며, 각 등급별로 다양한 혜택을 제공합니다.

## 멤버십 등급별 혜택

| 혜택 | 베이직 | 플러스 | 프리미엄 |
|---|---|---|---|
| 월 구독료 | 무료 | 29,000원 | 49,000원 |
| 기본 서비스 | ✅ | ✅ | ✅ |
| 대시보드 분석 | 기본 | 상세 | 프리미엄 |
| 캠페인 매칭 우선권 | - | ✅ | ✅✅ |
| 컵홀더 디자인 커스텀 | - | 기본 | 프리미엄 |
| 전담 매니저 | - | - | ✅ |
| 광고 수익 배분 | 기본 | +5% | +10% |
| 정산 주기 | 월 1회 | 월 2회 | 주 1회 |

## 베이직 멤버십

### 포함 혜택

| 혜택 | 내용 |
|---|---|
| 기본 서비스 이용 | 위드폼 파트너 기본 서비스 이용 |
| 기본 대시보드 | 일일 소진량, 기본 통계 확인 |
| 캠페인 매칭 | 기본 캠페인 매칭 |
| 월 정산 | 월 1회 정산 |

### 적합한 파트너

- 위드폼 서비스를 처음 시작하는 파트너
- 소규모 매장 운영자
- 기본 기능만 필요한 파트너

## 플러스 멤버십

### 포함 혜택

| 혜택 | 내용 |
|---|---|
| 베이직 혜택 전체 | 베이직 멤버십 모든 혜택 포함 |
| 상세 대시보드 | 고객 분석, 캠페인 성과 분석 등 |
| 캠페인 우선 매칭 | 캠페인 매칭 시 우선권 부여 |
| 컵홀더 디자인 커스텀 | 기본 커스텀 옵션 제공 |
| 광고 수익 +5% | 광고 수익 배분 5% 추가 |
| 월 2회 정산 | 15일, 말일 정산 |

### 적합한 파트너

- 캠페인 매칭을 더 많이 받고 싶은 파트너
- 데이터 분석을 활용하고 싶은 파트너
- 중규모 매장 운영자

## 프리미엄 멤버십

### 포함 혜택

| 혜택 | 내용 |
|---|---|
| 플러스 혜택 전체 | 플러스 멤버십 모든 혜택 포함 |
| 프리미엄 대시보드 | AI 분석, 맞춤 추천, 트렌드 분석 등 |
| 캠페인 최우선 매칭 | 캠페인 매칭 시 최우선권 부여 |
| 프리미엄 컵홀더 디자인 | 고급 커스텀 옵션 및 전문 디자인 지원 |
| 전담 매니저 | 1:1 전담 매니저 배정 |
| 광고 수익 +10% | 광고 수익 배분 10% 추가 |
| 주 1회 정산 | 매주 정산 |

### 적합한 파트너

- 프리미엄 서비스를 원하는 파트너
- 대규모 매장 또는 다점포 운영자
- 맞춤형 지원이 필요한 파트너

## 멤버십 변경

### 업그레이드

| 항목 | 내용 |
|---|---|
| 변경 시점 | 언제든지 업그레이드 가능 |
| 적용 시점 | 결제 완료 즉시 적용 |
| 요금 계산 | 일할 계산 적용 |

### 다운그레이드

| 항목 | 내용 |
|---|---|
| 변경 시점 | 다음 결제일 전까지 신청 |
| 적용 시점 | 다음 결제 주기부터 적용 |
| 환불 | 잔여 기간 환불 불가 |

## 참고

- 멤버십 구독은 선택 사항입니다
- 베이직 멤버십은 무료로 이용 가능합니다
- 멤버십 등급은 언제든지 변경할 수 있습니다
