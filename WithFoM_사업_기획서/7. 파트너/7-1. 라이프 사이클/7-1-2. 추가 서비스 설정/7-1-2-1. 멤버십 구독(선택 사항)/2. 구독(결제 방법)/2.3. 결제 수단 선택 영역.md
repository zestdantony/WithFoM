**2.3.** 결제 수단 선택 영역  


**2.3.1.** 결제 방법 선택  

　1: 신용/체크카드  
　　A. 라디오 버튼 또는 카드 형태 선택 UI  
　　B. 아이콘: 카드 아이콘 표시  
　　C. 설명: "국내 모든 신용카드 및 체크카드"  

　2: 계좌이체  
　　A. 라디오 버튼 또는 카드 형태 선택 UI  
　　B. 아이콘: 은행 아이콘 표시  
　　C. 설명: "자동이체(CMS) 지원"  

　3: 간편결제  
　　A. 라디오 버튼 또는 카드 형태 선택 UI  
　　B. 아이콘: 간편결제 서비스 로고 (카카오페이, 네이버페이, 토스페이 등)  
　　C. 설명: "카카오페이, 네이버페이, 토스페이 등"  


**2.3.2.** 카드 결제 정보 입력  

　1: 입력 필드 구성  
　　A. 카드 번호  
　　　　a. 형식: 16자리 숫자, 4자리씩 구분 표시 (XXXX-XXXX-XXXX-XXXX)  
　　　　b. 유효성 검사: 실시간 카드 번호 검증  
　　B. 유효 기간  
　　　　a. 형식: MM/YY (월/년)  
　　　　b. 입력 방식: 드롭다운 또는 텍스트 입력  
　　C. CVC  
　　　　a. 형식: 카드 뒷면 3자리 또는 앞면 4자리 (아메리칸 익스프레스)  
　　　　b. 보안: 입력 시 마스킹 처리 (●●●)  
　　D. 카드 비밀번호  
　　　　a. 형식: 카드 비밀번호 앞 2자리  
　　　　b. 보안: 입력 시 마스킹 처리 (●●)  
　　E. 카드사  
　　　　a. 자동 인식: 카드 번호 첫 자리로 카드사 자동 인식  
　　　　b. 수동 선택: 드롭다운에서 선택 가능  

　2: 입력 필드 레이아웃  
　　A. 세로 배치, 각 필드 간 적절한 간격  
　　B. 필수 필드 표시 (*)  
　　C. 에러 메시지 표시 영역 (필드 하단)  


**2.3.3.** 계좌이체 정보 입력  

　1: 입력 필드 구성  
　　A. 은행 선택  
　　　　a. 형식: 드롭다운 메뉴  
　　　　b. 옵션: 국내 모든 은행 목록  
　　B. 계좌번호  
　　　　a. 형식: 숫자만 입력  
　　　　b. 유효성 검사: 은행별 계좌번호 형식 검증  
　　C. 예금주명  
　　　　a. 형식: 한글, 영문, 숫자  
　　　　b. 유효성 검사: 실명 확인  
　　D. 생년월일  
　　　　a. 형식: YYYYMMDD 또는 YYYY-MM-DD  
　　　　b. 용도: 본인 확인  

　2: 자동이체(CMS) 안내  
　　A. 텍스트: "매월 결제일에 자동으로 출금됩니다"  
　　B. 스타일: 안내 텍스트, 작은 폰트  


**2.3.4.** 간편결제 연동  

　1: 간편결제 선택 시  
　　A. 해당 간편결제 서비스 인증 화면으로 이동  
　　B. 간편결제 앱 또는 웹 인증 완료 후 복귀  
　　C. 결제 수단 등록 완료 처리  

　2: 지원 간편결제 서비스  
　　A. 카카오페이  
　　B. 네이버페이  
　　C. 토스페이  
　　D. 기타 (추가 가능)
