**5.1.** 캠페인 캘린더 보드  

↳ 캠페인 기간 동안의 소진량 및 일정을 캘린더로 시각화  


**5.1.1.** 기본 기능  

**5.1.1.1.** 월 단위 캘린더 표시, 달력 넘겨가며 확인 가능  
**5.1.1.2.** 날짜별 소진량 숫자로 표시  
**5.1.1.3.** 보고 완료(그린) / 미보고(빨간색) 색상 구분  


**5.1.2.** 캠페인 일정 표시  

**5.1.2.1.** N번차 캠페인 매칭일, 배포 시작일, 입고일 표시  
**5.1.2.2.** 캠페인 종료일 표시 (예정일 및 실제 소진 완료일)  


**5.1.3.** 수정 기능  

**5.1.3.1.** 캘린더에서 날짜 클릭 시 해당 날짜 소진량 수정 가능  
**5.1.3.2.** 수정 가능 시간 (익일 오픈 후 2시간) 내에만 수정 버튼 활성화
