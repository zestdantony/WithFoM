**3.3.** 소진 보고 완료 상태 표시  


**3.3.1.** 보드 상태 표시  

**3.3.1.1.** 상태 정보  

　1: 완료/미완료 표시  
　　A. 보드 제목 오른쪽 옆에 작은 박스로 "완료/미완료" 표시  
　　B. 소진 보고 완료 시: "완료" 표시  
　　C. 소진 보고 미완료 시: "미완료" 표시  

　2: 보드 색상  
　　A. 미완료: 빨간색  
　　B. 완료: 그린 컬러  
　　C. 색상 변경은 소진 보고 제출 시 자동 반영  


**3.3.2.** 대시보드 연동 표시  

**3.3.2.1.** 연동 정보  

　1: 일일 소진 보드 상태  
　　A. 대시보드의 일일 소진 보고 보드에 상태 표시  
　　B. 완료/미완료 상태와 색상 연동  
　　C. 클릭 시 소진 보고 페이지로 이동  

　2: 캘린더 연동 표시  
　　A. 캠페인 캘린더에 소진량 표시  
　　B. 보고 완료 날짜는 그린 컬러로 표시  
　　C. 미보고 날짜는 빨간색으로 표시
