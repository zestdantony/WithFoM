**3.5.** 소진 보고 보드 구성  

↳ 대시보드 내 일일 소진 보고 보드 UI 구성  


**3.5.1.** 구성 요소  

**3.5.1.1.** 상단 - 전일 소진량 표시 + 수정 버튼  
**3.5.1.2.** 중간 - 오늘 소진량 입력창 + 제출 버튼  
**3.5.1.3.** 상태 표시 - "오늘 보고 완료/미완료"  
**3.5.1.4.** 보드 색상 - 미완료(빨간색) / 완료(그린)  


**3.5.2.** 클릭 시 이동  

**3.5.2.1.** 보드 클릭 → 사이드바의 일일 소진 보고 페이지로 이동
