**1.2.** 매장 배포 시작  


**1.2.1.** 배포 시작 절차  

**1.2.1.1.** 배포 시작 처리  

　1: 배포 시작 확인  
　　A. 입고 완료 확인 후 배포 시작 가능  
　　B. 배포 시작일 도달 시 배포 시작 버튼 활성화  
　　C. 배포 시작 버튼 클릭 시 배포 상태 변경  

　2: 배포 시작 안내 메시지  
　　A. 메시지:  
　　　a. '컵홀더 배포를 시작하시겠습니까?  
　　　b. 배포 시작 후 일일 소진 보고를 시작해 주세요.'  
　　B. 확인 버튼: 배포 시작 처리  


**1.2.2.** 배포 시작 후 안내  

**1.2.2.1.** 안내 사항  

　1: 배포 시작 완료 안내  
　　A. "배포가 시작되었습니다" 메시지 표시  
　　B. 일일 소진 보고 시작 안내  
　　C. 소진 보고 방법 가이드 링크 제공  

　2: 일일 소진 보고 활성화  
　　A. 배포 시작 후 일일 소진 보고 기능 활성화  
　　B. 매일 영업 종료 시점 소진량 입력 안내  
　　C. 소진 보고 미완료 시 알림 발송  


**1.2.3.** 컵홀더 배포 의무 및 책임  

**1.2.3.1.** 배포 의무 및 책임 사항  

　1: 배포 의무  
　　A. 매칭된 컵홀더는 캠페인 기간 내 전량 배포해야 함  
　　B. 광고주의 광고 컵홀더를 기간 내 100% 소진하는 것이 핵심 의무  
　　C. 파트너는 배포 일정에 맞춰 책임감 있게 컵홀더를 배포해야 함  

　2: 기간 내 정량 배포 미달성 시 조치  
　　A. 불가피한 사유로 기간 내 정량 배포가 어려울 경우 즉시 관리자에게 사유 보고 필요  
　　B. 합당한 사유: 천재지변, 매장 불가항력적 휴업, 시스템 오류 등  
　　C. 사유 보고 없이 배포 미완료 시 관리자 확인 연락 진행
