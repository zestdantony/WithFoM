**1.2.** 매장 배포 시작  

↳ 입고 완료 후 배포 시작 버튼 클릭으로 배포 시작  


**1.2.1.** 흐름  

**1.2.1.1.** 입고 완료 확인 → 배포 시작 버튼 활성화  
**1.2.1.2.** 배포 시작 버튼 클릭 → 배포 상태 변경  
**1.2.1.3.** 배포 시작 완료 메시지 표시  
**1.2.1.4.** 일일 소진 보고 기능 활성화  


**1.2.2.** 핵심 의무  

**1.2.2.1.** 매칭된 컵홀더는 캠페인 기간 내 전량 배포해야 함  
**1.2.2.2.** 기간 내 배포 미달성 시 즉시 관리자에게 사유 보고 필요  
**1.2.2.3.** 합당한 사유 (천재지변, 불가항력적 휴업, 시스템 오류) 보고 시 등급 불이익 없음
