**1.1.** 탈퇴 요청 절차  

↳ 탈퇴 요청 경로 및 확인 절차  


**1.1.1.** 탈퇴 요청 경로  

**1.1.1.1.** 파트너 포털 > 계정 설정 > 탈퇴 메뉴 접속  
**1.1.1.2.** 탈퇴 전 확인 사항 및 데이터 처리 안내 표시  


**1.1.2.** 탈퇴 사유 선택  

**1.1.2.1.** 사유: 서비스 불만족 / 비용 부담 / 매장 운영 중단 / 다른 서비스 이용 / 기타  
**1.1.2.2.** 상세 설명 입력 (선택), 기타 선택 시 직접 입력 필수  


**1.1.3.** 탈퇴 확인 절차  

**1.1.3.1.** 비밀번호 재입력으로 본인 확인  
**1.1.3.2.** 탈퇴 확인 체크박스 선택 필수  
**1.1.3.3.** [탈퇴 신청] 버튼 클릭 → 탈퇴 요청 완료  


• 상세: 7-2-13. 정보 수정 참조
