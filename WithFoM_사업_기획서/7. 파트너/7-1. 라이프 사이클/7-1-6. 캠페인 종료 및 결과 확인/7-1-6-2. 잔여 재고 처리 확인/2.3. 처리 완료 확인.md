**2.3.** 처리 완료 확인  

↳ 잔여 재고 처리 완료 확인 및 유의사항  


**2.3.1.** 확인 방법  

**2.3.1.1.** 파트너 포털 > 재고 관리 > 잔여 재고 탭 클릭  
**2.3.1.2.** 캠페인명, 잔여 수량, 처리 방법, 처리 기한, 처리 상태 확인  


**2.3.2.** 처리 방법별 절차  

**2.3.2.1.** 이월 처리 → 별도 조치 불필요, 다음 캠페인 시 자동 포함  
**2.3.2.2.** 반품 처리 → 안내 주소로 발송 → 운송장 번호 입력 → [반품 발송 완료] 클릭  
**2.3.2.3.** 폐기 처리 → 잔여 컵홀더 폐기 → [폐기 완료] 클릭  


**2.3.3.** 유의사항  

**2.3.3.1.** 처리 기한: 캠페인 종료 후 14일 이내 처리 완료 권장  
**2.3.3.2.** 이월 시 파트너가 보관 책임 (품질 유지)  
**2.3.3.3.** 미처리 시 처리 기한 경과 후 관리자에게 자동 에스컬레이션  


• 상세: 7-2-12. 캠페인 종료 관리 참조
