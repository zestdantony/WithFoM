**1.2.** 잔여 재고 처리 확인  


**1.2.1.** 개요  

캠페인 기간이 종료되었으나 배정된 컵홀더가 모두 소진되지 않은 경우, 잔여 재고 처리 방법을 확인합니다.  


**1.2.2.** 잔여 재고 발생 조건  

**1.2.2.1.** 발생 상황  

| 상황 | 설명 |  
|---|---|  
| 기간 만료 | 캠페인 종료일까지 컵홀더를 모두 소진하지 못함 |  
| 예상 대비 저조 | 방문객 수가 예상보다 적어 소진 속도가 느림 |  
| 휴무/이벤트 | 예상치 못한 휴무 또는 특별 이벤트로 인한 소진 지연 |  


**1.2.3.** 잔여 재고 처리 방법

**1.2.3.1.** 처리 방법 안내

　1: 다음 캠페인 이월 시 파트너 선택 사항
　　A. 동일 광고주의 다음 캠페인에서 재사용하는 경우
　　B. 파트너는 아래 두 가지 방법 중 선택하여 처리

| 처리 방법 | 설명 | 파트너 조치 |
|---|---|---|
| 보관 유지 | 다음 캠페인까지 매장에서 보관 | 별도 조치 불필요 |
| 지정 주소로 반송 | 위드폼 지정 창고로 반송 후 다음 캠페인 시 재입고 | 지정 주소로 반송 (배송비 위드폼 부담) |

**1.2.3.2.** 폐기 처리 (광고주 요청 시)

광고주가 폐기를 요청한 경우에만 폐기 처리를 진행합니다.

| 처리 방법 | 설명 | 파트너 조치 |
|---|---|---|
| 폐기 처리 | 광고주가 폐기를 요청한 경우 | 폐기 후 폐기 확인 버튼 클릭 |


**1.2.4.** 잔여 재고 확인 절차  

**1.2.4.1.** 확인 방법  

　1: 파트너 포털 접속  
　　A. 좌측 메뉴 > 재고 관리 > 잔여 재고 탭 클릭  

　2: 잔여 재고 현황 확인  

| 표시 항목 | 내용 |  
|---|---|  
| 캠페인명 | 해당 캠페인 이름 |  
| 잔여 수량 | 미소진 컵홀더 수량 |  
| 처리 방법 | 파트너가 선택한 처리 방법 |  
| 처리 기한 | 폐기/반품 시 처리 완료 기한 |  
| 처리 상태 | 대기 중 / 처리 완료 |  


**1.2.5.** 처리 완료 확인  

**1.2.5.1.** 처리 방법별 절차  

　1: 이월 처리  
　　A. 별도 조치 불필요  
　　B. 다음 캠페인 시작 시 자동으로 기존 재고 포함  

　2: 반품 처리  
　　A. 안내된 주소로 잔여 컵홀더 발송  
　　B. 운송장 번호 입력  
　　C. 파트너 포털 > 잔여 재고 > [반품 발송 완료] 버튼 클릭  

　3: 폐기 처리 (광고주 요청 시)  
　　A. 잔여 컵홀더 폐기  
　　B. 파트너 포털 > 잔여 재고 > [폐기 완료] 버튼 클릭  
　　C. 폐기 확인 메시지 확인 후 완료  


**1.2.6.** 유의사항  

**1.2.6.1.** 주의 사항  

| 항목 | 내용 |  
|---|---|  
| 처리 기한 | 캠페인 종료 후 14일 이내 처리 완료 권장 |  
| 보관 책임 | 이월 시 파트너가 보관 책임 (품질 유지) |  
| 미처리 시 | 처리 기한 경과 시 관리자에게 자동 에스컬레이션 |
