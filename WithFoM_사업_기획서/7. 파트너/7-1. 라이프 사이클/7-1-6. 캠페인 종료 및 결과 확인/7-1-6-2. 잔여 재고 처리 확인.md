# 7-1-6-2. 잔여 재고 처리 확인

## 개요
캠페인 기간이 종료되었으나 배정된 컵홀더가 모두 소진되지 않은 경우, 잔여 재고 처리 방법을 확인합니다.

---

## 1. 잔여 재고 발생 조건

| 상황 | 설명 |
|---|---|
| 기간 만료 | 캠페인 종료일까지 컵홀더를 모두 소진하지 못함 |
| 예상 대비 저조 | 방문객 수가 예상보다 적어 소진 속도가 느림 |
| 휴무/이벤트 | 예상치 못한 휴무 또는 특별 이벤트로 인한 소진 지연 |

---

## 2. 잔여 재고 처리 방법

광고주가 선택한 처리 방법에 따라 파트너에게 안내됩니다.

| 처리 방법 | 설명 | 파트너 조치 |
|---|---|---|
| 다음 캠페인 이월 | 동일 광고주의 다음 캠페인에서 재사용 | 보관 유지 (별도 조치 불필요) |
| 폐기 처리 | 광고주가 폐기를 선택한 경우 | 폐기 후 폐기 확인 버튼 클릭 |
| 반품 처리 | 광고주가 반품을 요청한 경우 | 지정 주소로 반송 (배송비 광고주 부담) |

---

## 3. 잔여 재고 확인 절차

### 3-1. 파트너 포털 접속
- 좌측 메뉴 > **재고 관리** > **잔여 재고** 탭 클릭

### 3-2. 잔여 재고 현황 확인

| 표시 항목 | 내용 |
|---|---|
| 캠페인명 | 해당 캠페인 이름 |
| 잔여 수량 | 미소진 컵홀더 수량 |
| 처리 방법 | 광고주가 선택한 처리 방법 |
| 처리 기한 | 폐기/반품 시 처리 완료 기한 |
| 처리 상태 | 대기 중 / 처리 완료 |

---

## 4. 처리 완료 확인

### 4-1. 이월 처리
- 별도 조치 불필요
- 다음 캠페인 시작 시 자동으로 기존 재고 포함

### 4-2. 폐기 처리
1. 잔여 컵홀더 폐기
2. 파트너 포털 > 잔여 재고 > **[폐기 완료]** 버튼 클릭
3. 폐기 확인 메시지 확인 후 완료

### 4-3. 반품 처리
1. 안내된 주소로 잔여 컵홀더 발송
2. 운송장 번호 입력
3. 파트너 포털 > 잔여 재고 > **[반품 발송 완료]** 버튼 클릭

---

## 5. 유의사항

| 항목 | 내용 |
|---|---|
| 처리 기한 | 캠페인 종료 후 14일 이내 처리 완료 권장 |
| 보관 책임 | 이월 시 파트너가 보관 책임 (품질 유지) |
| 미처리 시 | 처리 기한 경과 시 관리자에게 자동 에스컬레이션 |
