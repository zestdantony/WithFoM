**3.1.** 실 사용 기간 계산 방식  

↳ 등급 산정 기간은 실제 캠페인 참여 기간만 계산  


**3.1.1.** 계산 원칙  

**3.1.1.1.** 달력상 경과 기간이 아닌 실제 캠페인 참여 기간 기준  
**3.1.1.2.** 캠페인 미매칭 기간은 등급 산정에서 제외  
**3.1.1.3.** 미매칭 기간이 길어도 기존 등급 산정 기간에 영향 없음  


**3.1.2.** 참여 기간 인정 기준  

**3.1.2.1.** 캠페인 매칭 확정 후 배포 시작일부터 종료일까지  
**3.1.2.2.** 중도 포기 기간은 제외  
**3.1.2.3.** 휴업 신청 시 합당한 사유 보고 → 해당 기간 정지 처리  


**3.1.3.** 등급별 실 사용 기간 요구사항  

**3.1.3.1.** 브론즈~골드: 실 사용 기간 6개월 필요  
**3.1.3.2.** 플래티넘: 실 사용 기간 1년 필요  


**3.1.4.** 미매칭 기간과 등급 관계  

**3.1.4.1.** 등급 산정: 미매칭 기간은 산정 기간에서 제외  
**3.1.4.2.** 등급 유지: 6개월 이내 새 캠페인 매칭 시 등급 유지  
**3.1.4.3.** 등급 변경: 6개월 연속 미매칭 시 한 단계 다운  

**3.1.4.4.** 미매칭 기간 예외  
　　A. 휴업 신청 기간은 미매칭 기간에서 제외  
　　B. 휴업 복귀 후 미매칭 카운트 재시작  


**3.1.5.** 중도 포기 처리  

**3.1.5.1.** 포기 전까지 참여 기간: 등급 산정에 포함  
**3.1.5.2.** 포기 시점: 미매칭 기간 카운트 시작  


**3.1.6.** 미매칭 사전 알림  

**3.1.6.1.** 1차 알림 (3개월 경과)  
　　A. 제목: "등급 유지 안내"  
　　B. 본문: "마지막 캠페인 종료 후 3개월이 지났습니다. {현재등급} 등급 유지를 위해 새로운 캠페인에 참여가 시급합니다. (미참여 시 {다운그레이드등급}로 조정)"  
　　C. CTA 버튼 (멤버십 상태별 분기)  
　　　a. 베이직: "매칭 확률 UP" → 멤버십 구독 페이지  
　　　b. 플러스: "프리미엄으로 매칭 확률 UP" → 멤버십 업그레이드 페이지  
　　　c. 프리미엄 + 플래티넘: "매칭 시까지 등급이 유지됩니다" (CTA 없음, 정보성 안내)  
　　　d. 프리미엄 + 골드 이하: "등급 UP으로 매칭 확률 UP" → 파트너 등급 현황 페이지  

**3.1.6.2.** 최종 알림 (5개월 3주차)  
　　A. 제목: "[최종 알림] 등급 조정 예정"  
　　B. 본문: "등급 조정까지 7일 남았습니다. {현재등급} 등급과 혜택을 유지하기 위해 새 캠페인 참여가 시급합니다. (미참여 시 {다운그레이드등급}로 조정)"  
　　C. CTA 버튼 (멤버십 상태별 분기)  
　　　a. 베이직: "매칭 확률 UP" → 멤버십 구독 페이지  
　　　b. 플러스: "프리미엄으로 매칭 확률 UP" → 멤버십 업그레이드 페이지  
　　　c. 프리미엄 + 플래티넘: "매칭 시까지 등급이 유지됩니다" (CTA 없음, 정보성 안내)  
　　　d. 프리미엄 + 골드 이하: "등급 UP으로 매칭 확률 UP" → 파트너 등급 현황 페이지  

**3.1.6.3.** 알림 채널  
　　A. 푸시 알림: 필수 발송  
　　B. 앱 내 알림: 필수 (알림 센터에 저장)  
　　C. 이메일: 선택 (알림 설정에서 활성화한 경우)  


• 상세: 7-2-4. 파트너 등급 > 등급 기준·혜택 참조
