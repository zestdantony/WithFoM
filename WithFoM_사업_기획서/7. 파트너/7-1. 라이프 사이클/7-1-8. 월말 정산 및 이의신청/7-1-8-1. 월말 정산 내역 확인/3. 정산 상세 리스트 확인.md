**8.3.** 정산 상세 리스트 확인  


**8.3.0.** 목적  

**8.3.0.1.** 리스트의 역할  

　1: 정산 상세 리스트의 역할  
　　A. 위의 요약/카드에 나온 금액이 어떤 라인 아이템들의 합인지를 보여주는 상세 표  
　　B. 각 정산 항목의 상세 내역을 한눈에 확인 가능  


**8.3.1.** 상단 필터  

**8.3.1.1.** 필터 기능  

　1: 항목 타입 필터  
　　A. 컵홀더 / 멤버십 / 인센티브 / 리워드 / 기타  
　　B. 필터 선택 시 해당 항목만 표시  

　2: 기간 필터  
　　A. 선택된 월 내에서 추가로 일/주 단위 좁혀 보기 가능 (옵션)  
　　B. 상세 기간 선택 가능  

　3: 캠페인 필터  
　　A. 특정 캠페인 관련 항목만 조회  
　　B. 월별 정산 시 선택 사항  


**8.3.2.** 리스트 컬럼 구성  

**8.3.2.1.** 컬럼 정보  

　1: 기본 정보  
　　A. 기준일 또는 정산 기준 기간 (예: 2026-03, N회차 캠페인 등)  
　　B. 항목 타입 (컵홀더 / 멤버십 / 인센티브 / 리워드 / 기타)  
　　C. 상세 내용  
　　D. 예: "3월 컵홀더 정산(플러스 등급 50% 할인 반영)"  
　　E. 예: "3월 플러스 멤버십 구독료"  
　　F. 예: "플래티넘 인센티브 – 3월 소진 8,000개 기준"  

　2: 수량 및 단가  
　　A. 수량/기준 (컵홀더 개수, 멤버십 개월 수, 인센티브 대상 수량 등)  
　　B. 단가/요율 (컵홀더 단가, 인센티브 단가, 할인율 등)  

　3: 금액 및 상태  
　　A. 금액 (+ 또는 – 표시 포함)  
　　B. 정산 상태 (예정 / 확정 / 청구완료 / 입금완료)  


**8.3.3.** 페이지네이션  

**8.3.3.1.** 페이지 구성  

　1: 페이지 구성  
　　A. 1페이지당 20개 표시  
　　B. 최대 5페이지 (100개) 정도 기본 설정  
　　C. 페이지 이동 버튼 제공
