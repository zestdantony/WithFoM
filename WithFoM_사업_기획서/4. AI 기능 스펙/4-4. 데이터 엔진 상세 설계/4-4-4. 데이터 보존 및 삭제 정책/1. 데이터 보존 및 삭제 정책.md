# 1: 데이터 보존 및 삭제 정책

| 데이터 유형 | 보존 기간 | 삭제 방식 |
|---|---|---|
| QR/LOT 로그 | 3년 | 3년 경과 후 자동 삭제 |
| 행동 로그 | 3년 | 3년 경과 후 자동 삭제 |
| 거래 로그 | 5년 (세무 목적) | 5년 경과 후 자동 삭제 |
| 운영 로그 | 3년 | 3년 경과 후 자동 삭제 |
| 개인정보 | 탈퇴 후 30일 | 30일 경과 후 Hard Delete |
| 통계/집계 데이터 | 영구 보존 | 비식별화 상태로 보존 |
