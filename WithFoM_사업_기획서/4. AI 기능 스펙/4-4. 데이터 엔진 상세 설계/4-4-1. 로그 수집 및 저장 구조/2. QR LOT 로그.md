# 2: QR/LOT 로그 (스캔 추적)

소비자가 QR 코드를 스캔하는 순간부터 기록이 시작된다. 컵홀더가 "어느 매장에서, 언제, 누구에게" 노출되었는지 추적할 수 있다.

| 필드명 | 설명 | 예시 |
|---|---|---|
| scan_id | 스캔 이벤트 고유 ID | SCN_20240115_001234 |
| timestamp | 스캔 시각 (KST) | 2024-01-15 14:23:45 |
| campaign_id | 캠페인 ID | CMP_2024_0089 |
| partner_id | 파트너 매장 ID | PTN_CAFE_001 |
| lot_code | 생산 배치 번호 | LOT_2024W02_A1 |
| source_type | 유입 채널 | cup / banner / invite |
| device_fingerprint | 기기 고유 식별자 (해시) | d7f8a9b2c3e4... |
| user_agent | 브라우저/기기 정보 | Mobile Safari / iOS 17.2 |
| geo_location | GPS 좌표 (동의 시) | 37.5012, 127.0396 |
| ip_address | 접속 IP (해시 처리) | 192.168.xxx.xxx |

• 참조: WithFoM_Master_Planning_v3.10.md 4장 4-4절 4-4-1
