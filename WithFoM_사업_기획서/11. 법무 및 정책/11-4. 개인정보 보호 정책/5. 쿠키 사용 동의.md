# 5: 쿠키 사용 동의

## 개요

위드폼 웹 포털 이용 시 서비스 제공을 위한 쿠키 사용에 관한 동의 항목으로, **선택 동의 항목**입니다.

---

## 제1조 (쿠키의 정의)

쿠키는 웹사이트를 방문할 때 웹 브라우저를 통해 이용자의 컴퓨터에 저장되는 작은 텍스트 파일입니다. 쿠키는 이용자가 방문한 웹사이트의 정보를 저장하여 다음 방문 시 더 편리하게 이용할 수 있도록 도와줍니다.

---

## 제2조 (쿠키 사용 목적)

주식회사 위드폼(이하 "회사")은 다음의 목적을 위해 쿠키를 사용합니다:

| 사용 목적 | 상세 내용 |
|---|---|
| 필수 쿠키 | 로그인 상태 유지, 보안 인증, 서비스 기본 기능 제공 |
| 편의성 쿠키 | 언어 설정, 화면 크기 등 사용자 환경 설정 저장 |
| 분석 쿠키 | 서비스 이용 통계 분석, 사용자 경험 개선 |
| 마케팅 쿠키 | 맞춤형 광고 제공 (동의 시) |

---

## 제3조 (사용하는 쿠키의 종류)

### 3-1. 필수 쿠키 (동의 불필요)

| 쿠키명 | 목적 | 보유 기간 |
|---|---|---|
| 세션 쿠키 | 로그인 상태 유지 | 브라우저 종료 시 삭제 |
| 보안 쿠키 | 보안 인증 | 브라우저 종료 시 삭제 |
| 기능 쿠키 | 서비스 기본 기능 제공 | 1년 |

※ 필수 쿠키는 서비스 이용에 필수적이므로 동의 없이 사용됩니다.

### 3-2. 선택 쿠키 (동의 필요)

| 쿠키명 | 목적 | 보유 기간 |
|---|---|---|
| 분석 쿠키 | Google Analytics 등 분석 도구 | 2년 |
| 마케팅 쿠키 | 맞춤형 광고 제공 | 1년 |
| 편의성 쿠키 | 사용자 환경 설정 저장 | 1년 |

---

## 제4조 (쿠키 설정 방법)

파트너는 브라우저 설정을 통해 쿠키 저장을 거부하거나 삭제할 수 있습니다:

| 브라우저 | 설정 방법 |
|---|---|
| Chrome | 설정 > 개인정보 및 보안 > 쿠키 및 기타 사이트 데이터 |
| Safari | 환경설정 > 개인정보 보호 > 쿠키 및 웹사이트 데이터 차단 |
| Edge | 설정 > 쿠키 및 사이트 권한 > 쿠키 및 사이트 데이터 |
| Firefox | 옵션 > 개인정보 보호 > 쿠키 및 사이트 데이터 |

※ 쿠키를 거부할 경우, 일부 서비스 이용에 제한이 있을 수 있습니다.

---

## 제5조 (제3자 쿠키)

회사는 서비스 개선을 위해 다음과 같은 제3자 쿠키를 사용할 수 있습니다:

| 제공업체 | 쿠키 목적 | 보유 기간 |
|---|---|---|
| Google Analytics | 웹사이트 이용 통계 분석 | 2년 |
| (광고 플랫폼) | 맞춤형 광고 제공 | 1년 |

---

## 제6조 (쿠키 동의 관리)

### 6-1. 웹 포털에서 관리

- 웹 포털 접속 시 쿠키 동의 팝업 표시
- 설정 > 개인정보 > 쿠키 설정에서 언제든지 변경 가능

### 6-2. 동의 철회

- 쿠키 동의 철회 시 해당 쿠키 즉시 삭제
- 다만, 필수 쿠키는 서비스 이용을 위해 계속 사용됩니다.

---

## 제7조 (쿠키 사용 거부 시 영향)

| 쿠키 유형 | 거부 시 영향 |
|---|---|
| 필수 쿠키 | 서비스 이용 불가 (로그인 불가) |
| 분석 쿠키 | 서비스 이용 통계 분석 불가 (서비스 이용 가능) |
| 마케팅 쿠키 | 맞춤형 광고 제공 불가 (서비스 이용 가능) |
| 편의성 쿠키 | 환경 설정 저장 불가 (서비스 이용 가능) |

---

## 제8조 (모바일 앱과의 차이)

| 구분 | 웹 포털 | 모바일 앱 |
|---|---|---|
| 쿠키 사용 | ✅ 사용 | ❌ 사용 안 함 |
| 대체 기술 | - | 앱 내부 저장소 사용 |

※ 모바일 앱에서는 쿠키 대신 앱 내부 저장소를 사용합니다.

---

## 부칙

- 본 동의서는 2026년 1월 1일부터 시행합니다.
- 본 동의는 선택사항이며, 필수 쿠키를 제외한 선택 쿠키에 대한 동의입니다.
- 필수 쿠키는 서비스 이용을 위해 동의 없이 사용됩니다.
