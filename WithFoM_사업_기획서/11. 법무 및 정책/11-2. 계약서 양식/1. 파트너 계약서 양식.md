# 1: 파트너 계약서 양식

## 개요

위드폼 파트너 서비스 이용을 위한 계약서 양식입니다. 파트너 가입 시 매장 상세 정보 입력이 완료되면 자동으로 생성되며, 계약서 내용 확인, 서명, 다운로드 및 이메일 수신 절차를 포함합니다.

---

## 제1장 계약서 자동 생성 및 확인

### 제1조 (계약서 자동 생성)

매장 상세 정보 입력이 완료되면 자동으로 생성된 계약서를 확인합니다. 계약서에는 입력한 모든 정보가 자동으로 반영됩니다.

| 단계 | 내용 |
|---|---|
| 매장 상세 정보 입력 완료 | 7-1-1-7. 매장 상세 정보 설정 모든 항목 입력 완료 |
| 계약서 자동 생성 | 입력한 정보를 바탕으로 계약서 자동 생성 |
| 계약서 화면 표시 | 생성된 계약서가 화면에 표시됨 |

### 제2조 (계약서에 포함되는 정보)

| 정보 항목 | 출처 | 비고 |
|---|---|---|
| 대표자 성명 | 7-1-1-3. 본인 인증 → 7-1-1-5. 대표자명 | 본인 인증 정보 자동 기입 |
| 사업자등록번호 | 7-1-1-6. 사업자 인증 | |
| 사업자명 | 7-1-1-6. 사업자 인증 (국세청 API) | |
| 매장명 | 7-1-1-5. 매장 기본 정보 입력 | |
| 사업장 소재지 | 7-1-1-5. 매장 기본 정보 입력 (매장 주소) | |
| 연락처 | 7-1-1-3. 본인 인증 → 7-1-1-5. 연락처 | 본인 인증 정보 자동 기입 (수정 가능) |
| 이메일 | 7-1-1-5. 매장 기본 정보 입력 | |
| 정산 계좌 | 7-1-1-5. 매장 기본 정보 입력 | |
| 서비스 타입 | 7-1-1-7. 매장 상세 정보 설정 (리워드/오더 선택) | |
| 계약 일자 | 계약서 생성 일자 | |

### 제3조 (계약서 내용 구성)

#### 3-1. 계약서 주요 조항

| 구분 | 내용 |
|---|---|
| 제1조 | 목적 및 정의 |
| 제2조 | 계약의 체결 및 효력 |
| 제3조 | 서비스 이용 조건 |
| 제4조 | 파트너의 권리 및 의무 |
| 제5조 | 위드폼의 권리 및 의무 |
| 제6조 | 정산 및 지급 |
| 제7조 | 등급 및 혜택 |
| 제8조 | 계약 기간 및 해지 |
| 제9조 | 손해배상 및 면책 |
| 제10조 | 분쟁 해결 및 준거법 |
| 부칙 | 계약서 일자 및 서명란 |

#### 3-2. 주요 조항 요약

**제3조: 서비스 이용 조건**

| 항목 | 내용 |
|---|---|
| 컵홀더 배포 | 위드폼이 제공하는 컵홀더의 배포 및 관리 (필수) |
| 일일 소진 보고 | 영업 종료 후 일일 소진량 보고 의무 |

**제4조: 파트너의 권리 및 의무**

| 권리 | 의무 |
|---|---|
| 캠페인 매칭 및 수익 기회 | 정확한 매장 정보 제공 |
| 등급별 혜택 수령 | 일일 소진 보고 성실 이행 |
| 정산 금액 수령 | 서비스 이용약관 준수 |
| - | 부정 사용 방지 (FDS 위반 금지) |
| - | 소비자의 QR 스캔 유도 |

**제6조: 정산 및 지급**

| 항목 | 내용 |
|---|---|
| 정산 주기 | 매월 말일 기준 월말 정산 |
| 정산 금액 | 인센티브(플래티넘 등급) 지급 |
| 정산 계좌 | 파트너가 등록한 정산 계좌로 입금 |
| 정산 확정 | 정산 내역 확인 후 확정 (이의 신청 7일 이내) |

**제7조: 등급 및 혜택**

| 등급 | 혜택 |
|---|---|
| 브론즈 | 위드폼 서비스 이용 |
| 실버 | 컵홀더 무료 제공 |
| 골드 | 컵홀더 무료 제공 + 우선 매칭 + 멤버십 10% 할인 |
| 플래티넘 | 컵홀더 무료 제공 + 최우선 매칭 + 인센티브 지급 |

**제8조: 계약 기간 및 해지**

| 항목 | 내용 |
|---|---|
| 계약 기간 | 2년 |
| 자동 연장 | 2년 후 자동 연장 (별도 해지 의사 표시 없는 경우) |
| 자동 연장 사전 고지 | 자동 연장 30일 전 이메일, SMS, 앱 푸시로 고지 |
| 파트너 해지 | 30일 전 서면 통보, 미처리 주문/미정산 없어야 함 |
| 조기 해지 | 최소 2년 전 계약 해지 시 인센티브 반환 |
| 강제 해지 | 파트너의 잘못(약관 위반 등)으로 인한 해지 시 인센티브 반환 |
| 위드폼 해지 | 서비스 이용약관 중대 위반 시 즉시 해지 가능 |
| 데이터 보관 | 해지 후 30일 유예 기간, 이후 영구 삭제 |

※ 자동 연장 사전 고지 내용: 계약 만료일, 자동 연장 여부, 해지 방법

### 제4조 (계약서 확인 방법)

#### 4-1. 1단계: 계약서 화면 확인

- 매장 상세 정보 입력 완료 후
- 자동으로 생성된 계약서 화면 표시
- 계약서 전체 내용 스크롤하여 확인

#### 4-2. 2단계: 정보 정확성 확인

| 확인 항목 | 확인 방법 |
|---|---|
| 파트너 정보 | 대표자명, 사업자등록번호, 사업자명 등 정확성 확인 |
| 매장 정보 | 매장명, 주소, 연락처 등 정확성 확인 |
| 계약 조건 | 서비스 이용 조건, 정산 조건 등 확인 |

#### 4-3. 3단계: 계약서 내용 검토

- 계약서 전체 조항 읽기
- 이해가 되지 않는 부분 확인
- 필요 시 고객센터 문의

### 제5조 (정보 수정)

계약서에 표시된 정보가 잘못된 경우:

| 수정 방법 | 내용 |
|---|---|
| 이전 단계로 이동 | 해당 정보가 입력된 단계로 돌아가서 수정 |
| 정보 수정 후 재생성 | 정보 수정 후 계약서 자동 재생성 |
| 최종 확인 | 수정된 계약서 내용 재확인 |

### 제6조 (계약서 확인 체크리스트)

계약서 확인 시 아래 항목을 체크하세요.

- 대표자 성명 정확함
- 사업자등록번호 정확함 (10자리)
- 사업자명 정확함
- 매장명 정확함
- 사업장 소재지 정확함
- 연락처 정확함
- 이메일 정확함
- 정산 계좌 정보 정확함
- 서비스 타입 정확함
- 계약서 전체 내용 검토 완료

## 제2장 계약서 서명

### 제7조 (서명 절차)

#### 7-1. 1단계: 계약서 내용 최종 확인

- 계약서 전체 내용 재확인
- 모든 정보 정확성 확인
- 계약 조건 이해 확인

#### 7-2. 2단계: 서명 진행

| 단계 | 내용 |
|---|---|
| 서명 버튼 클릭 | "서명하기" 버튼 클릭 |
| 서명 방법 선택 | 직접 서명 또는 전자서명 선택 |
| 서명 완료 | 서명 완료 후 확인 |

#### 7-3. 3단계: 서명 확인

- 서명 완료 확인 팝업 표시
- 서명된 계약서 미리보기
- "확인" 버튼 클릭하여 확정

### 제8조 (서명 방법)

#### 8-1. 직접 서명

| 방법 | 설명 |
|---|---|
| 터치 서명 | 화면에 손가락으로 직접 서명 |
| 스타일러스 서명 | 스타일러스 펜으로 서명 (지원 기기) |
| 서명 확인 | 서명 후 미리보기로 확인 |

**서명 시 주의사항:**
- 명확하고 읽기 쉬운 서명
- 서명 영역 내에 서명
- 서명 후 재확인

#### 8-2. 전자서명

| 방법 | 설명 |
|---|---|
| 전자서명 인증 | 공인인증서 또는 간편인증을 통한 전자서명 |
| 서명 완료 | 전자서명 완료 후 자동으로 계약서에 반영 |

### 제9조 (서명 완료 후)

#### 9-1. 계약서 확정

| 단계 | 내용 |
|---|---|
| 서명 완료 | 서명이 계약서에 반영됨 |
| 계약서 저장 | 서명된 계약서가 시스템에 저장됨 |
| 다음 단계 진행 | 계약서 다운로드 및 이메일 수신 단계로 진행 |

#### 9-2. 서명 수정

서명이 마음에 들지 않는 경우:

| 방법 | 내용 |
|---|---|
| 서명 취소 | "다시 서명" 버튼으로 서명 취소 |
| 재서명 | 새로운 서명 진행 |
| 서명 확인 | 재서명 후 최종 확인 |

### 제10조 (서명 관련 주의사항)

| 주의사항 | 내용 |
|---|---|
| 법적 효력 | 서명 완료 시 계약서가 법적 구속력을 가짐 |
| 서명 전 확인 | 서명 전 반드시 계약서 전체 내용 확인 |
| 서명 후 수정 | 서명 완료 후 계약서 내용 수정 불가 |
| 서명 취소 | 서명 완료 전에만 서명 취소 가능 |

### 제11조 (개인사업자 vs 법인사업자)

#### 11-1. 개인사업자

| 항목 | 내용 |
|---|---|
| 서명자 | 대표자 본인 서명 |
| 서명 방법 | 직접 서명 또는 전자서명 |

#### 11-2. 법인사업자

| 항목 | 내용 |
|---|---|
| 서명자 | 대표자 또는 법인 인감 |
| 서명 방법 | 대표자 서명 또는 법인 인감 날인 (전자서명) |

### 제12조 (서명 완료 확인)

서명 완료 후 확인 사항:

- 서명이 명확하게 표시됨
- 서명이 계약서에 정상적으로 반영됨
- 서명 완료 확인 팝업 확인
- 다음 단계(계약서 다운로드) 진행 가능

## 제3장 계약서 다운로드 및 이메일 수신

### 제13조 (계약서 다운로드)

#### 13-1. 다운로드 방법

| 방법 | 설명 |
|---|---|
| 앱 내 다운로드 | 앱 내에서 직접 PDF 파일 다운로드 |
| 다운로드 버튼 | "계약서 다운로드" 버튼 클릭 |
| 파일 저장 | 기기 저장 공간에 PDF 파일 저장 |

#### 13-2. 다운로드 파일 정보

| 항목 | 내용 |
|---|---|
| 파일 형식 | PDF (Portable Document Format) |
| 파일명 | WithFoM_파트너_계약서_[매장명]_[날짜].pdf |
| 파일 크기 | 약 500KB ~ 1MB |
| 포함 내용 | 계약서 전체 내용 및 서명 |

#### 13-3. 다운로드 절차

- 서명 완료 후 "계약서 다운로드" 버튼 클릭
- 다운로드 진행 상태 확인
- 다운로드 완료 확인
- 파일 저장 위치 확인

### 제14조 (이메일 수신)

#### 14-1. 이메일 자동 발송

| 항목 | 내용 |
|---|---|
| 발송 시점 | 계약서 서명 완료 즉시 |
| 수신 주소 | 가입 시 입력한 이메일 주소 |
| 발송 내용 | 서명된 계약서 PDF 파일 첨부 |

#### 14-2. 이메일 내용

| 항목 | 내용 |
|---|---|
| 제목 | [위드폼 파트너] 파트너 계약서 |
| 본문 | 계약서 서명 완료 안내 및 계약서 첨부 |
| 첨부 파일 | 서명된 계약서 PDF 파일 |

#### 14-3. 이메일 수신 확인

| 확인 방법 | 내용 |
|---|---|
| 이메일 수신함 확인 | 등록한 이메일 주소의 수신함 확인 |
| 스팸 폴더 확인 | 스팸 폴더에 분류되었을 수 있음 |
| 이메일 재발송 | 미수신 시 "이메일 재발송" 버튼 클릭 |

### 제15조 (계약서 보관)

#### 15-1. 보관 방법

| 방법 | 설명 |
|---|---|
| 앱 내 보관 | 앱 내에서 계약서 확인 및 보관 |
| 다운로드 보관 | 기기에 다운로드하여 보관 |
| 이메일 보관 | 이메일로 받은 계약서 보관 |
| 클라우드 보관 | 클라우드 스토리지에 업로드하여 보관 (선택) |

#### 15-2. 보관 권장사항

| 항목 | 내용 |
|---|---|
| 다중 보관 | 여러 방법으로 보관 권장 |
| 백업 | 중요한 문서이므로 백업 권장 |
| 접근 가능성 | 필요 시 쉽게 접근할 수 있는 위치에 보관 |

### 제16조 (계약서 재다운로드)

#### 16-1. 재다운로드 방법

| 방법 | 설명 |
|---|---|
| 앱 내 메뉴 | 설정 > 계약서 > 계약서 다운로드 |
| 이메일 재발송 | 설정 > 계약서 > 이메일 재발송 |

#### 16-2. 재다운로드 시점

| 시점 | 내용 |
|---|---|
| 계약서 분실 시 | 계약서를 분실한 경우 |
| 백업 필요 시 | 추가 백업이 필요한 경우 |
| 이메일 미수신 시 | 이메일을 받지 못한 경우 |

### 제17조 (다운로드 및 이메일 수신 오류 해결)

#### 17-1. 다운로드 오류

| 오류 | 해결 방법 |
|---|---|
| 다운로드 실패 | 네트워크 연결 확인 후 재시도 |
| 저장 공간 부족 | 저장 공간 확보 후 재시도 |
| 파일 손상 | 재다운로드 시도 |

#### 17-2. 이메일 미수신

| 원인 | 해결 방법 |
|---|---|
| 이메일 주소 오류 | 이메일 주소 확인 및 수정 |
| 스팸 분류 | 스팸 폴더 확인 |
| 이메일 서버 오류 | 이메일 재발송 요청 |

### 제18조 (계약서 확인)

#### 18-1. 다운로드한 계약서 확인

| 확인 항목 | 내용 |
|---|---|
