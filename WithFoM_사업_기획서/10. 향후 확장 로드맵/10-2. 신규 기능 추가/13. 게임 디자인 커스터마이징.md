# 13: 게임 디자인 커스터마이징

## 개요

파트너가 자신의 브랜드에 맞게 게임 디자인을 커스터마이징할 수 있는 기능입니다.

## 현재 상태

| 항목 | 내용 |
|---|---|
| 현재 디자인 | 위드폼 기본 디자인 자동 적용 |
| 게임 유형별 | 기본 색상 및 아이콘 제공 |
| 표시 정보 | 매장명 + 게임 유형명 자동 표시 |


## 확장 기능

### 1. 커스텀 색상 설정

| 항목 | 내용 |
|---|---|
| 배경 색상 | 브랜드 컬러에 맞는 배경 설정 |
| 강조 색상 | 버튼, 당첨 효과 등 강조 색상 설정 |
| 텍스트 색상 | 가독성을 고려한 텍스트 색상 설정 |


### 2. 로고 삽입

| 항목 | 내용 |
|---|---|
| 위치 | 게임 화면 상단 또는 중앙 |
| 크기 | 최대 200x100px |
| 형식 | PNG, JPG (투명 배경 권장) |


### 3. 커스텀 효과

| 항목 | 내용 |
|---|---|
| 당첨 효과 | 파티클, 폭죽 등 커스텀 당첨 효과 |
| 사운드 | 커스텀 배경음악, 효과음 |
| 애니메이션 | 브랜드 캐릭터 애니메이션 삽입 |


## 요금 정책 (예정)

| 플랜 | 제공 범위 |
|---|---|
| 베이직 | 기본 디자인만 |
| 플러스 | 색상 커스터마이징 |
| 프리미엄 | 색상 + 로고 삽입 + 커스텀 효과 |


## 구현 우선순위

| 순위 | 기능 | 예상 시기 |
|---|---|---|
| 1 | 커스텀 색상 설정 | Phase 2 |
| 2 | 로고 삽입 | Phase 2 |
| 3 | 커스텀 효과 | Phase 3 |
