# 14: 알림톡 쿠폰 첨부 발송

## 개요

플러스/프리미엄 멤버십 파트너가 고객에게 알림톡 발송 시 보유한 쿠폰을 함께 첨부하여 발송할 수 있는 기능입니다.


## 대상 멤버십

| 플랜 | 이용 가능 여부 |
|---|---|
| 베이직 | ❌ 불가 |
| 플러스 | ✅ 가능 |
| 프리미엄 | ✅ 가능 |


## 기능 상세

### 1. 쿠폰 첨부 발송

| 항목 | 내용 |
|---|---|
| 첨부 가능 쿠폰 | 파트너가 생성한 활성화된 쿠폰 |
| 첨부 개수 | 알림톡 1건당 쿠폰 1개 첨부 |
| 쿠폰 형태 | 할인 쿠폰, 포인트 쿠폰 |
| 자동 발급 | 고객이 알림톡 수신 시 쿠폰 자동 발급 |


### 2. 알림톡 발송 유형

| 유형 | 설명 | 쿠폰 첨부 |
|---|---|---|
| 휴면 고객 케어 | 오랫동안 방문하지 않은 고객에게 재방문 유도 | ✅ 가능 |
| 생일 축하 | 고객 생일에 축하 메시지 발송 | ✅ 가능 |
| 이벤트 안내 | 매장 이벤트, 신메뉴 안내 | ✅ 가능 |
| 쿠폰 만료 임박 안내 | 보유 쿠폰 만료 전 안내 | ✅ 가능 |


### 3. 발송 월 한도

| 플랜 | 월 발송 한도 | 쿠폰 첨부 발송 한도 |
|---|---|---|
| 플러스 | 30회 | 30회 (전체 한도 공유) |
| 프리미엄 | 300회 | 300회 (전체 한도 공유) |


### 4. 쿠폰 첨부 시 알림톡 구성

```
[매장명] 고객님, 안녕하세요!

[메시지 본문]

🎁 쿠폰이 도착했어요!
[쿠폰명] - [할인 내용]
유효기간: [시작일] ~ [종료일]

[쿠폰 받기] 버튼
```


## 발송 플로우

| 단계 | 내용 |
|---|---|
| 1 | 파트너가 알림톡 발송 대상 선택 |
| 2 | 메시지 작성 |
| 3 | 첨부할 쿠폰 선택 (선택 사항) |
| 4 | 미리보기 확인 |
| 5 | 발송 |
| 6 | 고객 수신 시 쿠폰 자동 발급 |


## 고객 수신 후 동작

| 항목 | 내용 |
|---|---|
| 쿠폰 받기 클릭 | 위드폼 앱/웹으로 이동, 쿠폰 자동 발급 |
| 비회원인 경우 | 간편 회원가입 후 쿠폰 발급 |
| 이미 수신한 쿠폰 | "이미 발급된 쿠폰입니다" 안내 |
| 쿠폰 확인 | 마이페이지 > 쿠폰함에서 확인 |


## 통계 및 분석

| 항목 | 내용 |
|---|---|
| 발송 수 | 알림톡 발송 건수 |
| 쿠폰 첨부 발송 수 | 쿠폰이 포함된 발송 건수 |
| 쿠폰 수신율 | 쿠폰 받기 클릭 비율 |
| 쿠폰 사용율 | 발급된 쿠폰 중 사용 비율 |
| 재방문율 | 알림톡 수신 후 방문 비율 |


## 구현 우선순위

| 순위 | 기능 | 예상 시기 |
|---|---|---|
| 1 | 휴면 고객 케어 쿠폰 첨부 | Phase 2 |
| 2 | 이벤트 안내 쿠폰 첨부 | Phase 2 |
| 3 | 생일 축하 쿠폰 첨부 | Phase 2 |
| 4 | 발송 통계 대시보드 | Phase 3 |
