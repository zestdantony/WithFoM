# 9: AI 메뉴 이미지 생성

## 개요

메뉴명과 설명을 기반으로 AI가 자동으로 메뉴 이미지를 생성하는 기능입니다. 파트너가 직접 사진을 찍거나 디자인하지 않아도 고품질 메뉴 이미지를 사용할 수 있습니다.


## 주요 기능

### 1. AI 이미지 생성 프로세스

　1: 입력 정보 기반 생성  
　　A. 메뉴명 분석 (예: "아메리카노", "딸기 케이크")  
　　B. 메뉴 설명 분석 (예: "진한 에스프레소에 물을 더한 커피")  
　　C. 카테고리 정보 참조 (음료, 디저트 등)  

　2: 이미지 스타일 선택  
　　A. 사실적 (Realistic): 실제 사진처럼 생성  
　　B. 일러스트 (Illustration): 그림체 스타일  
　　C. 미니멀 (Minimal): 심플한 디자인  
　　D. 빈티지 (Vintage): 따뜻한 색감의 클래식 스타일  

　3: 생성 및 선택  
　　A. 3~4개 이미지 후보 생성  
　　B. 파트너가 원하는 이미지 선택  
　　C. 선택한 이미지 자동 적용  


### 2. 고급 커스터마이징 옵션

| 옵션 | 설명 |
|---|---|
| 배경색 | 원하는 배경색 선택 또는 투명 배경 |
| 그릇/컵 스타일 | 다양한 용기 스타일 선택 |
| 토핑/장식 | 추가 토핑이나 장식 요소 |
| 분위기 | 밝은/어두운/따뜻한/시원한 느낌 |


### 3. 일괄 생성 기능

　1: 다중 메뉴 선택  
　　A. 이미지가 없는 메뉴 일괄 선택  
　　B. 선택한 메뉴에 대해 AI 이미지 일괄 생성  

　2: 스타일 통일  
　　A. 모든 메뉴에 동일한 스타일 적용  
　　B. 매장 전체 메뉴의 시각적 일관성 확보  


## 기술 요구사항

### 1. AI 모델 연동
- 이미지 생성 AI API 연동 (DALL-E, Midjourney API 등)
- 프롬프트 자동 생성 로직
- 생성된 이미지 품질 검증

### 2. API 설계

| 엔드포인트 | 메서드 | 설명 |
|---|---|---|
| /api/ai/image/generate | POST | AI 이미지 생성 요청 |
| /api/ai/image/status/{job_id} | GET | 생성 상태 조회 |
| /api/ai/image/select | POST | 생성된 이미지 중 선택 |
| /api/ai/image/batch | POST | 일괄 생성 요청 |

### 3. 요청 데이터 구조

```json
{
  "menu_name": "아메리카노",
  "menu_description": "진한 에스프레소에 물을 더한 커피",
  "category": "커피",
  "style": "realistic",
  "options": {
    "background": "white",
    "container": "ceramic_cup",
    "mood": "warm"
  }
}
```


## 비즈니스 모델

| 플랜 | 제공 범위 |
|---|---|
| 베이직 | 월 5회 생성 |
| 플러스 | 월 30회 생성 |
| 프리미엄 | 무제한 생성 + 고급 커스터마이징 |

### 추가 결제 옵션
- 추가 생성권 구매: 10회/5,000원
- 프리미엄 스타일 팩: 3,000원/월


## 출시 로드맵

| Phase | 기능 | 예상 출시일 |
|---|---|---|
| Phase 0 (현재) | 기능 없음 | - |
| Phase 1 | 기본 AI 이미지 생성 (단일 메뉴) | 런칭 +12개월 |
| Phase 2 | 스타일 선택 및 커스터마이징 | 런칭 +15개월 |
| Phase 3 | 일괄 생성 및 프리미엄 기능 | 런칭 +18개월 |


## 기대 효과

1. **진입 장벽 최소화**: 사진 촬영/편집 기술 불필요
2. **시간 절약**: 수동 이미지 제작 대비 90% 이상 시간 절감
3. **품질 향상**: 일관된 고품질 이미지
4. **차별화**: 경쟁사 대비 혁신적 기능으로 파트너 유치
5. **매출 기여**: 프리미엄 기능으로 추가 수익 창출

- 상업적 사용 가능
- 타 플랫폼에서도 사용 가능
