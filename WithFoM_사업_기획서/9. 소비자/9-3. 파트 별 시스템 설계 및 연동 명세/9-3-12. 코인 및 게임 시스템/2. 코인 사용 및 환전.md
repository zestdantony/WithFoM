# 2: 코인 사용 및 환전


## 2.1: 위드 코인 사용

**사용처**:
- 위드폼 내 모든 결제에서 현금처럼 사용 가능
- 파트너 매장 주문 시 결제 금액에서 차감
- 1 위드 코인 = 100원


## 2.2: 폼 코인 사용

**사용처**:
- 게임 이용 시 소모
- 파트너 게임 1회 = 폼 코인 1개
- 광고주 게임 = 광고주 설정에 따라 상이 (상품 금액별 차등)

**파트너 전용 폼 코인**:
- 발행한 파트너 매장의 게임에서만 사용 가능
- 타 매장 게임에서는 사용 불가
- 환전 불가


## 2.3: 폼 코인 → 위드 코인 환전

| 항목 | 내용 |
|---|---|
| 환전 비율 | 폼 코인 10개 → 위드 코인 1개 |
| 최소 환전 단위 | 폼 코인 10개 이상 |
| 환전 가능 대상 | 폼 코인 (일반)만 가능 |
| 환전 불가 대상 | 파트너 전용 폼 코인 |
| 역방향 환전 | 위드 코인 → 폼 코인 **불가** |
| 유효기간 갱신 | 환전 완료 시점부터 1년 재시작 |

**환전 시 안내 문구**:
1. "폼 코인 [N]개를 위드 코인 [N/10]개로 환전합니다."
2. "환전된 위드 코인의 유효기간은 환전일로부터 1년입니다."
3. "환전 후에는 다시 폼 코인으로 변경할 수 없습니다."
4. "정말 환전하시겠습니까?" [환전] [취소]


## 2.4: 코인 현황 확인

**마이페이지 > 코인 관리**:
- 보유 위드 코인 수량 및 원화 가치
- 보유 폼 코인 수량 (일반 + 파트너별 분리 표시)
- 만료 예정 코인 알림
- 환전 버튼
- 사용/적립 내역 조회


• 참조: 위드폼 코인 시스템 정책 정의서
