# 3: 게임 이용 방법


## 3.1: 게임 진입 경로별 이용 조건

| 진입 경로 | 이용 조건 | 비고 |
|---|---|---|
| **컵홀더 QR 스캔** | **무료** 이용 | 광고주 캠페인 게임 참여 (코인 소모 없음) |
| **웹/앱 내에서 진입** | **코인** 소모 | 파트너 자체 게임만 해당 (폼 코인 1개) |


## 3.2: 컵홀더 QR 스캔 게임

**무제한 무료 이용**:
- 컵홀더에 인쇄된 QR 코드 스캔 시 게임 진입
- 코인 소모 없이 무료로 게임 참여
- 캠페인 기간 동안 무제한 참여 가능
- 100% 당첨 (꽝 없음)
- 1~6등 광고주 리워드 + 7등 파트너 리워드 구성


## 3.3: 앱/웹 내 게임

**파트너 게임**:
- 폼 코인 1개 소모
- 파트너가 설정한 리워드 당첨
- 파트너 전용 폼 코인도 사용 가능 (해당 매장 게임에 한함)
- 꽝 포함/미포함은 파트너 설정에 따라 결정

**광고주 캠페인 게임**:
- 컵홀더 QR 스캔으로 무료 참여 (코인 소모 없음)
- 광고주 부담으로 운영되므로 소비자 비용 없음
- 1~6등 광고주 리워드, 7등 파트너 리워드 (파트너 참여 시)


## 3.4: 파트너 제공 무료 게임

**무료 이용 조건**:
- 파트너가 코인을 적립하여 제공하는 무료 게임
- 파트너가 설정한 대상 고객만 이용 가능 (신규/단골/VIP/휴면/전체)
- **1인당 하루 1회만** 무료 이용 가능
- 1회 초과 시 고객 본인의 폼 코인 사용

**무료 게임 표시**:
- 무료 이용 가능 시: "[무료] 게임하기" 버튼
- 이미 이용한 경우: "[코인 1개] 게임하기" 버튼


## 3.5: 게임 유형

| 게임 유형 | 슬롯 수 | 설명 |
|---|---|---|
| **룰렛** | 6칸 | 원형 룰렛 회전 후 멈춘 칸의 리워드 당첨 |
| **뽑기 (가챠)** | 12캡슐 | 회전 원통에서 캡슐 사출 → 번호 매칭 리워드 당첨 |
| **스크래치** | 16칸 (4×4) | 터치/드래그로 긁기 → 같은 번호 3개 일치 시 당첨 |
| **사다리** | 6줄 | 시작점 선택 → 자동 이동 → 도착점 리워드 당첨 |


## 3.6: 당첨 상품 수령

**상품 종류**:
- 파트너 쿠폰 (할인, 사이즈업, 추가 증정)
- 포인트 쿠폰
- 스탬프 추가 적립
- 파트너 전용 폼 코인
- 광고주 브랜드 쿠폰

**수령 방법**:
- 당첨 즉시 마이페이지 > 쿠폰함에 저장
- 사용 시 해당 매장에서 쿠폰 제시


## 3.7: 게임 일시정지 상태

**파트너 게임 일시정지 시 소비자 표시**:
- 모든 리워드가 소진되면 파트너 게임이 자동 일시정지
- 게임 배너에 "준비 중" 오버레이 표시
- 게임 진입 버튼 비활성화

**진입 시도 시 안내**:

| 진입 방식 | 안내 내용 |
|---|---|
| **코인 사용 진입 시도** | "현재 리워드를 준비 중이에요. 코인은 차감되지 않습니다." |
| **무료 게임 기회 보유** | 무료 기회 유지(소멸 없음) + "게임 재개 후 사용 가능합니다" |

**재개 시**:
- "준비 중" 오버레이 해제만 처리
- 별도 메시지·푸시 알림 없음

**광고주 캠페인 게임**:
- 파트너 게임 일시정지와 무관하게 정상 운영
