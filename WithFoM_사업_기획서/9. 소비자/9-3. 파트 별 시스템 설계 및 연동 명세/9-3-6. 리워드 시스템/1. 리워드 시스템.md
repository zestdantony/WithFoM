# 1: 리워드 시스템

**리워드 종류**  

| 종류 | 설명 |  
|---|---|  
| 광고주 리워드(1 ~ 6등) | 제품 할인 쿠폰, 무료 증정, E-Com 포인트, 샘플 키트 |  
| 파트너 리워드(7th Slot) | 매장 전용 혜택 (사이즈업, 디저트 증정 등) |  
| 스탬프 | 특정 금액 이상 결제 시 적립, 완성 시 보상 쿠폰 발급 |  

**파트너 리워드 사용 플로우**  

| 단계 | 내용 |  
|---|---|  
| 1 | 소비자 [사용하기] → 1회용 사용 토큰 생성 |  
| 2 | 파트너 대시보드에 승인 요청 전송 |  
| 3 | 파트너 "승인" 또는 1분 후 자동 승인 |  
| 4 | 잘못 승인 시 파트너 [복구] 가능 |  

**리워드 사용 제한 (소비자 기타 정책)**  

| 항목 | 내용 |  
|---|---|  
| 참여 제한 | 기기 기준 1일 10회 |  
| 친구 초대 | 초대한 사람 1회 재도전, 초대받은 친구 1회 기회 |  
| 쿠폰/리워드 제한 | 1회 사용, 중복 사용 금지, 1회 3,000원 / 1일 5,000원 한도 (광고주 설정값) |
