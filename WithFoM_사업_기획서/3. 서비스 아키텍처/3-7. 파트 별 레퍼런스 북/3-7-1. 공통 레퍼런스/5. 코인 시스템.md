# 5: 코인 시스템


## 5.1: 코인 종류 및 가치 체계

| 코인 종류 | 가치 | 환산 |
|---|---|---|
| **위드 코인** | 1개 = 100원 | 현금과 동일하게 결제 시 사용 |
| **폼 코인** | 1개 = 10원 | 10개 = 위드 코인 1개 = 100원 |


## 5.2: 코인별 용도

| 코인 종류 | 용도 | 비고 |
|---|---|---|
| **위드 코인** | 결제 시 현금처럼 사용 | 위드폼 내 모든 결제에 사용 가능 |
| **폼 코인** | 게임 이용 시 사용 | 위드 코인으로 환전 가능 |


## 5.3: 폼 코인 유형

| 유형 | 발행 주체 | 사용 범위 | 환전 가능 여부 |
|---|---|---|---|
| **폼 코인 (일반)** | 위드폼 | 위드폼 내 어디서든 사용 | 위드 코인으로 환전 가능 |
| **파트너 전용 폼 코인** | 파트너 | 발행한 파트너 스토어에서만 사용 | 환전 불가 |


## 5.4: 환전 규칙

| 항목 | 내용 |
|---|---|
| 환전 방향 | 폼 코인 → 위드 코인 (일방향만 가능) |
| 최소 환전 단위 | 폼 코인 10개 이상 (= 위드 코인 1개) |
| 환전 비율 | 폼 코인 10개 = 위드 코인 1개 |
| 유효기간 시작 | 환전 완료 시점부터 1년 |
| 환전 불가 | 위드 코인 → 폼 코인 환전 불가 |
| 파트너 전용 폼 코인 | 환전 불가 (해당 매장에서만 사용) |

**환전 시 안내 문구**:
- "폼 코인 [N]개가 위드 코인 [N/10]개로 환전됩니다."
- "환전된 위드 코인의 유효기간은 환전일로부터 1년입니다."
- "환전 후에는 다시 폼 코인으로 변경할 수 없습니다."


## 5.5: 유효기간

| 코인 종류 | 유효기간 | 기산점 |
|---|---|---|
| 위드 코인 | 1년 | 적립일 또는 환전일 |
| 폼 코인 (일반) | 1년 | 적립일 |
| 파트너 전용 폼 코인 | 1년 | 적립일 |


## 5.6: 소비자 폼 코인 적립 조건

| 적립 조건 | 적립량 | 코인 유형 | 비고 |
|---|---|---|---|
| 실 결제금액 1만원당 | 1개 | 폼 코인 | 누적 금액 기준 |
| 단독 컵홀더 QR 회원가입 | 1개 | 폼 코인 | 최초 1회 |
| 콜라보 컵홀더 QR 회원가입 | 1개 | 파트너 전용 폼 코인 | 최초 1회 |
| 파트너 매장 비치 QR 회원가입 | 1개 | 파트너 전용 폼 코인 | 최초 1회 |
| 그 외 경로 회원가입 | 1개 | 폼 코인 | 최초 1회 |
| 파트너 스토어 단골 지정 | 1개 | 파트너 전용 폼 코인 | 매장당 최초 1회 |
| 친구 초대 (초대자) | 1개 | 폼 코인 | 초대 성공 시 |
| 친구 초대 (피초대자) | 1개 | 폼 코인 | 가입 완료 시 |


## 5.7: 남용 방지 정책

| 항목 | 정책 |
|---|---|
| 회원 탈퇴 후 재가입 | 회원가입 적립 불가 |
| 단골 지정 취소 후 재등록 | 단골 지정 적립 불가 |


## 5.8: 게임 이용 방식

| 진입 경로 | 이용 조건 | 비고 |
|---|---|---|
| 컵홀더 QR 스캔 | **무제한** 무료 이용 | 광고주/파트너 캠페인 |
| 웹/앱 내에서 진입 | **코인**으로 이용 | 폼 코인 소모 |


## 5.9: 게임 1회당 폼 코인 소모량

| 게임 유형 | 소모량 | 비고 |
|---|---|---|
| 파트너 게임 | 폼 코인 1개 | 고정 |
| 광고주 게임 | 광고주 설정에 따라 상이 | 상품 금액별 차등 설정 |


## 5.10: 파트너 폼 코인 구매

| 항목 | 내용 |
|---|---|
| 구매 단가 | 폼 코인 1개당 10원 |
| 구매 단위 | 1,000개 단위 (= 10,000원) |
| 용도 | 파트너 스토어 내 무료 게임 제공에 사용 |
