# 1: 파트너 KPI (5종)

**적용 파트**: 7-2 파트너 화면 아키텍처 (대시보드), 7-3 파트너 시스템 설계

| 번호 | KPI 명 | 정의 |
|---|---|---|
| 1 | 오늘 방문·참여 현황 | 오늘 QR 스캔 수, 게임/설문 완료 수, 리워드 사용 건수 |
| 2 | 오늘 매출 및 평균 객단가 | 오늘 주문 건수, 총 결제 금액, 1인당 평균 결제 금액 |
| 3 | 오늘 컵홀더 소진량 vs 예상 소진량 | 오늘 입력 소진량과 일일 평균 소진량 비교 |
| 4 | 최근 30일 재방문율 | 최소 2회 이상 방문 고객 비율 |
| 5 | 리워드 사용률 | 발행된 매장 전용 쿠폰/포인트 중 실제 사용 비율 |
