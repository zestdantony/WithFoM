# 1: 광고주 라이프 사이클

| 단계 | 내용 |
|---|---|
| 회원가입 | 회사 정보, 담당자, 정산 계좌 입력 |
| 캠페인 생성 | 10단계 위자드로 캠페인 설계 (보증금 10% 선결제 후 AI 활성화) |
| 승인 대기 | 관리자 검수 후 승인/수정 요청/반려 |
| 캠페인 운영 | 실시간 KPI 모니터링, E-Com 주문·배송 관리 |
| 정산 | 월말 정산 내역 확인, 이의신청 가능 |
| 탈퇴/휴면 | 진행 중 캠페인·미정산 없을 시 탈퇴 가능, 데이터 보존 정책 적용 |
