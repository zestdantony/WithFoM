# 파트너·광고주 계좌이체 본인 확인 방법 제안

## 📋 배경

파트너와 광고주는 **개인사업자 또는 법인사업자**이며, 이미 **사업자 인증을 완료한 상태**입니다.

**이미 확보한 정보:**
- ✅ 사업자등록번호 (국세청 API 인증 완료)
- ✅ 사업자명
- ✅ 대표자명
- ✅ 사업장 주소

**계좌이체 결제 시 필요한 본인 확인:**
- 개인사업자: 대표자 본인 확인
- 법인사업자: 법인 계좌 또는 대표자 계좌 확인

---

## 🎯 제안: 사업자 유형별 본인 확인 방법

### **방법 1: 계좌 인증(1원 입금) 방식 (권장)**

#### 개인사업자

| 단계 | 내용 | 비고 |
|---|---|---|
| 1. 계좌 정보 입력 | 은행명, 계좌번호, 예금주명 입력 | 예금주명 = 대표자명 (이미 인증 완료) |
| 2. 계좌 인증 | 1원 입금 인증 | 생년월일 불필요 |
| 3. 본인 확인 완료 | 예금주명이 대표자명과 일치 확인 | 사업자 인증 시 확인된 대표자명과 비교 |

**장점:**
- ✅ 생년월일 입력 불필요 (개인정보 수집 최소화)
- ✅ 이미 인증된 대표자명과 예금주명 비교로 간단한 본인 확인
- ✅ 사용자 편의성 향상
- ✅ 전자금융거래법 본인 확인 요건 충족

**본인 확인 로직:**
```
1. 사용자가 입력한 예금주명
2. 사업자 인증 시 확인된 대표자명
3. 두 값 일치 여부 확인
4. 일치 + 1원 입금 인증 완료 = 본인 확인 완료
```

---

#### 법인사업자

| 단계 | 내용 | 비고 |
|---|---|---|
| 1. 사업자 유형 선택 | 개인사업자 / 법인사업자 선택 | 사업자등록번호로 자동 판별 가능 |
| 2. 계좌 정보 입력 | 은행명, 계좌번호, 예금주명 입력 | 법인 계좌 또는 대표자 계좌 |
| 3. 계좌 인증 | 1원 입금 인증 | 생년월일 불필요 |
| 4. 본인 확인 완료 | 예금주명 확인 | 법인명 또는 대표자명과 일치 확인 |

**법인 계좌인 경우:**
- 예금주명 = 법인명 (사업자 인증 시 확인된 사업자명)
- 법인 계좌 인증 완료 = 본인 확인 완료

**대표자 계좌인 경우:**
- 예금주명 = 대표자명 (사업자 인증 시 확인된 대표자명)
- 대표자 계좌 인증 완료 = 본인 확인 완료

**장점:**
- ✅ 생년월일 입력 불필요
- ✅ 법인 계좌 또는 대표자 계좌 모두 사용 가능
- ✅ 사업자 인증 정보 활용으로 추가 입력 최소화

---

### **방법 2: 생년월일 + 계좌 인증 조합 (보안 강화 옵션)**

**필요한 경우에만 사용 (고액 결제, 특정 PG사 정책 등)**

| 사업자 유형 | 입력 항목 | 본인 확인 방법 |
|---|---|---|
| 개인사업자 | 생년월일 + 계좌 정보 | 생년월일 + 1원 입금 인증 |
| 법인사업자 | 생년월일 (대표자) + 계좌 정보 | 대표자 생년월일 + 1원 입금 인증 |

**단점:**
- ❌ 개인정보 수집 증가
- ❌ 입력 단계 증가로 전환율 저하 가능
- ❌ 법인사업자의 경우 대표자 생년월일 수집 필요성 의문

**권장:** 방법 1 (계좌 인증만) 우선 사용, 필요 시 방법 2 적용

## 📊 사업자 유형별 본인 확인 방법 비교

| 방법 | 개인사업자 | 법인사업자 | 생년월일 필요 | 권장도 |
|---|---|---|---|---|
| **계좌 인증만** | ✅ 예금주명 = 대표자명 확인 | ✅ 예금주명 = 법인명/대표자명 확인 | ❌ 불필요 | ⭐⭐⭐⭐⭐ |
| **생년월일 + 계좌 인증** | ✅ 생년월일 + 예금주명 확인 | ✅ 대표자 생년월일 + 예금주명 확인 | ✅ 필요 | ⭐⭐⭐ |

## 🔧 구현 제안

### **1단계: 사업자 유형 자동 판별**

```
사업자등록번호 → 국세청 API → 사업자 유형 확인
- 개인사업자: 사업자등록번호 1자리가 1, 2, 3
- 법인사업자: 사업자등록번호 1자리가 4, 5, 6, 7, 8, 9
```

### **2단계: 계좌 정보 입력 화면**

**개인사업자:**
```
입력 항목:
- 은행명 (드롭다운)
- 계좌번호
- 예금주명 (자동 입력: 대표자명, 수정 가능)

안내 문구:
"입력하신 예금주명이 대표자명과 일치해야 합니다."
```

**법인사업자:**
```
입력 항목:
- 계좌 유형 선택 (법인 계좌 / 대표자 계좌)
- 은행명 (드롭다운)
- 계좌번호
- 예금주명 (자동 입력: 법인명 또는 대표자명, 수정 가능)

안내 문구:
"법인 계좌: 법인명과 일치해야 합니다."
"대표자 계좌: 대표자명과 일치해야 합니다."
```

### **3단계: 계좌 인증 (1원 입금)**

```
1. "인증하기" 버튼 클릭
2. 1원 입금 안내 (은행명, 계좌번호, 입금자명 표시)
3. 사용자가 1원 입금
4. 입금 확인 (PG사 또는 은행 API 연동)
5. 예금주명 확인
   - 개인사업자: 예금주명 = 대표자명 일치 확인
   - 법인사업자: 예금주명 = 법인명 또는 대표자명 일치 확인
6. 인증 완료
```

## ⚖️ 법적 요구사항 충족

### 전자금융거래법 본인 확인 의무

| 요구사항 | 충족 방법 |
|---|---|
| 계좌 등록 시 본인 확인 | ✅ 1원 입금 인증 |
| 예금주 본인 확인 | ✅ 예금주명 = 사업자 인증 정보 일치 확인 |
| 부정 계좌 등록 방지 | ✅ 사업자 인증 정보와 예금주명 비교 |

**결론:** 생년월일 없이도 법적 요구사항 충족 가능

---

## 🎯 최종 권장 사항

### **권장 방법: 계좌 인증(1원 입금)만 사용**

**이유:**
1. ✅ **이미 사업자 인증 완료** → 대표자명/법인명 확인 완료
2. ✅ **예금주명 비교** → 사업자 인증 정보와 일치 여부 확인으로 본인 확인 가능
3. ✅ **생년월일 불필요** → 개인정보 수집 최소화
4. ✅ **사용자 편의성** → 입력 단계 감소로 전환율 향상
5. ✅ **법적 요구사항 충족** → 전자금융거래법 본인 확인 의무 충족

### **예외 적용: 생년월일 필요 시**

다음의 경우에만 생년월일 추가 요구:
- PG사 정책상 생년월일 필수인 경우
- 고액 결제(예: 100만원 이상) 시 보안 강화 필요
- 특정 금융기관 정책 요구
