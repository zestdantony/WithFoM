# 파트너 독립 게임 기능 추가 기획 변경 보고서

## 1. 변경 사항 요약

### 1-1. 핵심 변경 내용
- **기존**: 파트너 리워드는 광고주 게임의 7등 슬롯에만 포함
- **변경**: 파트너도 독립적인 게임을 생성하여 리워드 제공 가능
- **유지**: 광고주 게임에 파트너 리워드가 들어가는 혼합형 게임은 그대로 유지

### 1-2. 추가 기획 목적
컵홀더가 매칭되지 않은 파트너, 매칭을 기다리는 파트너, 테이크아웃이 아닌 매장을 이용하는 소비자들에게도 리워드를 제공하기 위함

### 1-3. 명명 규칙 정의
- **파트너 매장 / 매장**: 오프라인 매장을 지칭
- **파트너 스토어**: 온라인상의 파트너 매장을 지칭 (위드폼 웹/앱 내에서의 파트너 매장)
- **단골 매장 → 단골 스토어**: 모든 문서에서 "단골 매장"을 "단골 스토어"로 변경

---

## 2. 상세 변경 사항

### 2-1. 파트너 포털 UI 변경

#### 2-1-1. 리워드•스탬프 카테고리 > 설정 하위 구조 변경
**기존 구조:**
```
3: 설정
  A. 리워드
  B. 스탬프
```

**변경 후 구조:**
```
3: 설정
  1: 게임
  2: 리워드
  3: 스탬프
```

#### 2-1-2. 게임 설정 항목 추가
- **위치**: `7-2-7-3. 설정` > `1: 게임`
- **기능**:
  - 게임 유형 선택: 룰렛, 뽑기, 사다리 등
  - 게임 유형 선택 후 리워드 삽입 버튼 생성
  - 리워드 삽입 버튼을 통해 게임과 리워드 연동
  - 리워드 설정에서 설정한 리워드가 자동으로 게임에 적용됨

#### 2-1-3. 리워드 활성화와 게임 활성화 연동
- **리워드 활성화 시**: 게임도 자동으로 활성화됨 (연동)
- **리워드 비활성화 시**: 게임도 자동으로 비활성화됨 (연동)

---

### 2-2. 소비자 게임 참여 방식

#### 2-2-1. 파트너 스토어 게임 참여 조건
**게임 참여 기회 획득 조건:**
1. **단골 스토어 첫 등록**: 소비자가 위드폼 웹/앱에서 파트너 스토어를 단골 스토어로 등록하면 즉시 게임 1회 가능
2. **친구 초대**: 
   - 초대한 친구: 어느 파트너 스토어든 게임 기회 획득
   - 초대받은 친구: 어느 파트너 스토어든 게임 기회 획득
3. **스탬프 적립**: 파트너 스토어에서 스탬프 1개를 얻으면 게임 기회 획득

#### 2-2-2. 게임 참여 제한 정책
- **광고주 QR 게임**: 무제한 참여 가능 (기존 유지)
- **파트너 스토어 게임**: 횟수 제한, 조건부 기회 (신규)
- **게임 기회 획득 조건 추가 가능성**: 향후 확장 가능 (현재 미정)

---

### 2-3. 리워드 적용 방식

#### 2-3-1. 리워드 공유 원칙
- **리워드 설정에서 설정한 리워드**는 **양쪽 게임에서 동일하게 적용**
  - 광고주 게임 (7등 슬롯)
  - 파트너 스토어 게임

#### 2-3-2. 게임별 리워드 구성
- **광고주 게임**: 1~6등 광고주 리워드 + 7등 파트너 리워드 (기존 유지)
- **파트너 스토어 게임**: 파트너가 설정한 리워드만 사용 (신규)

#### 2-3-3. 리워드 당첨 확률
- **파트너 스토어 게임의 리워드 당첨 확률**: 광고주 게임과 동일하게 적용

## 5. 세부 구현 사항

### 5-1. 파트너 포털 - 게임 설정 UI

#### 5-1-1. 게임 유형 선택
- 룰렛
- 뽑기
- 사다리
- (추가 게임 유형 확장 가능)

#### 5-1-2. 게임 생성 프로세스
1. 게임 유형 선택
2. **리워드 삽입 버튼 생성** (게임 유형 선택 후 자동 생성)
3. 리워드 삽입 버튼 클릭하여 리워드 설정에서 설정한 리워드와 게임 연동
4. 리워드 활성화 시 게임도 자동 활성화 (연동)
5. 소비자 게임 참여 가능

#### 5-1-3. 게임 활성화 조건
- 리워드가 활성화되어 있어야 함 (리워드 활성화 시 게임도 자동 활성화)
- 게임 유형이 선택되어 있어야 함
- 리워드가 최소 1개 이상 설정되어 있어야 함

---

### 5-2. 소비자 - 파트너 스토어 게임 참여

#### 5-2-1. 게임 기회 획득 조건 상세
1. **단골 스토어 첫 등록**
   - 소비자가 위드폼 웹/앱에서 파트너 스토어를 단골 스토어로 등록
   - 등록 즉시 게임 1회 기회 획득

2. **친구 초대**
   - **초대한 친구**: 어느 파트너 스토어든 게임 기회 획득
   - **초대받은 친구**: 어느 파트너 스토어든 게임 기회 획득
   - (기존 친구 초대 보상 규칙과 연동)

3. **스탬프 적립**
   - 파트너 스토어에서 스탬프 1개를 적립하면 게임 기회 획득
   - 스탬프 적립 조건: 결제 금액 충족 + 단골 스토어 지정

4. **향후 확장 가능성** (현재 미정)
   - 게임 기회 획득 조건 추가 가능

#### 5-2-2. 게임 참여 제한
- **광고주 QR 게임**: 무제한 (기존 유지)
- **파트너 스토어 게임**: 조건부 기회 (단골 스토어 등록, 친구 추천, 스탬프 적립 시에만 가능)

---

### 5-3. 리워드 공유 메커니즘

#### 5-3-1. 리워드 설정과 게임 연동
- 파트너가 `2: 리워드`에서 설정한 리워드는 자동으로 다음에 적용:
  1. 광고주 게임의 7등 슬롯 (기존)
  2. 파트너 스토어 게임 (신규)

#### 5-3-2. 게임별 리워드 구성
- **광고주 게임**: 1~6등 (광고주 리워드) + 7등 (파트너 리워드)
- **파트너 스토어 게임**: 파트너 리워드만 사용 (광고주 리워드 없음)

#### 5-3-3. 리워드 당첨 확률
- **파트너 스토어 게임의 리워드 당첨 확률**: 광고주 게임과 동일하게 적용
