# 3: 리워드 사용 취소 방법  

　A. 복구 정책  
　　　a. 복구 가능 주체  
　　　　aa. 파트너만 복구 가능합니다. (매장 실수 가능성 고려)  
　　　b. 복구 시 작동  
　　　　aa. 파트너 측 "사용 완료 → 복구" 버튼을 클릭합니다.  
　　　　bb. 소비자 계정에 쿠폰/포인트가 즉시 복원됩니다.  
　　　c. 이상 감지  
　　　　aa. 관리자 및 파트너 대표 계정에 이상 감지 알림이 발송됩니다.  
　　　　bb. 동일 직원이 반복 시 위험 레벨이 상승합니다.  
　B. 취소 불가 정책  
　　　a. 소비자는 사용 완료 후 취소 불가합니다.  
　　　b. 소비자가 "사용하기" 버튼을 클릭한 후에는 취소할 수 없습니다.
