# 4: 플러스 매장관리 화면(벤치마크 + 단골·재방문 개선 실행 중심)  

　A. 상단 KPI 카드(6개 권장)  
　　a. 방문  
　　b. 재방문  
　　c. 리워드 발행  
　　d. 리워드 사용  
　　e. 스탬프 적립  
　　f. 휴면 고객 수(30일 이상 미방문)  
　B. 현황 탭(보드)  
　　a. 내 매장 추이 차트 보드  
　　aa. 지표 선택(방문/재방문/리워드 발행/리워드 사용/스탬프 적립)  
　　bb. 단위 선택(일별/주별/월별)  
　　cc. 날짜 클릭 시 상세 패널 노출(가능하면 실행 이력 마커 표시)  
　　b. 고객 리스트 바로가기 보드  
　　aa. 기간 별 랭킹 조회 연결  
　　bb. 정렬 프리셋 버튼 노출  
　　• 가입순(기본), 결제순, 방문순, 스탬프 사용순, 쿠폰 사용순, 포인트 사용순  
　　cc. 날짜 검색 프리셋 노출  
　　• 일, 주, 월, 년, 기간  
　C. 벤치마크 탭(핵심: 비교군 설정/검색/저장)  
　　a. 벤치마크 상단 컨트롤 영역(고정 노출)  
　　aa. 현재 적용중인 비교군(벤치마크 그룹) 이름을 칩(태그) 형태로 고정 노출  
　　bb. "비교군 변경" 버튼 노출(설정 드로어/모달 오픈)  
　　cc. "비교군 저장/즐겨찾기/기본값" 버튼 노출(권한 정책 적용)  
　　dd. 비교군 품질 정보 노출(작게)  
　　• 비교군 매장 수 N  
　　• 최근 30일 데이터 커버리지 %  
　　• 마지막 업데이트 일시  
　　• 표본 부족 경고 배지(최소 기준 미달)  
　　b. 벤치마크 결과 보드(기본 지표 5개)  
　　aa. 지표 카드 5개 고정(방문/재방문/리워드 발행/리워드 사용/스탬프 적립)  
　　bb. 지표 카드 공통 구성  
　　• 내 매장 값  
　　• 비교군 평균 값  
　　• "내 매장/비교군 평균 = XX%" 메인 수치  
　　• 상태 라벨(좋음/보통/개선 필요)  
　　• "원인 힌트" 버튼  
　　c. 지표 상세 비교(선택 토글)  
　　aa. 기간별 비교 라인(내 매장 vs 비교군 평균)  
　　bb. 급변 구간 마커(이상치)  
　　d. 비교군 설정 드로어/모달(단계형)  
　　aa. 1단계: 비교 방식 선택 탭  
　　• 반경 기반(지도)  
　　• 타지역 동일 상권(상권 유형/조건 기반)  
　　• 유사 매장 조건 기반(프로필 필터)  
　　• 사용자 지정 그룹(직접 선택/즐겨찾기)  
　　bb. 2단계: 범위 설정(비교 방식별 노출)  
　　• 반경 기반: 지도 + 반경 선택(기본 500m, 옵션 300m/1km/2km 등)  
　　• 타지역 동일 상권: 지역 범위 선택(전국/시도/도시/구/동) + 검색 입력창(자동완성)  
　　• 유사 매장 조건 기반: 권역 선택(전국 또는 특정 권역)  
　　cc. 3단계: 필터 설정(선택, 제공 권장)  
　　• 프랜차이즈 제외 토글  
　　• 신규 오픈 매장 제외(예: 30일 미만 제외)  
　　• 데이터 보유 기준(예: 최근 30일 중 70% 이상 데이터 보유)  
　　• 업종(카페/베이커리/디저트 등)  
　　dd. 4단계: 비교군 미리보기(적용 전 품질 확인)  
　　• 비교군 매장 수 N  
　　• 데이터 커버리지 %  
　　• 표본 부족 경고 및 원인 안내  
　　• 조건 완화 제안 버튼(반경 확장/필터 완화/범위 확대)  
　　ee. 5단계: 적용/저장  
　　• "적용" 버튼  
　　• "비교군 저장(이름 입력)" 버튼  
　　• "기본 비교군으로 설정" 토글  
　　• 권한 정책(권장)  
　　• 대표: 저장/기본값 지정 가능  
　　• 직원: 임시 적용 가능(저장/기본값 제한) 또는 권한 옵션 제공  
　　e. 반경 기반(지도) 설정 상세  
　　aa. 매장 핀 고정 + 반경 원 시각화  
　　bb. 반경 변경 시 표본/커버리지 실시간 갱신  
　　f. 타지역 동일 상권 설정 상세(상권 유형/조건 기반)  
　　aa. 상권 유형 템플릿 선택 영역 노출(초기 5~7개)  
　　• 대학가 상권  
　　• 오피스 상권  
　　• 주거 상권  
　　• 관광/유동 상권  
　　• 역세권/환승 중심 상권  
　　• 복합 상권(상업+주거 혼합)  
　　bb. 템플릿 카드 공통 구성  
　　• 템플릿명  
　　• 설명 1~2줄  
　　• 추천 필터(기본값 자동 체크)  
　　• 주의 문구(표본 부족/해석 유의)  
　　cc. 지역/상권 검색 입력창(자동완성)  
　　• 검색 대상 카테고리 구분 노출(지역/역·교통/상권 키워드/추천 상권)  
　　• 검색 결과 선택 시 후보 상권 리스트업(표본/커버리지 요약 포함)  
　　• 후보 상권 1개 또는 복수 선택 가능(복수 선택 시 비교군 확장)  
　　dd. 표본 우선 추천 토글(표본 부족 자동 회피 모드)  
　　g. 유사 매장 조건 기반 설정 상세(프로필 필터)  
　　aa. 기본 필터(권장)  
　　• 업종  
　　• 가격대(평균 결제금액 구간)  
　　• 좌석 규모(구간)  
　　• 영업 시간대(오전형/야간형/종일형)  
　　• 주말 비중(주말형/평일형)  
　　bb. 필터 증가 시 표본 감소 경고 실시간 노출  
　　cc. 추천 완화 버튼(필터 자동 완화)  
　　h. 사용자 지정 그룹 설정 상세  
　　aa. 매장 검색 입력창(매장명/지역/상권 키워드)  
　　bb. 결과 리스트 체크 선택으로 비교군 구성  
　　cc. 선택 비교군 요약(매장 수/커버리지)  
　　dd. 저장/즐겨찾기/기본값 지정 연동  
　　i. 비교군 추천 기능(선택 제공, 제공 시 가치 큼)  
　　aa. "추천 비교군 자동 선택" 버튼 노출  
　　bb. 추천 후보 2~3개 리스트업  
　　• 반경 500m 기반 후보  
　　• 타지역 동일 상권 유형 후보  
　　• 유사 매장 조건 기반 후보  
　　cc. 추천은 수정 가능 상태로 제공(사용자 확정 전 적용 없음)  
　　j. 비교군 품질 규칙(신뢰도 관리)  
　　aa. 최소 표본 수(N) 기준 적용(정책 값 확정 필요)  
　　bb. 최소 커버리지 기준 적용(정책 값 확정 필요)  
　　cc. 표본/커버리지 부족 시 "주의 배지 + 해결 버튼" 노출  
　　dd. 해결 버튼(반경 확장/필터 완화/범위 확대)  
　D. 인사이트·실행 탭(반자동 실행 중심)  
　　a. 오늘의 인사이트 카드(최대 3개)  
　　aa. 지표 기반 문구 + 비교군 대비 % 노출  
　　bb. 원인 힌트(가설 2~3개)  
　　cc. 실행 버튼 노출  
　　• 스탬프 혜택 단계 추가  
　　• 휴면 고객 알림톡 발송(월 30회)  
　　• 리워드 혜택/노출 점검  
　　b. 휴면 케어 위젯(월 30회)  
　　aa. 휴면 고객 수 노출  
　　bb. 이번 달 잔여 발송 수 노출  
　　cc. 템플릿 선택/미리보기 노출  
　　dd. 예약/즉시 발송 버튼 노출(권한 정책 적용)  
　　c. 스탬프 바로 연결 위젯  
　　aa. 스탬프 적립 저조 시 우선 노출  
　　bb. 적립 조건 점검/혜택 단계 추가 연결
