# 3: 스탬프  

　A. 스탬프 생성  
　　a. 스탬프 카드 생성 박스  
　　aa. 스탬프 카드 이름 입력창  
　　bb. 스탬프 1개당 적립 기준 금액 입력창  
　　cc. 유효기간 입력창  
　　• 유효기간 미설정 체크박스  
　　• 체크박스 클릭 시 기간 설정 불가, 무기한 표시  
　　• 유효기간 기간 선택  
　　• 한달  
　　• 세달  
　　• 유효기간(미설정 시 무기한)  
　　dd. 박스 아래 확인/추가 버튼  
　　• 확인 버튼 → "스탬프 카드" 박스 생성  
　　• 스탬프 1개당 적립 기준, 유효기간 표시  
　　ee. "N개 적립 시 혜택" 설정 박스(스탬프 카드 박스 우측)  
　　• 3개의 체크박스와 입력창  
　　• 각각의 입력창에 N개 당 혜택 설정  
　B. 스탬프 요청/승인 흐름(대시보드 연동)  
　　a. 고객이 스탬프 적립 가능 컨디션일 경우 파트너 대시보드의 리워드 사용 보드에 박스 생성  
　　aa. 스탬프 적립 가능 컨디션:  
　　• 결제 금액 충족  
　　• 단골 스토어 지정 여부  
　　bb. 스탬프 적립 가능 컨디션일 경우 자동 적립  
　　b. 위드폼 오더 미사용 + 스탬프 기능 활성화 매장의 경우:  
　　aa. 소비자가 스탬프 적립 조건을 만족하면, 앱/웹에서 "스탬프 적립 요청" 가능  
　　bb. 스탬프 적립 가능 컨디션일 경우 자동 적립  
　　cc. 파트너 대시보드의 리워드 사용 보드에 스탬프 박스 생성  
　　c. 결제 금액 충족 + 스탬프 시스템을 모르는 고객 안내 메뉴얼  
　　aa. 단골 스토어 지정 + 스탬프 적립 가능 여부 안내.  
　　d. 첫 방문 고객 안내 메뉴얼  
　　aa. 위드폼에서 카페 검색 후 단골 스토어 지정 시 스탬프 카드 자동 생성됨을 안내
